数据结构概述 ​

📚 概述 ​

数据结构是计算机科学的基础，它研究数据的组织、存储和管理方式。选择合适的数据结构对于算法的效率至关重要，不同的数据结构在不同的操作上有着不同的性能特点。数据结构可以分为线性结构和非线性结构，每种结构都有其特定的应用场景和优缺点。掌握数据结构不仅能帮助我们编写更高效的程序，还能培养我们的抽象思维和问题解决能力。本章将系统介绍各种常见的数据结构，包括它们的特点、实现方法、时间复杂度分析以及实际应用场景。

🎯 学习目标 ​

  • 理解数据结构的基本概念和分类
  • 掌握各种数据结构的特点和应用场景
  • 学会分析数据结构操作的时间复杂度
  • 熟悉数据结构的实现方法和优化技巧
  • 理解数据结构与算法的关系
  • 学会根据问题选择合适的数据结构
  • 掌握数据结构在实际开发中的应用
  • 了解高级数据结构的设计思想

         * 数组特点：

         * 2. 随机访问 O(1)

         * 4. 固定大小（静态数组）

        System.out.println("数组特点和操作:");
        
        // 静态数组
        int[] staticArray = {10, 20, 30, 40, 50};
        System.out.println("静态数组: " + Arrays.toString(staticArray));

        long startTime = System.nanoTime();
        int element = staticArray[2];
        long endTime = System.nanoTime();

        List<Integer> dynamicArray = new ArrayList<>();
        System.out.println("\n动态数组操作:");

        startTime = System.nanoTime();
        for (int i = 0; i < 5; i++) {

        startTime = System.nanoTime();
        dynamicArray.add(2, 25);

    public static void demonstrateQueues() {

         * 队列特点：
         * 1. FIFO (First In First Out)
         * 2. 队尾入队，队头出队

        for (int i = 1; i <= 5; i++) {
            queue.offer(i * 10);
            System.out.println("enqueue " + (i * 10) + ": " + queue);
        }
        
        // 查看队头 - O(1)
        System.out.println("队头元素: " + queue.peek());
        
        // 出队 - O(1)
        System.out.println("\n出队操作:");
        while (!queue.isEmpty()) {
            int element = queue.poll();
            System.out.println("dequeue " + element + ": " + queue);
        }
        
        // 双端队列
        System.out.println("\n双端队列 (Deque) 操作:");
        Deque<Integer> deque = new ArrayDeque<>();

        deque.addLast(20);
        deque.addFirst(5);
        deque.addLast(25);
        System.out.println("双端队列: " + deque);
        
        System.out.println("从前端移除: " + deque.removeFirst() + ", 结果: " + deque);
        System.out.println("从后端移除: " + deque.removeLast() + ", 结果: " + deque);
        
        // 优先队列
        System.out.println("\n优先队列 (Priority Queue) 操作:");
        PriorityQueue<Integer> priorityQueue = new PriorityQueue<>();
        
        int[] values = {30, 10, 50, 20, 40};
        for (int value : values) {
            priorityQueue.offer(value);
        }
        System.out.println("添加元素后: " + priorityQueue);
        
        System.out.println("按优先级出队:");

    // 辅助方法：括号匹配
    public static boolean isValidParentheses(String s) {
        Stack<Character> stack = new Stack<>();
        Map<Character, Character> mapping = new HashMap<>();
        mapping.put(')', '(');
        mapping.put(']', '[');
        mapping.put('}', '{');
        
        for (char c : s.toCharArray()) {
            if (mapping.containsKey(c)) {
                if (stack.isEmpty() || stack.pop() != mapping.get(c)) {
                    return false;
                }
            } else {
                stack.push(c);
            }
        }
        
        return stack.isEmpty();
    }
}

// 简单链表实现
class SimpleLinkedList<T> {
    private Node<T> head;
    private int size;
    
    private static class Node<T> {
        T data;
        Node<T> next;
        
        Node(T data) {
            this.data = data;
        }
    }
    
    public void addFirst(T data) {
        Node<T> newNode = new Node<>(data);
        newNode.next = head;
        head = newNode;
        size++;
    }
    
    public boolean contains(T data) {
        Node<T> current = head;
        while (current != null) {
            if (current.data.equals(data)) {
                return true;
            }
            current = current.next;
        }
        return false;
    }
    
    public boolean remove(T data) {
        if (head == null) return false;
        
        if (head.data.equals(data)) {
            head = head.next;
            size--;
            return true;
        }
        
        Node<T> current = head;
        while (current.next != null) {
            if (current.next.data.equals(data)) {
                current.next = current.next.next;
                size--;
                return true;
            }
            current = current.next;
        }
        
        return false;
    }
    
    public int size() {
        return size;
    }
    
    @Override
    public String toString() {
        StringBuilder sb = new StringBuilder("[");
        Node<T> current = head;
        while (current != null) {
            sb.append(current.data);
            if (current.next != null) {
                sb.append(", ");
            }
            current = current.next;
        }
        sb.append("]");
        return sb.toString();
    }
}

class TreeNode {
    int val;
    TreeNode left;
    TreeNode right;
    
    TreeNode(int val) {
        this.val = val;
    }

// 二叉树
class BinaryTree {
    TreeNode root;
    
    public void preorderTraversal(TreeNode node) {
        if (node != null) {
            System.out.print(node.val + " ");
            preorderTraversal(node.left);
            preorderTraversal(node.right);
        }
    }
    
    public void inorderTraversal(TreeNode node) {
        if (node != null) {
            inorderTraversal(node.left);
            System.out.print(node.val + " ");
            inorderTraversal(node.right);
        }
    }
    
    public void postorderTraversal(TreeNode node) {
        if (node != null) {
            postorderTraversal(node.left);
            postorderTraversal(node.right);
            System.out.print(node.val + " ");
        }
    }
    
    public void levelOrderTraversal() {
        if (root == null) return;
        
        Queue<TreeNode> queue = new LinkedList<>();
        queue.offer(root);
        
        while (!queue.isEmpty()) {
            TreeNode node = queue.poll();
            System.out.print(node.val + " ");
            
            if (node.left != null) queue.offer(node.left);
            if (node.right != null) queue.offer(node.right);
        }
    }
}

// 二叉搜索树
class BinarySearchTree {
    private TreeNode root;
    
    public void insert(int val) {
        root = insertRec(root, val);
    }
    
    private TreeNode insertRec(TreeNode root, int val) {
        if (root == null) {
            return new TreeNode(val);
        }
        
        if (val < root.val) {
            root.left = insertRec(root.left, val);
        } else if (val > root.val) {
            root.right = insertRec(root.right, val);
        }
        
        return root;
    }
    
    public boolean search(int val) {
        return searchRec(root, val);
    }
    
    private boolean searchRec(TreeNode root, int val) {
        if (root == null) return false;
        if (root.val == val) return true;
        
        return val < root.val ? searchRec(root.left, val) : searchRec(root.right, val);
    }
    
    public void inorderTraversal() {
        inorderRec(root);
    }
    
    private void inorderRec(TreeNode root) {
        if (root != null) {
            inorderRec(root.left);
            System.out.print(root.val + " ");
            inorderRec(root.right);
        }
    }
}

// 图
class Graph {
    private int vertices;
    private List<List<Integer>> adjList;
    
    public Graph(int vertices) {
        this.vertices = vertices;
        adjList = new ArrayList<>(vertices);
        for (int i = 0; i < vertices; i++) {
            adjList.add(new ArrayList<>());
        }
    }
    
    public void addEdge(int src, int dest) {
        adjList.get(src).add(dest);
        adjList.get(dest).add(src); // 无向图
    }
    
    public void printGraph() {
        for (int i = 0; i < vertices; i++) {
            System.out.println("顶点 " + i + ": " + adjList.get(i));
        }
    }
    
    public void DFS(int start) {
        boolean[] visited = new boolean[vertices];
        DFSUtil(start, visited);
    }
    
    private void DFSUtil(int vertex, boolean[] visited) {
        visited[vertex] = true;
        System.out.print(vertex + " ");
        
        for (int neighbor : adjList.get(vertex)) {
            if (!visited[neighbor]) {
                DFSUtil(neighbor, visited);
            }
        }
    }
    
    public void BFS(int start) {
        boolean[] visited = new boolean[vertices];
        Queue<Integer> queue = new LinkedList<>();
        
        visited[start] = true;
        queue.offer(start);
        
        while (!queue.isEmpty()) {
            int vertex = queue.poll();
            System.out.print(vertex + " ");
            
            for (int neighbor : adjList.get(vertex)) {
                if (!visited[neighbor]) {
                    visited[neighbor] = true;
                    queue.offer(neighbor);
                }
            }
        }
    }
}

💡 最佳实践总结 ​

数据结构选择原则 ​

  1. 分析操作频率：确定哪些操作最频繁
  2. 考虑数据规模：大数据量时复杂度更重要
  3. 权衡时间空间：在时间和空间效率间找平衡
  4. 考虑实际场景：结合具体应用需求

实现技巧 ​

  1. 合理使用泛型：提高代码复用性
  2. 注意边界条件：处理空数据结构的情况
  3. 优化内存使用：避免不必要的内存分配
  4. 保证线程安全：多线程环境下的数据结构选择

性能优化 ​

  1. 预分配空间：避免频繁的内存重分配
  2. 批量操作：减少单次操作的开销
  3. 缓存友好：考虑CPU缓存的影响
  4. 懒加载：延迟初始化大型数据结构

常见陷阱 ​

  1. 忽视常数因子：小数据量时常数因子很重要
  2. 过度优化：在不必要的地方进行优化
  3. 忽略内存局部性：数组通常比链表更快
  4. 不考虑并发：多线程环境下的数据竞争

使用场景指南 ​

选择数组当：

  • 需要频繁随机访问元素
  • 内存使用要求严格
  • 数据大小相对固定

选择链表当：

  • 频繁在头部插入/删除
  • 数据大小变化很大
  • 不需要随机访问

选择栈当：

  • 需要LIFO操作
  • 实现递归算法
  • 处理嵌套结构

选择队列当：

  • 需要FIFO操作
  • 实现BFS算法
  • 任务调度系统

选择哈希表当：

  • 需要快速查找
  • 键值对存储
  • 去重操作

选择树当：

  • 需要有序数据
  • 范围查询
  • 层次结构数据

选择图当：

  • 表示复杂关系
  • 网络结构
  • 路径问题

💡 重点总结：

  1. 数据结构是组织和存储数据的方式，直接影响算法效率
  2. 线性结构包括数组、链表、栈、队列，适用于顺序处理
  3. 非线性结构包括树、图、哈希表、堆，适用于复杂关系
  4. 不同数据结构在不同操作上有不同的时间复杂度特点
  5. 选择数据结构要考虑操作频率、数据规模、时间空间权衡
  6. 实际应用中要结合具体场景选择最合适的数据结构
  7. 理解数据结构的实现原理有助于更好地使用和优化
  8. 数据结构是算法设计的基础，两者密不可分
