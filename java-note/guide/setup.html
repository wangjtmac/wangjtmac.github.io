环境搭建指南 ​

📚 概述 ​

本指南将帮助您搭建完整的 Java 开发环境，包括 JDK 安装、IDE 配置、构建工具设置等。

🎯 环境要求 ​

系统要求 ​

  • 操作系统：Windows 10+、macOS 10.14+、Linux（Ubuntu 18.04+）
  • 内存：至少 4GB RAM（推荐 8GB+）
  • 存储空间：至少 5GB 可用空间
  • 网络：稳定的互联网连接

☕ JDK 安装 ​

1. 选择 JDK 版本 ​

bash
# 推荐版本（LTS 长期支持版本）
- Java 8 (LTS) - 企业级应用广泛使用
- Java 11 (LTS) - 现代特性与稳定性平衡
- Java 17 (LTS) - 最新 LTS 版本，推荐新项目使用
- Java 21 (LTS) - 最新 LTS 版本（2023年发布）

2. 下载 JDK ​

Oracle JDK ​

bash
# 官方网站
https://www.oracle.com/java/technologies/downloads/

# 注意：Oracle JDK 商业使用需要许可证

OpenJDK（推荐） ​

bash
# Adoptium（推荐）
https://adoptium.net/

# Amazon Corretto
https://aws.amazon.com/corretto/

# Microsoft OpenJDK
https://www.microsoft.com/openjdk

3. 安装 JDK ​

Windows 安装 ​

bash
# 1. 下载 .msi 安装包
# 2. 双击运行安装程序
# 3. 按照向导完成安装
# 4. 配置环境变量

# 设置 JAVA_HOME
JAVA_HOME=C:\Program Files\Eclipse Adoptium\jdk-17.0.8.101-hotspot

# 添加到 PATH
PATH=%JAVA_HOME%\bin;%PATH%

macOS 安装 ​

bash
# 方法1：使用 Homebrew（推荐）
brew install openjdk@17

# 方法2：下载 .pkg 安装包
# 从官网下载并安装

# 配置环境变量（添加到 ~/.zshrc 或 ~/.bash_profile）
export JAVA_HOME=/opt/homebrew/opt/openjdk@17/libexec/openjdk.jdk/Contents/Home
export PATH=$JAVA_HOME/bin:$PATH

Linux 安装 ​

bash
# Ubuntu/Debian
sudo apt update
sudo apt install openjdk-17-jdk

# CentOS/RHEL
sudo yum install java-17-openjdk-devel

# 或者使用 SDKMAN（推荐）
curl -s "https://get.sdkman.io" | bash
source "$HOME/.sdkman/bin/sdkman-init.sh"
sdk install java 17.0.8-tem

4. 验证安装 ​

bash
# 检查 Java 版本
java -version

# 检查编译器版本
javac -version

# 检查环境变量
echo $JAVA_HOME  # Linux/macOS
echo %JAVA_HOME% # Windows

🔧 构建工具 ​

1. Maven ​

安装 Maven ​

bash
# Windows（使用 Chocolatey）
choco install maven

# macOS（使用 Homebrew）
brew install maven

# Linux
sudo apt install maven  # Ubuntu/Debian
sudo yum install maven  # CentOS/RHEL

# 验证安装
mvn -version

配置 Maven ​

xml
<!-- settings.xml 配置文件位置 -->
<!-- Windows: %USERPROFILE%\.m2\settings.xml -->
<!-- Linux/macOS: ~/.m2/settings.xml -->

<?xml version="1.0" encoding="UTF-8"?>
<settings xmlns="http://maven.apache.org/SETTINGS/1.0.0">
  <mirrors>
    <!-- 阿里云镜像（中国用户推荐） -->
    <mirror>
      <id>aliyun</id>
      <mirrorOf>central</mirrorOf>
      <name>Aliyun Central</name>
      <url>https://maven.aliyun.com/repository/central</url>
    </mirror>
  </mirrors>
  
  <profiles>
    <profile>
      <id>jdk-17</id>
      <properties>
        <maven.compiler.source>17</maven.compiler.source>
        <maven.compiler.target>17</maven.compiler.target>
      </properties>
    </profile>
  </profiles>
  
  <activeProfiles>
    <activeProfile>jdk-17</activeProfile>
  </activeProfiles>
</settings>

🚀 创建第一个项目 ​

1. 使用 Maven 创建项目 ​

bash
# 创建标准 Maven 项目
mvn archetype:generate \
  -DgroupId=com.example \
  -DartifactId=my-java-app \
  -DarchetypeArtifactId=maven-archetype-quickstart \
  -DinteractiveMode=false

# 进入项目目录
cd my-java-app

# 编译项目
mvn compile

# 运行测试
mvn test

# 打包项目
mvn package

2. 项目结构 ​

my-java-app/
├── pom.xml                 # Maven 配置文件
├── src/
│   ├── main/
│   │   └── java/
│   │       └── com/
│   │           └── example/
│   │               └── App.java
│   └── test/
│       └── java/
│           └── com/
│               └── example/
│                   └── AppTest.java
└── target/                 # 编译输出目录

3. Hello World 程序 ​

java
// src/main/java/com/example/App.java
package com.example;

public class App {
    public static void main(String[] args) {
        System.out.println("Hello, Java World!");
        
        // 显示 Java 版本信息
        System.out.println("Java Version: " + System.getProperty("java.version"));
        System.out.println("Java Home: " + System.getProperty("java.home"));
        System.out.println("OS Name: " + System.getProperty("os.name"));
    }
}

🔍 版本管理工具 ​

SDKMAN（推荐） ​

bash
# 安装 SDKMAN
curl -s "https://get.sdkman.io" | bash
source "$HOME/.sdkman/bin/sdkman-init.sh"

# 列出可用的 Java 版本
sdk list java

# 安装特定版本
sdk install java 17.0.8-tem
sdk install java 11.0.20-tem

# 切换版本
sdk use java 17.0.8-tem

# 设置默认版本
sdk default java 17.0.8-tem

# 查看当前版本
sdk current java

🛡️ 常见问题解决 ​

1. JAVA_HOME 未设置 ​

bash
# 错误信息
Error: JAVA_HOME is not defined correctly

# 解决方案
# Windows
set JAVA_HOME=C:\Program Files\Java\jdk-17

# Linux/macOS
export JAVA_HOME=/usr/lib/jvm/java-17-openjdk

2. 编码问题 ​

bash
# 编译时指定编码
javac -encoding UTF-8 HelloWorld.java

# Maven 项目配置
<properties>
    <project.build.sourceEncoding>UTF-8</project.build.sourceEncoding>
    <maven.compiler.source>17</maven.compiler.source>
    <maven.compiler.target>17</maven.compiler.target>
</properties>

3. 权限问题（Linux/macOS） ​

bash
# 给予执行权限
chmod +x /path/to/java

# 使用 sudo 安装（如果需要）
sudo apt install openjdk-17-jdk

🎯 下一步 ​

环境搭建完成后，建议：

  1. 熟悉 IDE：学习快捷键和调试功能
  2. 练习基础语法：编写简单的 Java 程序
  3. 学习构建工具：掌握 Maven 或 Gradle 的基本使用
  4. 版本控制：学习 Git 的基本操作
  5. 开始系统学习：按照 学习路径 进行

💡 提示：环境搭建是学习的第一步，建议选择一套工具组合并熟练掌握，而不是频繁更换工具。推荐组合：OpenJDK 17 + IntelliJ IDEA + Maven + Git。
