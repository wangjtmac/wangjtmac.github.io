Java 集合框架详解 ​

📚 概述 ​

Java 集合框架（Java Collections Framework）是 Java 中用于存储和操作对象集合的统一架构。它提供了一套设计良好的接口和类，用于处理各种数据结构。

🎯 学习目标 ​

  • 理解集合框架的整体架构
  • 掌握 Collection 和 Map 接口体系
  • 熟悉常用集合类的特点和使用场景
  • 理解集合的性能特征和选择原则
  • 掌握集合的遍历和操作方法

🏗️ 集合框架架构 ​

核心接口层次结构 ​

Collection (接口)
├── List (接口) - 有序集合，允许重复
│   ├── ArrayList (类)
│   ├── LinkedList (类)
│   └── Vector (类)
├── Set (接口) - 无序集合，不允许重复
│   ├── HashSet (类)
│   ├── LinkedHashSet (类)
│   └── TreeSet (类)
└── Queue (接口) - 队列接口
    ├── PriorityQueue (类)
    └── Deque (接口)
        └── ArrayDeque (类)

Map (接口) - 键值对映射
├── HashMap (类)
├── LinkedHashMap (类)
├── TreeMap (类)
└── Hashtable (类)

集合框架的优势 ​

  1. 统一的接口：提供一致的编程接口
  2. 高性能实现：针对不同场景优化的实现
  3. 互操作性：不同集合类型之间可以相互转换
  4. 可扩展性：可以自定义集合实现
  5. 线程安全选项：提供线程安全的集合类

📋 Collection 接口体系 ​

Collection 接口 ​

java
public interface Collection<E> extends Iterable<E> {
    // 基本操作
    int size();
    boolean isEmpty();
    boolean contains(Object o);
    Iterator<E> iterator();
    Object[] toArray();
    
    // 修改操作
    boolean add(E e);
    boolean remove(Object o);
    
    // 批量操作
    boolean containsAll(Collection<?> c);
    boolean addAll(Collection<? extends E> c);
    boolean removeAll(Collection<?> c);
    boolean retainAll(Collection<?> c);
    void clear();
}

List 接口 - 有序集合 ​

java
// List 特点：有序、可重复、支持索引访问
List<String> list = new ArrayList<>();
list.add("Apple");
list.add("Banana");
list.add("Apple");  // 允许重复

// 索引访问
String first = list.get(0);
list.set(1, "Orange");

// 常用实现类比较
ArrayList<String> arrayList = new ArrayList<>();    // 动态数组，随机访问快
LinkedList<String> linkedList = new LinkedList<>(); // 双向链表，插入删除快
Vector<String> vector = new Vector<>();             // 线程安全的动态数组

Set 接口 - 无重复集合 ​

java
// Set 特点：无序、不可重复
Set<String> set = new HashSet<>();
set.add("Apple");
set.add("Banana");
set.add("Apple");   // 重复元素不会被添加
System.out.println(set.size()); // 输出：2

// 常用实现类比较
HashSet<String> hashSet = new HashSet<>();           // 基于哈希表，无序
LinkedHashSet<String> linkedHashSet = new LinkedHashSet<>(); // 保持插入顺序
TreeSet<String> treeSet = new TreeSet<>();           // 基于红黑树，自动排序

Queue 接口 - 队列 ​

java
// Queue 特点：FIFO（先进先出）
Queue<String> queue = new LinkedList<>();
queue.offer("First");   // 入队
queue.offer("Second");
String head = queue.poll(); // 出队

// PriorityQueue - 优先队列
PriorityQueue<Integer> pq = new PriorityQueue<>();
pq.offer(3);
pq.offer(1);
pq.offer(2);
System.out.println(pq.poll()); // 输出：1（最小值优先）

🗺️ Map 接口体系 ​

Map 接口 ​

java
public interface Map<K,V> {
    // 基本操作
    int size();
    boolean isEmpty();
    boolean containsKey(Object key);
    boolean containsValue(Object value);
    V get(Object key);
    V put(K key, V value);
    V remove(Object key);
    
    // 批量操作
    void putAll(Map<? extends K, ? extends V> m);
    void clear();
    
    // 视图操作
    Set<K> keySet();
    Collection<V> values();
    Set<Map.Entry<K, V>> entrySet();
}

🔄 集合的遍历方式 ​

1. 传统 for 循环（仅适用于 List） ​

java
List<String> list = Arrays.asList("A", "B", "C");
for (int i = 0; i < list.size(); i++) {
    System.out.println(list.get(i));
}

2. 增强 for 循环（for-each） ​

java
List<String> list = Arrays.asList("A", "B", "C");
for (String item : list) {
    System.out.println(item);
}

3. Iterator 迭代器 ​

java
List<String> list = Arrays.asList("A", "B", "C");
Iterator<String> iterator = list.iterator();
while (iterator.hasNext()) {
    String item = iterator.next();
    System.out.println(item);
    // iterator.remove(); // 安全删除
}

4. Stream API（Java 8+） ​

java
List<String> list = Arrays.asList("A", "B", "C");
list.stream()
    .filter(s -> s.length() > 1)
    .forEach(System.out::println);

🛠️ 实践示例 ​

示例1：学生成绩管理 ​

java
public class StudentGradeManager {
    private Map<String, List<Integer>> studentGrades;
    
    public StudentGradeManager() {
        this.studentGrades = new HashMap<>();
    }
    
    public void addGrade(String student, int grade) {
        studentGrades.computeIfAbsent(student, k -> new ArrayList<>()).add(grade);
    }
    
    public double getAverageGrade(String student) {
        List<Integer> grades = studentGrades.get(student);
        if (grades == null || grades.isEmpty()) {
            return 0.0;
        }
        return grades.stream().mapToInt(Integer::intValue).average().orElse(0.0);
    }
    
    public Set<String> getTopStudents(int count) {
        return studentGrades.entrySet().stream()
            .sorted((e1, e2) -> Double.compare(
                getAverageGrade(e2.getKey()), 
                getAverageGrade(e1.getKey())))
            .limit(count)
            .map(Map.Entry::getKey)
            .collect(Collectors.toCollection(LinkedHashSet::new));
    }
}

示例2：单词频率统计 ​

java
public class WordFrequency {
    public static Map<String, Integer> countWords(String text) {
        Map<String, Integer> wordCount = new HashMap<>();
        String[] words = text.toLowerCase().split("\\W+");
        
        for (String word : words) {
            if (!word.isEmpty()) {
                wordCount.put(word, wordCount.getOrDefault(word, 0) + 1);
            }
        }
        
        return wordCount;
    }
    
    public static void main(String[] args) {
        String text = "Java is great. Java is powerful. Programming in Java is fun.";
        Map<String, Integer> result = countWords(text);
        
        // 按频率排序输出
        result.entrySet().stream()
            .sorted(Map.Entry.<String, Integer>comparingByValue().reversed())
            .forEach(entry -> 
                System.out.println(entry.getKey() + ": " + entry.getValue()));
    }
}

🎯 选择指南 ​

何时使用 List ​

  • 需要保持元素的插入顺序
  • 需要通过索引访问元素
  • 允许重复元素
  • 推荐：ArrayList（随机访问多）、LinkedList（插入删除多）

何时使用 Set ​

  • 需要确保元素唯一性
  • 不需要索引访问
  • 推荐：HashSet（性能优先）、TreeSet（需要排序）、LinkedHashSet（保持顺序）

何时使用 Map ​

  • 需要键值对映射关系
  • 需要通过键快速查找值
  • 推荐：HashMap（性能优先）、TreeMap（需要排序）、LinkedHashMap（保持顺序）

🔒 线程安全考虑 ​

非线程安全的集合 ​

java
// 这些集合在多线程环境下不安全
List<String> list = new ArrayList<>();
Set<String> set = new HashSet<>();
Map<String, String> map = new HashMap<>();

线程安全的解决方案 ​

java
// 1. 使用 Collections 工具类
List<String> syncList = Collections.synchronizedList(new ArrayList<>());
Set<String> syncSet = Collections.synchronizedSet(new HashSet<>());
Map<String, String> syncMap = Collections.synchronizedMap(new HashMap<>());

// 2. 使用并发集合类
List<String> copyOnWriteList = new CopyOnWriteArrayList<>();
Set<String> concurrentSet = ConcurrentHashMap.newKeySet();
Map<String, String> concurrentMap = new ConcurrentHashMap<>();
