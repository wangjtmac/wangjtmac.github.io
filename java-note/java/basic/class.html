Java 类与对象详解 ​

📚 概述 ​

类（Class）是面向对象编程的基础，它是创建对象的模板或蓝图。对象（Object）是类的实例，是具有状态和行为的实体。理解类与对象的关系以及如何正确设计和使用它们是掌握 Java 编程的关键。

🎯 学习目标 ​

  • 深入理解类和对象的概念及其关系
  • 掌握类的定义语法和各种成员的使用
  • 熟练使用构造方法和方法重载
  • 理解 this 和 super 关键字的作用
  • 掌握静态成员和实例成员的区别
  • 学会正确重写 Object 类的方法
  • 理解封装原则和不可变对象的设计
  • 掌握对象的生命周期和内存管理

// 类是对象的模板/蓝图
public class Car {
    // 属性（状态）
    private String brand;
    private String model;
    private int year;
    private double price;

    // 构造方法
    public Car(String brand, String model, int year, double price) {
        this.brand = brand;
        this.model = model;
        this.year = year;
        this.price = price;
    }

    // 方法（行为）
    public void start() {
        System.out.println(brand + " " + model + " 启动了");
    }

    public void stop() {
        System.out.println(brand + " " + model + " 停止了");
    }

    public void displayInfo() {
        System.out.printf("%s %s (%d年) - ¥%.2f%n", brand, model, year, price);
    }

    // getter 和 setter 方法
    public String getBrand() { return brand; }
    public String getModel() { return model; }
    public int getYear() { return year; }
    public double getPrice() { return price; }

    public void setPrice(double price) {
        if (price > 0) {
            this.price = price;
        }
    }
}

// 使用类创建对象
public class CarDemo {
    public static void main(String[] args) {
        // 创建对象（实例化）
        Car car1 = new Car("丰田", "凯美瑞", 2023, 180000);
        Car car2 = new Car("本田", "雅阁", 2022, 175000);

        // 使用对象
        car1.displayInfo();
        car1.start();

        car2.displayInfo();
        car2.start();

        // 修改对象状态
        car1.setPrice(170000);
        car1.displayInfo();

        // 对象比较
        System.out.println("car1 == car2: " + (car1 == car2)); // false，不同对象
        System.out.println("car1.getBrand().equals(car2.getBrand()): " +
                          car1.getBrand().equals(car2.getBrand())); // false，不同品牌
    }
}

public class ConstructorExample {
    private String name;
    private int age;
    private String email;

    // 1. 默认构造方法（如果没有定义任何构造方法，编译器会自动提供）
    // public ConstructorExample() { }

    // 2. 无参构造方法
    public ConstructorExample() {
        this("未知", 0, "unknown@example.com");
        System.out.println("无参构造方法被调用");
    }

    // 3. 有参构造方法
    public ConstructorExample(String name) {
        this(name, 0, "unknown@example.com");
        System.out.println("单参数构造方法被调用");
    }

    public ConstructorExample(String name, int age) {
        this(name, age, "unknown@example.com");
        System.out.println("双参数构造方法被调用");
    }

    // 4. 完整构造方法
    public ConstructorExample(String name, int age, String email) {
        // 参数验证
        if (name == null || name.trim().isEmpty()) {
            throw new IllegalArgumentException("姓名不能为空");
        }
        if (age < 0 || age > 150) {
            throw new IllegalArgumentException("年龄必须在0-150之间");
        }
        if (email == null || !email.contains("@")) {
            throw new IllegalArgumentException("邮箱格式无效");
        }

        this.name = name;
        this.age = age;
        this.email = email;
        System.out.println("完整构造方法被调用");
    }

    // 5. 拷贝构造方法
    public ConstructorExample(ConstructorExample other) {
        this(other.name, other.age, other.email);
        System.out.println("拷贝构造方法被调用");
    }

    @Override
    public String toString() {
        return String.format("Person{name='%s', age=%d, email='%s'}", name, age, email);
    }
}

2. 构造方法链和 this() 调用 ​

java
public class ConstructorChaining {
    private String firstName;
    private String lastName;
    private int age;
    private String department;
    private double salary;

    // 最基础的构造方法
    public ConstructorChaining(String firstName, String lastName, int age, String department, double salary) {
        this.firstName = firstName;
        this.lastName = lastName;
        this.age = age;
        this.department = department;
        this.salary = salary;
        System.out.println("主构造方法执行完成");
    }

    // 委托给主构造方法
    public ConstructorChaining(String firstName, String lastName, int age) {
        this(firstName, lastName, age, "未分配", 0.0);
        System.out.println("三参数构造方法执行完成");
    }

    // 委托给三参数构造方法
    public ConstructorChaining(String firstName, String lastName) {
        this(firstName, lastName, 0);
        System.out.println("双参数构造方法执行完成");
    }

    // 委托给双参数构造方法
    public ConstructorChaining(String fullName) {
        this(parseFirstName(fullName), parseLastName(fullName));
        System.out.println("单参数构造方法执行完成");
    }

    // 委托给单参数构造方法
    public ConstructorChaining() {
        this("Unknown Unknown");
        System.out.println("无参数构造方法执行完成");
    }

    // 辅助方法
    private static String parseFirstName(String fullName) {
        String[] parts = fullName.split(" ");
        return parts.length > 0 ? parts[0] : "Unknown";
    }

    private static String parseLastName(String fullName) {
        String[] parts = fullName.split(" ");
        return parts.length > 1 ? parts[1] : "Unknown";
    }

    @Override
    public String toString() {
        return String.format("Employee{%s %s, age=%d, dept='%s', salary=%.2f}",
                           firstName, lastName, age, department, salary);
    }

    public static void main(String[] args) {
        System.out.println("=== 构造方法链演示 ===");
        ConstructorChaining emp = new ConstructorChaining();
        System.out.println(emp);

        System.out.println("\n=== 带参数的构造 ===");
        ConstructorChaining emp2 = new ConstructorChaining("John Doe");
        System.out.println(emp2);
    }
}

🔧 方法重载详解 ​

1. 方法重载的规则 ​

java
public class MethodOverloadingExample {

    // 1. 参数个数不同
    public void print(String message) {
        System.out.println("单个字符串: " + message);
    }

    public void print(String message, int times) {
        for (int i = 0; i < times; i++) {
            System.out.println(message);
        }
    }

    public void print(String message, int times, String separator) {
        for (int i = 0; i < times; i++) {
            System.out.print(message);
            if (i < times - 1) {
                System.out.print(separator);
            }
        }
        System.out.println();
    }

    // 2. 参数类型不同
    public void calculate(int a, int b) {
        System.out.println("整数计算: " + (a + b));
    }

    public void calculate(double a, double b) {
        System.out.println("浮点数计算: " + (a + b));
    }

    public void calculate(String a, String b) {
        System.out.println("字符串连接: " + (a + b));
    }

    // 3. 参数顺序不同
    public void createUser(String name, int age) {
        System.out.println("创建用户: " + name + ", 年龄: " + age);
    }

    public void createUser(int id, String name) {
        System.out.println("创建用户: ID=" + id + ", 姓名: " + name);
    }

    // 4. 可变参数的重载
    public void sum(int... numbers) {
        int total = 0;
        for (int num : numbers) {
            total += num;
        }
        System.out.println("可变参数求和: " + total);
    }

    public void sum(int a, int b) {
        System.out.println("两个参数求和: " + (a + b));
    }

    // 注意：仅返回类型不同不能构成重载
    // public int calculate(int a, int b) { return a + b; } // 编译错误

    // 5. 数组参数的重载
    public void process(int[] array) {
        System.out.println("处理整数数组，长度: " + array.length);
    }

    public void process(String[] array) {
        System.out.println("处理字符串数组，长度: " + array.length);
    }

    public void process(Object[] array) {
        System.out.println("处理对象数组，长度: " + array.length);
    }

    public static void main(String[] args) {
        MethodOverloadingExample example = new MethodOverloadingExample();

        // 测试不同的重载方法
        example.print("Hello");
        example.print("Hello", 3);
        example.print("Hello", 3, " | ");

        example.calculate(5, 3);
        example.calculate(5.5, 3.2);
        example.calculate("Hello", "World");

        example.createUser("Alice", 25);
        example.createUser(1001, "Bob");

        example.sum(1, 2, 3, 4, 5);
        example.sum(10, 20);

        example.process(new int[]{1, 2, 3});
        example.process(new String[]{"a", "b", "c"});
        example.process(new Object[]{1, "hello", 3.14});
    }
}

1. this 关键字的使用 ​

java
public class ThisExample {
    private String name;
    private int age;
    private String email;

    public ThisExample(String name, int age, String email) {
        // 1. this 用于区分参数和实例变量
        this.name = name;
        this.age = age;
        this.email = email;
    }

    public ThisExample(String name, int age) {
        // 2. this() 调用其他构造方法
        this(name, age, "unknown@example.com");
    }

    public ThisExample(String name) {
        // 3. this() 必须是构造方法的第一行
        this(name, 0);
    }

    // 4. this 用于返回当前对象（链式调用）
    public ThisExample setName(String name) {
        this.name = name;
        return this; // 返回当前对象
    }

    public ThisExample setAge(int age) {
        this.age = age;
        return this;
    }

    public ThisExample setEmail(String email) {
        this.email = email;
        return this;
    }

    // 5. this 用于方法参数
    public void copyFrom(ThisExample other) {
        this.name = other.name;
        this.age = other.age;
        this.email = other.email;
    }

    public boolean equals(ThisExample other) {
        if (this == other) return true; // 同一个对象
        if (other == null) return false;

        return this.name.equals(other.name) &&
               this.age == other.age &&
               this.email.equals(other.email);
    }

    // 6. 将当前对象传递给其他方法
    public void registerToService() {
        UserService.register(this); // 传递当前对象
    }

    @Override
    public String toString() {
        return String.format("ThisExample{name='%s', age=%d, email='%s'}",
                           this.name, this.age, this.email);
    }

    public static void main(String[] args) {
        // 链式调用演示
        ThisExample person = new ThisExample("Alice")
            .setAge(25)
            .setEmail("alice@example.com");

        System.out.println(person);

        // 对象比较
        ThisExample person2 = new ThisExample("Alice", 25, "alice@example.com");
        System.out.println("相等: " + person.equals(person2));
    }
}

// 模拟服务类
class UserService {
    public static void register(ThisExample user) {
        System.out.println("注册用户: " + user);
    }
}

2. super 关键字的使用 ​

java
// 父类
class Animal {
    protected String name;
    protected int age;

    public Animal(String name, int age) {
        this.name = name;
        this.age = age;
        System.out.println("Animal 构造方法执行");
    }

    public void eat() {
        System.out.println(name + " 正在吃东西");
    }

    public void sleep() {
        System.out.println(name + " 正在睡觉");
    }

    public void makeSound() {
        System.out.println(name + " 发出声音");
    }

    public void displayInfo() {
        System.out.println("动物: " + name + ", 年龄: " + age);
    }
}

// 子类
class Dog extends Animal {
    private String breed;
    private String owner;

    public Dog(String name, int age, String breed, String owner) {
        // 1. super() 调用父类构造方法，必须是第一行
        super(name, age);
        this.breed = breed;
        this.owner = owner;
        System.out.println("Dog 构造方法执行");
    }

    public Dog(String name, int age, String breed) {
        // 2. 调用本类的其他构造方法
        this(name, age, breed, "未知主人");
    }

    @Override
    public void makeSound() {
        // 3. super 调用父类方法
        super.makeSound(); // 先调用父类方法
        System.out.println(name + " 汪汪叫");
    }

    @Override
    public void displayInfo() {
        // 4. 扩展父类方法
        super.displayInfo(); // 调用父类的显示信息
        System.out.println("品种: " + breed + ", 主人: " + owner);
    }

    // 5. 访问父类的 protected 成员
    public void showParentInfo() {
        System.out.println("父类的 name: " + super.name);
        System.out.println("父类的 age: " + super.age);
        // 注意：如果没有同名变量，super 可以省略
        System.out.println("直接访问 name: " + name);
        System.out.println("直接访问 age: " + age);
    }

    public void wagTail() {
        System.out.println(name + " 摇尾巴");
    }
}

// 演示类
public class SuperExample {
    public static void main(String[] args) {
        System.out.println("=== 创建 Dog 对象 ===");
        Dog dog = new Dog("旺财", 3, "金毛", "张三");

        System.out.println("\n=== 调用方法 ===");
        dog.eat();        // 继承自父类
        dog.makeSound();  // 重写的方法，会调用 super.makeSound()
        dog.wagTail();    // 子类特有方法

        System.out.println("\n=== 显示信息 ===");
        dog.displayInfo(); // 重写的方法，会调用 super.displayInfo()

        System.out.println("\n=== 访问父类成员 ===");
        dog.showParentInfo();
    }
}

3. this 和 super 的区别 ​

java
public class ThisVsSuper {

    static class Parent {
        protected String name = "Parent";

        public Parent() {
            System.out.println("Parent 构造方法");
        }

        public void method() {
            System.out.println("Parent method");
        }
    }

    static class Child extends Parent {
        private String name = "Child";

        public Child() {
            super(); // 调用父类构造方法
            System.out.println("Child 构造方法");
        }

        @Override
        public void method() {
            System.out.println("Child method");
        }

        public void demonstrateThisVsSuper() {
            // this 引用当前对象
            System.out.println("this.name: " + this.name);     // Child
            System.out.println("super.name: " + super.name);   // Parent

            // 方法调用
            this.method();  // 调用当前类的方法
            super.method(); // 调用父类的方法

            // 构造方法调用
            // this() 和 super() 只能在构造方法中使用，且必须是第一行
        }
    }

    public static void main(String[] args) {
        Child child = new Child();
        child.demonstrateThisVsSuper();
    }
}

public class MemberTypes {
    // 静态常量
    public static final String COMPANY_NAME = "ABC Corp";

    // 静态变量
    private static int totalEmployees = 0;

    // 实例变量
    private String name;
    private int id;
    private double salary;

    // 静态初始化块
    static {
        System.out.println("静态初始化块执行");
        totalEmployees = 0;
    }

    // 实例初始化块
    {
        System.out.println("实例初始化块执行");
        totalEmployees++;
    }

    // 构造方法
    public MemberTypes(String name, int id, double salary) {
        this.name = name;
        this.id = id;
        this.salary = salary;
        System.out.println("构造方法执行");
    }

    // 实例方法
    public void work() {
        System.out.println(name + " 正在工作");
    }

    public void displayInfo() {
        System.out.printf("员工: %s (ID: %d), 薪资: %.2f%n", name, id, salary);
        System.out.println("公司: " + COMPANY_NAME); // 访问静态成员
        System.out.println("总员工数: " + totalEmployees); // 访问静态成员
    }

    // 静态方法
    public static int getTotalEmployees() {
        return totalEmployees;
    }

    public static void printCompanyInfo() {
        System.out.println("公司名称: " + COMPANY_NAME);
        System.out.println("员工总数: " + totalEmployees);
        // System.out.println(name); // 编译错误：静态方法不能访问实例成员
    }

    // 静态方法不能被重写，但可以被隐藏
    public static void staticMethod() {
        System.out.println("MemberTypes 的静态方法");
    }
}

// 演示执行顺序
public class MemberDemo {
    public static void main(String[] args) {
        System.out.println("=== 程序开始 ===");

        System.out.println("\n=== 第一次访问类 ===");
        System.out.println("总员工数: " + MemberTypes.getTotalEmployees());

        System.out.println("\n=== 创建第一个对象 ===");
        MemberTypes emp1 = new MemberTypes("张三", 1001, 8000);

        System.out.println("\n=== 创建第二个对象 ===");
        MemberTypes emp2 = new MemberTypes("李四", 1002, 9000);

        System.out.println("\n=== 使用对象 ===");

1. equals() 方法重写 ​

java
import java.util.Objects;

public class Person {
    private String name;
    private int age;
    private String email;

    public Person(String name, int age, String email) {
        this.name = name;
        this.age = age;
        this.email = email;
    }

    // 正确重写 equals 方法
    @Override
    public boolean equals(Object obj) {
        // 1. 检查是否是同一个对象
        if (this == obj) return true;

        // 2. 检查是否为 null
        if (obj == null) return false;

        // 3. 检查类型是否相同
        if (getClass() != obj.getClass()) return false;

        // 4. 类型转换
        Person person = (Person) obj;

        // 5. 比较关键字段
        return age == person.age &&
               Objects.equals(name, person.name) &&
               Objects.equals(email, person.email);
    }

    // 重写 equals 必须同时重写 hashCode
    @Override
    public int hashCode() {
        return Objects.hash(name, age, email);
    }

    // 重写 toString 方便调试
    @Override
    public String toString() {
        return String.format("Person{name='%s', age=%d, email='%s'}", name, age, email);
    }

    // getter 方法
    public String getName() { return name; }
    public int getAge() { return age; }
    public String getEmail() { return email; }
}

// equals 方法测试
public class EqualsDemo {
    public static void main(String[] args) {
        Person p1 = new Person("Alice", 25, "alice@example.com");
        Person p2 = new Person("Alice", 25, "alice@example.com");
        Person p3 = new Person("Bob", 30, "bob@example.com");
        Person p4 = p1; // 同一个对象

        System.out.println("=== equals 测试 ===");
        System.out.println("p1.equals(p2): " + p1.equals(p2)); // true
        System.out.println("p1.equals(p3): " + p1.equals(p3)); // false
        System.out.println("p1.equals(p4): " + p1.equals(p4)); // true
        System.out.println("p1.equals(null): " + p1.equals(null)); // false

        System.out.println("\n=== hashCode 测试 ===");
        System.out.println("p1.hashCode(): " + p1.hashCode());
        System.out.println("p2.hashCode(): " + p2.hashCode());
        System.out.println("p3.hashCode(): " + p3.hashCode());

        System.out.println("\n=== toString 测试 ===");

2. 不可变类的设计 ​

java
import java.util.*;

// 不可变类示例
public final class ImmutablePerson {
    private final String name;
    private final int age;
    private final List<String> hobbies;
    private final Map<String, String> properties;

    public ImmutablePerson(String name, int age, List<String> hobbies, Map<String, String> properties) {
        this.name = name;
        this.age = age;
        // 防御性拷贝
        this.hobbies = hobbies != null ? List.copyOf(hobbies) : List.of();
        this.properties = properties != null ? Map.copyOf(properties) : Map.of();
    }

    // 只提供 getter，不提供 setter
    public String getName() {
        return name;
    }

    public int getAge() {
        return age;
    }

    // 返回不可变集合
    public List<String> getHobbies() {
        return hobbies; // 已经是不可变的
    }

    public Map<String, String> getProperties() {
        return properties; // 已经是不可变的
    }

    // 修改操作返回新对象
    public ImmutablePerson withName(String newName) {
        return new ImmutablePerson(newName, this.age, this.hobbies, this.properties);
    }

    public ImmutablePerson withAge(int newAge) {
        return new ImmutablePerson(this.name, newAge, this.hobbies, this.properties);
    }

    public ImmutablePerson addHobby(String hobby) {
        List<String> newHobbies = new ArrayList<>(this.hobbies);
        newHobbies.add(hobby);
        return new ImmutablePerson(this.name, this.age, newHobbies, this.properties);
    }

    @Override
    public boolean equals(Object obj) {
        if (this == obj) return true;
        if (obj == null || getClass() != obj.getClass()) return false;

        ImmutablePerson that = (ImmutablePerson) obj;
        return age == that.age &&
               Objects.equals(name, that.name) &&
               Objects.equals(hobbies, that.hobbies) &&
               Objects.equals(properties, that.properties);
    }

    @Override
    public int hashCode() {
        return Objects.hash(name, age, hobbies, properties);
    }

    @Override
    public String toString() {
        return String.format("ImmutablePerson{name='%s', age=%d, hobbies=%s, properties=%s}",
                           name, age, hobbies, properties);
    }
}

// 不可变类测试
public class ImmutableDemo {

        ImmutablePerson person1 = new ImmutablePerson("Alice", 25, hobbies, props);

        ImmutablePerson person2 = person1.withAge(26);

1. 封装的实现 ​

java
public class BankAccount {
    // 私有字段，外部无法直接访问
    private String accountNumber;
    private String accountHolder;
    private double balance;
    private boolean isActive;

    // 构造方法
    public BankAccount(String accountNumber, String accountHolder, double initialBalance) {
        this.accountNumber = accountNumber;
        this.accountHolder = accountHolder;
        this.balance = initialBalance >= 0 ? initialBalance : 0;
        this.isActive = true;
    }

    // 受控的访问方法
    public String getAccountNumber() {
        return accountNumber;
    }

    public String getAccountHolder() {
        return accountHolder;
    }

    public double getBalance() {
        return isActive ? balance : 0; // 只有活跃账户才能查询余额
    }

    public boolean isActive() {
        return isActive;
    }

    // 业务方法，包含验证逻辑
    public boolean deposit(double amount) {
        if (!isActive) {
            System.out.println("账户已冻结，无法存款");
            return false;
        }

        if (amount <= 0) {
            System.out.println("存款金额必须大于0");
            return false;
        }

        balance += amount;
        System.out.printf("存款成功，金额: %.2f，余额: %.2f%n", amount, balance);
        return true;
    }

    public boolean withdraw(double amount) {
        if (!isActive) {
            System.out.println("账户已冻结，无法取款");
            return false;
        }

        if (amount <= 0) {
            System.out.println("取款金额必须大于0");
            return false;
        }

        if (amount > balance) {
            System.out.println("余额不足，无法取款");
            return false;
        }

        balance -= amount;
        System.out.printf("取款成功，金额: %.2f，余额: %.2f%n", amount, balance);
        return true;
    }

    public boolean transfer(BankAccount targetAccount, double amount) {
        if (this.withdraw(amount)) {
            if (targetAccount.deposit(amount)) {
                System.out.printf("转账成功，从 %s 转到 %s，金额: %.2f%n",
                                this.accountNumber, targetAccount.accountNumber, amount);
                return true;
            } else {
                // 如果目标账户存款失败，需要回滚
                this.deposit(amount);
                System.out.println("转账失败，已回滚");
                return false;
            }
        }
        return false;
    }

    // 管理方法
    public void freezeAccount() {
        isActive = false;
        System.out.println("账户已冻结");
    }

    public void activateAccount() {
        isActive = true;
        System.out.println("账户已激活");
    }

    @Override
    public String toString() {
        return String.format("BankAccount{accountNumber='%s', holder='%s', balance=%.2f, active=%s}",
                           accountNumber, accountHolder, balance, isActive);
    }
}

🧭 构造器与重载 ​

  • 无参/有参构造器可重载；第一行可显式调用 this(...) 或 super(...)。
  • 重载分派基于参数列表（数量、类型、顺序），与返回值无关。
java
class Ctor {
    private final int x;
    private final int y;

    public Ctor() { this(0, 0); }
    public Ctor(int x) { this(x, 0); }
    public Ctor(int x, int y) { this.x = x; this.y = y; }
}

🧩 this 与 super ​

  • this 引用当前对象；super 引用父类成员/构造器。
  • 构造器链：子类构造器必须先完成父类构造。
java
class Parent { Parent(){ System.out.println("P"); } }
class Child extends Parent {
    Child(){ super(); System.out.println("C"); }
}

⚙️ static 的要点 ​

  • 静态字段/方法归属于类；常用于工具方法、共享常量、工厂。
  • 初始化顺序：父类静态 → 子类静态 → 父类实例 → 子类实例。
java
public class Order {

🧪 equals/hashCode/toString ​

  • 覆盖 equals 时应同时覆盖 hashCode（保持相等对象哈希一致）。
  • toString 便于调试与日志；可读性优于拼接。
java
public class Point {
    private final int x, y;
    @Override public boolean equals(Object o){
        if (this == o) return true;
        if (!(o instanceof Point p)) return false;
        return x == p.x && y == p.y;
    }
    @Override public int hashCode(){ return Objects.hash(x, y); }
    @Override public String toString(){ return "Point(" + x + "," + y + ")"; }
}

💡 最佳实践 ​

类设计原则 ​

  • 单一职责：一个类应该只有一个改变的理由
  • 封装数据：使用 private 字段和公共方法
  • 验证输入：在构造方法和 setter 中验证参数
  • 不可变性：优先设计不可变类
  • 重写方法：正确重写 equals、hashCode 和 toString

常见错误避免 ​

  • 不要公开可变字段
  • 重写 equals 时必须重写 hashCode
  • 构造方法中不要调用可重写的方法
  • 避免在静态方法中访问实例成员
