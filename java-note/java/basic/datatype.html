Java 数据类型详解 ​

📚 概述 ​

Java 是一种强类型语言，所有变量都必须先声明类型才能使用。Java 的数据类型分为两大类：基本数据类型（Primitive Types）和引用数据类型（Reference Types）。

🎯 学习目标 ​

  • 掌握 Java 中所有基本数据类型的特点和使用
  • 理解基本类型与引用类型的区别
  • 熟悉类型转换的规则和注意事项
  • 掌握包装类型的使用和自动装箱/拆箱机制
  • 理解常量和字面量的概念

📊 数据类型分类 ​

Java 数据类型
├── 基本数据类型 (Primitive Types)
│   ├── 整数类型
│   │   ├── byte (8位)
│   │   ├── short (16位)
│   │   ├── int (32位)
│   │   └── long (64位)
│   ├── 浮点类型
│   │   ├── float (32位)
│   │   └── double (64位)
│   ├── 字符类型
│   │   └── char (16位)
│   └── 布尔类型
│       └── boolean (1位)
└── 引用数据类型 (Reference Types)
    ├── 类 (Class)
    ├── 接口 (Interface)
    ├── 数组 (Array)
    └── 枚举 (Enum)

🔢 基本数据类型详解 ​

1. 整数类型 ​

byte 类型 ​

java
// byte: 8位有符号整数
byte b1 = 127;          // 最大值
byte b2 = -128;         // 最小值
byte b3 = 0;            // 默认值

// 取值范围：-128 到 127 (-2^7 到 2^7-1)
System.out.println("byte 最小值: " + Byte.MIN_VALUE);  // -128
System.out.println("byte 最大值: " + Byte.MAX_VALUE);  // 127
System.out.println("byte 字节数: " + Byte.BYTES);      // 1

// 使用场景：节省内存空间，处理二进制数据
byte[] data = {0x01, 0x02, 0x03};  // 字节数组

short 类型 ​

java
// short: 16位有符号整数
short s1 = 32767;       // 最大值
short s2 = -32768;      // 最小值
short s3 = 0;           // 默认值

// 取值范围：-32768 到 32767 (-2^15 到 2^15-1)
System.out.println("short 最小值: " + Short.MIN_VALUE);  // -32768
System.out.println("short 最大值: " + Short.MAX_VALUE);  // 32767
System.out.println("short 字节数: " + Short.BYTES);      // 2

// 使用场景：比 int 节省内存，但比 byte 范围大
short year = 2024;      // 年份
short port = 8080;      // 端口号

int 类型 ​

java
// int: 32位有符号整数（最常用的整数类型）
int i1 = 2147483647;    // 最大值
int i2 = -2147483648;   // 最小值
int i3 = 0;             // 默认值

// 取值范围：-2147483648 到 2147483647 (-2^31 到 2^31-1)
System.out.println("int 最小值: " + Integer.MIN_VALUE);
System.out.println("int 最大值: " + Integer.MAX_VALUE);
System.out.println("int 字节数: " + Integer.BYTES);     // 4

// 不同进制表示
int decimal = 100;      // 十进制
int binary = 0b1100100; // 二进制（Java 7+）
int octal = 0144;       // 八进制
int hex = 0x64;         // 十六进制

// 使用下划线分隔（Java 7+）
int million = 1_000_000;
int binary2 = 0b1010_0001_1000_0101;

long 类型 ​

java
// long: 64位有符号整数
long l1 = 9223372036854775807L;  // 最大值，注意 L 后缀
long l2 = -9223372036854775808L; // 最小值
long l3 = 0L;                    // 默认值

// 取值范围：-2^63 到 2^63-1
System.out.println("long 最小值: " + Long.MIN_VALUE);
System.out.println("long 最大值: " + Long.MAX_VALUE);
System.out.println("long 字节数: " + Long.BYTES);      // 8

// 使用场景：需要大整数的计算
long timestamp = System.currentTimeMillis();  // 时间戳
long fileSize = 1024L * 1024L * 1024L;        // 文件大小（GB）

2. 浮点类型 ​

float 类型 ​

java
// float: 32位单精度浮点数
float f1 = 3.14f;       // 注意 f 后缀
float f2 = 1.23e-4f;    // 科学计数法
float f3 = 0.0f;        // 默认值

// 特殊值
float positiveInfinity = Float.POSITIVE_INFINITY;  // 正无穷
float negativeInfinity = Float.NEGATIVE_INFINITY;  // 负无穷
float notANumber = Float.NaN;                      // 非数字

System.out.println("float 最小值: " + Float.MIN_VALUE);      // 最小正值
System.out.println("float 最大值: " + Float.MAX_VALUE);
System.out.println("float 字节数: " + Float.BYTES);          // 4

// 精度问题示例
float a = 0.1f;
float b = 0.2f;
float c = a + b;
System.out.println(c);  // 可能不是精确的 0.3

double 类型 ​

java
// double: 64位双精度浮点数（默认浮点类型）
double d1 = 3.14159265359;  // 不需要后缀
double d2 = 1.23e-10;       // 科学计数法
double d3 = 0.0;            // 默认值

System.out.println("double 最小值: " + Double.MIN_VALUE);
System.out.println("double 最大值: " + Double.MAX_VALUE);
System.out.println("double 字节数: " + Double.BYTES);        // 8

// 精确计算建议使用 BigDecimal
import java.math.BigDecimal;
BigDecimal bd1 = new BigDecimal("0.1");
BigDecimal bd2 = new BigDecimal("0.2");
BigDecimal bd3 = bd1.add(bd2);
System.out.println(bd3);  // 精确的 0.3

3. 字符类型 ​

char 类型 ​

java
// char: 16位 Unicode 字符
char c1 = 'A';          // 字符字面量
char c2 = 65;           // ASCII 码值
char c3 = '\u0041';     // Unicode 编码
char c4 = '\0';         // 默认值（空字符）

// 常用转义字符
char newline = '\n';    // 换行
char tab = '\t';        // 制表符
char backslash = '\\';  // 反斜杠
char quote = '\'';      // 单引号
char doubleQuote = '\"'; // 双引号

// 中文字符
char chinese = '中';

// 字符运算
char ch = 'A';

System.out.println(ch);

// 字符判断
System.out.println(Character.isLetter('A'));    // true
System.out.println(Character.isDigit('5'));     // true
System.out.println(Character.isWhitespace(' ')); // true

4. 布尔类型 ​

boolean 类型 ​

java
// boolean: 逻辑类型，只有 true 和 false
boolean flag1 = true;

// 逻辑运算
boolean and = true && false;    // false

// int i = true;   // 编译错误

🔄 类型转换 ​

1. 自动类型转换（隐式转换） ​

java
// 小范围类型自动转换为大范围类型
byte b = 10;
short s = b;        // byte → short
int i = s;          // short → int
long l = i;         // int → long
float f = l;        // long → float
double d = f;       // float → double

// 转换路径：byte → short → int → long → float → double
//                    ↑
//                  char

// 字符类型转换
char c = 'A';
int ascii = c;      // char → int，得到 ASCII 值 65

2. 强制类型转换（显式转换） ​

java
// 大范围类型转换为小范围类型需要强制转换
double d = 3.14;
float f = (float) d;    // 可能丢失精度
int i = (int) f;        // 丢失小数部分，i = 3
short s = (short) i;
byte b = (byte) s;

// 注意数据溢出
int big = 300;
byte small = (byte) big;  // 结果是 44，发生溢出

// 字符转换
int num = 65;
char ch = (char) num;     // 得到 'A'

3. 表达式中的类型提升 ​

java
// 算术运算中的自动类型提升
byte b1 = 10;
byte b2 = 20;
// byte result = b1 + b2;  // 编译错误！
int result = b1 + b2;      // 正确，byte 运算结果提升为 int

// 混合类型运算
int i = 10;
double d = 3.14;
double result2 = i + d;    // int 自动提升为 double

// char 参与运算
char c = 'A';
int ascii = c + 1;         // 得到 66

📦 包装类型 ​

1. 基本类型对应的包装类 ​

java
// 每个基本类型都有对应的包装类
byte    → Byte
short   → Short
int     → Integer
long    → Long
float   → Float
double  → Double
char    → Character
boolean → Boolean

2. 自动装箱和拆箱 ​

java
// 自动装箱（基本类型 → 包装类型）
Integer i = 100;           // 等价于 Integer.valueOf(100)
Double d = 3.14;           // 等价于 Double.valueOf(3.14)
Boolean b = true;          // 等价于 Boolean.valueOf(true)

// 自动拆箱（包装类型 → 基本类型）
int num = i;               // 等价于 i.intValue()
double value = d;          // 等价于 d.doubleValue()
boolean flag = b;          // 等价于 b.booleanValue()

// 运算中的装箱拆箱
Integer a = 100;
Integer b = 200;
Integer c = a + b;         // 拆箱运算后装箱

3. 包装类型的缓存机制 ​

java
// Integer 缓存 -128 到 127
Integer a1 = 127;
Integer a2 = 127;
System.out.println(a1 == a2);  // true，使用缓存

Integer b1 = 128;
Integer b2 = 128;
System.out.println(b1 == b2);  // false，超出缓存范围

// 正确的比较方式
System.out.println(b1.equals(b2));  // true

// 其他包装类型的缓存
Boolean.TRUE 和 Boolean.FALSE     // Boolean 缓存
Character 0-127                   // Character 缓存
Byte 全部值 (-128 到 127)         // Byte 缓存
Short -128 到 127                 // Short 缓存
Long -128 到 127                  // Long 缓存

public class DataTypeSelection {

    }

java
public class TypeConversionExample {

        String str = "123";

示例3：精度处理 ​

java
import java.math.BigDecimal;
import java.math.RoundingMode;

public class PrecisionExample {
    public static void main(String[] args) {
        // 浮点数精度问题
        double a = 0.1;
        double b = 0.2;
        double c = a + b;
        System.out.println(c);  // 0.30000000000000004
        
        // 使用 BigDecimal 解决精度问题
        BigDecimal bd1 = new BigDecimal("0.1");
        BigDecimal bd2 = new BigDecimal("0.2");
        BigDecimal bd3 = bd1.add(bd2);
        System.out.println(bd3);  // 0.3
        
        // 金额计算示例
        BigDecimal price = new BigDecimal("99.99");
        BigDecimal quantity = new BigDecimal("3");
        BigDecimal total = price.multiply(quantity);
        
        // 四舍五入到两位小数
        BigDecimal rounded = total.setScale(2, RoundingMode.HALF_UP);
        System.out.println("总价: " + rounded);
    }
}

⚠️ 注意事项和最佳实践 ​

1. 数据类型选择原则 ​

java
// 1. 优先使用基本类型，性能更好
int count = 0;              // 好
Integer count2 = 0;         // 避免，除非需要 null 值

// 2. 根据数据范围选择合适的类型
byte age = 25;              // 年龄用 byte 足够
int population = 1000000;   // 人口数用 int
long timestamp = System.currentTimeMillis();  // 时间戳用 long

// 3. 浮点数默认使用 double
double price = 99.99;       // 好
float price2 = 99.99f;      // 除非内存敏感

2. 常见陷阱 ​

java
// 1. 整数除法截断
int a = 5;
int b = 2;
double result = a / b;      // 结果是 2.0，不是 2.5
double correct = (double) a / b;  // 正确结果 2.5

// 2. 浮点数比较
double x = 0.1 + 0.2;
// if (x == 0.3) { ... }    // 可能为 false
if (Math.abs(x - 0.3) < 1e-10) { ... }  // 正确的比较方式

// 3. 包装类型的 null 值
Integer num = null;
// int value = num;         // NullPointerException
int value = (num != null) ? num : 0;  // 安全的拆箱

// 4. 字符编码问题
char c1 = '中';             // 正确
// char c2 = '中国';        // 错误，char 只能存储一个字符

3. 性能考虑 ​

java
// 1. 避免不必要的装箱拆箱
List<Integer> list = new ArrayList<>();
for (int i = 0; i < 1000; i++) {
    list.add(i);  // 每次都会装箱，性能影响
}

// 2. 使用基本类型数组而不是包装类型数组
int[] intArray = new int[1000];      // 好
Integer[] integerArray = new Integer[1000];  // 内存占用更大

// 3. 字符串拼接使用 StringBuilder
StringBuilder sb = new StringBuilder();
for (int i = 0; i < 1000; i++) {
    sb.append(i);  // 高效
}
String result = sb.toString();

💡 重点总结：

  1. Java 有 8 种基本数据类型，每种都有特定的用途和范围
  2. 类型转换遵循从小到大自动转换，从大到小需要强制转换
  3. 包装类型提供了更多功能，但性能略低于基本类型
  4. 浮点数计算存在精度问题，金融计算建议使用 BigDecimal
  5. 选择合适的数据类型对程序性能和内存使用很重要
