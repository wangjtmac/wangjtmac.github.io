Java Lambda 表达式详解 ​

📚 概述 ​

Lambda 表达式是 Java 8 引入的一个重要特性，它允许我们以更简洁的方式表示匿名函数。Lambda 表达式本质上是一个匿名函数，可以作为参数传递给方法，或者赋值给函数式接口类型的变量。它的引入使得 Java 支持了函数式编程范式，让代码更加简洁、可读性更强。Lambda 表达式的语法形式为 (parameters) -> expression 或 (parameters) -> { statements; }，其中参数列表、箭头符号和函数体是三个基本组成部分。Lambda 表达式只能用于函数式接口（只有一个抽象方法的接口），编译器会进行类型推断来确定 Lambda 表达式的具体类型。掌握 Lambda 表达式是现代 Java 开发的基础，它与 Stream API、函数式接口等特性紧密结合，是编写高质量 Java 代码的重要工具。

🎯 学习目标 ​

  • 深入理解 Lambda 表达式的语法和语义
  • 掌握 Lambda 表达式的各种使用形式
  • 理解 Lambda 表达式与函数式接口的关系
  • 学会使用方法引用简化 Lambda 表达式
  • 掌握 Lambda 表达式的变量捕获机制
  • 理解 Lambda 表达式的类型推断
  • 学会在实际项目中应用 Lambda 表达式
  • 了解 Lambda 表达式的性能特点和最佳实践

// Lambda 表达式基础语法演示
import java.util.*;
import java.util.function.*;

public class LambdaSyntaxDemo {
    
    public static void main(String[] args) {
        demonstrateBasicSyntax();
        demonstrateParameterVariations();
        demonstrateBodyVariations();
        demonstrateTypeInference();
    }
    
    // 基础语法演示
    public static void demonstrateBasicSyntax() {
        System.out.println("=== Lambda 基础语法演示 ===");
        
        /*
         * Lambda 表达式的基本语法：
         * (parameter1, parameter2, ...) -> expression
         * (parameter1, parameter2, ...) -> { statements; }
         * 
         * 组成部分：
         * 1. 参数列表 - 用括号包围，多个参数用逗号分隔
         * 2. 箭头符号 - 分隔参数列表和函数体
         * 3. 函数体 - 表达式或语句块
         */
        
        System.out.println("Lambda 表达式基础语法:");
        
        // 1. 最简单的 Lambda 表达式
        Runnable simpleRunnable = () -> System.out.println("Hello Lambda!");
        simpleRunnable.run();
        
        // 2. 带参数的 Lambda 表达式
        Consumer<String> stringConsumer = (String s) -> System.out.println("消费: " + s);
        stringConsumer.accept("Lambda 参数");
        
        // 3. 带返回值的 Lambda 表达式
        Function<Integer, Integer> square = (Integer x) -> x * x;
        System.out.println("平方: " + square.apply(5));
        
        // 4. 多参数 Lambda 表达式
        BinaryOperator<Integer> add = (Integer a, Integer b) -> a + b;
        System.out.println("加法: " + add.apply(3, 7));
        
        // 5. 多行 Lambda 表达式
        Function<String, String> processString = (String input) -> {
            String trimmed = input.trim();
            String upperCase = trimmed.toUpperCase();
            return "处理结果: " + upperCase;
        };
        System.out.println(processString.apply("  hello world  "));
        
        System.out.println();
    }
    
    // 参数变化演示
    public static void demonstrateParameterVariations() {
        System.out.println("=== 参数变化演示 ===");
        
        /*
         * 参数列表的变化形式：
         * 1. 无参数: () -> expression
         * 2. 单参数: param -> expression 或 (param) -> expression
         * 3. 多参数: (param1, param2) -> expression
         * 4. 类型推断: 编译器自动推断参数类型
         */
        
        System.out.println("参数列表变化:");
        
        // 1. 无参数
        Supplier<String> noParams = () -> "无参数 Lambda";
        System.out.println(noParams.get());
        
        // 2. 单参数（可省略括号）
        Function<String, Integer> singleParam = s -> s.length();
        System.out.println("单参数（省略括号）: " + singleParam.apply("Hello"));
        
        Function<String, Integer> singleParamWithParens = (s) -> s.length();
        System.out.println("单参数（保留括号）: " + singleParamWithParens.apply("World"));
        
        // 3. 多参数（必须有括号）
        BiFunction<String, String, String> multiParams = (s1, s2) -> s1 + " " + s2;
        System.out.println("多参数: " + multiParams.apply("Hello", "Lambda"));
        
        // 4. 显式类型声明
        BiFunction<Integer, Integer, Integer> explicitTypes = (Integer a, Integer b) -> a * b;
        System.out.println("显式类型: " + explicitTypes.apply(4, 6));
        
        // 5. 类型推断
        BiFunction<Integer, Integer, Integer> inferredTypes = (a, b) -> a * b;
        System.out.println("类型推断: " + inferredTypes.apply(4, 6));
        
        // 6. 混合使用（不推荐）
        // BiFunction<Integer, Integer, Integer> mixed = (Integer a, b) -> a * b; // 编译错误
        
        System.out.println();
    }
    
    // 函数体变化演示
    public static void demonstrateBodyVariations() {
        System.out.println("=== 函数体变化演示 ===");
        
        /*
         * 函数体的形式：
         * 1. 表达式: param -> expression
         * 2. 语句块: param -> { statements; }
         * 3. 返回值处理
         */
        
        System.out.println("函数体变化:");
        
        // 1. 单表达式（自动返回）
        Function<Integer, Integer> expressionBody = x -> x * 2;
        System.out.println("表达式体: " + expressionBody.apply(5));
        
        // 2. 语句块（需要显式 return）
        Function<Integer, Integer> statementBody = x -> {
            int result = x * 2;
            System.out.println("计算中间结果: " + result);
            return result;
        };
        System.out.println("语句块体: " + statementBody.apply(5));
        
        // 3. 无返回值的语句块
        Consumer<String> voidBody = message -> {
            System.out.println("开始处理消息");
            System.out.println("消息内容: " + message);
            System.out.println("处理完成");
        };
        voidBody.accept("测试消息");
        
        // 4. 复杂逻辑的语句块
        Function<List<Integer>, Integer> complexBody = numbers -> {
            if (numbers == null || numbers.isEmpty()) {
                return 0;
            }
            
            int sum = 0;
            for (Integer num : numbers) {
                if (num != null && num > 0) {
                    sum += num;
                }
            }
            
            return sum;
        };
        
        List<Integer> testNumbers = Arrays.asList(1, 2, 3, 4, 5);
        System.out.println("复杂逻辑结果: " + complexBody.apply(testNumbers));
        
        System.out.println();
    }
    
    // 类型推断演示
    public static void demonstrateTypeInference() {
        System.out.println("=== 类型推断演示 ===");
        
        /*
         * Lambda 表达式的类型推断：
         * 1. 目标类型推断
         * 2. 参数类型推断
         * 3. 返回类型推断
         * 4. 泛型类型推断
         */
        
        System.out.println("类型推断:");
        
        // 1. 目标类型推断
        Predicate<String> isLong = s -> s.length() > 5;  // 推断为 Predicate<String>
        Function<String, Boolean> isLongFunc = s -> s.length() > 5;  // 推断为 Function<String, Boolean>
        
        System.out.println("目标类型推断 - Predicate: " + isLong.test("Hello World"));
        System.out.println("目标类型推断 - Function: " + isLongFunc.apply("Hello World"));
        
        // 2. 参数类型推断
        List<String> words = Arrays.asList("apple", "banana", "cherry");
        
        // 编译器推断参数类型为 String
        words.stream()
             .filter(word -> word.length() > 5)  // word 被推断为 String
             .map(word -> word.toUpperCase())    // word 被推断为 String
             .forEach(word -> System.out.println("长单词: " + word));  // word 被推断为 String
        
        // 3. 泛型类型推断
        List<Integer> numbers = Arrays.asList(1, 2, 3, 4, 5);
        
        // 编译器推断泛型类型
        Optional<Integer> max = numbers.stream()
            .filter(n -> n > 2)     // n 被推断为 Integer
            .max((a, b) -> a - b);  // a, b 被推断为 Integer
        
        System.out.println("最大值: " + max.orElse(0));
        
        // 4. 复杂泛型推断
        Map<String, List<Integer>> groupedData = new HashMap<>();
        groupedData.put("evens", Arrays.asList(2, 4, 6));
        groupedData.put("odds", Arrays.asList(1, 3, 5));
        
        groupedData.entrySet().stream()
            .filter(entry -> entry.getValue().size() > 2)  // entry 被推断为 Map.Entry<String, List<Integer>>
            .forEach(entry -> System.out.println("组: " + entry.getKey() + ", 大小: " + entry.getValue().size()));
        
        // 5. 类型推断的限制
        // 有时需要显式类型声明
        Comparator<String> lengthComparator = (String s1, String s2) -> s1.length() - s2.length();
        
        // 或者使用方法引用
        Comparator<String> lengthComparatorRef = Comparator.comparing(String::length);
        
        words.sort(lengthComparator);
        System.out.println("按长度排序: " + words);
        
        System.out.println();
    }
}

// 方法引用详细演示
import java.util.*;
import java.util.function.*;

public class MethodReferenceDemo {
    
    public static void main(String[] args) {
        demonstrateStaticMethodReference();
        demonstrateInstanceMethodReference();
        demonstrateConstructorReference();
        demonstrateArbitraryObjectMethodReference();
    }
    
    // 静态方法引用演示
    public static void demonstrateStaticMethodReference() {
        System.out.println("=== 静态方法引用演示 ===");
        
        /*
         * 静态方法引用语法：
         * ClassName::staticMethodName
         * 
         * 等价于：
         * (args) -> ClassName.staticMethodName(args)
         */
        
        System.out.println("静态方法引用:");
        
        List<String> numbers = Arrays.asList("1", "2", "3", "4", "5");
        
        // 1. Lambda 表达式形式
        List<Integer> lambdaParsed = numbers.stream()
            .map(s -> Integer.parseInt(s))
            .collect(Collectors.toList());
        System.out.println("Lambda 解析: " + lambdaParsed);
        
        // 2. 静态方法引用形式
        List<Integer> methodRefParsed = numbers.stream()
            .map(Integer::parseInt)
            .collect(Collectors.toList());
        System.out.println("方法引用解析: " + methodRefParsed);
        
        // 3. 更多静态方法引用示例
        List<Double> values = Arrays.asList(1.1, 2.2, 3.3, 4.4);
        
        // Math.sqrt 方法引用
        List<Double> sqrtValues = values.stream()
            .map(Math::sqrt)
            .collect(Collectors.toList());
        System.out.println("平方根: " + sqrtValues);
        
        // String.valueOf 方法引用
        List<String> stringValues = values.stream()
            .map(String::valueOf)
            .collect(Collectors.toList());
        System.out.println("转字符串: " + stringValues);
        
        // 4. 自定义静态方法引用
        List<Integer> nums = Arrays.asList(1, 2, 3, 4, 5);
        
        List<Integer> doubled = nums.stream()
            .map(MathUtils::doubleValue)  // 引用自定义静态方法
            .collect(Collectors.toList());
        System.out.println("加倍: " + doubled);
        
        List<Boolean> evenChecks = nums.stream()
            .map(MathUtils::isEven)
            .collect(Collectors.toList());
        System.out.println("偶数检查: " + evenChecks);
        
        System.out.println();
    }
    
    // 实例方法引用演示
    public static void demonstrateInstanceMethodReference() {
        System.out.println("=== 实例方法引用演示 ===");
        
        /*
         * 实例方法引用语法：
         * instance::instanceMethodName
         * 
         * 等价于：
         * (args) -> instance.instanceMethodName(args)
         */
        
        System.out.println("实例方法引用:");
        
        List<String> words = Arrays.asList("apple", "banana", "cherry", "date");
        
        // 1. 字符串实例方法引用
        String prefix = "Fruit: ";
        
        // Lambda 表达式形式
        List<String> lambdaPrefixed = words.stream()
            .map(word -> prefix.concat(word))
            .collect(Collectors.toList());
        System.out.println("Lambda 前缀: " + lambdaPrefixed);
        
        // 实例方法引用形式
        List<String> methodRefPrefixed = words.stream()
            .map(prefix::concat)
            .collect(Collectors.toList());
        System.out.println("方法引用前缀: " + methodRefPrefixed);
        
        // 2. 自定义对象的实例方法引用
        StringProcessor processor = new StringProcessor();
        
        List<String> processed = words.stream()
            .map(processor::process)  // 引用实例方法
            .collect(Collectors.toList());
        System.out.println("自定义处理: " + processed);
        
        List<Integer> lengths = words.stream()
            .map(processor::getLength)
            .collect(Collectors.toList());
        System.out.println("长度计算: " + lengths);
        
        // 3. 系统对象的实例方法引用
        PrintStream out = System.out;
        words.forEach(out::println);  // 等价于 word -> System.out.println(word)
        
        // 4. 集合实例方法引用
        List<String> result = new ArrayList<>();
        words.stream()
             .filter(word -> word.length() > 4)
             .forEach(result::add);  // 等价于 word -> result.add(word)
        System.out.println("过滤结果: " + result);
        
        System.out.println();
    }
    
    // 构造器引用演示
    public static void demonstrateConstructorReference() {
        System.out.println("=== 构造器引用演示 ===");
        
        /*
         * 构造器引用语法：
         * ClassName::new
         * 
         * 等价于：
         * (args) -> new ClassName(args)
         */
        
        System.out.println("构造器引用:");
        
        List<String> names = Arrays.asList("Alice", "Bob", "Charlie");
        List<Integer> ages = Arrays.asList(25, 30, 35);
        
        // 1. 无参构造器引用
        Supplier<List<String>> listSupplier = ArrayList::new;
        List<String> newList = listSupplier.get();
        System.out.println("新列表: " + newList);
        
        // 2. 有参构造器引用
        Function<String, StringBuilder> sbCreator = StringBuilder::new;
        StringBuilder sb = sbCreator.apply("Hello");
        System.out.println("StringBuilder: " + sb.toString());
        
        // 3. 创建对象列表
        List<Person> people = names.stream()
            .map(Person::new)  // 引用 Person(String name) 构造器
            .collect(Collectors.toList());
        System.out.println("人员列表: " + people);
        
        // 4. 数组构造器引用
        Function<Integer, String[]> arrayCreator = String[]::new;
        String[] stringArray = arrayCreator.apply(5);
        System.out.println("数组长度: " + stringArray.length);
        
        // 在 Stream 中使用数组构造器引用
        String[] wordsArray = names.stream()
            .toArray(String[]::new);  // 等价于 size -> new String[size]
        System.out.println("转换为数组: " + Arrays.toString(wordsArray));
        
        // 5. 复杂构造器引用
        BiFunction<String, Integer, Employee> employeeCreator = Employee::new;
        Employee emp = employeeCreator.apply("John", 28);
        System.out.println("员工: " + emp);
        
        System.out.println();
    }
    
    // 任意对象的实例方法引用演示
    public static void demonstrateArbitraryObjectMethodReference() {
        System.out.println("=== 任意对象的实例方法引用演示 ===");
        
        /*
         * 任意对象的实例方法引用语法：
         * ClassName::instanceMethodName
         * 
         * 等价于：
         * (obj, args) -> obj.instanceMethodName(args)
         */
        
        System.out.println("任意对象的实例方法引用:");
        
        List<String> words = Arrays.asList("apple", "BANANA", "Cherry", "DATE");
        
        // 1. 字符串方法引用
        List<String> lowerCaseWords = words.stream()
            .map(String::toLowerCase)  // 等价于 s -> s.toLowerCase()
            .collect(Collectors.toList());
        System.out.println("转小写: " + lowerCaseWords);
        
        List<Integer> lengths = words.stream()
            .map(String::length)  // 等价于 s -> s.length()
            .collect(Collectors.toList());
        System.out.println("长度: " + lengths);
        
        // 2. 排序中的方法引用
        List<String> sortedWords = words.stream()
            .sorted(String::compareToIgnoreCase)  // 等价于 (s1, s2) -> s1.compareToIgnoreCase(s2)
            .collect(Collectors.toList());
        System.out.println("忽略大小写排序: " + sortedWords);
        
        // 3. 对象方法引用
        List<Person> people = Arrays.asList(
            new Person("Alice"),
            new Person("Bob"),
            new Person("Charlie")
        );
        
        List<String> personNames = people.stream()
            .map(Person::getName)  // 等价于 person -> person.getName()
            .collect(Collectors.toList());
        System.out.println("人员姓名: " + personNames);
        
        // 4. 复杂对象方法引用
        List<Employee> employees = Arrays.asList(
            new Employee("Alice", 25),
            new Employee("Bob", 30),
            new Employee("Charlie", 35)
        );
        
        // 按年龄排序
        employees.sort(Comparator.comparing(Employee::getAge));  // 等价于 emp -> emp.getAge()
        System.out.println("按年龄排序: " + employees);
        
        // 按姓名排序
        employees.sort(Comparator.comparing(Employee::getName));
        System.out.println("按姓名排序: " + employees);
        
        System.out.println();
    }
}

1. 变量捕获规则和限制 ​

// Lambda 变量捕获详细演示
import java.util.*;
import java.util.function.*;

public class LambdaCaptureDemo {

    private static String staticField = "静态字段";
    private String instanceField = "实例字段";

    public static void main(String[] args) {
        LambdaCaptureDemo demo = new LambdaCaptureDemo();
        demo.demonstrateLocalVariableCapture();
        demo.demonstrateFieldCapture();
        demo.demonstrateCaptureRestrictions();
    }

    // 局部变量捕获演示
    public void demonstrateLocalVariableCapture() {
        System.out.println("=== 局部变量捕获演示 ===");

        /*
         * 局部变量捕获规则：
         * 1. 只能捕获 final 或 effectively final 变量
         * 2. 不能修改被捕获的局部变量
         * 3. 编译器会检查变量的 final 性质
         */

        System.out.println("局部变量捕获:");

        // 1. 捕获 final 变量
        final String finalVar = "final 变量";
        Supplier<String> supplier1 = () -> "捕获: " + finalVar;
        System.out.println(supplier1.get());

        // 2. 捕获 effectively final 变量
        String effectivelyFinal = "effectively final 变量";
        Supplier<String> supplier2 = () -> "捕获: " + effectivelyFinal;
        System.out.println(supplier2.get());
        // effectivelyFinal = "修改"; // 这行代码会导致编译错误

        // 3. 基本类型捕获
        int number = 42;
        IntSupplier intSupplier = () -> number * 2;
        System.out.println("捕获基本类型: " + intSupplier.getAsInt());

        // 4. 对象引用捕获
        List<String> list = new ArrayList<>();
        list.add("初始元素");

        Consumer<String> addToList = item -> list.add(item);  // 可以修改对象内容
        addToList.accept("新元素");
        System.out.println("修改捕获的对象: " + list);

        // 5. 复杂对象捕获
        Map<String, Integer> scores = new HashMap<>();
        scores.put("Alice", 95);
        scores.put("Bob", 87);

        BiConsumer<String, Integer> updateScore = (name, score) -> {
            scores.put(name, score);  // 可以修改 Map 内容
            System.out.println("更新分数: " + name + " = " + score);
        };

        updateScore.accept("Charlie", 92);
        System.out.println("最终分数: " + scores);

        System.out.println();
    }

    // 字段捕获演示
    public void demonstrateFieldCapture() {
        System.out.println("=== 字段捕获演示 ===");

        /*
         * 字段捕获规则：
         * 1. 可以访问和修改实例字段
         * 2. 可以访问和修改静态字段
         * 3. 字段访问没有 final 限制
         */

        System.out.println("字段捕获:");

        // 1. 访问实例字段
        Supplier<String> instanceFieldGetter = () -> "访问实例字段: " + instanceField;
        System.out.println(instanceFieldGetter.get());

        // 2. 修改实例字段
        Consumer<String> instanceFieldSetter = value -> {
            instanceField = value;
            System.out.println("修改实例字段为: " + value);
        };
        instanceFieldSetter.accept("新的实例字段值");
        System.out.println("当前实例字段: " + instanceField);

        // 3. 访问静态字段
        Supplier<String> staticFieldGetter = () -> "访问静态字段: " + staticField;
        System.out.println(staticFieldGetter.get());

        // 4. 修改静态字段
        Consumer<String> staticFieldSetter = value -> {
            staticField = value;
            System.out.println("修改静态字段为: " + value);
        };
        staticFieldSetter.accept("新的静态字段值");
        System.out.println("当前静态字段: " + staticField);

        System.out.println();
    }

    // 捕获限制演示
    public void demonstrateCaptureRestrictions() {
        System.out.println("=== 捕获限制演示 ===");

        /*
         * 捕获限制：
         * 1. 不能修改局部变量
         * 2. 循环变量的特殊处理
         * 3. 异常处理中的变量捕获
         */

        System.out.println("捕获限制:");

        // 1. 使用数组或对象包装解决计数问题
        int[] counterArray = {0};
        List<Runnable> tasks = new ArrayList<>();

        for (int i = 0; i < 5; i++) {
            final int index = i;  // 捕获循环变量
            tasks.add(() -> {
                counterArray[0]++;
                System.out.println("任务 " + index + " 执行，计数器: " + counterArray[0]);
            });
        }

        tasks.forEach(Runnable::run);

        // 2. 使用原子类解决计数问题
        java.util.concurrent.atomic.AtomicInteger atomicCounter =
            new java.util.concurrent.atomic.AtomicInteger(0);

        List<Runnable> atomicTasks = new ArrayList<>();
        for (int i = 0; i < 3; i++) {
            final int index = i;
            atomicTasks.add(() -> {
                int count = atomicCounter.incrementAndGet();
                System.out.println("原子任务 " + index + " 执行，计数器: " + count);
            });
        }

        atomicTasks.forEach(Runnable::run);

        // 3. 异常处理中的变量捕获
        String resource = "重要资源";

        Runnable riskyTask = () -> {
            try {
                System.out.println("处理资源: " + resource);
                if (Math.random() > 0.5) {
                    throw new RuntimeException("随机异常");
                }
                System.out.println("资源处理成功");
            } catch (Exception e) {
                System.out.println("处理资源 " + resource + " 时发生异常: " + e.getMessage());
            }
        };

        riskyTask.run();

        System.out.println();
    }
}

import java.util.*;
import java.util.stream.*;
import java.util.function.*;
import java.util.concurrent.*;

public class LambdaApplicationDemo {

        demonstrateCollectionProcessing();

    // 集合处理应用
    public static void demonstrateCollectionProcessing() {
        System.out.println("=== 集合处理应用演示 ===");

        List<Employee> employees = Arrays.asList(
            new Employee("Alice", 25, "Engineering", 75000),
            new Employee("Bob", 30, "Marketing", 65000),
            new Employee("Charlie", 35, "Engineering", 85000),
            new Employee("Diana", 28, "Sales", 60000),
            new Employee("Eve", 32, "Engineering", 80000)
        );

        System.out.println("集合处理应用:");

        // 1. 数据过滤和转换
        List<String> highEarners = employees.stream()
            .filter(emp -> emp.getSalary() > 70000)
            .map(Employee::getName)
            .collect(Collectors.toList());
        System.out.println("高收入员工: " + highEarners);

        // 2. 分组和聚合
        Map<String, Double> avgSalaryByDept = employees.stream()
            .collect(Collectors.groupingBy(
                Employee::getDepartment,
                Collectors.averagingDouble(Employee::getSalary)
            ));
        System.out.println("各部门平均薪资: " + avgSalaryByDept);

        // 3. 复杂业务逻辑
        Optional<Employee> topPerformer = employees.stream()
            .filter(emp -> "Engineering".equals(emp.getDepartment()))
            .max(Comparator.comparing(Employee::getSalary));

        topPerformer.ifPresent(emp ->
            System.out.println("工程部最高薪资: " + emp.getName() + " - $" + emp.getSalary()));

        // 4. 数据验证
        boolean allAdults = employees.stream()
            .allMatch(emp -> emp.getAge() >= 18);
        System.out.println("所有员工都是成年人: " + allAdults);

        // 5. 自定义收集器
        String employeeSummary = employees.stream()
            .map(emp -> emp.getName() + "(" + emp.getDepartment() + ")")
            .collect(Collectors.joining(", ", "员工列表: [", "]"));
        System.out.println(employeeSummary);

        System.out.println();
    }

    // 事件处理应用
    public static void demonstrateEventHandling() {
        System.out.println("=== 事件处理应用演示 ===");

        System.out.println("事件处理应用:");

        // 1. 事件监听器
        EventManager eventManager = new EventManager();

        // 注册不同类型的事件处理器
        eventManager.addEventListener("user.login", event ->
            System.out.println("用户登录事件: " + event.getData()));

        eventManager.addEventListener("user.logout", event ->
            System.out.println("用户登出事件: " + event.getData()));

        eventManager.addEventListener("order.created", event -> {
            System.out.println("订单创建事件: " + event.getData());
            // 可以在这里添加复杂的业务逻辑
        });

        // 触发事件
        eventManager.fireEvent(new Event("user.login", "用户 Alice 登录"));
        eventManager.fireEvent(new Event("order.created", "订单 #12345"));
        eventManager.fireEvent(new Event("user.logout", "用户 Alice 登出"));

        // 2. 条件事件处理
        eventManager.addConditionalListener("order.created",
            event -> event.getData().contains("#"),  // 条件
            event -> System.out.println("处理有效订单: " + event.getData())  // 处理器
        );

        eventManager.fireEvent(new Event("order.created", "订单 #67890"));
        eventManager.fireEvent(new Event("order.created", "无效订单"));

        System.out.println();
    }

    // 异步处理应用
    public static void demonstrateAsynchronousProcessing() {
        System.out.println("=== 异步处理应用演示 ===");

        System.out.println("异步处理应用:");

        ExecutorService executor = Executors.newFixedThreadPool(3);

        // 1. 异步任务执行
        List<CompletableFuture<String>> futures = Arrays.asList(
            CompletableFuture.supplyAsync(() -> processData("数据1"), executor),
            CompletableFuture.supplyAsync(() -> processData("数据2"), executor),
            CompletableFuture.supplyAsync(() -> processData("数据3"), executor)
        );

        // 2. 组合异步操作
        CompletableFuture<List<String>> allResults = CompletableFuture.allOf(
            futures.toArray(new CompletableFuture[0])
        ).thenApply(v ->
            futures.stream()
                   .map(CompletableFuture::join)
                   .collect(Collectors.toList())
        );

        // 3. 异步结果处理
        allResults.thenAccept(results -> {
            System.out.println("所有异步任务完成:");
            results.forEach(result -> System.out.println("  " + result));
        }).join();

        // 4. 异步链式处理
        CompletableFuture<String> chainedResult = CompletableFuture
            .supplyAsync(() -> "初始数据", executor)
            .thenApply(data -> data.toUpperCase())
            .thenApply(data -> "处理后: " + data)
            .thenCompose(data -> CompletableFuture.supplyAsync(() -> data + " [完成]", executor));

        System.out.println("链式处理结果: " + chainedResult.join());

        executor.shutdown();
        System.out.println();
    }

    // 函数式编程应用
    public static void demonstrateFunctionalProgramming() {
        System.out.println("=== 函数式编程应用演示 ===");

        System.out.println("函数式编程应用:");

        // 1. 高阶函数
        Function<Integer, Function<Integer, Integer>> adder = x -> y -> x + y;
        Function<Integer, Integer> add5 = adder.apply(5);
        System.out.println("高阶函数结果: " + add5.apply(10));

        // 2. 函数组合
        Function<String, String> addPrefix = s -> "前缀-" + s;
        Function<String, String> addSuffix = s -> s + "-后缀";
        Function<String, String> toUpper = String::toUpperCase;

        Function<String, String> composed = addPrefix
            .andThen(addSuffix)
            .andThen(toUpper);

        System.out.println("函数组合结果: " + composed.apply("测试"));

        // 3. 柯里化
        TriFunction<Integer, Integer, Integer, Integer> add3 = (a, b, c) -> a + b + c;
        Function<Integer, Function<Integer, Function<Integer, Integer>>> curriedAdd =
            a -> b -> c -> add3.apply(a, b, c);

        int curriedResult = curriedAdd.apply(1).apply(2).apply(3);
        System.out.println("柯里化结果: " + curriedResult);

        // 4. 函数式数据处理管道
        List<String> words = Arrays.asList("apple", "banana", "cherry", "date");

        String result = words.stream()
            .filter(word -> word.length() > 4)
            .map(String::toUpperCase)
            .map(word -> "[" + word + "]")
            .collect(Collectors.joining(" | "));

        System.out.println("数据处理管道: " + result);

        // 5. 惰性求值
        Supplier<String> lazyComputation = () -> {
            System.out.println("执行昂贵的计算...");
            return "计算结果";
        };

        System.out.println("惰性求值创建完成");
        System.out.println("获取结果: " + lazyComputation.get());

        System.out.println();
    }

    private static String processData(String data) {
        try {
            Thread.sleep(1000); // 模拟耗时操作
        } catch (InterruptedException e) {
            Thread.currentThread().interrupt();
        }
        return "处理完成: " + data;
    }

class Event {
    private String type;
    private String data;

    public Event(String type, String data) {
        this.type = type;
        this.data = data;
    }

    public String getType() { return type; }
    public String getData() { return data; }
}

class EventManager {
    private Map<String, List<Consumer<Event>>> listeners = new HashMap<>();

    public void addEventListener(String eventType, Consumer<Event> listener) {
        listeners.computeIfAbsent(eventType, k -> new ArrayList<>()).add(listener);
    }

    public void addConditionalListener(String eventType, Predicate<Event> condition, Consumer<Event> listener) {
        addEventListener(eventType, event -> {
            if (condition.test(event)) {
                listener.accept(event);
            }
        });
    }

    public void fireEvent(Event event) {
        List<Consumer<Event>> eventListeners = listeners.get(event.getType());
        if (eventListeners != null) {
            eventListeners.forEach(listener -> listener.accept(event));
        }
    }
}

💡 最佳实践总结 ​

代码质量 ​

  1. 保持简洁：Lambda 表达式应该简短且易于理解
  2. 使用方法引用：当 Lambda 只是调用现有方法时，优先使用方法引用
  3. 避免副作用：Lambda 表达式应该是纯函数，避免修改外部状态
  4. 合理命名：为复杂的 Lambda 表达式使用有意义的变量名

性能考虑 ​

  1. 避免装箱拆箱：使用原始类型特化的函数式接口
  2. 注意捕获开销：减少不必要的变量捕获
  3. 合理使用并行流：在适当的场景下使用并行处理
  4. 缓存 Lambda 实例：对于重复使用的 Lambda，考虑缓存

可读性提升 ​

  1. 适当分解：将复杂的 Lambda 表达式分解为多个简单的步骤
  2. 使用有意义的参数名：即使编译器可以推断类型，也要使用清晰的参数名
  3. 添加注释：为复杂的业务逻辑添加必要的注释
  4. 保持一致性：在项目中保持 Lambda 使用风格的一致性

常见陷阱 ​

  1. 变量捕获陷阱：注意 effectively final 的限制
  2. 类型推断失败：在复杂场景下可能需要显式类型声明
  3. 异常处理：Lambda 中的异常处理需要特别注意
  4. 调试困难：复杂的 Lambda 链可能难以调试

💡 重点总结：

  1. Lambda 表达式是 Java 8 引入的重要特性，支持函数式编程范式
  2. 基本语法为 (parameters) -> expression 或 (parameters) -> { statements; }
  3. 方法引用是 Lambda 表达式的简化形式，提高代码可读性
  4. 变量捕获遵循 final 或 effectively final 规则，字段访问无此限制
  5. Lambda 表达式只能用于函数式接口，编译器进行类型推断
  6. 在集合处理、事件处理、异步编程等场景中有广泛应用
  7. 正确使用 Lambda 表达式可以让代码更简洁、更易读、更易维护
  8. 需要注意性能影响、调试难度等潜在问题，遵循最佳实践
