React 学习之旅 ​

欢迎来到我的 React 学习笔记！这里记录了我系统学习 React 技术栈的全过程，包括基础概念、高级特性、相关生态以及源码分析。

学习路线 ​

  1. React 基础 - JSX、组件、Props、State、生命周期等
  2. React Hooks - useState、useEffect、useContext、useReducer 等
  3. React Router - 路由配置、参数传递、编程式导航等
  4. Redux - 状态管理、Actions、Reducers、中间件等
  5. 开发工具 - Babel、SWC等编译工具、CSS方案（CSS Modules、CSS-in-JS、原子化CSS）
  6. Next.js - SSR、SSG、API Routes 等
  7. 源码解析 - Fiber 架构、调和过程、Hooks 实现等
