事件处理 ​

React 中的事件处理与 DOM 事件处理非常相似，但有一些语法上的差异。本章将介绍 React 中事件处理的基本概念和常见模式。

基本语法 ​

React 事件处理的基本语法与 HTML 有以下区别：

  1. React 事件使用小驼峰命名（camelCase），而不是全小写
  2. 使用 JSX 语法时，需要传入函数作为事件处理器，而不是字符串
  3. 在 React 中不能通过返回 false 阻止默认行为，必须显式调用 preventDefault

HTML 与 React 事件处理对比 ​

HTML:

html
<button onclick="handleClick()">点击</button>

React:

jsx
<button onClick={handleClick}>点击</button>

事件处理函数 ​

基本事件处理 ​

jsx
import React from 'react';

function Button() {
  const handleClick = () => {
    console.log('按钮被点击了');
  };

  return <button onClick={handleClick}>点击我</button>;
}

阻止默认行为 ​

jsx
import React from 'react';

function Form() {
  const handleSubmit = (e) => {
    e.preventDefault(); // 阻止表单默认提交行为
    console.log('表单已提交');
  };

  return (
    <form onSubmit={handleSubmit}>
      <button type="submit">提交</button>
    </form>
  );
}

传递参数给事件处理函数 ​

有两种常见的方式可以向事件处理函数传递参数：

  1. 使用箭头函数：
jsx
<button onClick={(e) => handleClick(id, e)}>删除</button>
  1. 使用 bind 方法：
jsx
<button onClick={handleClick.bind(this, id)}>删除</button>

完整示例：

jsx
import React from 'react';

function ItemList() {
  const items = [
    { id: 1, name: '项目 1' },
    { id: 2, name: '项目 2' },
    { id: 3, name: '项目 3' }
  ];

  const handleDelete = (id, e) => {
    e.stopPropagation(); // 阻止事件冒泡
    console.log(`删除项目 ${id}`);
  };

  const handleItemClick = (id) => {
    console.log(`点击项目 ${id}`);
  };

  return (
    <ul>
      {items.map(item => (
        <li key={item.id} onClick={() => handleItemClick(item.id)}>
          {item.name}
          <button onClick={(e) => handleDelete(item.id, e)}>删除</button>
        </li>
      ))}
    </ul>
  );
}

在类组件中处理事件 ​

在类组件中，事件处理函数通常是类的方法。需要注意的是，在 JavaScript 中，类方法默认不会绑定 this，因此需要手动绑定。

方法 1：在构造函数中绑定 ​

jsx
import React, { Component } from 'react';

class Button extends Component {
  constructor(props) {
    super(props);
    // 在构造函数中绑定 this
    this.handleClick = this.handleClick.bind(this);
  }

  handleClick() {
    console.log('按钮被点击了');
    console.log(this.props); // 可以访问 this
  }

  render() {
    return <button onClick={this.handleClick}>点击我</button>;
  }
}

方法 2：使用类属性语法和箭头函数 ​

jsx
import React, { Component } from 'react';

class Button extends Component {
  // 使用类属性语法和箭头函数，自动绑定 this
  handleClick = () => {
    console.log('按钮被点击了');
    console.log(this.props); // 可以访问 this
  };

  render() {
    return <button onClick={this.handleClick}>点击我</button>;
  }
}

方法 3：在回调中使用箭头函数 ​

jsx
import React, { Component } from 'react';

class Button extends Component {
  handleClick() {
    console.log('按钮被点击了');
    console.log(this.props); // 可以访问 this
  }

  render() {
    // 在回调中使用箭头函数
    return <button onClick={() => this.handleClick()}>点击我</button>;
  }
}

常见的 React 事件 ​

React 支持多种事件类型，以下是一些常见的事件：

鼠标事件 ​

  • onClick - 点击事件
  • onDoubleClick - 双击事件
  • onMouseEnter - 鼠标进入元素
  • onMouseLeave - 鼠标离开元素
  • onMouseMove - 鼠标在元素内移动
jsx
function MouseEvents() {
  return (
    <div 
      onMouseEnter={() => console.log('鼠标进入')}
      onMouseLeave={() => console.log('鼠标离开')}
    >
      <button onClick={() => console.log('点击')}>点击</button>
      <button onDoubleClick={() => console.log('双击')}>双击</button>
    </div>
  );
}

表单事件 ​

  • onChange - 输入框值变化
  • onSubmit - 表单提交
  • onFocus - 获得焦点
  • onBlur - 失去焦点
jsx
function FormEvents() {
  const [value, setValue] = React.useState('');

  const handleSubmit = (e) => {
    e.preventDefault();
    console.log('表单提交，值为:', value);
  };

  return (
    <form onSubmit={handleSubmit}>
      <input
        type="text"
        value={value}
        onChange={(e) => setValue(e.target.value)}
        onFocus={() => console.log('获得焦点')}
        onBlur={() => console.log('失去焦点')}
      />
      <button type="submit">提交</button>
    </form>
  );
}

键盘事件 ​

  • onKeyDown - 键盘按下
  • onKeyUp - 键盘释放
  • onKeyPress - 键盘按下并释放
jsx
function KeyboardEvents() {
  const handleKeyDown = (e) => {
    if (e.key === 'Enter') {
      console.log('按下了回车键');
    }
  };

  return (
    <input
      type="text"
      onKeyDown={handleKeyDown}
      onKeyUp={() => console.log('键盘释放')}
      placeholder="按下回车键"
    />
  );
}

合成事件 (SyntheticEvent) ​

React 使用合成事件系统来处理事件，它是原生 DOM 事件的跨浏览器包装器。合成事件具有与原生事件相同的接口，包括 stopPropagation() 和 preventDefault()，但它们在所有浏览器中的行为都相同。

jsx
function SyntheticEventDemo() {
  const handleClick = (e) => {
    console.log('事件类型:', e.type); // 'click'
    console.log('目标:', e.target); // 被点击的 DOM 元素
    console.log('当前目标:', e.currentTarget); // 处理事件的 DOM 元素
    
    // 阻止默认行为
    e.preventDefault();
    
    // 阻止事件冒泡
    e.stopPropagation();
  };

  return (
    <a href="https://example.com" onClick={handleClick}>
      点击我
    </a>
  );
}

事件池 ​

在 React 17 之前，合成事件对象会被池化（pooled）。这意味着事件对象会在事件回调执行后被回收并重用。因此，如果你需要在异步操作中访问事件属性，需要调用 e.persist()。

jsx
// React 16 及更早版本
function AsyncEventAccess() {
  const handleClick = (e) => {
    e.persist(); // 保留事件对象以便在异步操作中使用
    
    setTimeout(() => {
      console.log(e.type); // 'click'
    }, 1000);
  };

  return <button onClick={handleClick}>点击</button>;
}

在 React 17 及以后版本中，React 不再使用事件池，所以不再需要调用 e.persist()。

事件委托 (Event Delegation) ​

React 使用事件委托来提高性能。它不会将事件处理器直接附加到每个元素上，而是在 DOM 树的根节点上设置一个事件监听器，然后根据事件的目标元素来决定如何处理。

这种方法有几个优点：

  1. 减少内存使用，因为只需要少量的事件监听器
  2. 不需要在组件挂载和卸载时添加或移除事件监听器
  3. 即使 DOM 元素动态添加或删除，事件处理也能正常工作

自定义事件处理 ​

有时你可能需要创建自定义组件，并为其添加自定义事件处理。在 React 中，这通常通过 props 来实现：

jsx
function CustomButton({ onClick, children }) {
  const handleClick = (e) => {
    console.log('按钮被点击');
    // 调用父组件传递的 onClick 处理函数
    if (onClick) {
      onClick(e);
    }
  };

  return (
    <button onClick={handleClick} className="custom-button">
      {children}
    </button>
  );
}

// 使用自定义按钮
function App() {
  return (
    <CustomButton onClick={() => console.log('自定义按钮被点击')}>
      点击我
    </CustomButton>
  );
}

实践示例：表单处理 ​

下面是一个结合多种事件处理的表单示例：

jsx
import React, { useState } from 'react';
import './Form.css';

function ContactForm() {
  const [formData, setFormData] = useState({
    name: '',
    email: '',
    message: ''
  });
  
  const [errors, setErrors] = useState({});
  const [submitted, setSubmitted] = useState(false);

  // 处理输入变化
  const handleChange = (e) => {
    const { name, value } = e.target;
    setFormData({
      ...formData,
      [name]: value
    });
    
    // 清除相应的错误
    if (errors[name]) {
      setErrors({
        ...errors,
        [name]: null
      });
    }
  };

  // 处理表单验证
  const validateForm = () => {
    const newErrors = {};
    
    if (!formData.name.trim()) {
      newErrors.name = '请输入姓名';
    }
    
    if (!formData.email.trim()) {
      newErrors.email = '请输入邮箱';
    } else if (!/\S+@\S+\.\S+/.test(formData.email)) {
      newErrors.email = '请输入有效的邮箱地址';
    }
    
    if (!formData.message.trim()) {
      newErrors.message = '请输入留言内容';
    }
    
    setErrors(newErrors);
    return Object.keys(newErrors).length === 0;
  };

  // 处理表单提交
  const handleSubmit = (e) => {
    e.preventDefault();
    
    if (validateForm()) {
      // 模拟表单提交
      console.log('表单数据:', formData);
      
      // 显示成功消息
      setSubmitted(true);
      
      // 重置表单
      setFormData({
        name: '',
        email: '',
        message: ''
      });
      
      // 3秒后隐藏成功消息
      setTimeout(() => {
        setSubmitted(false);
      }, 3000);
    }
  };

  // 处理按键事件
  const handleKeyDown = (e) => {
    if (e.key === 'Enter' && e.ctrlKey) {
      handleSubmit(e);
    }
  };

  return (
    <div className="contact-form-container">
      <h2>联系我们</h2>
      
      {submitted && (
        <div className="success-message">
          表单提交成功！我们会尽快回复您。
        </div>
      )}
      
      <form onSubmit={handleSubmit} className="contact-form">
        <div className="form-group">
          <label htmlFor="name">姓名</label>
          <input
            type="text"
            id="name"
            name="name"
            value={formData.name}
            onChange={handleChange}
            onBlur={() => validateForm()}
            className={errors.name ? 'error' : ''}
          />
          {errors.name && <span className="error-message">{errors.name}</span>}
        </div>
        
        <div className="form-group">
          <label htmlFor="email">邮箱</label>
          <input
            type="email"
            id="email"
            name="email"
            value={formData.email}
            onChange={handleChange}
            onBlur={() => validateForm()}
            className={errors.email ? 'error' : ''}
          />
          {errors.email && <span className="error-message">{errors.email}</span>}
        </div>
        
        <div className="form-group">
          <label htmlFor="message">留言</label>
          <textarea
            id="message"
            name="message"
            value={formData.message}
            onChange={handleChange}
            onKeyDown={handleKeyDown}
            onBlur={() => validateForm()}
            className={errors.message ? 'error' : ''}
            rows={5}
            placeholder="按 Ctrl+Enter 快速提交"
          />
          {errors.message && <span className="error-message">{errors.message}</span>}
        </div>
        
        <button type="submit" className="submit-button">
          提交
        </button>
      </form>
      
      <p className="form-tip">
        提示：填写完表单后，可以按 Ctrl+Enter 快速提交
      </p>
    </div>
  );
}

export default ContactForm;
