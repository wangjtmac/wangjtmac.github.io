React Router 懒加载 ​

懒加载是一种优化技术，允许应用在需要时才加载特定的路由组件，而不是在初始加载时一次性加载所有组件。在 React Router 中实现懒加载可以显著减小初始包的大小，提高应用的首次加载性能。

基本概念 ​

懒加载基于以下两个 React API：

  1. React.lazy：允许动态导入组件，返回一个 Promise
  2. React.Suspense：在组件加载时显示回退内容

基本用法 ​

jsx
import { lazy, Suspense } from 'react';
import { Routes, Route } from 'react-router-dom';

// 懒加载组件
const Home = lazy(() => import('./pages/Home'));
const About = lazy(() => import('./pages/About'));
const Products = lazy(() => import('./pages/Products'));

function App() {
  return (
    <Suspense fallback={<div>加载中...</div>}>
      <Routes>
        <Route path="/" element={<Home />} />
        <Route path="/about" element={<About />} />
        <Route path="/products" element={<Products />} />
      </Routes>
    </Suspense>
  );
}

详细说明 ​

React.lazy ​

React.lazy 接受一个函数，该函数必须调用 import() 动态导入组件。它返回一个 Promise，解析为包含组件的模块。

jsx
const Home = lazy(() => import('./pages/Home'));

React.Suspense ​

React.Suspense 组件用于包装懒加载的组件，并在组件加载时显示回退内容。

jsx
<Suspense fallback={<LoadingSpinner />}>
  {/* 懒加载的组件 */}
</Suspense>

实现方式 ​

基础懒加载 ​

最简单的懒加载实现：

jsx
import { lazy, Suspense } from 'react';
import { Routes, Route } from 'react-router-dom';

const Home = lazy(() => import('./pages/Home'));
const About = lazy(() => import('./pages/About'));
const Contact = lazy(() => import('./pages/Contact'));

function App() {
  return (
    <div>
      <header>{/* 导航栏 */}</header>
      <main>
        <Suspense fallback={<div className="loading">页面加载中...</div>}>
          <Routes>
            <Route path="/" element={<Home />} />
            <Route path="/about" element={<About />} />
            <Route path="/contact" element={<Contact />} />
          </Routes>
        </Suspense>
      </main>
      <footer>{/* 页脚 */}</footer>
    </div>
  );
}

嵌套路由懒加载 ​

对于嵌套路由，可以在不同级别应用懒加载：

jsx
import { lazy, Suspense } from 'react';
import { Routes, Route } from 'react-router-dom';
import Layout from './components/Layout';

// 顶级路由
const Home = lazy(() => import('./pages/Home'));
const Dashboard = lazy(() => import('./pages/Dashboard'));

// 嵌套路由
const Products = lazy(() => import('./pages/Products'));
const ProductDetail = lazy(() => import('./pages/ProductDetail'));
const ProductReviews = lazy(() => import('./pages/ProductReviews'));

function App() {
  return (
    <Suspense fallback={<div>加载中...</div>}>
      <Routes>
        <Route path="/" element={<Layout />}>
          <Route index element={<Home />} />
          <Route path="dashboard" element={<Dashboard />} />
          <Route path="products" element={
            <Suspense fallback={<div>加载产品页面...</div>}>
              <Products />
            </Suspense>
          }>
            <Route path=":id" element={<ProductDetail />} />
            <Route path=":id/reviews" element={<ProductReviews />} />
          </Route>
        </Route>
      </Routes>
    </Suspense>
  );
}

按功能模块懒加载 ​

将相关的路由组件按功能模块分组懒加载：

jsx
import { lazy, Suspense } from 'react';
import { Routes, Route } from 'react-router-dom';

// 核心模块
const Home = lazy(() => import('./pages/Home'));
const About = lazy(() => import('./pages/About'));

// 用户模块
const UserRoutes = lazy(() => import('./routes/UserRoutes'));

// 产品模块
const ProductRoutes = lazy(() => import('./routes/ProductRoutes'));

// 管理模块
const AdminRoutes = lazy(() => import('./routes/AdminRoutes'));

function App() {
  return (
    <Suspense fallback={<div>加载中...</div>}>
      <Routes>
        <Route path="/" element={<Home />} />
        <Route path="/about" element={<About />} />
        
        {/* 用户相关路由 */}
        <Route path="/user/*" element={<UserRoutes />} />
        
        {/* 产品相关路由 */}
        <Route path="/products/*" element={<ProductRoutes />} />
        
        {/* 管理相关路由 */}
        <Route path="/admin/*" element={<AdminRoutes />} />
      </Routes>
    </Suspense>
  );
}

// UserRoutes.jsx
import { Routes, Route } from 'react-router-dom';

const Profile = lazy(() => import('./pages/Profile'));
const Settings = lazy(() => import('./pages/Settings'));
const Orders = lazy(() => import('./pages/Orders'));

function UserRoutes() {
  return (
    <Routes>
      <Route path="profile" element={<Profile />} />
      <Route path="settings" element={<Settings />} />
      <Route path="orders" element={<Orders />} />
    </Routes>
  );
}

预加载路由 ​

在用户可能导航到某个路由之前预加载该路由：

jsx
import { lazy, Suspense, useState, useEffect } from 'react';
import { Routes, Route, Link } from 'react-router-dom';

// 懒加载组件
const Home = lazy(() => import('./pages/Home'));
const About = lazy(() => import('./pages/About'));
const Contact = lazy(() => import('./pages/Contact'));

// 预加载函数
const preloadAbout = () => import('./pages/About');
const preloadContact = () => import('./pages/Contact');

function App() {
  return (
    <div>
      <nav>
        <Link to="/">首页</Link>
        <Link 
          to="/about" 
          onMouseEnter={preloadAbout} // 鼠标悬停时预加载
        >
          关于
        </Link>
        <Link 
          to="/contact" 
          onMouseEnter={preloadContact} // 鼠标悬停时预加载
        >
          联系我们
        </Link>
      </nav>
      
      <Suspense fallback={<div>加载中...</div>}>
        <Routes>
          <Route path="/" element={<Home />} />
          <Route path="/about" element={<About />} />
          <Route path="/contact" element={<Contact />} />
        </Routes>
      </Suspense>
    </div>
  );
}

自定义加载组件 ​

创建更丰富的加载体验：

jsx
import { lazy, Suspense } from 'react';
import { Routes, Route } from 'react-router-dom';
import LoadingSpinner from './components/LoadingSpinner';
import ErrorBoundary from './components/ErrorBoundary';

const Home = lazy(() => import('./pages/Home'));
const About = lazy(() => import('./pages/About'));
const Products = lazy(() => import('./pages/Products'));

function App() {
  return (
    <ErrorBoundary>
      <Suspense fallback={<LoadingSpinner />}>
        <Routes>
          <Route path="/" element={<Home />} />
          <Route path="/about" element={<About />} />
          <Route path="/products" element={<Products />} />
        </Routes>
      </Suspense>
    </ErrorBoundary>
  );
}

// LoadingSpinner.jsx
function LoadingSpinner() {
  return (
    <div className="loading-container">
      <div className="spinner"></div>
      <p>页面加载中，请稍候...</p>
    </div>
  );
}

最佳实践 ​

1. 合理分割代码 ​

  • 按路由分割：每个主要路由对应一个代码块
  • 按功能分割：相关功能组合成一个代码块
  • 避免过度分割：过多的小文件会增加 HTTP 请求数

2. 使用加载指示器 ​

为了提供良好的用户体验，始终提供加载指示器：

jsx
<Suspense fallback={
  <div className="loading">
    <div className="spinner"></div>
    <p>加载中...</p>
  </div>
}>
  {/* 路由内容 */}
</Suspense>

3. 处理加载错误 ​

使用错误边界捕获加载失败的情况：

jsx
import { Component } from 'react';

class ErrorBoundary extends Component {
  constructor(props) {
    super(props);
    this.state = { hasError: false };
  }

  static getDerivedStateFromError(error) {
    return { hasError: true };
  }

  componentDidCatch(error, errorInfo) {
    console.error("组件加载失败:", error, errorInfo);
  }

  render() {
    if (this.state.hasError) {
      return (
        <div className="error-container">
          <h2>加载失败</h2>
          <p>抱歉，页面加载出错。</p>
          <button onClick={() => window.location.reload()}>
            重试
          </button>
        </div>
      );
    }

    return this.props.children;
  }
}

4. 预加载关键路由 ​

预加载用户可能会访问的路由，减少等待时间：

jsx
// 预加载函数
const preloadRoutes = () => {
  // 预加载关键路由
  import('./pages/About');
  import('./pages/Products');
};

// 在应用初始化后预加载
useEffect(() => {
  // 等待主页面渲染完成
  const timer = setTimeout(preloadRoutes, 2000);
  return () => clearTimeout(timer);
}, []);

5. 结合 React Router Data API ​

在 React Router v6.4+ 中，结合 loader 和懒加载：

jsx
import { lazy } from 'react';
import { createBrowserRouter } from 'react-router-dom';

const Products = lazy(() => import('./pages/Products'));

// 产品数据加载函数
async function productsLoader() {
  const response = await fetch('/api/products');
  return response.json();
}

const router = createBrowserRouter([
  {
    path: '/',
    element: <Layout />,
    children: [
      { index: true, element: <Home /> },
      { 
        path: 'products',
        element: <Products />,
        loader: productsLoader
      }
    ]
  }
]);

注意事项 ​

  1. 初始加载体验：懒加载可能导致用户在导航到新路由时看到加载指示器，确保提供良好的加载体验。

  2. 网络条件：在网络条件不佳的情况下，懒加载可能导致更长的等待时间。考虑预加载关键路由。

  3. 服务器端渲染：在服务器端渲染环境中，需要特殊处理懒加载组件，以避免客户端和服务器端不匹配的问题。

  4. 包大小监控：使用工具如 Webpack Bundle Analyzer 监控分割后的包大小，确保合理分割。

示例：完整的懒加载实现 ​

jsx
import { lazy, Suspense, useEffect } from 'react';
import { Routes, Route, useLocation } from 'react-router-dom';
import Layout from './components/Layout';
import LoadingFallback from './components/LoadingFallback';
import ErrorBoundary from './components/ErrorBoundary';

// 懒加载页面组件
const Home = lazy(() => import('./pages/Home'));
const About = lazy(() => import('./pages/About'));
const Products = lazy(() => import('./pages/Products'));
const ProductDetail = lazy(() => import('./pages/ProductDetail'));
const Contact = lazy(() => import('./pages/Contact'));
const NotFound = lazy(() => import('./pages/NotFound'));

// 预加载映射
const preloadMap = {
  '/': () => import('./pages/Home'),
  '/about': () => import('./pages/About'),
  '/products': () => import('./pages/Products'),
  '/contact': () => import('./pages/Contact')
};

function App() {
  const location = useLocation();
  
  // 预加载当前路由可能相关的路由
  useEffect(() => {
    const currentPath = location.pathname;
    
    // 预加载相关路由
    if (currentPath === '/') {
      // 从首页可能导航到 About 和 Products
      preloadMap['/about']();
      preloadMap['/products']();
    } else if (currentPath.startsWith('/products')) {
      // 在产品页面可能导航到联系页
      preloadMap['/contact']();
    }
  }, [location]);
  
  return (
    <ErrorBoundary>
      <Layout>
        <Suspense fallback={<LoadingFallback />}>
          <Routes>
            <Route path="/" element={<Home />} />
            <Route path="/about" element={<About />} />
            <Route path="/products">
              <Route index element={<Products />} />
              <Route path=":id" element={<ProductDetail />} />
            </Route>
            <Route path="/contact" element={<Contact />} />
            <Route path="*" element={<NotFound />} />
          </Routes>
        </Suspense>
      </Layout>
    </ErrorBoundary>
  );
}

export default App;
