redirect API ​

redirect 是 React Router v6.4+ 提供的一个工具函数，用于在 loader 或 action 函数中执行重定向操作。它通过抛出一个特殊的响应对象来实现重定向，这个响应会被 React Router 捕获并处理。

基本用法 ​

jsx
import { redirect } from 'react-router-dom';

// 在 loader 函数中使用
export async function loader() {
  const user = await getUser();
  
  if (!user) {
    // 重定向到登录页面
    return redirect('/login');
  }
  
  return { user };
}

// 在 action 函数中使用
export async function action({ request }) {
  const formData = await request.formData();
  const result = await saveData(formData);
  
  if (result.success) {
    // 重定向到成功页面
    return redirect('/success');
  }
  
  return { error: result.error };
}

语法 ​

jsx
redirect(to, init);

参数 ​

  • to: 重定向的目标 URL 或路径
  • init (可选): 一个包含重定向选项的对象

返回值 ​

redirect 函数不直接返回值，而是抛出一个特殊的响应对象，这个对象会被 React Router 捕获并处理。

详细说明 ​

工作原理 ​

redirect 函数通过抛出一个带有特殊状态码（通常是 302）的 Response 对象来实现重定向。这个响应会被 React Router 捕获，然后执行导航操作。

重定向选项 ​

redirect 函数的第二个参数 init 可以包含以下选项：

  • status: 重定向的状态码，默认为 302（临时重定向）
  • headers: 要包含在响应中的额外头部信息
jsx
redirect('/login', { status: 301 }); // 永久重定向

示例 ​

基本重定向 ​

jsx
import { redirect } from 'react-router-dom';

export async function loader() {
  const isAuthenticated = await checkAuth();
  
  if (!isAuthenticated) {
    return redirect('/login');
  }
  
  return { isAuthenticated };
}

带状态码的重定向 ​

jsx
import { redirect } from 'react-router-dom';

export async function loader() {
  const isFeatureEnabled = await checkFeatureFlag('new-dashboard');
  
  if (isFeatureEnabled) {
    // 永久重定向到新仪表板
    return redirect('/new-dashboard', { status: 301 });
  }
  
  return { isFeatureEnabled };
}

带查询参数的重定向 ​

jsx
import { redirect } from 'react-router-dom';

export async function loader({ request }) {
  const url = new URL(request.url);
  const searchParams = url.searchParams;
  const category = searchParams.get('category');
  
  if (category === 'premium' && !(await isPremiumUser())) {
    // 重定向到升级页面，并传递返回 URL
    const returnTo = encodeURIComponent(url.pathname + url.search);
    return redirect(`/upgrade?returnTo=${returnTo}`);
  }
  
  return { category };
}

表单提交后重定向 ​

jsx
import { redirect } from 'react-router-dom';

export async function action({ request }) {
  const formData = await request.formData();
  const product = Object.fromEntries(formData);
  
  try {
    const result = await createProduct(product);
    // 重定向到新创建的产品详情页
    return redirect(`/products/${result.id}`);
  } catch (error) {
    // 不重定向，而是返回错误信息
    return { error: error.message };
  }
}

function NewProduct() {
  const actionData = useActionData();
  
  return (
    <Form method="post">
      <h1>创建新产品</h1>
      {actionData?.error && <p className="error">{actionData.error}</p>}
      <input name="name" placeholder="产品名称" required />
      <input name="price" type="number" placeholder="价格" required />
      <button type="submit">创建</button>
    </Form>
  );
}

条件重定向 ​

jsx
import { redirect } from 'react-router-dom';

export async function loader({ params }) {
  const { productId } = params;
  
  try {
    const product = await fetchProduct(productId);
    
    if (product.status === 'archived') {
      // 产品已归档，重定向到归档页面
      return redirect(`/archived-products/${productId}`);
    }
    
    if (product.status === 'discontinued') {
      // 产品已停产，重定向到替代产品
      return redirect(`/products/${product.replacementProductId}`);
    }
    
    return { product };
  } catch (error) {
    if (error.status === 404) {
      // 产品不存在，重定向到产品列表
      return redirect('/products');
    }
    throw error;
  }
}

带状态的重定向 ​

如果需要在重定向时传递状态，可以结合 redirect 和自定义头部：

jsx
import { redirect } from 'react-router-dom';

export async function action({ request }) {
  const formData = await request.formData();
  const result = await processPayment(formData);
  
  if (result.success) {
    // 创建包含状态的 URL
    const url = new URL('/payment-success', window.location.origin);
    
    // 使用自定义头部传递状态
    return redirect(url.toString(), {
      headers: {
        'X-Redirect-State': JSON.stringify({
          orderId: result.orderId,
          amount: result.amount
        })
      }
    });
  }
  
  return { error: result.error };
}

// 在目标页面中，可以使用 useLocation 获取状态
function PaymentSuccess() {
  const location = useLocation();
  const state = location.state;
  
  return (
    <div>
      <h1>支付成功！</h1>
      {state && (
        <div>
          <p>订单号: {state.orderId}</p>
          <p>金额: ¥{state.amount}</p>
        </div>
      )}
    </div>
  );
}

注意事项 ​

  1. 只能在 loader 或 action 中使用：redirect 函数只能在 loader 或 action 函数中使用，不能在组件渲染过程中使用。

  2. 抛出而非返回：redirect 函数通过抛出响应对象工作，而不是通过返回值。但在 React Router 中，你可以像返回值一样使用它。

  3. 与 useNavigate 的区别：redirect 用于服务器端或数据加载/提交过程中的重定向，而 useNavigate 用于组件内的客户端重定向。

  4. 状态传递限制：redirect 不直接支持像 useNavigate 那样传递状态，需要使用查询参数或自定义头部。

最佳实践 ​

  1. 使用相对路径：在大多数情况下，使用相对路径进行重定向，以便应用可以部署在不同的基本路径下。

  2. 处理认证重定向：在处理认证重定向时，保存原始 URL，以便用户登录后可以返回。

jsx
export async function loader({ request }) {
  const isAuthenticated = await checkAuth();
  
  if (!isAuthenticated) {
    const url = new URL(request.url);
    const returnTo = encodeURIComponent(url.pathname + url.search);
    return redirect(`/login?returnTo=${returnTo}`);
  }
  
  return { isAuthenticated };
}

  1. 区分临时和永久重定向：根据需要使用不同的状态码，如临时重定向（302）或永久重定向（301）。

  2. 提供有意义的重定向：确保重定向目标与用户期望一致，避免重定向循环。

相关 API ​

  • Form - 用于处理表单提交
