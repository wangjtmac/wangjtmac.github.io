Link API ​

Link 是 React Router 提供的核心组件，用于在应用中创建声明式导航链接。它渲染为一个可访问的 <a> 标签，但会拦截点击事件，避免整页刷新，实现客户端导航。

基本用法 ​

jsx
import { Link } from 'react-router-dom';

function Navigation() {
  return (
    <nav>
      <Link to="/">首页</Link>
      <Link to="/about">关于</Link>
      <Link to="/contact">联系我们</Link>
    </nav>
  );
}

属性 ​

基本属性 ​

属性类型默认值描述
tostring | object必填导航目标的 URL 或位置对象
replacebooleanfalse如果为 true，导航将替换历史堆栈中的当前条目，而不是添加新条目
stateanyundefined要保存到历史状态的状态
reloadDocumentbooleanfalse如果为 true，点击链接将导致整页刷新，绕过路由
preventScrollResetbooleanfalse如果为 true，阻止滚动位置重置
relativestringundefined可以是 "route" 或 "path"，指定 to 是相对于当前路由还是当前路径

位置对象 ​

to 属性可以是字符串或位置对象。位置对象具有以下属性：

jsx
<Link
  to={{
    pathname: "/courses",
    search: "?sort=name",
    hash: "#top",
  }}
>
  课程
</Link>

示例 ​

基本链接 ​

jsx
import { Link } from 'react-router-dom';

function Navigation() {
  return (
    <nav>
      <Link to="/">首页</Link>
      <Link to="/about">关于</Link>
      <Link to="/contact">联系我们</Link>
    </nav>
  );
}

带查询参数的链接 ​

jsx
<Link to="/products?category=electronics&sort=price">电子产品</Link>

带路径参数的链接 ​

jsx
<Link to={`/users/${userId}`}>查看用户资料</Link>

使用 replace 属性 ​

使用 replace 属性可以替换当前的历史记录，而不是添加新的记录：

jsx
<Link to="/login" replace>登录</Link>

传递状态 ​

使用 state 属性可以在导航时传递数据：

jsx
<Link 
  to="/products/123" 
  state={{ productName: "手机", price: 1999 }}
>
  查看产品详情
</Link>

在目标组件中，可以使用 useLocation 钩子访问这些状态：

jsx
import { useLocation } from 'react-router-dom';

function ProductDetail() {
  const location = useLocation();
  const { productName, price } = location.state || {};
  
  return (
    <div>
      <h1>产品详情</h1>
      {productName && <p>产品名称: {productName}</p>}
      {price && <p>价格: ¥{price}</p>}
    </div>
  );
}

相对路径导航 ​

使用 relative 属性可以指定导航是相对于当前路由还是当前路径：

jsx
<Link to="../sibling" relative="route">
  导航到兄弟路由
</Link>

自定义样式 ​

你可以为 Link 组件添加自定义样式：

jsx
<Link 
  to="/about" 
  className="nav-link" 
  style={{ color: 'blue', textDecoration: 'none' }}
>
  关于
</Link>

阻止滚动重置 ​

默认情况下，导航到新页面时会重置滚动位置。使用 preventScrollReset 属性可以阻止这一行为：

jsx
<Link to="/long-page" preventScrollReset={true}>
  保持滚动位置
</Link>

绕过路由器 ​

在某些情况下，你可能希望链接导致整页刷新。使用 reloadDocument 属性可以实现这一点：

jsx
<Link to="/external-resource" reloadDocument>
  外部资源
</Link>

与 HTML <a> 标签的区别 ​

Link 组件与 HTML <a> 标签的主要区别在于：

  1. Link 使用 to 属性而不是 href 属性
  2. Link 会拦截点击事件，使用客户端路由导航，而不是发送新的 HTTP 请求
  3. Link 提供了额外的属性，如 replace、state 和 relative

如果你需要链接到外部网站，应该使用普通的 <a> 标签：

jsx
<a href="https://reactjs.org" target="_blank" rel="noopener noreferrer">
  React 官网
</a>

最佳实践 ​

  1. 使用相对路径：在嵌套路由中，使用相对路径可以使代码更加灵活和可维护。

  2. 添加语义化标签：为了提高可访问性，考虑添加适当的 ARIA 属性：

jsx
<Link to="/about" aria-label="关于我们页面">
  关于
</Link>

  1. 避免空链接：始终为 Link 组件提供有效的 to 属性。

  2. 使用 NavLink 代替 Link：如果需要为活动链接添加样式，考虑使用 NavLink 组件。
