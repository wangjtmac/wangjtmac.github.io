React Router 操作 ​

React Router v6.4+ 引入了一套强大的数据操作 API，允许路由直接处理数据加载、表单提交和其他操作。这些操作使得路由不仅可以渲染 UI，还可以负责与之相关的数据交互，从而简化应用架构。

基本概念 ​

React Router 的操作主要包括以下几种：

  1. loader：用于在路由渲染前加载数据
  2. action：用于处理表单提交和其他数据修改操作
  3. fetcher：用于在不导航的情况下执行数据操作

loader 操作 ​

loader 函数用于在组件渲染前加载数据，类似于组件的 useEffect 钩子，但在导航完成前执行。

基本用法 ​

jsx
import { createBrowserRouter, useLoaderData } from 'react-router-dom';

// 定义 loader 函数
async function productsLoader() {
  const response = await fetch('/api/products');
  
  if (!response.ok) {
    throw new Response('加载产品失败', { status: response.status });
  }
  
  return response.json();
}

// 在组件中使用加载的数据
function Products() {
  const products = useLoaderData();
  
  return (
    <div>
      <h1>产品列表</h1>
      <ul>
        {products.map(product => (
          <li key={product.id}>{product.name}</li>
        ))}
      </ul>
    </div>
  );
}

// 在路由配置中使用 loader
const router = createBrowserRouter([
  {
    path: '/products',
    element: <Products />,
    loader: productsLoader
  }
]);

使用路由参数 ​

loader 函数可以接收包含路由参数的对象：

jsx
async function productLoader({ params }) {
  const { productId } = params;
  const response = await fetch(`/api/products/${productId}`);
  
  if (response.status === 404) {
    throw new Response('产品未找到', { status: 404 });
  }
  
  return response.json();
}

// 在路由配置中使用
{
  path: '/products/:productId',
  element: <ProductDetail />,
  loader: productLoader
}

处理 URL 查询参数 ​

loader 函数可以访问请求对象，从中获取查询参数：

jsx
async function searchLoader({ request }) {
  const url = new URL(request.url);
  const query = url.searchParams.get('q') || '';
  
  const response = await fetch(`/api/search?q=${query}`);
  return response.json();
}

action 操作 ​

action 函数用于处理表单提交和其他数据修改操作，通常与 <Form> 组件一起使用。

基本用法 ​

jsx
import { Form, redirect, useActionData } from 'react-router-dom';

// 定义 action 函数
async function createProductAction({ request }) {
  const formData = await request.formData();
  const product = Object.fromEntries(formData);
  
  try {
    const response = await fetch('/api/products', {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json',
      },
      body: JSON.stringify(product),
    });
    
    if (!response.ok) {
      return { error: '创建产品失败' };
    }
    
    const newProduct = await response.json();
    return redirect(`/products/${newProduct.id}`);
  } catch (error) {
    return { error: error.message };
  }
}

// 在组件中使用表单和 action
function NewProduct() {
  const actionData = useActionData();
  
  return (
    <div>
      <h1>创建新产品</h1>
      {actionData?.error && <p className="error">{actionData.error}</p>}
      
      <Form method="post">
        <div>
          <label>
            名称：
            <input name="name" required />
          </label>
        </div>
        <div>
          <label>
            价格：
            <input name="price" type="number" step="0.01" required />
          </label>
        </div>
        <div>
          <label>
            描述：
            <textarea name="description" />
          </label>
        </div>
        <button type="submit">创建产品</button>
      </Form>
    </div>
  );
}

// 在路由配置中使用 action
{
  path: '/products/new',
  element: <NewProduct />,
  action: createProductAction
}

处理不同的表单操作 ​

可以在一个 action 中处理多个表单操作：

jsx
async function productAction({ request, params }) {
  const formData = await request.formData();
  const intent = formData.get('intent');
  
  if (intent === 'delete') {
    await deleteProduct(params.productId);
    return redirect('/products');
  }
  
  if (intent === 'update') {
    const updates = Object.fromEntries(formData);
    await updateProduct(params.productId, updates);
    return { success: true };
  }
  
  throw new Response('不支持的操作', { status: 400 });
}

function ProductDetail() {
  const product = useLoaderData();
  const actionData = useActionData();
  
  return (
    <div>
      <h1>{product.name}</h1>
      
      {actionData?.success && <p>产品已更新！</p>}
      
      <Form method="post">
        <input name="name" defaultValue={product.name} />
        <input name="price" defaultValue={product.price} />
        <button name="intent" value="update">更新产品</button>
      </Form>
      
      <Form method="post">
        <button name="intent" value="delete">删除产品</button>
      </Form>
    </div>
  );
}

fetcher 操作 ​

fetcher 允许在不导航的情况下执行 loader 和 action 操作，适用于后台数据获取、部分表单提交等场景。

基本用法 ​

jsx
import { useFetcher } from 'react-router-dom';

function AddToWishlist({ productId }) {
  const fetcher = useFetcher();
  
  // 检查操作状态
  const isAdding = fetcher.state === 'submitting';
  
  return (
    <fetcher.Form method="post" action="/api/wishlist">
      <input type="hidden" name="productId" value={productId} />
      <button type="submit" disabled={isAdding}>
        {isAdding ? '添加中...' : '添加到心愿单'}
      </button>
    </fetcher.Form>
  );
}

加载数据而不导航 ​

jsx
import { useFetcher } from 'react-router-dom';
import { useEffect } from 'react';

function ProductSuggestions() {
  const fetcher = useFetcher();
  
  // 组件挂载时加载推荐产品
  useEffect(() => {
    if (fetcher.state === 'idle' && !fetcher.data) {
      fetcher.load('/api/suggestions');
    }
  }, [fetcher]);
  
  // 显示加载状态或数据
  if (fetcher.state === 'loading') {
    return <p>加载推荐中...</p>;
  }
  
  const suggestions = fetcher.data || [];
  
  return (
    <div>
      <h2>你可能还喜欢</h2>
      <ul>
        {suggestions.map(product => (
          <li key={product.id}>{product.name}</li>
        ))}
      </ul>
    </div>
  );
}

高级操作 ​

延迟数据加载 ​

使用 defer 函数可以实现数据的延迟加载：

jsx
import { defer, Await, useLoaderData } from 'react-router-dom';
import { Suspense } from 'react';

async function dashboardLoader() {
  // 立即加载关键数据
  const userPromise = fetchUser();
  
  // 延迟加载非关键数据
  const salesPromise = fetchSales();
  const reportsPromise = fetchReports();
  
  return defer({
    user: await userPromise, // 等待用户数据加载完成
    sales: salesPromise,     // 不等待销售数据
    reports: reportsPromise  // 不等待报告数据
  });
}

function Dashboard() {
  const { user, sales, reports } = useLoaderData();
  
  return (
    <div>
      <h1>欢迎，{user.name}</h1>
      
      <Suspense fallback={<p>加载销售数据...</p>}>
        <Await resolve={sales}>
          {(resolvedSales) => (
            <SalesChart data={resolvedSales} />
          )}
        </Await>
      </Suspense>
      
      <Suspense fallback={<p>加载报告数据...</p>}>
        <Await resolve={reports}>
          {(resolvedReports) => (
            <ReportsList reports={resolvedReports} />
          )}
        </Await>
      </Suspense>
    </div>
  );
}

乐观 UI 更新 ​

使用 fetcher.Form 和 optimisticData 实现乐观 UI 更新：

jsx
import { useFetcher } from 'react-router-dom';

function TodoItem({ todo }) {
  const fetcher = useFetcher();
  
  // 确定当前显示的完成状态
  const completed = 
    fetcher.formData?.get('completed') === 'true' ||
    todo.completed;
  
  return (
    <div className={completed ? 'completed' : ''}>
      <span>{todo.title}</span>
      
      <fetcher.Form method="post" action={`/todos/${todo.id}`}>
        <input 
          type="hidden" 
          name="completed" 
          value={!completed}
        />
        <button 
          type="submit"
          name="intent"
          value="toggle"
        >
          {completed ? '标记为未完成' : '标记为完成'}
        </button>
      </fetcher.Form>
    </div>
  );
}

并行数据加载 ​

使用 Promise.all 在 loader 中并行加载多个数据源：

jsx
async function productDetailLoader({ params }) {
  const { productId } = params;
  
  // 并行加载产品详情和评论
  const [product, reviews] = await Promise.all([
    fetch(`/api/products/${productId}`).then(r => r.json()),
    fetch(`/api/products/${productId}/reviews`).then(r => r.json())
  ]);
  
  return { product, reviews };
}

最佳实践 ​

1. 错误处理 ​

在 loader 和 action 中妥善处理错误：

jsx
async function productLoader({ params }) {
  try {
    const response = await fetch(`/api/products/${params.productId}`);
    
    if (response.status === 404) {
      throw new Response('产品未找到', { status: 404 });
    }
    
    if (!response.ok) {
      throw new Response('加载产品失败', { status: response.status });
    }
    
    return response.json();
  } catch (error) {
    if (error instanceof Response) throw error;
    throw new Response('加载产品时出错', { status: 500 });
  }
}

2. 表单验证 ​

在 action 中进行表单验证：

jsx
async function createUserAction({ request }) {
  const formData = await request.formData();
  const userData = Object.fromEntries(formData);
  
  // 验证表单数据
  const errors = {};
  
  if (!userData.username) {
    errors.username = '用户名是必填项';
  } else if (userData.username.length < 3) {
    errors.username = '用户名至少需要3个字符';
  }
  
  if (!userData.password) {
    errors.password = '密码是必填项';
  } else if (userData.password.length < 8) {
    errors.password = '密码至少需要8个字符';
  }
  
  if (Object.keys(errors).length > 0) {
    return { errors, userData: { ...userData, password: '' } };
  }
  
  // 继续处理有效表单
  try {
    const response = await createUser(userData);
    return redirect('/login');
  } catch (error) {
    return { errors: { form: error.message }, userData: { ...userData, password: '' } };
  }
}

使用 useRouteLoaderData 钩子在子路由中访问父路由的数据：

jsx
import { useRouteLoaderData } from 'react-router-dom';

function ProductReviews() {
  // 访问产品详情路由的数据
  const { product } = useRouteLoaderData('product-detail');
  
  return (
    <div>
      <h2>{product.name} 的评论</h2>
      {/* 评论列表 */}
    </div>
  );
}

// 路由配置
{
  path: '/products/:productId',
  id: 'product-detail', // 添加路由 ID

  element: <ProductDetail />,

    {
      path: 'reviews',

4. 请求中止 ​

使用 request.signal 处理请求中止：

jsx
async function searchLoader({ request }) {
  const url = new URL(request.url);
  const query = url.searchParams.get('q') || '';
  
  // 使用 request.signal 处理请求中止
  const response = await fetch(`/api/search?q=${query}`, {
    signal: request.signal
  });
  
  return response.json();
}

5. 状态管理 ​

使用 loader 和 action 替代部分全局状态管理：

jsx
// 用户认证状态管理
async function rootLoader() {
  const user = await getUserFromSession();
  return { user };
}

// 在任何组件中访问用户状态
function Header() {
  const { user } = useRouteLoaderData('root');
  
  return (
    <header>
      {user ? (
        <span>欢迎，{user.name}</span>
      ) : (
        <Link to="/login">登录</Link>
      )}
    </header>
  );
}

// 路由配置
{
  path: '/',
  id: 'root',
  loader: rootLoader,
  element: <Layout />
}

示例：完整的数据操作流程 ​

jsx
import {
  createBrowserRouter,
  RouterProvider,
  useLoaderData,
  useActionData,
  Form,
  redirect
} from 'react-router-dom';

// 产品列表加载器
async function productsLoader({ request }) {
  const url = new URL(request.url);
  const category = url.searchParams.get('category');
  
  const response = await fetch(
    category 
      ? `/api/products?category=${category}`
      : '/api/products'
  );
  
  if (!response.ok) {
    throw new Response('加载产品失败', { status: response.status });
  }
  
  return response.json();
}

// 产品详情加载器
async function productDetailLoader({ params }) {
  const { productId } = params;
  
  const [productResponse, reviewsResponse] = await Promise.all([
    fetch(`/api/products/${productId}`),
    fetch(`/api/products/${productId}/reviews`)
  ]);
  
  if (productResponse.status === 404) {
    throw new Response('产品未找到', { status: 404 });
  }
  
  const product = await productResponse.json();
  const reviews = await reviewsResponse.json();
  
  return { product, reviews };
}

// 产品操作处理
async function productAction({ request, params }) {
  const { productId } = params;
  const formData = await request.formData();
  const intent = formData.get('intent');
  
  if (intent === 'add-to-cart') {
    const quantity = Number(formData.get('quantity') || 1);
    
    try {
      await fetch('/api/cart', {
        method: 'POST',
        headers: {
          'Content-Type': 'application/json'
        },
        body: JSON.stringify({ productId, quantity })
      });
      
      return { success: true, message: '已添加到购物车' };
    } catch (error) {
      return { success: false, message: '添加到购物车失败' };
    }
  }
  
  if (intent === 'add-review') {
    const review = {
      rating: Number(formData.get('rating')),
      comment: formData.get('comment')
    };
    
    // 验证评论
    const errors = {};
    
    if (!review.rating) {
      errors.rating = '请选择评分';
    }
    
    if (!review.comment) {
      errors.comment = '请填写评论内容';
    }
    
    if (Object.keys(errors).length > 0) {
      return { success: false, errors };
    }
    
    // 提交评论
    try {
      await fetch(`/api/products/${productId}/reviews`, {
        method: 'POST',
        headers: {
          'Content-Type': 'application/json'
        },
        body: JSON.stringify(review)
      });
      
      return { success: true, message: '评论已提交' };
    } catch (error) {
      return { success: false, message: '提交评论失败' };
    }
  }
  
  throw new Response('不支持的操作', { status: 400 });
}

// 产品详情组件
function ProductDetail() {
  const { product, reviews } = useLoaderData();
  const actionData = useActionData();
  
  return (
    <div className="product-detail">
      <h1>{product.name}</h1>
      <p className="price">¥{product.price}</p>
      <p>{product.description}</p>
      
      {/* 添加到购物车表单 */}
      <Form method="post">
        <label>
          数量：
          <input 
            type="number" 
            name="quantity" 
            min="1" 
            defaultValue="1" 
          />
        </label>
        
        <button 
          type="submit" 
          name="intent" 
          value="add-to-cart"
        >
          添加到购物车
        </button>
      </Form>
      
      {actionData?.intent === 'add-to-cart' && (
        <p className={actionData.success ? 'success' : 'error'}>
          {actionData.message}
        </p>
      )}
      
      {/* 评论列表 */}
      <h2>用户评论</h2>
      {reviews.length > 0 ? (
        <ul className="reviews">
          {reviews.map(review => (
            <li key={review.id}>
              <div>评分: {review.rating}/5</div>
              <p>{review.comment}</p>
            </li>
          ))}
        </ul>
      ) : (
        <p>暂无评论</p>
      )}
      
      {/* 添加评论表单 */}
      <h3>添加评论</h3>
      <Form method="post">
        <div>
          <label>
            评分：
            <select name="rating">
              <option value="">选择评分</option>
              <option value="5">5 星</option>
              <option value="4">4 星</option>
              <option value="3">3 星</option>
              <option value="2">2 星</option>
              <option value="1">1 星</option>
            </select>
          </label>
          {actionData?.errors?.rating && (
            <p className="error">{actionData.errors.rating}</p>
          )}
        </div>
        
        <div>
          <label>
            评论：
            <textarea name="comment" rows="4" />
          </label>
          {actionData?.errors?.comment && (
            <p className="error">{actionData.errors.comment}</p>
          )}
        </div>
        
        <button 
          type="submit" 
          name="intent" 
          value="add-review"
        >
          提交评论
        </button>
      </Form>
      
      {actionData?.intent === 'add-review' && actionData.success && (
        <p className="success">{actionData.message}</p>
      )}
    </div>
  );
}

// 路由配置
const router = createBrowserRouter([
  {
    path: '/',
    element: <Layout />,
    children: [
      { index: true, element: <Home /> },
      { 
        path: 'products',
        element: <Products />,
        loader: productsLoader
      },
      {
        path: 'products/:productId',
        element: <ProductDetail />,
        loader: productDetailLoader,
        action: productAction
      }
    ]
  }
]);

// 应用入口
function App() {
  return <RouterProvider router={router} />;
}
