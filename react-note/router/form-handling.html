React Router 表单处理 ​

React Router v6.4+提供了强大的表单处理功能，通过Form组件和action函数，可以简化表单提交和数据处理流程。本文将详细介绍如何使用React Router处理表单。

表单处理概述 ​

传统的React表单处理通常需要：

  1. 创建表单状态
  2. 编写onChange处理函数
  3. 编写onSubmit处理函数
  4. 手动发送请求并处理响应
  5. 处理加载和错误状态

React Router的表单处理简化了这一过程：

  1. 使用<Form>组件替代HTML的<form>
  2. 定义action函数处理表单提交
  3. 自动处理加载状态和错误

Form 组件 ​

Form组件是HTML form元素的增强版，它会拦截表单提交，然后调用相应的action函数。

基本用法 ​

jsx
import { Form } from 'react-router-dom';

function ContactForm() {
  return (
    <Form method="post">
      <div>
        <label>
          姓名：
          <input type="text" name="name" />
        </label>
      </div>
      <div>
        <label>
          邮箱：
          <input type="email" name="email" />
        </label>
      </div>
      <div>
        <label>
          消息：
          <textarea name="message"></textarea>
        </label>
      </div>
      <button type="submit">提交</button>
    </Form>
  );
}

Form 组件属性 ​

Form组件接受以下属性：

  • method: 表单提交方法，可以是'get'或'post'（默认）、'put'、'patch'、'delete'
  • action: 表单提交的URL，默认为当前路由
  • encType: 编码类型，如'application/x-www-form-urlencoded'（默认）、'multipart/form-data'
  • replace: 布尔值，提交后是否替换历史记录
  • reloadDocument: 布尔值，是否执行传统的表单提交（刷新页面）
  • preventScrollReset: 布尔值，提交后是否阻止滚动重置
  • relative: 相对路径类型，'route'（默认）或'path'
jsx
<Form 
  method="post"
  action="/contacts/new"
  encType="multipart/form-data"
  replace
>
  {/* 表单字段 */}
</Form>

action 函数 ​

action函数在表单提交时执行，用于处理表单数据。

定义 action 函数 ​

jsx
// 在路由配置中定义action函数
async function contactAction({ request }) {
  const formData = await request.formData();
  const contactData = Object.fromEntries(formData);
  
  // 验证数据
  const errors = validateContact(contactData);
  if (Object.keys(errors).length > 0) {
    return { errors };
  }
  
  // 提交数据到API
  try {
    await saveContact(contactData);
    return redirect('/contacts');
  } catch (error) {
    return { error: '保存联系人失败' };
  }
}

// 在路由配置中使用
const router = createBrowserRouter([
  {
    path: '/contacts/new',
    element: <NewContact />,
    action: contactAction
  }
]);

action 函数参数 ​

action函数接收一个对象，包含以下属性：

  • request: 表单提交的Request对象
  • params: URL参数对象
  • context: 由router提供的上下文数据

处理表单数据 ​

request.formData()方法返回一个Promise，解析为FormData对象：

jsx
async function action({ request }) {
  const formData = await request.formData();
  
  // 获取单个字段
  const name = formData.get('name');
  const email = formData.get('email');
  
  // 获取多个值（如复选框）
  const interests = formData.getAll('interests');
  
  // 转换为普通对象
  const data = Object.fromEntries(formData);
  
  // 处理数据...
}

处理表单状态 ​

提交状态 ​

使用useNavigation钩子获取表单提交状态：

jsx
import { Form, useNavigation } from 'react-router-dom';

function ContactForm() {
  const navigation = useNavigation();
  const isSubmitting = navigation.state === 'submitting';
  
  return (
    <Form method="post">
      {/* 表单字段 */}
      
      <button type="submit" disabled={isSubmitting}>
        {isSubmitting ? '提交中...' : '提交'}
      </button>
      
      {navigation.state === 'submitting' && (
        <div className="spinner">加载中...</div>
      )}
    </Form>
  );
}

表单错误处理 ​

使用useActionData钩子获取action函数返回的数据，包括错误信息：

jsx
import { Form, useActionData } from 'react-router-dom';

function ContactForm() {
  const actionData = useActionData();
  const errors = actionData?.errors || {};
  const formError = actionData?.error;
  
  return (
    <Form method="post">
      {formError && <div className="error">{formError}</div>}
      
      <div>
        <label>
          姓名：
          <input type="text" name="name" />
        </label>
        {errors.name && <div className="error">{errors.name}</div>}
      </div>
      
      <div>
        <label>
          邮箱：
          <input type="email" name="email" />
        </label>
        {errors.email && <div className="error">{errors.email}</div>}
      </div>
      
      <button type="submit">提交</button>
    </Form>
  );
}

处理文件上传 ​

使用encType="multipart/form-data"属性处理文件上传：

jsx
import { Form, redirect } from 'react-router-dom';

function ProfileForm() {
  return (
    <Form 
      method="post" 
      encType="multipart/form-data"
    >
      <div>
        <label>
          姓名：
          <input type="text" name="name" />
        </label>
      </div>
      <div>
        <label>
          头像：
          <input type="file" name="avatar" />
        </label>
      </div>
      <button type="submit">更新资料</button>
    </Form>
  );
}

// action函数处理文件上传
async function profileAction({ request }) {
  const formData = await request.formData();
  const name = formData.get('name');
  const avatar = formData.get('avatar');
  
  // 处理文件
  if (avatar.size > 0) {
    // 使用FormData直接发送到服务器
    const response = await fetch('/api/upload', {
      method: 'POST',
      body: formData // 直接使用formData，不需要转换
    });
    
    if (!response.ok) {
      return { error: '上传失败' };
    }
  }
  
  return redirect('/profile');
}

使用 useFetcher 进行非导航提交 ​

有时我们需要提交表单但不想导航到新页面，比如点赞、收藏等操作。这时可以使用useFetcher钩子：

jsx
import { useFetcher } from 'react-router-dom';

function LikeButton({ postId, initialLikes }) {
  const fetcher = useFetcher();
  
  // 判断当前状态
  const isLiking = fetcher.state === 'submitting';
  
  // 获取最新的点赞数
  const likes = fetcher.data?.likes || initialLikes;
  
  return (
    <fetcher.Form method="post" action={`/posts/${postId}/like`}>
      <button type="submit" disabled={isLiking}>
        {isLiking ? '处理中...' : `点赞 (${likes})`}
      </button>
    </fetcher.Form>
  );
}

// 对应的action函数
async function likeAction({ params }) {
  const { postId } = params;
  
  // 调用API进行点赞
  const response = await fetch(`/api/posts/${postId}/like`, {
    method: 'POST'
  });
  
  if (!response.ok) {
    throw new Response('点赞失败', { status: 500 });
  }
  
  const data = await response.json();
  return data; // 返回更新后的点赞数
}

表单验证 ​

客户端验证 ​

可以使用HTML5表单验证或自定义验证：

jsx
<Form method="post" onSubmit={handleValidate}>
  <input 
    type="email" 
    name="email" 
    required 
    pattern="[a-z0-9._%+-]+@[a-z0-9.-]+\.[a-z]{2,}$" 
  />
  <button type="submit">提交</button>
</Form>

function handleValidate(event) {
  const form = event.currentTarget;
  if (!form.checkValidity()) {
    event.preventDefault();
    // 显示自定义错误消息
  }
}

服务端验证 ​

在action函数中进行验证，并返回错误信息：

jsx
async function registerAction({ request }) {
  const formData = await request.formData();
  const email = formData.get('email');
  const password = formData.get('password');
  
  const errors = {};
  
  // 验证邮箱
  if (!email || !email.includes('@')) {
    errors.email = '请输入有效的邮箱地址';
  }
  
  // 验证密码
  if (!password || password.length < 8) {
    errors.password = '密码至少需要8个字符';
  }
  
  // 如果有错误，返回错误信息
  if (Object.keys(errors).length > 0) {
    return { errors };
  }
  
  // 继续处理注册逻辑...
}

高级表单技巧 ​

动态表单字段 ​

使用JavaScript动态生成表单字段：

jsx
function DynamicForm() {
  const [fields, setFields] = useState([{ id: 1, value: '' }]);
  
  const addField = () => {
    const newId = fields.length > 0 ? Math.max(...fields.map(f => f.id)) + 1 : 1;
    setFields([...fields, { id: newId, value: '' }]);
  };
  
  return (
    <Form method="post">
      {fields.map((field, index) => (
        <div key={field.id}>
          <input
            type="text"
            name={`field-${field.id}`}
            placeholder={`字段 ${index + 1}`}
          />
        </div>
      ))}
      
      <button type="button" onClick={addField}>添加字段</button>
      <button type="submit">提交</button>
    </Form>
  );
}

表单重置 ​

使用useRef和Form组件的ref属性重置表单：

jsx
function ResetableForm() {
  const formRef = useRef();
  
  const handleReset = () => {
    formRef.current.reset();
  };
  
  return (
    <>
      <Form ref={formRef} method="post">
        {/* 表单字段 */}
        <div className="form-buttons">
          <button type="submit">提交</button>
          <button type="button" onClick={handleReset}>重置</button>
        </div>
      </Form>
    </>
  );
}

表单预填充 ​

使用useLoaderData钩子获取数据预填充表单：

jsx
// loader函数获取数据
async function contactLoader({ params }) {
  const contact = await fetchContact(params.contactId);
  return { contact };
}

// 组件中使用数据预填充表单
function EditContact() {
  const { contact } = useLoaderData();
  
  return (
    <Form method="post">
      <div>
        <label>
          姓名：
          <input 
            type="text" 
            name="name" 
            defaultValue={contact.name} 
          />
        </label>
      </div>
      <div>
        <label>
          邮箱：
          <input 
            type="email" 
            name="email" 
            defaultValue={contact.email} 
          />
        </label>
      </div>
      <button type="submit">更新</button>
    </Form>
  );
}

最佳实践 ​

  1. 使用命名约定：为表单字段使用一致的命名约定，便于在action函数中处理。

  2. 分离关注点：将表单UI、验证逻辑和提交处理分开。

  3. 适当使用客户端验证：对于简单验证使用HTML5表单验证，对于复杂验证使用JavaScript。

  4. 总是进行服务端验证：即使有客户端验证，也要在action函数中进行服务端验证。

  5. 提供清晰的错误消息：为用户提供具体、有帮助的错误消息。

  6. 处理加载状态：使用useNavigation钩子显示加载状态，提升用户体验。

  7. 使用TypeScript：为表单数据和错误定义类型，提高代码可维护性。

总结 ​

React Router的表单处理功能大大简化了表单提交和数据处理流程。通过Form组件和action函数，我们可以轻松实现表单提交、数据验证、错误处理和状态管理。这种声明式的表单处理方式使得代码更加清晰、可维护，同时提供了更好的用户体验。
