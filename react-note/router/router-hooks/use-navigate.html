useNavigate Hook ​

useNavigate 是 React Router v6 提供的一个 Hook，用于实现编程式导航。它返回一个函数，可以用来导航到不同的路由。

基本用法 ​

jsx
import { useNavigate } from 'react-router-dom';

function HomeButton() {
  const navigate = useNavigate();

  return (
    <button onClick={() => navigate('/')}>
      返回首页
    </button>
  );
}

语法 ​

jsx
const navigate = useNavigate();

返回值 ​

useNavigate 返回一个函数，该函数可以用以下方式调用：

  1. 导航到指定路径：

    jsx
    navigate(to, options);

  2. 相对于当前位置导航：

    jsx
    navigate(delta);

参数 ​

路径导航 ​

当使用 navigate(to, options) 形式时：

  • to：字符串或位置对象，表示导航目标
  • options：可选的配置对象，包含以下属性：
    • replace：布尔值，如果为 true，导航将替换历史堆栈中的当前条目，而不是添加新条目
    • state：任意值，要保存到历史状态的状态
    • relative：字符串，可以是 "route" 或 "path"，指定 to 是相对于当前路由还是当前路径

历史导航 ​

当使用 navigate(delta) 形式时：

  • delta：数字，表示在历史堆栈中前进或后退的步数（正数前进，负数后退）

示例 ​

基本导航 ​

jsx
import { useNavigate } from 'react-router-dom';

function ProductPage() {
  const navigate = useNavigate();

  return (
    <div>
      <h1>产品详情</h1>
      <button onClick={() => navigate('/products')}>
        返回产品列表
      </button>
    </div>
  );
}

带参数的导航 ​

jsx
import { useNavigate } from 'react-router-dom';

function ProductList() {
  const navigate = useNavigate();

  const viewProduct = (productId) => {
    navigate(`/products/${productId}`);
  };

  return (
    <div>
      <h1>产品列表</h1>
      <ul>
        <li onClick={() => viewProduct(1)}>产品 1</li>
        <li onClick={() => viewProduct(2)}>产品 2</li>
        <li onClick={() => viewProduct(3)}>产品 3</li>
      </ul>
    </div>
  );
}

使用 replace 选项 ​

使用 replace: true 选项可以替换当前的历史记录，而不是添加新的记录。这在登录成功后重定向等场景中很有用。

jsx
import { useNavigate } from 'react-router-dom';

function Login() {
  const navigate = useNavigate();

  const handleLogin = async (event) => {
    event.preventDefault();
    // 执行登录逻辑
    const success = await loginUser();
    
    if (success) {
      // 登录成功后替换当前历史记录，防止用户点击返回按钮回到登录页
      navigate('/dashboard', { replace: true });
    }
  };

  return (
    <form onSubmit={handleLogin}>
      {/* 登录表单 */}
      <button type="submit">登录</button>
    </form>
  );
}

传递状态 ​

使用 state 选项可以在导航时传递数据：

jsx
import { useNavigate, useLocation } from 'react-router-dom';

// 在一个组件中传递状态
function ProductList() {
  const navigate = useNavigate();

  const viewProductDetails = (product) => {
    navigate(`/products/${product.id}`, {
      state: { productName: product.name, price: product.price }
    });
  };

  return (
    <div>
      <h1>产品列表</h1>
      <button onClick={() => viewProductDetails({ id: 1, name: '手机', price: 1999 })}>
        查看手机详情
      </button>
    </div>
  );
}

// 在另一个组件中接收状态
function ProductDetails() {
  const location = useLocation();
  const { productName, price } = location.state || {};

  return (
    <div>
      <h1>产品详情</h1>
      {productName && <p>产品名称: {productName}</p>}
      {price && <p>价格: ¥{price}</p>}
    </div>
  );
}

历史导航 ​

使用数字参数可以在历史堆栈中前进或后退：

jsx
import { useNavigate } from 'react-router-dom';

function NavigationControls() {
  const navigate = useNavigate();

  return (
    <div>
      <button onClick={() => navigate(-1)}>返回上一页</button>
      <button onClick={() => navigate(-2)}>返回前两页</button>
      <button onClick={() => navigate(1)}>前进一页</button>
    </div>
  );
}

相对路由导航 ​

使用 relative 选项可以指定导航是相对于当前路由还是当前路径：

jsx
import { useNavigate } from 'react-router-dom';

function NestedComponent() {
  const navigate = useNavigate();

  return (
    <div>
      {/* 相对于当前路由导航 */}
      <button onClick={() => navigate('../sibling', { relative: 'route' })}>
        导航到兄弟路由
      </button>
      
      {/* 相对于当前路径导航 */}
      <button onClick={() => navigate('../other', { relative: 'path' })}>
        导航到其他路径
      </button>
    </div>
  );
}

注意事项 ​

  1. 只能在 React 组件或自定义 Hook 中使用：useNavigate 是一个 React Hook，因此只能在 React 组件或自定义 Hook 中使用。

  2. 不能在条件语句中使用：与其他 React Hook 一样，useNavigate 不能在条件语句、循环或嵌套函数中使用。

  3. 与 useLocation 配合使用：如果需要访问通过 state 传递的数据，需要使用 useLocation Hook。

  4. 路径解析：默认情况下，路径是相对于当前路由解析的。如果需要改变这一行为，可以使用 relative 选项。

与旧版本的区别 ​

在 React Router v5 中，编程式导航是通过 useHistory Hook 实现的。在 v6 中，useHistory 被替换为 useNavigate，API 也有所变化：

jsx
// React Router v5
import { useHistory } from 'react-router-dom';

function Example() {
  const history = useHistory();
  
  const handleClick = () => {
    history.push('/home');
  };
}

// React Router v6
import { useNavigate } from 'react-router-dom';

function Example() {
  const navigate = useNavigate();
  
  const handleClick = () => {
    navigate('/home');
  };
}

相关 API ​

  • Link - 声明式导航组件
  • NavLink - 带有活动状态的导航链接组件
  • useLocation - 访问当前位置
  • useParams - 访问 URL 参数
