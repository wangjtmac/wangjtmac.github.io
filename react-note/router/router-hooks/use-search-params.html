useSearchParams Hook ​

useSearchParams 是 React Router 提供的一个 Hook，用于读取和修改 URL 的查询字符串。它返回一个类似于 React 的 useState 的数组，包含当前的查询参数和一个更新查询参数的函数。

基本用法 ​

jsx
import { useSearchParams } from 'react-router-dom';

function ProductList() {
  const [searchParams, setSearchParams] = useSearchParams();
  
  // 获取查询参数
  const category = searchParams.get('category');
  const sort = searchParams.get('sort') || 'name';
  
  // 更新查询参数
  const handleSortChange = (e) => {
    setSearchParams({ category, sort: e.target.value });
  };
  
  return (
    <div>
      <h1>产品列表</h1>
      {category && <p>分类: {category}</p>}
      <p>排序方式: {sort}</p>
      
      <select value={sort} onChange={handleSortChange}>
        <option value="name">按名称</option>
        <option value="price">按价格</option>
        <option value="date">按日期</option>
      </select>
    </div>
  );
}

语法 ​

jsx
const [searchParams, setSearchParams] = useSearchParams(defaultInit);

参数 ​

  • defaultInit (可选): 默认的查询参数，可以是 URLSearchParams 对象、查询字符串或对象。

返回值 ​

useSearchParams 返回一个包含两个元素的数组：

  1. searchParams: 当前的查询参数，是一个 URLSearchParams 对象
  2. setSearchParams: 更新查询参数的函数

详细说明 ​

searchParams ​

searchParams 是一个 URLSearchParams 对象，提供了以下常用方法：

  • get(key): 获取指定键的第一个值
  • getAll(key): 获取指定键的所有值
  • has(key): 检查是否存在指定键
  • set(key, value): 设置指定键的值
  • append(key, value): 添加指定键的值
  • delete(key): 删除指定键
  • entries(): 返回所有键值对的迭代器
  • forEach(callback): 对每个键值对执行回调函数
  • toString(): 返回查询字符串表示

setSearchParams ​

setSearchParams 函数用于更新查询参数，它接受以下类型的参数：

  • 对象: { key1: value1, key2: value2 }
  • 字符串: "key1=value1&key2=value2"
  • URLSearchParams 对象

它还接受一个可选的第二个参数，用于指定导航选项：

jsx
setSearchParams(newParams, { replace: true });

示例 ​

基本示例 ​

jsx
import { useSearchParams } from 'react-router-dom';

function SearchComponent() {
  const [searchParams, setSearchParams] = useSearchParams();
  const query = searchParams.get('q') || '';
  
  const handleSearch = (e) => {
    e.preventDefault();
    const formData = new FormData(e.target);
    const newQuery = formData.get('query');
    setSearchParams({ q: newQuery });
  };
  
  return (
    <div>
      <form onSubmit={handleSearch}>
        <input 
          name="query" 
          defaultValue={query} 
          placeholder="搜索..." 
        />
        <button type="submit">搜索</button>
      </form>
      
      {query && <p>搜索结果: "{query}"</p>}
    </div>
  );
}

多个查询参数 ​

jsx
import { useSearchParams } from 'react-router-dom';

function FilterableList() {
  const [searchParams, setSearchParams] = useSearchParams();
  
  const category = searchParams.get('category') || 'all';
  const sort = searchParams.get('sort') || 'name';
  const page = parseInt(searchParams.get('page') || '1', 10);
  
  const updateFilters = (newFilters) => {
    // 合并现有参数和新参数
    const newParams = { category, sort, page, ...newFilters };
    setSearchParams(newParams);
  };
  
  return (
    <div>
      <div className="filters">
        <select 
          value={category} 
          onChange={(e) => updateFilters({ category: e.target.value })}
        >
          <option value="all">所有分类</option>
          <option value="electronics">电子产品</option>
          <option value="clothing">服装</option>
          <option value="books">图书</option>
        </select>
        
        <select 
          value={sort} 
          onChange={(e) => updateFilters({ sort: e.target.value })}
        >
          <option value="name">按名称排序</option>
          <option value="price">按价格排序</option>
          <option value="date">按日期排序</option>
        </select>
      </div>
      
      <div className="pagination">
        <button 
          disabled={page === 1} 
          onClick={() => updateFilters({ page: page - 1 })}
        >
          上一页
        </button>
        <span>第 {page} 页</span>
        <button onClick={() => updateFilters({ page: page + 1 })}>
          下一页
        </button>
      </div>
    </div>
  );
}

保留现有参数 ​

使用扩展运算符保留现有参数：

jsx
import { useSearchParams } from 'react-router-dom';

function SearchFilters() {
  const [searchParams, setSearchParams] = useSearchParams();
  
  // 将 URLSearchParams 对象转换为普通对象
  const params = Object.fromEntries(searchParams.entries());
  
  const updateFilter = (key, value) => {
    setSearchParams({
      ...params,
      [key]: value
    });
  };
  
  return (
    <div>
      <button onClick={() => updateFilter('category', 'electronics')}>
        电子产品
      </button>
      <button onClick={() => updateFilter('category', 'clothing')}>
        服装
      </button>
      <button onClick={() => updateFilter('sort', 'price')}>
        按价格排序
      </button>
    </div>
  );
}

删除参数 ​

jsx
import { useSearchParams } from 'react-router-dom';

function FilterTags() {
  const [searchParams, setSearchParams] = useSearchParams();
  
  // 将 URLSearchParams 对象转换为普通对象
  const params = Object.fromEntries(searchParams.entries());
  
  const removeFilter = (key) => {
    const newParams = { ...params };
    delete newParams[key];
    setSearchParams(newParams);
  };
  
  return (
    <div className="filter-tags">
      {Object.entries(params).map(([key, value]) => (
        <span key={key} className="filter-tag">
          {key}: {value}
          <button onClick={() => removeFilter(key)}>×</button>
        </span>
      ))}
    </div>
  );
}

import { useSearchParams } from 'react-router-dom';

function ProductGrid() {
  // 提供默认参数
  const [searchParams, setSearchParams] = useSearchParams({
    view: 'grid',
    sort: 'name',
    page: '1'
  });
  
  const view = searchParams.get('view');
  const sort = searchParams.get('sort');

  return (

注意事项 ​

  1. 参数类型：URLSearchParams 对象中的所有值都是字符串，需要手动转换为其他类型。
jsx
const page = parseInt(searchParams.get('page') || '1', 10);
const showDetails = searchParams.get('details') === 'true';

  1. 保留现有参数：setSearchParams 会替换所有现有参数，如果需要保留其他参数，需要手动合并。

  2. 与 useLocation 的区别：useSearchParams 专注于查询参数的处理，而 useLocation 提供完整的位置信息。

  3. URL 编码：URLSearchParams 会自动处理 URL 编码和解码，无需手动调用 encodeURIComponent 或 decodeURIComponent。

最佳实践 ​

  1. 转换为对象：将 URLSearchParams 对象转换为普通对象，便于操作。
jsx
const params = Object.fromEntries(searchParams.entries());
  1. 默认值：为查询参数提供默认值，避免 null 或 undefined。
jsx
const sort = searchParams.get('sort') || 'name';
  1. 类型转换：根据需要将字符串值转换为适当的类型。
jsx
const page = parseInt(searchParams.get('page') || '1', 10);
const filters = JSON.parse(searchParams.get('filters') || '{}');
  1. 参数验证：验证查询参数的有效性，提供回退值。
jsx
const validSortOptions = ['name', 'price', 'date'];
const sort = validSortOptions.includes(searchParams.get('sort'))
  ? searchParams.get('sort')
  : 'name';
