组件实战 ​

本文将通过一系列实际案例，展示如何设计和实现高质量的React组件。我们将涵盖常见UI组件的实现、组件设计模式、性能优化以及组件测试等方面。

设计可复用的按钮组件 ​

按钮是几乎所有应用中最常用的组件之一。一个好的按钮组件应该具有高度的可定制性和可访问性。

基础按钮组件 ​

jsx
import React from 'react';
import './Button.css'; // 假设我们有一个对应的CSS文件

const Button = ({ 
  children, 
  type = 'button', 
  variant = 'primary', 
  size = 'medium', 
  disabled = false, 
  onClick, 
  className = '', 
  ...rest 
}) => {
  const baseClass = 'btn';
  const classes = `${baseClass} ${baseClass}--${variant} ${baseClass}--${size} ${className}`;
  
  return (
    <button
      type={type}
      className={classes}
      disabled={disabled}
      onClick={onClick}
      {...rest}
    >
      {children}
    </button>
  );
};

export default Button;

实现可控制的表单组件 ​

表单组件是应用中另一个常见的组件类型。下面我们实现一个可复用的输入框组件。

基础输入框组件 ​

jsx
import React, { useState } from 'react';
import './Input.css';

const Input = ({ 
  label, 
  id, 
  error, 
  helperText, 
  value, 
  onChange, 
  type = 'text',
  required = false,
  ...rest 
}) => {
  const [focused, setFocused] = useState(false);
  const inputId = id || `input-${Math.random().toString(36).substr(2, 9)}`;
  
  const handleFocus = () => setFocused(true);
  const handleBlur = () => setFocused(false);
  
  const inputClass = `input ${focused ? 'input--focused' : ''} ${error ? 'input--error' : ''}`;
  
  return (
    <div className="input-container">
      {label && (
        <label htmlFor={inputId} className="input-label">
          {label}
          {required && <span className="input-required">*</span>}
        </label>
      )}
      <input
        id={inputId}
        type={type}
        className={inputClass}
        value={value}
        onChange={onChange}
        onFocus={handleFocus}
        onBlur={handleBlur}
        aria-invalid={!!error}
        aria-describedby={error || helperText ? `${inputId}-helper` : undefined}
        required={required}
        {...rest}
      />
      {(error || helperText) && (
        <div 
          id={`${inputId}-helper`} 
          className={`input-helper ${error ? 'input-helper--error' : ''}`}
        >
          {error || helperText}
        </div>
      )}
    </div>
  );
};

export default Input;

自定义表单钩子 ​

创建一个自定义钩子来简化表单状态管理：

jsx
import { useState } from 'react';

function useForm(initialValues = {}) {
  const [values, setValues] = useState(initialValues);
  const [errors, setErrors] = useState({});
  const [touched, setTouched] = useState({});
  
  const handleChange = (e) => {
    const { name, value } = e.target;
    setValues(prev => ({ ...prev, [name]: value }));
  };
  
  const handleBlur = (e) => {
    const { name } = e.target;
    setTouched(prev => ({ ...prev, [name]: true }));
  };
  
  const reset = () => {
    setValues(initialValues);
    setErrors({});
    setTouched({});
  };
  
  const setFieldValue = (name, value) => {
    setValues(prev => ({ ...prev, [name]: value }));
  };
  
  const setFieldError = (name, error) => {
    setErrors(prev => ({ ...prev, [name]: error }));
  };
  
  return {
    values,
    errors,
    touched,
    handleChange,
    handleBlur,
    reset,
    setFieldValue,
    setFieldError
  };
}

// 使用示例
function LoginForm() {
  const { values, errors, handleChange, handleBlur } = useForm({
    email: '',
    password: ''
  });
  
  const handleSubmit = (e) => {
    e.preventDefault();
    console.log('提交表单', values);
  };
  
  return (
    <form onSubmit={handleSubmit}>
      <Input
        label="邮箱"
        name="email"
        type="email"
        value={values.email}
        onChange={handleChange}
        onBlur={handleBlur}
        error={errors.email}
        required
      />
      <Input
        label="密码"
        name="password"
        type="password"
        value={values.password}
        onChange={handleChange}
        onBlur={handleBlur}
        error={errors.password}
        required
      />
      <Button type="submit">登录</Button>
    </form>
  );
}

实现可复用的模态框组件 ​

模态框是另一个常见的UI组件，它需要处理焦点管理、键盘交互和可访问性。

jsx
import React, { useEffect, useRef } from 'react';
import ReactDOM from 'react-dom';
import './Modal.css';

const Modal = ({ 
  isOpen, 
  onClose, 
  title, 
  children, 
  footer 
}) => {
  const modalRef = useRef(null);
  
  useEffect(() => {
    if (isOpen) {
      // 保存之前的焦点元素
      const previousFocus = document.activeElement;
      
      // 将焦点移到模态框
      modalRef.current?.focus();
      
      // 禁止背景滚动
      document.body.style.overflow = 'hidden';
      
      // 处理ESC键
      const handleKeyDown = (e) => {
        if (e.key === 'Escape') onClose();
      };
      
      window.addEventListener('keydown', handleKeyDown);
      
      return () => {
        // 清理
        window.removeEventListener('keydown', handleKeyDown);
        document.body.style.overflow = '';
        previousFocus?.focus();
      };
    }
  }, [isOpen, onClose]);
  
  if (!isOpen) return null;
  
  return ReactDOM.createPortal(
    <div className="modal-overlay" onClick={onClose}>
      <div 
        className="modal"
        ref={modalRef}
        onClick={e => e.stopPropagation()}
        tabIndex={-1}
        role="dialog"
        aria-modal="true"
        aria-labelledby="modal-title"
      >
        <div className="modal-header">
          <h2 id="modal-title" className="modal-title">{title}</h2>
          <button 
            className="modal-close"
            onClick={onClose}
            aria-label="关闭"
          >
            &times;
          </button>
        </div>
        <div className="modal-body">
          {children}
        </div>
        {footer && (
          <div className="modal-footer">
            {footer}
          </div>
        )}
      </div>
    </div>,
    document.body
  );
};

// 使用示例
function App() {
  const [isModalOpen, setIsModalOpen] = useState(false);
  
  return (
    <div>
      <Button onClick={() => setIsModalOpen(true)}>打开模态框</Button>
      
      <Modal
        isOpen={isModalOpen}
        onClose={() => setIsModalOpen(false)}
        title="示例模态框"
        footer={
          <>
            <Button variant="secondary" onClick={() => setIsModalOpen(false)}>
              取消
            </Button>
            <Button onClick={() => console.log('确认')}>
              确认
            </Button>
          </>
        }
      >
        <p>这是模态框的内容</p>
      </Modal>
    </div>
  );
}

实现可复用的数据表格组件 ​

数据表格是展示和操作数据的重要组件，下面实现一个简单但功能完整的表格组件：

jsx
import React, { useState } from 'react';
import './Table.css';

const Table = ({ 
  columns, 
  data, 
  onRowClick,
  sortable = false,
  pagination = false,
  pageSize = 10
}) => {
  const [sortConfig, setSortConfig] = useState({ key: null, direction: 'asc' });
  const [currentPage, setCurrentPage] = useState(1);
  
  // 排序逻辑
  const sortedData = React.useMemo(() => {
    if (!sortConfig.key) return data;
    
    return [...data].sort((a, b) => {
      if (a[sortConfig.key] < b[sortConfig.key]) {
        return sortConfig.direction === 'asc' ? -1 : 1;
      }
      if (a[sortConfig.key] > b[sortConfig.key]) {
        return sortConfig.direction === 'asc' ? 1 : -1;
      }
      return 0;
    });
  }, [data, sortConfig]);
  
  // 分页逻辑
  const paginatedData = React.useMemo(() => {
    if (!pagination) return sortedData;
    
    const start = (currentPage - 1) * pageSize;
    return sortedData.slice(start, start + pageSize);
  }, [sortedData, pagination, currentPage, pageSize]);
  
  // 排序处理
  const handleSort = (key) => {
    if (!sortable) return;
    
    let direction = 'asc';
    if (sortConfig.key === key && sortConfig.direction === 'asc') {
      direction = 'desc';
    }
    setSortConfig({ key, direction });
  };
  
  // 分页控件
  const totalPages = Math.ceil(data.length / pageSize);
  
  const renderPagination = () => {
    return (
      <div className="table-pagination">
        <button
          disabled={currentPage === 1}
          onClick={() => setCurrentPage(p => Math.max(1, p - 1))}
        >
          上一页
        </button>
        <span>
          {currentPage} / {totalPages}
        </span>
        <button
          disabled={currentPage === totalPages}
          onClick={() => setCurrentPage(p => Math.min(totalPages, p + 1))}
        >
          下一页
        </button>
      </div>
    );
  };
  
  return (
    <div className="table-container">
      <table className="table">
        <thead>
          <tr>
            {columns.map(column => (
              <th 
                key={column.key} 
                onClick={() => handleSort(column.key)}
                className={sortable ? 'sortable' : ''}
              >
                {column.title}
                {sortable && sortConfig.key === column.key && (
                  <span className="sort-indicator">
                    {sortConfig.direction === 'asc' ? ' ↑' : ' ↓'}
                  </span>
                )}
              </th>
            ))}
          </tr>
        </thead>
        <tbody>
          {paginatedData.map((row, index) => (
            <tr 
              key={row.id || index} 
              onClick={() => onRowClick && onRowClick(row)}
              className={onRowClick ? 'clickable' : ''}
            >
              {columns.map(column => (
                <td key={column.key}>
                  {column.render ? column.render(row) : row[column.key]}
                </td>
              ))}
            </tr>
          ))}
        </tbody>
      </table>
      
      {pagination && renderPagination()}
    </div>
  );
};

// 使用示例
function UserTable() {
  const columns = [
    { key: 'name', title: '姓名' },
    { key: 'age', title: '年龄' },
    { key: 'email', title: '邮箱' },
    { 
      key: 'actions', 
      title: '操作',
      render: (row) => (
        <div>
          <Button size="small" onClick={() => console.log('编辑', row)}>编辑</Button>
          <Button size="small" variant="danger" onClick={() => console.log('删除', row)}>删除</Button>
        </div>
      )
    }
  ];
  
  const data = [
    { id: 1, name: '张三', age: 28, email: 'zhangsan@example.com' },
    { id: 2, name: '李四', age: 32, email: 'lisi@example.com' },
    // 更多数据...
  ];
  
  return (
    <Table 
      columns={columns} 
      data={data}
      sortable
      pagination
      pageSize={5}
      onRowClick={(row) => console.log('点击行', row)}
    />
  );
}

实现无限滚动列表组件 ​

无限滚动是处理大量数据的常用模式，下面实现一个简单的无限滚动组件：

jsx
import React, { useState, useEffect, useRef, useCallback } from 'react';
import './InfiniteScroll.css';

const InfiniteScroll = ({ 
  loadItems, 
  renderItem, 
  hasMore, 
  loading,
  initialItems = [],
  loadingIndicator = <div>加载中...</div>,
  endMessage = <div>没有更多数据了</div>,
  threshold = 300
}) => {
  const [items, setItems] = useState(initialItems);
  const [page, setPage] = useState(1);
  const [isLoading, setIsLoading] = useState(false);
  const observer = useRef(null);
  const lastItemRef = useRef(null);
  
  // 加载更多数据
  const loadMore = useCallback(async () => {
    if (isLoading || !hasMore) return;
    
    setIsLoading(true);
    try {
      const newItems = await loadItems(page);
      setItems(prevItems => [...prevItems, ...newItems]);
      setPage(prevPage => prevPage + 1);
    } catch (error) {
      console.error('加载数据失败', error);
    } finally {
      setIsLoading(false);
    }
  }, [isLoading, hasMore, loadItems, page]);
  
  // 设置Intersection Observer
  useEffect(() => {
    if (loading) return;
    
    if (observer.current) {
      observer.current.disconnect();
    }
    
    observer.current = new IntersectionObserver(
      entries => {
        if (entries[0].isIntersecting && hasMore) {
          loadMore();
        }
      },
      { rootMargin: `0px 0px ${threshold}px 0px` }
    );
    
    if (lastItemRef.current) {
      observer.current.observe(lastItemRef.current);
    }
    
    return () => {
      if (observer.current) {
        observer.current.disconnect();
      }
    };
  }, [loading, hasMore, loadMore, threshold]);
  
  return (
    <div className="infinite-scroll">
      <div className="infinite-scroll-content">
        {items.map((item, index) => (
          <div 
            key={item.id || index} 
            ref={index === items.length - 1 ? lastItemRef : null}
          >
            {renderItem(item)}
          </div>
        ))}
      </div>
      
      {isLoading && loadingIndicator}
      {!hasMore && !isLoading && endMessage}
    </div>
  );
};

// 使用示例
function PostList() {
  const [hasMore, setHasMore] = useState(true);
  
  const loadPosts = async (page) => {
    // 模拟API请求
    const response = await fetch(`/api/posts?page=${page}&limit=10`);
    const data = await response.json();
    
    if (data.length < 10) {
      setHasMore(false);
    }
    
    return data;
  };
  
  const renderPost = (post) => (
    <div className="post">
      <h3>{post.title}</h3>
      <p>{post.content}</p>
    </div>
  );
  
  return (
    <InfiniteScroll
      loadItems={loadPosts}
      renderItem={renderPost}
      hasMore={hasMore}
      loading={false}
      initialItems={[]}
    />
  );
}

组件设计最佳实践 ​

在实现这些组件的过程中，我们遵循了一些重要的设计原则：

1. 组件接口设计 ​

  • Props API清晰：每个组件都有明确定义的props，包括默认值和类型
  • 使用扩展运算符：通过{...rest}传递未明确处理的props，增强灵活性
  • 合理的默认值：为常用props提供合理的默认值，减少使用者的配置负担

2. 可组合性 ​

  • 组件分层：基础组件（如Button）可以被更高级组件（如IconButton）复用
  • children属性：通过children属性允许组件包含任意内容
  • 渲染属性：如Table组件的column.render函数，允许自定义渲染逻辑

3. 可访问性 ​

  • 语义化HTML：使用正确的HTML元素（如button而非div）
  • ARIA属性：添加适当的ARIA属性，如aria-label、aria-modal等
  • 键盘交互：支持键盘导航和操作，如模态框的ESC键关闭

4. 性能优化 ​

  • 记忆化：使用React.useMemo避免不必要的计算
  • 惰性初始化：如useForm钩子中的初始值
  • 条件渲染：只在需要时渲染组件，如Modal组件的条件渲染

5. 状态管理 ​

  • 单一数据源：保持状态的单一来源，避免状态同步问题
  • 受控与非受控模式：支持两种模式，增加灵活性
  • 自定义钩子：抽象复杂的状态逻辑，如useForm钩子

单元测试 ​

使用Jest和React Testing Library测试组件的基本功能：

jsx
// Button组件测试示例
import { render, screen, fireEvent } from '@testing-library/react';
import Button from './Button';

describe('Button组件', () => {
  test('渲染按钮文本', () => {
    render(<Button>点击我</Button>);
    expect(screen.getByText('点击我')).toBeInTheDocument();
  });
  
  test('点击按钮触发onClick', () => {
    const handleClick = jest.fn();
    render(<Button onClick={handleClick}>点击我</Button>);
    fireEvent.click(screen.getByText('点击我'));
    expect(handleClick).toHaveBeenCalledTimes(1);
  });
  
  test('禁用按钮', () => {
    render(<Button disabled>点击我</Button>);
    expect(screen.getByText('点击我')).toBeDisabled();
  });
  
  test('应用正确的CSS类', () => {
    render(<Button variant="secondary" size="large">点击我</Button>);
    const button = screen.getByText('点击我');
    expect(button).toHaveClass('btn');
    expect(button).toHaveClass('btn--secondary');
    expect(button).toHaveClass('btn--large');
  });
});
