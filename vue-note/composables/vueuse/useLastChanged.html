useLastChanged ​

基本信息 ​

  • 类别: 动画/状态
  • 包名: @vueuse/core
  • 功能: 跟踪值最后变化的时间

描述 ​

useLastChanged 函数用于跟踪一个响应式值最后一次变化的时间戳。当被监视的值发生变化时，它会自动更新时间戳。这对于需要知道数据何时更新、实现自动刷新逻辑或基于时间的状态管理非常有用。

类型定义 ​

typescript
/**
 * 跟踪值最后变化的时间
 *
 * @param source - 要监视的源值
 * @param options - 配置选项
 */
export declare function useLastChanged(
  source: MaybeRef<any>,
  options?: UseLastChangedOptions
): ComputedRef<number>

export interface UseLastChangedOptions extends WatchOptions {
  /**
   * 初始时间戳
   * @default 0
   */
  initialValue?: number
  
  /**
   * 时间戳生成函数
   */
  timestamp?: () => number
}

参数 ​

参数名类型默认值说明
sourceMaybeRef<any>必填要监视的响应式值
optionsUseLastChangedOptions{}配置选项

配置选项 ​

选项名类型默认值说明
initialValuenumber0初始时间戳值
timestamp() => numberDate.now自定义时间戳生成函数
deepbooleanfalse是否深度监听对象变化
immediatebooleanfalse是否立即触发一次
flush'pre' | 'post' | 'sync''pre'触发时机

返回值 ​

返回一个计算属性 ComputedRef<number>，表示被监视值最后一次变化的时间戳。

使用示例 ​

基本用法 ​

js
import { ref } from 'vue'
import { useLastChanged } from '@vueuse/core'

const counter = ref(0)
const lastChanged = useLastChanged(counter)

// 当 counter 变化时，lastChanged 会自动更新为当前时间戳
counter.value++ // lastChanged.value 将被更新为当前时间戳

自定义初始值和时间戳函数 ​

js
import { ref } from 'vue'
import { useLastChanged } from '@vueuse/core'

const source = ref('hello')
const lastChanged = useLastChanged(source, {
  initialValue: Date.now(), // 使用当前时间作为初始值
  timestamp: () => performance.now(), // 使用高精度时间
})

深度监听对象变化 ​

js
import { reactive } from 'vue'
import { useLastChanged } from '@vueuse/core'

const user = reactive({
  name: 'John',
  settings: {
    theme: 'dark',
  },
})

// 深度监听对象变化
const lastChanged = useLastChanged(user, { deep: true })

// 当嵌套属性变化时，也会触发更新
user.settings.theme = 'light' // lastChanged.value 将被更新

结合其他函数使用 ​

js
import { ref, computed } from 'vue'
import { useLastChanged, useTimeAgo } from '@vueuse/core'

const message = ref('Hello')
const lastChanged = useLastChanged(message)

// 计算最后更新时间的相对描述
const lastUpdatedAgo = useTimeAgo(lastChanged)

// 使用示例
message.value = 'Updated message'
console.log(lastUpdatedAgo.value) // 输出："刚刚"

工作原理 ​

useLastChanged 函数的工作原理如下：

  1. 创建一个响应式引用，初始值为 options.initialValue 或默认为 0
  2. 使用 watch 监听源值的变化
  3. 当源值变化时，使用 options.timestamp 函数（默认为 Date.now）生成新的时间戳
  4. 将新的时间戳赋值给响应式引用

简化的实现如下：

js
function useLastChanged(source, options = {}) {
  const {
    initialValue = 0,
    timestamp = () => Date.now(),
    ...watchOptions
  } = options
  
  const changed = ref(initialValue)
  
  watch(
    source,
    () => {
      changed.value = timestamp()
    },
    watchOptions
  )
  
  return computed(() => changed.value)
}

应用场景 ​

  1. 数据刷新指示器：显示数据最后更新的时间
  2. 自动刷新逻辑：基于上次更新时间决定是否需要刷新数据
  3. 用户活动跟踪：记录用户最后一次交互的时间
  4. 缓存失效策略：基于数据变化时间实现缓存失效
  5. 变化频率分析：监控值变化的频率，用于性能优化

示例：创建数据刷新指示器 ​

vue
<template>
  <div class="data-panel">
    <h3>用户数据</h3>
    <div class="data-content">
      <div v-for="(user, index) in users" :key="index" class="user-item">
        {{ user.name }} - {{ user.email }}
      </div>
    </div>
    <div class="data-footer">
      <span>最后更新: {{ lastUpdatedText }}</span>
      <button @click="refreshData">刷新数据</button>
    </div>
  </div>
</template>

<script setup>
import { ref, computed } from 'vue'
import { useLastChanged, useTimeAgo } from '@vueuse/core'

const users = ref([
  { name: '张三', email: 'zhangsan@example.com' },
  { name: '李四', email: 'lisi@example.com' },
])

const lastChanged = useLastChanged(users, {
  initialValue: Date.now(),
  deep: true,
})

const lastUpdatedText = useTimeAgo(lastChanged)

const refreshData = () => {
  // 模拟数据刷新
  users.value = [
    ...users.value,
    { name: '王五', email: 'wangwu@example.com' },
  ]
  // lastChanged 会自动更新
}
</script>

示例：自动刷新机制 ​

vue
<template>
  <div class="auto-refresh-panel">
    <h3>实时数据</h3>
    <div class="data-content">
      {{ data }}
    </div>
    <div class="refresh-status">
      <div :class="['status-indicator', { 'needs-refresh': needsRefresh }]"></div>
      <span>{{ statusText }}</span>
      <button @click="refreshData">手动刷新</button>
    </div>
  </div>
</template>

<script setup>
import { ref, computed, watch } from 'vue'
import { useLastChanged, useTimestamp } from '@vueuse/core'

const data = ref('初始数据')
const lastChanged = useLastChanged(data, { initialValue: Date.now() })

// 当前时间戳
const now = useTimestamp()

// 计算是否需要刷新（超过30秒未更新）
const needsRefresh = computed(() => {
  return now.value - lastChanged.value > 30000
})

// 状态文本
const statusText = computed(() => {
  if (needsRefresh.value) {
    return '数据可能已过期，请刷新'
  }
  const seconds = Math.floor((now.value - lastChanged.value) / 1000)
  return `数据更新于 ${seconds} 秒前`
})

// 自动刷新逻辑
watch(needsRefresh, (needs) => {
  if (needs) {
    console.log('数据已过期，可以自动刷新')
    // 可以在这里触发自动刷新
  }
})

const refreshData = () => {
  data.value = `刷新的数据 - ${new Date().toLocaleTimeString()}`
  // lastChanged 会自动更新
}
</script>

.status-indicator {

  display: inline-block;
  margin-right: 5px;
}

}

与其他函数的对比 ​

函数相似点不同点
useLastChanged跟踪值的变化返回最后变化的时间戳
useTimestamp提供时间戳按固定间隔更新，不依赖于值的变化
useTimeAgo与时间相关将时间戳转换为相对时间描述
watch监听值的变化更通用，可执行任意回调函数

最佳实践 ​

  1. 合理设置初始值：

    • 如果需要表示"从未变化过"，可以使用 0 作为初始值
    • 如果需要表示"初始状态也是一次变化"，可以使用 Date.now() 作为初始值

  2. 选择合适的监听选项：

    • 对于对象类型，考虑使用 { deep: true } 监听嵌套属性的变化
    • 如果需要立即获取初始时间戳，使用 { immediate: true }

  3. 结合其他函数使用：

    • 与 useTimeAgo 结合显示相对时间
    • 与 useTimestamp 结合实现基于时间的逻辑

  4. 性能考虑：

    • 对于频繁变化的值，考虑使用 { flush: 'post' } 减少更新频率
    • 避免不必要的深度监听，特别是对于大型对象

总结 ​

useLastChanged 是一个简单而实用的函数，它提供了跟踪值最后变化时间的能力。通过与其他时间相关函数结合使用，可以实现丰富的基于时间的功能，如数据刷新指示器、自动刷新逻辑、用户活动跟踪等。它的实现简洁而高效，是状态管理和用户界面反馈的有力工具。
