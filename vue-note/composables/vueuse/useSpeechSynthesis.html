VueUse - useSpeechSynthesis ​

useSpeechSynthesis 组合式函数提供了与 Web Speech API 的 SpeechSynthesis 接口进行交互的能力，允许你在浏览器中实现文本到语音的转换。

它封装了 SpeechSynthesisUtterance 对象和 SpeechSynthesis 接口，提供了播放、暂停、停止语音以及获取语音状态（如是否正在播放、支持状态、错误信息等）的功能。

基本用法 ​

通过传入文本内容和可选的配置项，你可以轻松地将文本转换为语音。

vue
<template>
  <div>
    <p>该浏览器是否支持语音合成: {{ isSupported }}</p>
    <p>正在播放: {{ isPlaying }}</p>
    <p>状态: {{ status }}</p>
    <p>错误: {{ error?.message }}</p>
    <label for="text-to-speak">要朗读的文本:</label>
    <input id="text-to-speak" v-model="text" type="text" />
    <br />
    <label for="lang">语言:</label>
    <input id="lang" v-model="lang" type="text" />
    <br />
    <label for="pitch">音高 (0-2):</label>
    <input id="pitch" v-model="pitch" type="range" min="0" max="2" step="0.1" />
    <span>{{ pitch }}</span>
    <br />
    <label for="rate">语速 (0.1-10):</label>
    <input id="rate" v-model="rate" type="range" min="0.1" max="10" step="0.1" />
    <span>{{ rate }}</span>
    <br />
    <label for="volume">音量 (0-1):</label>
    <input id="volume" v-model="volume" type="range" min="0" max="1" step="0.1" />
    <span>{{ volume }}</span>
    <br />
    <button @click="speak" :disabled="!isSupported || isPlaying">
      朗读
    </button>
    <button @click="toggle" :disabled="!isSupported">
      {{ isPlaying ? '暂停' : '继续' }}
    </button>
    <button @click="stop" :disabled="!isSupported">
      停止
    </button>
  </div>
</template>

<script setup>
import { ref } from 'vue';
import { useSpeechSynthesis } from '@vueuse/core';

const text = ref('Hello, VueUse Speech Synthesis!');
const lang = ref('en-US');
const pitch = ref(1);
const rate = ref(1);
const volume = ref(1);

const {
  isSupported,
  isPlaying,
  status,
  error,
  speak,
  toggle,
  stop,
} = useSpeechSynthesis(text, {
  lang,
  pitch,
  rate,
  volume,
});
</script>

类型定义 ​

typescript
import type { MaybeRef, MaybeRefOrGetter } from 'vue'
import type { ConfigurableWindow } from '../_configurable'

export type UseSpeechSynthesisStatus = 'init' | 'play' | 'pause' | 'end'

export interface UseSpeechSynthesisOptions extends ConfigurableWindow {
  /**
   * Language for SpeechSynthesis
   *
   * @default 'en-US'
   */
  lang?: MaybeRefOrGetter<string>
  /**
   * Gets and sets the pitch at which the utterance will be spoken at.
   *
   * @default 1
   */
  pitch?: MaybeRefOrGetter<SpeechSynthesisUtterance['pitch']>
  /**
   * Gets and sets the speed at which the utterance will be spoken at.
   *
   * @default 1
   */
  rate?: MaybeRefOrGetter<SpeechSynthesisUtterance['rate']>
  /**
   * Gets and sets the voice that will be used to speak the utterance.
   */
  voice?: MaybeRef<SpeechSynthesisVoice>
  /**
   * Gets and sets the volume that the utterance will be spoken at.
   *
   * @default 1
   */
  volume?: MaybeRefOrGetter<SpeechSynthesisUtterance['volume']>
}

export interface UseSpeechSynthesisReturn {
  isSupported: Ref<boolean>
  isPlaying: Ref<boolean>
  status: Ref<UseSpeechSynthesisStatus>
  utterance: Ref<SpeechSynthesisUtterance>
  error: Ref<SpeechSynthesisErrorEvent | undefined>
  stop: () => void
  toggle: (value?: boolean) => void
  speak: () => void
}

declare function useSpeechSynthesis(
  text: MaybeRefOrGetter<string>,
  options?: UseSpeechSynthesisOptions
): UseSpeechSynthesisReturn

参数 ​

useSpeechSynthesis 函数接收两个参数：

  • text: MaybeRefOrGetter<string>。要转换为语音的文本内容。可以是 string 或 Ref<string> 或 Getter<string>。
  • options: UseSpeechSynthesisOptions (可选)。一个配置对象，用于自定义语音合成的行为。包含以下属性：
    • lang: MaybeRefOrGetter<string>。语音的语言，默认为 'en-US'。可以是 string 或 Ref<string> 或 Getter<string>。
    • pitch: MaybeRefOrGetter<SpeechSynthesisUtterance['pitch']>。语音的音高，范围从 0 到 2，默认为 1。可以是 number 或 Ref<number> 或 Getter<number>。
    • rate: MaybeRefOrGetter<SpeechSynthesisUtterance['rate']>。语音的语速，范围从 0.1 到 10，默认为 1。可以是 number 或 Ref<number> 或 Getter<number>。
    • voice: MaybeRef<SpeechSynthesisVoice>。用于朗读的语音。可以是一个 SpeechSynthesisVoice 对象或 Ref<SpeechSynthesisVoice>。默认由浏览器自动选择。
    • volume: MaybeRefOrGetter<SpeechSynthesisUtterance['volume']>。语音的音量，范围从 0 到 1，默认为 1。可以是 number 或 Ref<number> 或 Getter<number>。
    • window: Window (可选)。指定监听事件的窗口对象，默认为当前窗口。

返回值 ​

useSpeechSynthesis 返回一个包含以下属性的对象：

  • isSupported: Ref<boolean>。一个布尔型的 ref，指示当前浏览器是否支持 Web Speech API 的语音合成功能。
  • isPlaying: Ref<boolean>。一个布尔型的 ref，指示当前语音是否正在播放。
  • status: Ref<UseSpeechSynthesisStatus>。一个字符串型的 ref，表示语音的当前状态，可能的值为 'init' (初始化), 'play' (播放中), 'pause' (暂停), 'end' (结束)。
  • utterance: Ref<SpeechSynthesisUtterance>。一个 SpeechSynthesisUtterance 实例的 ref，包含了当前语音合成请求的详细信息和事件处理器。
  • error: Ref<SpeechSynthesisErrorEvent | undefined>。一个 SpeechSynthesisErrorEvent 实例的 ref，如果发生错误，则包含错误信息；否则为 undefined。
  • stop: () => void。一个函数，用于停止当前的语音播放。
  • toggle: (value?: boolean) => void。一个函数，用于切换语音的播放/暂停状态。如果传入 value，则强制设置为该值。
  • speak: () => void。一个函数，用于开始播放语音。如果当前有正在播放的语音，会先停止再重新播放。

工作原理 ​

useSpeechSynthesis 的核心工作原理是利用浏览器提供的 SpeechSynthesis 接口和 SpeechSynthesisUtterance 对象。

  1. 特性检测: 首先通过 useSupported 判断当前环境是否支持 speechSynthesis API。
  2. SpeechSynthesisUtterance 创建: 当 text 或相关选项变化时，会创建一个新的 SpeechSynthesisUtterance 实例，并设置其语言、音高、语速和音量。
  3. 事件监听: 为 utterance 对象注册 onstart, onpause, onresume, onend, onerror 等事件监听器，以便实时更新 isPlaying 和 status 等响应式状态。
  4. 控制方法: speak 方法会先取消当前正在播放的所有语音，然后使用 synth.speak() 播放新的 utterance。stop 方法调用 synth.cancel() 停止所有语音。toggle 方法则根据 isPlaying 的状态调用 synth.resume() 或 synth.pause()。
  5. 响应式同步: 通过 watch 监听 lang 和 voice 的变化，以更新 utterance 对象。同时，isPlaying 的变化也会触发 resume 或 pause 操作。
  6. 资源清理: 在组件卸载时，通过 tryOnScopeDispose 确保停止语音播放，释放资源。

使用场景 ​

  • 辅助功能: 为视觉障碍用户提供文本朗读功能。
  • 通知和提醒: 在特定事件发生时通过语音进行提醒，例如订单状态更新、新消息通知等。
  • 多媒体应用: 用于有声读物、语言学习应用等。
  • 交互式向导: 作为用户界面的一部分，提供语音指导和反馈。
  • 数据可视化: 将图表或数据的关键信息转化为语音输出，方便用户理解。

注意事项 ​

  • 浏览器兼容性: 虽然现代浏览器对 SpeechSynthesis API 的支持度较高，但仍需注意兼容性问题。可以使用 isSupported 来判断。
  • 用户交互: 某些浏览器可能需要用户与页面进行一次交互后才能播放语音，以避免自动播放。
  • 语音库: 浏览器提供的语音可能因操作系统和浏览器而异，且质量参差不齐。可以考虑提供语音选择功能。
  • 音量控制: volume 选项控制的是单个 utterance 的音量，而不是系统音量。
  • 语音队列: SpeechSynthesis 接口会维护一个语音队列。如果频繁调用 speak 而不先 cancel，可能会导致语音堆积。

实际应用示例 ​

1. 播放用户输入文本 ​

一个简单的文本转语音工具，用户输入文本后点击按钮即可朗读。

vue
<template>
  <div>
    <input v-model="textToSpeak" placeholder="输入要朗读的文本" />
    <button @click="speakText">朗读</button>
    <button @click="stopSpeech">停止</button>
  </div>
</template>

<script setup>
import { ref } from 'vue';
import { useSpeechSynthesis } from '@vueuse/core';

const textToSpeak = ref('');
const {
  isSupported,
  speak,
  stop: stopSpeech,
} = useSpeechSynthesis(textToSpeak);

const speakText = () => {
  if (isSupported.value && textToSpeak.value) {
    speak();
  } else if (!textToSpeak.value) {
    alert('请输入要朗读的文本！');
  }
};
</script>

2. 状态语音播报 ​

在特定状态变化时自动播报语音提示，例如在倒计时结束时提醒用户。

vue
<template>
  <div>
    <p>倒计时: {{ timeLeft }} 秒</p>
    <button @click="startCountdown">开始倒计时</button>
  </div>
</template>

<script setup>
import { ref, watch } from 'vue';
import { useSpeechSynthesis } from '@vueuse/core';

const timeLeft = ref(10);
const isCounting = ref(false);

const {
  isSupported,
  speak,
} = useSpeechSynthesis(ref('倒计时结束！'));

const startCountdown = () => {
  if (isCounting.value) return;
  timeLeft.value = 10;
  isCounting.value = true;
  const interval = setInterval(() => {
    timeLeft.value--;
    if (timeLeft.value <= 0) {
      clearInterval(interval);
      isCounting.value = false;
      if (isSupported.value) {
        speak();
      }
    }
  }, 1000);
};

watch(timeLeft, (newValue) => {
  if (newValue > 0 && newValue <= 3) {
    if (isSupported.value) {
      speak(ref(`${newValue} 秒`));
    }
  }
});
</script>

3. 多语言支持 ​

根据用户选择的语言切换语音合成的语言。

vue
<template>
  <div>
    <select v-model="selectedLang">
      <option value="en-US">English (US)</option>
      <option value="zh-CN">中文 (简体)</option>
      <option value="ja-JP">日本語</option>
    </select>
    <input v-model="textForLang" placeholder="输入文本" />
    <button @click="speakWithLang">朗读</button>
  </div>
</template>

<script setup>
import { ref, watch } from 'vue';
import { useSpeechSynthesis } from '@vueuse/core';

const selectedLang = ref('en-US');
const textForLang = ref('Hello World!');

const {
  isSupported,
  speak,
} = useSpeechSynthesis(textForLang, {
  lang: selectedLang,
});

const speakWithLang = () => {
  if (isSupported.value && textForLang.value) {
    speak();
  }
};

watch(selectedLang, (newLang) => {
  // 当语言变化时，可以重新设置文本或直接朗读新语言下的文本
  if (newLang === 'zh-CN') {
    textForLang.value = '你好世界！';
  } else if (newLang === 'ja-JP') {
    textForLang.value = 'こんにちは世界！';
  } else {
    textForLang.value = 'Hello World!';
  }
});
</script>
