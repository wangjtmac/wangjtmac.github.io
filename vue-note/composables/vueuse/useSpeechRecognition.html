useSpeechRecognition ​

响应式地获取语音识别（Speech Recognition）的状态和结果。

useSpeechRecognition 是 VueUse 库中的一个组合式函数，它提供了一种简单且响应式的方式来集成 Web Speech API 中的语音识别功能。这个函数允许你的 Vue 应用程序将用户的语音输入转换为文本，并在语音识别过程中提供实时状态（是否正在监听、是否是最终结果）和识别结果。它非常适用于需要语音命令、语音输入或语音转文本功能的场景。

基本用法 ​

ts
import { useSpeechRecognition } from '@vueuse/core'
import { ref, watchEffect } from 'vue'

const { 
  isSupported,  // 是否支持语音识别 API
  isListening,  // 是否正在监听语音
  isFinal,      // 当前识别结果是否为最终结果
  result,       // 识别到的文本结果
  error,        // 可能发生的错误对象
  start,        // 开始监听语音
  stop,         // 停止监听语音
  toggle        // 切换监听状态
} = useSpeechRecognition({
  lang: 'zh-CN',        // 设置语言为中文
  continuous: true,     // 持续监听
  interimResults: true  // 返回临时结果
})

// 实时打印识别结果
watchEffect(() => {
  if (result.value) {
    console.log(`识别结果: ${result.value} (最终结果: ${isFinal.value ? '是' : '否'})`)
  }
  if (error.value) {
    console.error('语音识别错误:', error.value)
  }
})

// 示例：在模板中显示状态和结果
// <template>
//   <p>支持语音识别: {{ isSupported ? '是' : '否' }}</p>
//   <p>正在监听: {{ isListening ? '是' : '否' }}</p>
//   <p>最终结果: {{ isFinal ? '是' : '否' }}</p>
//   <p>识别结果: {{ result }}</p>
//   <button @click="toggle">{{ isListening ? '停止监听' : '开始监听' }}</button>
// </template>

useSpeechRecognition 返回的 result 是一个响应式引用，会随着语音识别的进行实时更新。isListening 和 isFinal 提供了当前语音识别过程的状态。

类型定义 ​

ts
import type { MaybeRefOrGetter, ShallowRef } from 'vue'
import type { ConfigurableWindow } from '../_configurable'

export interface UseSpeechRecognitionOptions extends ConfigurableWindow {
  /**
   * 控制每次识别是否返回连续结果，或只返回单个结果。
   *
   * @default true
   */
  continuous?: boolean
  /**
   * 控制是否返回临时结果（true）或不返回（false）。临时结果是尚未最终确定的结果。
   *
   * @default true
   */
  interimResults?: boolean
  /**
   * 语音识别的语言。
   *
   * @default 'en-US'
   */
  lang?: MaybeRefOrGetter<string>
  /**
   * 一个数字，表示每个结果返回的最大替代方案数量。
   *
   * @see https://developer.mozilla.org/en-US/docs/Web/API/SpeechRecognition/maxAlternatives
   * @default 1
   */
  maxAlternatives?: number
}

export interface SpeechRecognition extends EventTarget {
  grammars: SpeechGrammarList
  lang: string
  continuous: boolean
  interimResults: boolean
  maxAlternatives: number
  serviceURI: string

  onaudiostart: ((this: SpeechRecognition, ev: Event) => any) | null
  onaudioend: ((this: SpeechRecognition, ev: Event) => any) | null
  onend: ((this: SpeechRecognition, ev: Event) => any) | null
  onerror: ((this: SpeechRecognition, ev: SpeechRecognitionErrorEvent) => any) | null
  onnomatch: ((this: SpeechRecognition, ev: SpeechRecognitionEvent) => any) | null
  onresult: ((this: SpeechRecognition, ev: SpeechRecognitionEvent) => any) | null
  onsoundstart: ((this: SpeechRecognition, ev: Event) => any) | null
  onsoundend: ((this: SpeechRecognition, ev: Event) => any) | null
  onspeechstart: ((this: SpeechRecognition, ev: Event) => any) | null
  onspeechend: ((this: SpeechRecognition, ev: Event) => any) | null

  abort(): void
  start(): void
  stop(): void
}

export interface SpeechRecognitionEvent extends Event {
  readonly resultIndex: number
  readonly results: SpeechRecognitionResultList
}

export interface SpeechRecognitionErrorEvent extends Event {
  readonly error: SpeechRecognitionErrorCode
  readonly message: string
}

declare function useSpeechRecognition(
  options?: UseSpeechRecognitionOptions,
): {
  isSupported: ShallowRef<boolean>
  isListening: ShallowRef<boolean>
  isFinal: ShallowRef<boolean>
  recognition: SpeechRecognition | undefined
  result: ShallowRef<string>
  error: ShallowRef<SpeechRecognitionErrorEvent | undefined>
  toggle: (value?: boolean) => void
  start: () => void
  stop: () => void
}

参数 ​

useSpeechRecognition 函数接受一个可选的 options 对象作为参数，用于配置语音识别行为：

  1. options?: UseSpeechRecognitionOptions：可选的配置对象，可以包含以下属性：
    • continuous?: boolean：可选，默认为 true。控制是否返回连续的识别结果。如果设置为 true，语音识别器会在用户持续说话时不断返回结果；如果设置为 false，则在每次用户停顿后返回单个结果。
    • interimResults?: boolean：可选，默认为 true。控制是否返回临时（非最终）的识别结果。临时结果会在识别过程中实时更新，最终结果会在用户停止说话或识别器确定后返回。
    • lang?: MaybeRefOrGetter<string>：可选，默认为 'en-US'。设置语音识别的语言。例如 'zh-CN' 表示中文，'en-US' 表示美式英语。可以是一个响应式引用。
    • maxAlternatives?: number：可选，默认为 1。每个识别结果返回的最大替代方案数量。例如，如果设置为 5，则对于同一段语音，可能会返回 5 个不同的识别结果（按置信度排序）。
    • window?: Window (来自 ConfigurableWindow)：可选。允许你指定一个自定义的 Window 对象，通常用于测试环境或 SSR（服务端渲染）环境。

返回值 ​

useSpeechRecognition 函数返回一个包含以下属性的响应式对象：

  • isSupported: ShallowRef<boolean>：一个浅层响应式引用，指示当前浏览器是否支持 Web Speech Recognition API。如果为 false，则其他功能将不可用。
  • isListening: ShallowRef<boolean>：一个浅层响应式引用，指示语音识别器当前是否正在监听用户的语音。
  • isFinal: ShallowRef<boolean>：一个浅层响应式引用，指示当前的 result 是否为最终的识别结果。在 continuous 模式下，通常只有在用户停顿或识别会话结束时才会变为 true。
  • recognition: SpeechRecognition | undefined：底层的 SpeechRecognition 实例。如果浏览器不支持该 API，则为 undefined。
  • result: ShallowRef<string>：一个浅层响应式引用，包含当前识别到的文本结果。在 interimResults 为 true 时，它会实时更新。
  • error: ShallowRef<SpeechRecognitionErrorEvent | undefined>：一个浅层响应式引用，如果语音识别过程中发生错误，将包含 SpeechRecognitionErrorEvent 对象，否则为 undefined。
  • toggle: (value?: boolean) => void：一个函数，用于切换语音识别的监听状态。可以不传参数（则切换当前状态），也可以传入 true (开始监听) 或 false (停止监听)。
  • start: () => void：一个函数，用于开始语音识别监听。
  • stop: () => void：一个函数，用于停止语音识别监听。

工作原理 ​

useSpeechRecognition 的核心工作原理是封装了浏览器原生的 Web Speech API 中的 SpeechRecognition 接口：

  1. API 支持检测：在初始化时，它会通过 useSupported 检查当前 window 对象是否提供了 SpeechRecognition 或 webkitSpeechRecognition 构造函数，以确定浏览器是否支持此功能。
  2. 实例化 SpeechRecognition：如果支持，它会创建一个 SpeechRecognition 的实例。
  3. 配置选项：将 continuous、interimResults、lang 和 maxAlternatives 等选项直接应用于 recognition 实例的相应属性。
  4. 事件监听：通过为 recognition 实例绑定事件监听器（如 onstart, onresult, onerror, onend），来捕获语音识别过程中的各种事件：
    • onstart: 当语音识别服务开始监听时触发，更新 isListening 为 true。
    • onresult: 当识别到语音结果时触发。它会从事件对象中提取最新的 transcript（识别文本），并更新 result 和 isFinal 状态。
    • onerror: 当语音识别发生错误时触发，更新 error 状态。
    • onend: 当语音识别服务结束时触发（无论是通过 stop() 调用还是自动停止），更新 isListening 为 false。
  5. 响应式状态：内部使用 shallowRef 来创建 isListening、isFinal、result 和 error 等响应式引用，确保这些状态的变化能够触发 Vue 组件的重新渲染。
  6. 生命周期管理：使用 tryOnScopeDispose 确保在组件卸载时调用 stop() 函数，从而正确地停止语音识别服务，防止内存泄漏和不必要的资源占用。
  7. 语言动态切换：lang 参数是一个 MaybeRefOrGetter 类型，允许在运行时动态改变识别语言。通过 watch 监听 lang 的变化，并在非监听状态下更新 recognition.lang。

使用场景 ​

  • 语音助手/聊天机器人：作为语音输入模块，将用户的口语转换为文本，然后进行处理。
  • 无障碍应用：为残障人士提供语音输入功能，提升可访问性。
  • 语音笔记/速记工具：快速记录语音内容为文本。
  • 教育应用：语音练习、发音检测等。
  • 智能家居控制：通过语音命令控制智能设备。
  • 游戏：通过语音命令进行游戏操作。
  • 表单语音输入：替代键盘输入，简化表单填写。

注意事项 ​

  • 浏览器兼容性：Web Speech API（尤其是 SpeechRecognition）在不同浏览器上的支持程度和实现细节可能有所差异。在使用前，务必检查 Can I use? 以确认目标浏览器的支持情况。在某些浏览器中可能需要用户授权麦克风权限。
  • 网络连接：部分语音识别服务可能依赖于网络连接（特别是当使用云端识别引擎时）。确保用户设备有稳定的网络连接。
  • 用户隐私：涉及到麦克风输入时，务必告知用户数据如何被使用和存储，遵守相关隐私政策和法规。
  • 错误处理：由于网络、权限或设备问题，语音识别可能会失败。应始终实现健全的错误处理机制（通过 error ref），并向用户提供清晰的反馈。
  • 离线识别：浏览器原生的 SpeechRecognition API 通常需要在线服务。如果需要离线语音识别，可能需要集成第三方库或自行部署模型。
  • 语言设置：正确设置 lang 参数对于获得准确的识别结果至关重要。
  • 性能考量：长时间持续监听语音可能会消耗较多设备资源，根据实际需求合理控制监听的开始和停止。

实际应用示例 ​

以下是一个简单的 Vue 组件，演示如何使用 useSpeechRecognition 来实现一个基本的语音转文本输入框，并显示当前的识别状态和结果。

vue
<template>
  <div class="demo-container">
    <h2>useSpeechRecognition 演示</h2>

    <p class="note">
      点击"开始监听"按钮，然后对着麦克风说话。你说的内容会实时显示在文本框中。
    </p>

    <div v-if="!isSupported" class="unsupported-warning">
      您的浏览器不支持 Web Speech Recognition API。请尝试使用 Chrome 浏览器。
    </div>

    <div v-else class="speech-controls">
      <button @click="toggle" :disabled="!isSupported">
        {{ isListening ? '停止监听' : '开始监听' }}
      </button>
      <button @click="result = ''" :disabled="!result" class="clear-button">
        清空结果
      </button>
    </div>

    <div class="status-info">
      <p><b>支持 API:</b> {{ isSupported ? '是 ✅' : '否 ❌' }}</p>
      <p><b>正在监听:</b> <span :class="{ active: isListening }">{{ isListening ? '是...' : '否' }}</span></p>
      <p><b>最终结果:</b> <span :class="{ final: isFinal }">{{ isFinal ? '是' : '否' }}</span></p>
      <p v-if="error" class="error-message"><b>错误:</b> {{ error.message }}</p>
    </div>

    <div class="result-box">
      <textarea 
        v-model="result"
        rows="8"
        placeholder="你说的话将显示在这里..."
        readonly
      ></textarea>
    </div>
