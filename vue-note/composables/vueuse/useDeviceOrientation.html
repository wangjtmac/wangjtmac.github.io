useDeviceOrientation ​

响应式设备方向传感器 API

useDeviceOrientation 是 VueUse 库中的一个传感器类组合式函数，用于获取设备的物理方向信息，包括设备围绕每个轴的旋转角度。这个函数封装了浏览器的 DeviceOrientationEvent API，提供了响应式的设备方向监控。

基本用法 ​

ts
import { useDeviceOrientation } from '@vueuse/core'

const {
  isAbsolute,  // 一个布尔值，表示设备是否提供绝对方向数据
  alpha,       // 设备围绕 z 轴（屏幕法线）的旋转角度
  beta,        // 设备围绕 x 轴（从左到右）的旋转角度
  gamma,       // 设备围绕 y 轴（从前到后）的旋转角度
  isSupported, // 是否支持 DeviceOrientationEvent API
} = useDeviceOrientation()

类型定义 ​

ts
export interface DeviceOrientationOptions extends ConfigurableWindow {}

/**
 * 响应式 DeviceOrientationEvent
 */
export declare function useDeviceOrientation(options?: ConfigurableWindow): UseDeviceOrientationReturn

export interface UseDeviceOrientationReturn {
  isSupported: Ref<boolean>
  isAbsolute: Ref<boolean>
  alpha: Ref<number | null>
  beta: Ref<number | null>
  gamma: Ref<number | null>
}

参数 ​

参数类型默认值描述
optionsConfigurableWindow{}配置选项，例如指定要使用的 window 对象

配置选项 ​

ConfigurableWindow 接口包含以下选项：

选项类型默认值描述
windowWindowdefaultWindow要使用的窗口对象

返回值 ​

useDeviceOrientation 函数返回一个包含以下属性的对象：

属性类型描述
isSupportedRef<boolean>当前环境是否支持 DeviceOrientationEvent API
isAbsoluteRef<boolean>一个布尔值，表示设备是否提供绝对方向数据。true 表示设备提供了从地球坐标系算起的方向（例如，设备朝向北方），false 则表示方向是相对于设备启动时的初始方向。
alphaRef<number | null>设备围绕 z 轴（屏幕法线）的旋转角度，范围 0 到 360 度。通常表示罗盘方向，0 度代表正北。
betaRef<number | null>设备围绕 x 轴（从左到右穿过屏幕）的旋转角度，范围 -180 到 180 度。表示设备的前后倾斜。
gammaRef<number | null>设备围绕 y 轴（从前到后穿过屏幕）的旋转角度，范围 -90 到 90 度。表示设备的左右倾斜。

工作原理 ​

useDeviceOrientation 使用浏览器的 DeviceOrientationEvent API 来获取设备的物理方向信息。它监听 deviceorientation 事件，并将事件数据（absolute、alpha、beta、gamma）转换为响应式引用。这些值反映了设备相对于地球坐标系或设备初始方向的姿态。

源码实现 ​

typescript
import type { Ref } from 'vue'
import type { ConfigurableWindow } from '../_configurable'
import { shallowRef } from 'vue'
import { defaultWindow } from '../_configurable'
import { useEventListener } from '../useEventListener'
import { useSupported } from '../useSupported'

export function useDeviceOrientation(options: ConfigurableWindow = {}) {
  const { window = defaultWindow } = options
  const isSupported = useSupported(() => window && 'DeviceOrientationEvent' in window)

  const isAbsolute = shallowRef(false)
  const alpha: Ref<number | null> = shallowRef(null)
  const beta: Ref<number | null> = shallowRef(null)
  const gamma: Ref<number | null> = shallowRef(null)

  if (window && isSupported.value) {
    useEventListener(window, 'deviceorientation', (event) => {
      isAbsolute.value = event.absolute
      alpha.value = event.alpha
      beta.value = event.beta
      gamma.value = event.gamma
    }, { passive: true })
  }

  return {
    isSupported,
    isAbsolute,
    alpha,
    beta,
    gamma,
  }
}

export type UseDeviceOrientationReturn = ReturnType<typeof useDeviceOrientation>

使用场景 ​

useDeviceOrientation 主要用于以下场景：

  1. 3D 渲染和游戏：根据设备方向调整 3D 场景的视角或控制游戏角色。
  2. 地图和导航应用：指示用户在真实世界中的方向，例如罗盘功能。
  3. 增强现实 (AR)：将虚拟对象叠加到现实世界中时，需要根据设备方向精确对齐。
  4. 图像查看器：根据设备旋转自动调整图像方向。
  5. 辅助功能：为某些特定需求的用户提供基于设备方向的交互。
  6. 教学和实验：在科学或物理教学中模拟和展示物体的姿态变化。

注意事项 ​

  1. 浏览器兼容性：DeviceOrientationEvent API 在不同浏览器和设备上的支持程度不同。在使用前应检查 isSupported 属性。
  2. 数据精度和稳定性：传感器数据可能受到硬件限制和环境干扰（如磁场）的影响，导致精度不高或数据抖动。在需要高精度或平滑数据的应用中，可能需要进行数据滤波或融合。
  3. 绝对方向与相对方向：isAbsolute 属性指示方向数据是绝对的（相对于地球坐标系）还是相对的（相对于设备启动时的初始方向）。在某些设备或浏览器上，可能只提供相对方向数据。
  4. 轴定义：理解 alpha、beta、gamma 分别代表的轴向（z 轴、x 轴、y 轴）以及其正向旋转方向非常重要。
  5. 性能考虑：持续监听设备方向事件可能会消耗电池电量，应在不需要时停止监听或降低更新频率。

与其他函数的对比 ​

函数描述区别
useDeviceOrientation监控设备的方向（姿态）关注设备在空间中的姿态，即设备如何倾斜和旋转。
useDeviceMotion监控设备的运动（加速度和旋转速率）关注设备的运动状态，即设备正在如何移动和加速。它提供的是瞬时变化量，而不是设备的绝对或相对姿态。

示例2：3D 立方体旋转 ​

vue
<template>
  <div class="cube-container">
    <h2>3D 立方体旋转</h2>
    <div v-if="isSupported" class="cube-content">
      <div class="cube-wrapper">
        <div class="cube" :style="cubeStyle">
          <div class="face front">前</div>
          <div class="face back">后</div>
          <div class="face right">右</div>
          <div class="face left">左</div>
          <div class="face top">上</div>
          <div class="face bottom">下</div>
        </div>
      </div>
      <p>拖动或倾斜设备来旋转立方体</p>
      <div class="orientation-values">
        <p>Alpha: {{ formatValue(alpha) }}°</p>
        <p>Beta: {{ formatValue(beta) }}°</p>
        <p>Gamma: {{ formatValue(gamma) }}°</p>
      </div>
    </div>
    <div v-else class="not-supported">
      <p>您的设备或浏览器不支持设备方向传感器 API</p>
    </div>
  </div>
</template>

<script setup>
import { computed } from 'vue'
import { useDeviceOrientation } from '@vueuse/core'

const { alpha, beta, gamma, isSupported } = useDeviceOrientation()

const cubeStyle = computed(() => {
  // 为了更直观地演示，可能需要对传感器值进行调整
  // beta 影响围绕 X 轴的旋转（上下倾斜）
  // gamma 影响围绕 Y 轴的旋转（左右倾斜）
  // alpha 影响围绕 Z 轴的旋转（平面旋转，罗盘方向）
  const xRotate = beta.value !== null ? beta.value : 0
  const yRotate = gamma.value !== null ? gamma.value : 0
  const zRotate = alpha.value !== null ? -alpha.value : 0 // 反向旋转以匹配常见预期

  return {
    transform: `rotateX(${-xRotate}deg) rotateY(${yRotate}deg) rotateZ(${zRotate}deg)`,
  }
})

function formatValue(value) {
  if (value === null || value === undefined)
    return 'N/A'
  return value.toFixed(2)
}
</script>

示例3：组件用法 ​

VueUse 为 useDeviceOrientation 提供了一个组件版本，可以直接在模板中使用：

vue
<template>
  <div>
    <h2>设备方向组件</h2>
    
    <UseDeviceOrientation v-slot="{ alpha, beta, gamma, isSupported }">
      <div v-if="isSupported">
        <p>
          Alpha: {{ formatNumber(alpha) }}°,
          Beta: {{ formatNumber(beta) }}°,
          Gamma: {{ formatNumber(gamma) }}°
        </p>
      </div>
      <div v-else>
        您的设备不支持设备方向传感器
      </div>
    </UseDeviceOrientation>
  </div>
</template>

<script setup>
import { UseDeviceOrientation } from '@vueuse/core'

function formatNumber(value) {
  return value !== null ? value.toFixed(2) : 'N/A'
}
</script>
