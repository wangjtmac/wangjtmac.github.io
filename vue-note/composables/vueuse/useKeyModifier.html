useKeyModifier ​

响应式地跟踪修饰键（Modifier Key）的状态。

useKeyModifier 是 VueUse 库中的一个组合式函数，它提供了一种简单且响应式的方式来获取键盘修饰键（如 Alt、Control、Shift、CapsLock 等）的当前按下状态。它利用 KeyboardEvent.getModifierState() API 来检测修饰键的状态，并能监听鼠标和键盘事件以实时更新状态，从而帮助你构建依赖于修饰键组合的交互逻辑。

基本用法 ​

ts
import { useKeyModifier } from '@vueuse/core'
import { watchEffect } from 'vue'

const isShiftPressed = useKeyModifier('Shift')
const isCtrlPressed = useKeyModifier('Control')
const isCapsLockOn = useKeyModifier('CapsLock')

watchEffect(() => {
  console.log(`Shift键是否按下: ${isShiftPressed.value}`)
  console.log(`Control键是否按下: ${isCtrlPressed.value}`)
  console.log(`大小写锁定键是否开启: ${isCapsLockOn.value}`)
})

// 示例：在模板中根据修饰键状态显示不同内容
// <template>
//   <p v-if="isShiftPressed">Shift 键已按下！</p>
//   <p>请按住 Alt 键查看提示。</p>
//   <p v-if="isAltPressed">你按住了 Alt 键！</p>
// </template>

useKeyModifier 返回的 ref 会在相应的修饰键状态变化时自动更新。默认情况下，它会监听鼠标和键盘的 down/up 事件。

类型定义 ​

ts
import type { ShallowRef } from 'vue'
import type { ConfigurableDocument } from '../_configurable'
import type { WindowEventName } from '../useEventListener'

export type KeyModifier = 'Alt' | 'AltGraph' | 'CapsLock' | 'Control' | 'Fn' | 'FnLock' | 'Meta' | 'NumLock' | 'ScrollLock' | 'Shift' | 'Symbol' | 'SymbolLock'

export interface UseModifierOptions<Initial> extends ConfigurableDocument {
  /**
   * 将提示更新修饰符状态的事件名称。
   *
   * @default ['mousedown', 'mouseup', 'keydown', 'keyup']
   */
  events?: WindowEventName[]

  /**
   * 返回的 ref 的初始值。
   *
   * @default null
   */
  initial?: Initial
}

export type UseKeyModifierReturn<Initial> = ShallowRef<Initial extends boolean ? boolean : boolean | null>

declare function useKeyModifier<Initial extends boolean | null>(modifier: KeyModifier, options?: UseModifierOptions<Initial>): UseKeyModifierReturn<Initial>

参数 ​

useKeyModifier 函数接受以下两个参数：

  1. modifier: KeyModifier：

    • 类型：KeyModifier (一个字符串字面量联合类型，包括 'Alt', 'AltGraph', 'CapsLock', 'Control', 'Fn', 'FnLock', 'Meta', 'NumLock', 'ScrollLock', 'Shift', 'Symbol', 'SymbolLock')
    • 描述：你想要跟踪状态的修饰键名称。这些名称与 KeyboardEvent.key 属性的值相对应。

  2. options?: UseModifierOptions<Initial>：可选的配置对象，可以包含以下属性：

    • events?: WindowEventName[]：可选，默认为 ['mousedown', 'mouseup', 'keydown', 'keyup']。一个事件名称数组，这些事件的发生会触发修饰键状态的更新。你可以自定义这些事件以优化性能或满足特定需求。
    • initial?: Initial：可选，默认为 null。返回的 ref 的初始值。如果设置为 boolean 类型（如 false），则返回的 ref 也将是 Ref<boolean> 类型，而不会是 Ref<boolean | null>。
    • document?: Document (来自 ConfigurableDocument)：可选。允许你指定一个自定义的 Document 对象，通常用于测试环境或非浏览器环境。

返回值 ​

useKeyModifier 函数返回一个 ShallowRef<boolean | null> 或 ShallowRef<boolean> 类型的响应式引用：

  • ShallowRef<boolean | null>：如果 initial 选项未设置或设置为 null，返回的 ref 初始值为 null，并在收到第一个相关事件后更新为 boolean 类型。true 表示修饰键当前被按下或开启（如 CapsLock），false 表示未按下或关闭。
  • ShallowRef<boolean>：如果 initial 选项设置为 boolean 类型（例如 false），返回的 ref 将始终为 boolean 类型。

工作原理 ​

useKeyModifier 的实现利用了 KeyboardEvent.getModifierState() 方法和事件监听器：

  1. 事件监听：它通过 useEventListener 在 document 对象上监听一系列指定的事件（默认为 mousedown, mouseup, keydown, keyup）。
  2. getModifierState()：当任何一个监听的事件触发时，回调函数会接收到一个 KeyboardEvent 或 MouseEvent 对象。它会检查事件对象是否具有 getModifierState 方法（现代浏览器普遍支持）。
  3. 状态提取与更新：如果 getModifierState 方法可用，它会调用 evt.getModifierState(modifier) 来获取指定修饰键的当前状态（true 或 false），并将结果赋值给内部的 shallowRef，从而实现响应式更新。
  4. shallowRef：使用 shallowRef 而不是 ref 可以稍微优化性能，因为修饰键的状态通常是一个简单的布尔值，不需要深度响应式。

使用场景 ​

  • 键盘快捷键：实现需要同时按下修饰键和其他键的快捷键功能（例如 Ctrl + S 保存）。
  • 交互式设计：当用户按住特定修饰键时，改变 UI 元素的行为或显示额外信息（例如按住 Alt 键拖动复制元素）。
  • 文本编辑器：检测 CapsLock 状态，以在用户输入密码时提供提示。
  • 游戏控制：在游戏中使用修饰键进行特殊操作或组合技能。
  • 辅助功能：根据修饰键状态调整界面的可访问性。

注意事项 ​

  • 浏览器兼容性：KeyboardEvent.getModifierState() 方法在现代浏览器中得到了广泛支持。对于非常旧的浏览器，可能需要考虑 Polyfill 或提供回退方案。
  • 事件类型：getModifierState() 可以用于 KeyboardEvent 和 MouseEvent。这意味着即使是鼠标点击事件，也可以获取到键盘修饰键的状态。
  • 持久性：getModifierState() 返回的是当前修饰键的实时状态，而不是修饰键是否在事件发生时被按下。例如，如果 Shift 键在 keydown 事件发生后被释放，那么在随后的 keyup 事件中，getModifierState('Shift') 将返回 false。
  • 性能：默认监听四个事件通常不会引起明显的性能问题，但如果你有非常特殊的性能要求，可以考虑限制 events 选项。

实际应用示例 ​

以下是一个简单的 Vue 组件，演示如何使用 useKeyModifier 来实时显示 Shift、Control 和 CapsLock 键的状态，并根据状态改变文本提示：

vue
<template>
  <div class="demo-container">
    <h2>useKeyModifier 演示</h2>

    <p class="note">
      尝试按下或松开 <b>Shift</b>、<b>Control</b> 键，以及切换 <b>CapsLock</b> 键，观察状态变化。
    </p>

    <div class="modifier-status-box">
      <div :class="['status-item', isShiftPressed ? 'active' : '']">
        <span class="key-name">Shift</span>
        <span class="key-state">{{ isShiftPressed ? '按下' : '未按下' }}</span>
      </div>
      <div :class="['status-item', isCtrlPressed ? 'active' : '']">
        <span class="key-name">Control</span>
        <span class="key-state">{{ isCtrlPressed ? '按下' : '未按下' }}</span>
      </div>
      <div :class="['status-item', isAltPressed ? 'active' : '']">
        <span class="key-name">Alt</span>
        <span class="key-state">{{ isAltPressed ? '按下' : '未按下' }}</span>
      </div>
      <div :class="['status-item', isCapsLockOn ? 'active' : '']">
        <span class="key-name">CapsLock</span>
        <span class="key-state">{{ isCapsLockOn ? '开启' : '关闭' }}</span>
      </div>
    </div>

    <div class="code-section">
      <h3>代码示例</h3>
      <pre><code>
import { useKeyModifier } from '@vueuse/core'

const isShiftPressed = useKeyModifier('Shift')
const isCtrlPressed = useKeyModifier('Control')

// 你可以根据这些响应式状态来控制其他逻辑，例如：
// if (isCtrlPressed.value && isShiftPressed.value) {
//   // 执行 Ctrl+Shift 组合键操作
// }
      </code></pre>
    </div>
  </div>
</template>
