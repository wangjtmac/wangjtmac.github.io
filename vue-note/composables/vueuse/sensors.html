VueUse - 传感器（Sensors） ​

传感器（Sensors）分类下的函数主要用于监听用户交互和设备传感器，帮助开发者更方便地处理用户输入和设备状态。这些函数使得与用户交互和设备状态相关的功能实现变得简单而直观。

函数列表 ​

鼠标/触摸相关 ​

  • onClickOutside - 监听元素外部点击
  • onKeyStroke - 监听键盘按键
  • onLongPress - 监听长按事件
  • onStartTyping - 监听用户开始输入
  • useMouse - 跟踪鼠标位置
  • useMouseInElement - 跟踪鼠标在元素内的位置
  • useMousePressed - 跟踪鼠标按下状态
  • useScroll - 跟踪滚动位置
  • useScrollLock - 锁定页面滚动

设备传感器 ​

  • useBattery - 获取设备电池状态
  • useDeviceMotion - 获取设备运动信息
  • useDeviceOrientation - 获取设备方向
  • useDevicePixelRatio - 获取设备像素比
  • useDevicesList - 获取可用媒体设备列表
  • useGeolocation - 获取地理位置
  • usePermission - 获取权限状态

其他传感器 ​

  • useIdle - 检测用户空闲状态
  • useInfiniteScroll - 无限滚动
  • useKeyModifier - 检测修饰键状态
  • useParallax - 视差效果
  • usePointer - 统一的指针事件（鼠标和触摸）
  • usePreferredDark - 检测用户是否偏好深色模式
  • usePreferredLanguages - 获取用户偏好语言

详细使用示例 ​

1. onLongPress ​

onLongPress 函数用于监听元素的长按事件，可以用来区分普通点击和长按操作。

基本用法 ​

vue
<template>
  <div>
    <button ref="target">
      {{ isLongPressed ? '已长按' : '长按我' }}
    </button>
  </div>
</template>

<script setup>
import { ref } from 'vue';
import { onLongPress } from '@vueuse/core';

const target = ref(null);
const isLongPressed = ref(false);

onLongPress(target, () => {
  isLongPressed.value = true;
  setTimeout(() => {
    isLongPressed.value = false;
  }, 1000);
});
</script>

自定义长按时间 ​

默认情况下，长按需要按住 500 毫秒才会触发，你可以通过 delay 选项自定义这个时间：

js
// 设置为 1000ms (1秒) 长按时间
onLongPress(target, () => {
  console.log('长按 1 秒触发');
}, { delay: 1000 });

监听长按的不同阶段 ​

onLongPress 支持监听长按的开始、触发和结束三个阶段：

js
const pressState = ref('idle');

onLongPress(target, {
  onStart: () => {
    pressState.value = 'pressing';
    console.log('长按开始');
  },
  onPress: () => {
    pressState.value = 'pressed';
    console.log('长按触发');
  },
  onEnd: () => {
    console.log('长按结束');
    setTimeout(() => {
      pressState.value = 'idle';
    }, 300);
  }
});

源码实现解析 ​

onLongPress 的核心实现原理是通过计时器来判断用户是否长时间按压元素：

  1. 监听元素的 mousedown/touchstart 事件，开始计时
  2. 如果在计时结束前触发了 mouseup/touchend 事件，则取消计时器
  3. 如果计时器正常结束，则触发长按回调函数
  4. 同时监听 mouseleave/touchcancel 事件，确保用户移出元素时取消长按

实际应用场景 ​

  • 移动应用中的删除确认（长按删除）
  • 游戏中的蓄力操作
  • 编辑器中的特殊功能触发
  • 上下文菜单的替代方案
  • 长按拖动排序
  • 长按复制文本
  • 长按放大查看图片

配置选项 ​

ts
interface OnLongPressOptions {
  // 触发长按事件所需的时间（毫秒）
  delay?: number; // 默认为 500
  // 是否阻止默认行为
  preventDefault?: boolean; // 默认为 true
  // 修饰符选项
  modifiers?: {
    prevent?: boolean;
    stop?: boolean;
  };
}

返回值 ​

onLongPress 函数返回一个清理函数，调用该函数可以移除事件监听器：

js
const stop = onLongPress(target, () => {
  console.log('长按触发');
});

// 当不再需要监听长按事件时
stop();

2. useMouse ​

useMouse 函数用于跟踪鼠标在页面或特定元素内的位置。

基本用法 ​

vue
<template>
  <div>
    <p>鼠标坐标 (页面): x: {{ x }}, y: {{ y }}</p>
    <div
      ref="target"
      style="width: 300px; height: 200px; background-color: #f0f0f0;"
    >
      <p>鼠标坐标 (元素内): x: {{ elementX }}, y: {{ elementY }}</p>
    </div>
  </div>
</template>

<script setup>
import { ref } from "vue";
import { useMouse } from "@vueuse/core";

const target = ref(null);
// 页面鼠标位置
const { x, y } = useMouse();
// 元素内鼠标位置
const { x: elementX, y: elementY } = useMouse({ target });
</script>

源码实现解析 ​

通过研究 VueUse 源码，我们可以深入了解 useMouse 的实现原理：

  1. 参数配置：

    typescript
    export interface UseMouseOptions
  extends ConfigurableWindow,
    ConfigurableEventFilter {
  type?: UseMouseCoordType | UseMouseEventExtractor; // 坐标类型或自定义提取器
  target?: MaybeRefOrGetter<Window | EventTarget | null | undefined>; // 监听事件的目标元素
  touch?: boolean; // 是否监听触摸事件，默认为true
  scroll?: boolean; // 是否监听滚动事件，默认为true
  resetOnTouchEnds?: boolean; // 触摸结束时是否重置坐标，默认为false
  initialValue?: Position; // 初始值
}

  2. 坐标类型：

    • page：相对于整个文档的坐标（默认）
    • client：相对于浏览器视窗的坐标
    • screen：相对于屏幕的坐标
    • movement：相对于上一次位置的移动量

  3. 事件处理：

    • 使用 useEventListener 监听 mousemove 和 dragover 事件
    • 如果启用触摸，还会监听 touchstart 和 touchmove 事件
    • 如果启用滚动且类型为 page，还会监听 scroll 事件

  4. 坐标计算：

    typescript
    const UseMouseBuiltinExtractors: Record<
  UseMouseCoordType,
  UseMouseEventExtractor
> = {
  page: (event) => [event.pageX, event.pageY],
  client: (event) => [event.clientX, event.clientY],
  screen: (event) => [event.screenX, event.screenY],
  movement: (event) =>
    event instanceof MouseEvent ? [event.movementX, event.movementY] : null,
};

  5. 滚动处理： 当页面滚动时，useMouse 会调整鼠标位置，确保 page 类型的坐标始终正确：

    typescript
    const scrollHandler = () => {
  if (!_prevMouseEvent || !window) return;
  const pos = extractor(_prevMouseEvent);

  if (_prevMouseEvent instanceof MouseEvent && pos) {
    x.value = pos[0] + window.scrollX - _prevScrollX;
    y.value = pos[1] + window.scrollY - _prevScrollY;
  }
};

  6. 自定义提取器： 可以提供自定义函数来从事件中提取坐标，例如：

    typescript
    const extractor: UseMouseEventExtractor = (event) =>
  event instanceof MouseEvent ? [event.offsetX, event.offsetY] : null;

  7. 返回值：

    typescript
    return {
  x, // x坐标，响应式
  y, // y坐标，响应式
  sourceType, // 事件来源类型（'mouse'、'touch'或null）
};

这种实现方式使得 useMouse 非常灵活，可以适应各种不同的使用场景。通过配置不同的参数，我们可以获取不同类型的鼠标坐标，满足各种交互需求。

组件用法 ​

VueUse 官方提供了组件版本的 API，但这些组件不在核心包 @vueuse/core 中，而是在一个单独的包 @vueuse/components 中。要使用这些组件，需要先安装该包：

bash
npm i @vueuse/core @vueuse/components

然后就可以导入并使用组件版本的 API，例如 UseMouse：

vue
<template>
  <div>
    <UseMouse v-slot="{ x, y, sourceType }">
      <div>
        <p>鼠标坐标: x: {{ x }}, y: {{ y }}</p>
        <p>鼠标类型: {{ sourceType }}</p>
      </div>
    </UseMouse>
  </div>
</template>

<script setup>
import { UseMouse } from "@vueuse/components";
</script>

UseMouse 组件的源码实现非常简洁：

typescript
export const UseMouse = /* #__PURE__ */ defineComponent<UseMouseOptions>({
  name: "UseMouse",
  props: ["touch", "resetOnTouchEnds", "initialValue"] as unknown as undefined,
  setup(props, { slots }) {
    const data = reactive(useMouse(props));

    return () => {
      if (slots.default) return slots.default(data);
    };
  },
});

它只是对 useMouse 函数的一个简单封装，通过 Vue 的作用域插槽将鼠标数据传递给子组件。

如果没有安装 @vueuse/components 包，我们也可以很容易地创建自己的组件来实现类似的功能：

vue
<template>
  <div>
    <MouseTracker v-slot="{ x, y, sourceType }">
      <div>
        <p>鼠标坐标: x: {{ x }}, y: {{ y }}</p>
        <p>鼠标类型: {{ sourceType }}</p>
      </div>
    </MouseTracker>
  </div>
</template>

<script setup>
import MouseTracker from "./MouseTracker.vue";
</script>

自定义的MouseTracker组件实现如下：

vue
<template>
  <div>
    <slot :x="x" :y="y" :sourceType="sourceType" />
  </div>
</template>

<script setup>
import { useMouse } from "@vueuse/core";

/**
 * MouseTracker组件
 *
 * 使用VueUse的useMouse函数跟踪鼠标位置，
 * 并通过作用域插槽向子组件提供鼠标数据
 */
const { x, y, sourceType } = useMouse();
</script>

组件用法的优点是不需要在每个使用鼠标位置的组件中重复调用useMouse函数，而是通过 Vue 的作用域插槽（v-slot）直接获取鼠标位置数据。这种方式在某些场景下可以让代码更简洁，特别是当你只需要在特定组件内使用鼠标位置数据时。

返回值 ​

  • x: 鼠标在页面上的 X 坐标
  • y: 鼠标在页面上的 Y 坐标
  • sourceType: 事件来源类型（mouse、touch 等）

当提供 target 参数时，返回的坐标是相对于目标元素的位置。

应用场景 ​

  • 自定义光标跟踪
  • 元素跟随鼠标移动
  • 鼠标位置相关的交互效果
  • 图像放大镜效果

3. onClickOutside ​

onClickOutside 函数用于监听元素外部的点击事件。

基本用法 ​

vue
<template>
  <div>
    <div
      ref="target"
      :class="{ active: isActive }"
      @click="isActive = true"
      style="padding: 20px; border: 1px solid #ccc;"
    >
      <p v-if="isActive">点击此区域外部将重置状态</p>
      <p v-else>点击此区域激活</p>
    </div>
    <p>状态: {{ isActive ? "激活" : "未激活" }}</p>
    <p>点击外部次数: {{ outsideClickCount }}</p>
  </div>
</template>

<script setup>
import { ref } from "vue";
import { onClickOutside } from "@vueuse/core";

const target = ref(null);
const isActive = ref(false);
const outsideClickCount = ref(0);

onClickOutside(target, () => {
  if (isActive.value) {
    isActive.value = false;
    outsideClickCount.value++;
  }
});
</script>

<style scoped>
.active {
  background-color: #42b983;
  color: white;
}
</style>

参数 ​

  • target: 目标元素的引用
  • handler: 当点击发生在目标元素外部时调用的回调函数
  • options: 可选配置项（如事件类型、是否忽略右键点击等）

应用场景 ​

  • 下拉菜单自动关闭
  • 模态框点击外部关闭
  • 弹出提示自动隐藏
  • 自定义选择器点击外部取消选择

4. useScroll ​

useScroll 函数用于跟踪元素的滚动位置。

基本用法 ​

vue
<template>
  <div>
    <div class="scroll-info">
      <p>滚动位置: x: {{ x }}, y: {{ y }}</p>
      <p>是否到达底部: {{ isScrollBottom ? "是" : "否" }}</p>
      <p>滚动进度: {{ scrollProgress.toFixed(2) }}%</p>
    </div>
    <div class="progress-bar">
      <div class="progress" :style="{ width: `${scrollProgress}%` }"></div>
    </div>
    <div
      ref="scrollContainer"
      style="height: 300px; overflow-y: auto; border: 1px solid #ddd;"
    >
      <div
        v-for="i in 20"
        :key="i"
        style="padding: 20px; border-bottom: 1px solid #eee;"
      >
        滚动项 #{{ i }}
      </div>
    </div>
  </div>
</template>

<script setup>
import { ref, computed } from "vue";
import { useScroll } from "@vueuse/core";

const scrollContainer = ref(null);
const { x, y, isScrolling, arrivedState } = useScroll(scrollContainer);

const isScrollBottom = computed(() => arrivedState.bottom);
const scrollProgress = computed(() => {
  if (!scrollContainer.value) return 0;

  const { scrollTop, scrollHeight, clientHeight } = scrollContainer.value;
  const maxScroll = scrollHeight - clientHeight;

  if (maxScroll <= 0) return 100;
  return (scrollTop / maxScroll) * 100;
});
</script>

<style scoped>
.scroll-info {
  margin-bottom: 10px;
  padding: 10px;
  background-color: #f5f5f5;
  border-radius: 4px;
}
.progress-bar {
  height: 10px;
  background-color: #eee;
  border-radius: 5px;
  overflow: hidden;
  margin-bottom: 10px;
}
.progress {
  height: 100%;
  background-color: #42b983;
  transition: width 0.2s ease;
}
</style>

返回值 ​

  • x: 水平滚动位置
  • y: 垂直滚动位置
  • isScrolling: 是否正在滚动
  • arrivedState: 滚动到达状态（top, right, bottom, left）

应用场景 ​

  • 滚动进度指示器
  • 滚动到底部加载更多
  • 滚动位置同步
  • 基于滚动位置的动画效果

5. useInfiniteScroll ​

useInfiniteScroll 函数用于实现无限滚动加载。

基本用法 ​

vue
<template>
  <div>
    <div class="info-bar">
      <p>已加载项目数: {{ items.length }}</p>
      <p>加载状态: {{ isLoading ? "加载中..." : "空闲" }}</p>
      <button @click="resetItems">重置列表</button>
    </div>
    <div
      ref="infiniteScrollEl"
      style="height: 400px; overflow-y: auto; border: 1px solid #ddd;"
    >
      <div
        v-for="(item, index) in items"
        :key="index"
        style="padding: 15px; border-bottom: 1px solid #eee;"
      >
        <h4>项目 #{{ item.id }}</h4>
        <p>{{ item.content }}</p>
      </div>
      <div v-if="isLoading" style="padding: 20px; text-align: center;">
        加载更多数据...
      </div>
      <div
        v-if="reachedLimit"
        style="padding: 20px; text-align: center; font-style: italic;"
      >
        已加载所有数据
      </div>
    </div>
  </div>
</template>

<script setup>
import { ref, computed } from "vue";
import { useInfiniteScroll } from "@vueuse/core";

const infiniteScrollEl = ref(null);
const items = ref([]);
const isLoading = ref(false);
const page = ref(1);
const limit = 50; // 最多加载50个项目
const reachedLimit = computed(() => items.value.length >= limit);

// 初始化项目
const initItems = () => {
  items.value = Array.from({ length: 10 }, (_, i) => ({
    id: i + 1,
    content: `这是一个示例内容项目，ID为 ${
      i + 1
    }。你可以继续滚动加载更多内容。`,
  }));
  page.value = 1;
};

// 加载更多项目
const loadMoreItems = async () => {
  if (isLoading.value || reachedLimit.value) return;

  isLoading.value = true;

  // 模拟API请求延迟
  await new Promise((resolve) => setTimeout(resolve, 1000));

  const newItems = Array.from({ length: 5 }, (_, i) => {
    const id = items.value.length + i + 1;
    return {
      id,
      content: `这是一个示例内容项目，ID为 ${id}。你可以继续滚动加载更多内容。`,
    };
  });

  items.value = [...items.value, ...newItems];
  page.value++;
  isLoading.value = false;
};

// 重置列表
const resetItems = () => {
  initItems();
};

// 初始化
initItems();

// 使用 useInfiniteScroll
useInfiniteScroll(
  infiniteScrollEl,
  () => {
    if (!reachedLimit.value) {
      loadMoreItems();
    }
  },
  { distance: 10 }
);
</script>

<style scoped>
.info-bar {
  display: flex;
  justify-content: space-between;
  align-items: center;
  padding: 10px;
  background-color: #f5f5f5;
  border-radius: 4px;
  margin-bottom: 10px;
}
.info-bar button {
  padding: 5px 10px;
  background-color: #42b983;
  color: white;
  border: none;
  border-radius: 4px;
  cursor: pointer;
}
</style>

参数 ​

  • element: 滚动容器元素
  • onLoadMore: 加载更多数据的回调函数
  • options: 配置选项，如触发距离、方向等

应用场景 ​

  • 社交媒体信息流
  • 商品列表无限加载
  • 文章阅读无限加载
  • 图片库无限滚动

6. useDeviceOrientation ​

useDeviceOrientation 函数用于获取设备的方向信息。

基本用法 ​

vue
<template>
  <div>
    <div class="orientation-info">
      <p>Alpha (z轴旋转): {{ alpha !== null ? alpha.toFixed(2) : "N/A" }}°</p>
      <p>Beta (x轴旋转): {{ beta !== null ? beta.toFixed(2) : "N/A" }}°</p>
      <p>Gamma (y轴旋转): {{ gamma !== null ? gamma.toFixed(2) : "N/A" }}°</p>
    </div>
    <div class="device-visualization">
      <div
        class="device-model"
        :style="{
          transform: `rotateZ(${alpha || 0}deg) rotateX(${
            beta || 0
          }deg) rotateY(${gamma || 0}deg)`,
        }"
      ></div>
    </div>
    <p class="note">注意: 需要在移动设备上才能获取完整的方向数据</p>
  </div>
</template>

<script setup>
import { useDeviceOrientation } from "@vueuse/core";

const { alpha, beta, gamma } = useDeviceOrientation();
</script>

<style scoped>
.orientation-info {
  padding: 10px;
  background-color: #f5f5f5;
  border-radius: 4px;
  margin-bottom: 20px;
}
.device-visualization {
  height: 200px;
  display: flex;
  align-items: center;
  justify-content: center;
  perspective: 500px;
}
.device-model {
  width: 100px;
  height: 180px;
  background-color: #333;
  border-radius: 10px;

自定义传感器函数 ​

除了使用 VueUse 提供的传感器函数外，我们还可以基于 Vue 的组合式 API 创建自定义的传感器函数。以下是一个自定义键盘方向键监听函数的示例：

useKeyboardDirection ​

js
import { ref, onMounted, onUnmounted } from "vue";

/**
 * 监听键盘方向键的按下状态
 * @returns {{
 *   up: Ref<boolean>,
 *   down: Ref<boolean>,
 *   left: Ref<boolean>,
 *   right: Ref<boolean>,
 *   pressed: Ref<boolean>
 * }}
 */
export function useKeyboardDirection() {
  const up = ref(false);
  const down = ref(false);
  const left = ref(false);
  const right = ref(false);
  const pressed = ref(false);

  // 按键映射
  const KEY_MAP = {
    ArrowUp: "up",
    ArrowDown: "down",
    ArrowLeft: "left",
    ArrowRight: "right",
    w: "up",
    s: "down",
    a: "left",
    d: "right",
  };

  // 更新按键状态
  const updatePressed = () => {
    pressed.value = up.value || down.value || left.value || right.value;
  };

  // 处理按键按下
  const handleKeyDown = (e) => {
    const direction = KEY_MAP[e.key];
    if (direction) {
      e.preventDefault();
      // 设置对应方向的状态为 true
      if (direction === "up") up.value = true;
      if (direction === "down") down.value = true;
      if (direction === "left") left.value = true;
      if (direction === "right") right.value = true;

      updatePressed();
    }
  };

  // 处理按键释放
  const handleKeyUp = (e) => {
    const direction = KEY_MAP[e.key];
    if (direction) {
      e.preventDefault();
      // 设置对应方向的状态为 false
      if (direction === "up") up.value = false;
      if (direction === "down") down.value = false;
      if (direction === "left") left.value = false;
      if (direction === "right") right.value = false;

      updatePressed();
    }
  };

  // 在组件挂载时添加事件监听
  onMounted(() => {
    window.addEventListener("keydown", handleKeyDown);
    window.addEventListener("keyup", handleKeyUp);
  });

  // 在组件卸载时移除事件监听
  onUnmounted(() => {
    window.removeEventListener("keydown", handleKeyDown);
    window.removeEventListener("keyup", handleKeyUp);
  });

  return {
    up,
    down,
    left,
    right,
    pressed,
  };
}

<script setup>
import { ref, onMounted, onUnmounted } from "vue";
import { useKeyboardDirection } from "./useKeyboardDirection";

// 使用自定义的键盘方向传感器
const { up, down, left, right, pressed } = useKeyboardDirection();

// 玩家位置
const playerX = ref(150);
const playerY = ref(150);
const speed = 3;

// 游戏循环
let animationFrameId = null;

const gameLoop = () => {
  // 根据按键状态更新玩家位置
  if (up.value) playerY.value = Math.max(0, playerY.value - speed);
  if (down.value) playerY.value = Math.min(300 - 20, playerY.value + speed);
  if (left.value) playerX.value = Math.max(0, playerX.value - speed);
  if (right.value) playerX.value = Math.min(300 - 20, playerX.value + speed);

  // 继续游戏循环
  animationFrameId = requestAnimationFrame(gameLoop);
};

// 在组件挂载时启动游戏循环
onMounted(() => {
  animationFrameId = requestAnimationFrame(gameLoop);
});

// 在组件卸载时停止游戏循环
onUnmounted(() => {
  if (animationFrameId) {
    cancelAnimationFrame(animationFrameId);
  }
});
</script>

技术优势 ​

VueUse 传感器函数的主要优势在于：

  1. 简化复杂 API：将复杂的浏览器 API 封装成简单易用的组合式函数
  2. 响应式集成：完全集成 Vue 的响应式系统，可以直接在模板中使用
  3. 类型安全：提供完整的 TypeScript 类型定义
  4. 跨平台兼容：处理了不同浏览器和设备的兼容性问题

注意事项 ​

  1. 部分传感器函数需要用户授予权限才能使用（如地理位置、设备列表等）
  2. 在服务端渲染环境中，这些函数通常会返回默认值或空值
  3. 部分功能在不同浏览器中的支持程度不同，使用前应检查兼容性
  4. 使用传感器数据时应考虑性能影响，必要时应用节流或防抖技术
  5. 对于需要频繁更新的传感器数据，应避免在每次更新时进行复杂计算
  6. 在组件卸载时确保清理所有事件监听，避免内存泄漏
