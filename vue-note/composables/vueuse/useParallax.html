useParallax ​

轻松创建视差（Parallax）效果。如果设备不支持方向传感器，它会回退到使用鼠标位置。

useParallax 是 VueUse 库中的一个组合式函数，它提供了一种简单的方式来创建视差效果。视差效果是一种常见的网页设计技术，通过让背景元素比前景元素移动得慢，从而在滚动或鼠标移动时产生深度和沉浸感。useParallax 自动检测设备是否支持 DeviceOrientation API，如果支持则利用设备的倾斜和滚动数据，否则会优雅地回退到使用鼠标在元素内的位置来模拟视差效果。

基本用法 ​

ts
import { ref } from 'vue'
import { useParallax } from '@vueuse/core'

const target = ref(null)
const { roll, tilt, source } = useParallax(target)

// roll: 元素的水平倾斜度，值在 -0.5 到 0.5 之间
// tilt: 元素的垂直倾斜度，值在 -0.5 到 0.5 之间
// source: 检测到的传感器来源，可以是 'mouse' 或 'deviceOrientation'

你可以将 roll 和 tilt 值应用于 CSS transform 属性，例如 rotateX、rotateY 或 translate，来创建动态的视差效果。

类型定义 ​

ts
import type { ComputedRef } from 'vue'
import type { ConfigurableWindow } from '../_configurable'
import type { MaybeElementRef } from '../unrefElement'

export interface UseParallaxOptions extends ConfigurableWindow {
  /**
   * 设备方向倾斜度调整函数。
   * 输入范围通常为 [-90, 90]，输出应调整到 [-0.5, 0.5] 或您希望的范围。
   */
  deviceOrientationTiltAdjust?: (i: number) => number
  /**
   * 设备方向滚动度调整函数。
   * 输入范围通常为 [-180, 180]，输出应调整到 [-0.5, 0.5] 或您希望的范围。
   */
  deviceOrientationRollAdjust?: (i: number) => number
  /**
   * 鼠标倾斜度调整函数。
   * 输入范围通常为 [-1, 1]，输出应调整到 [-0.5, 0.5] 或您希望的范围。
   */
  mouseTiltAdjust?: (i: number) => number
  /**
   * 鼠标滚动度调整函数。
   * 输入范围通常为 [-1, 1]，输出应调整到 [-0.5, 0.5] 或您希望的范围。
   */
  mouseRollAdjust?: (i: number) => number
}

export interface UseParallaxReturn {
  /**
   * 滚动值。缩放至 `-0.5 ~ 0.5`。
   * 当源是 'deviceOrientation' 时，表示设备的 Beta 轴（X轴）旋转。
   * 当源是 'mouse' 时，表示鼠标在元素内的垂直位置。
   */
  roll: ComputedRef<number>
  /**
   * 倾斜值。缩放至 `-0.5 ~ 0.5`。
   * 当源是 'deviceOrientation' 时，表示设备的 Gamma 轴（Y轴）旋转。
   * 当源是 'mouse' 时，表示鼠标在元素内的水平位置。
   */
  tilt: ComputedRef<number>
  /**
   * 传感器来源，可以是 `mouse` 或 `deviceOrientation`。
   */
  source: ComputedRef<'deviceOrientation' | 'mouse'>
}

declare function useParallax(
  target: MaybeElementRef,
  options?: UseParallaxOptions,
): UseParallaxReturn

参数 ​

useParallax 函数接受以下两个参数：

  1. target: MaybeElementRef：

    • 类型：MaybeElementRef (可以是 HTMLElement、SVGElement、Vue ref 对象或 null/undefined)
    • 描述：用于监听鼠标事件或作为设备方向基准的目标元素。通常是一个包裹视差内容的容器。

  2. options?: UseParallaxOptions：可选的配置对象，可以包含以下属性：

    • deviceOrientationTiltAdjust?: (i: number) => number：一个调整函数，用于处理来自 deviceOrientation 的倾斜（gamma 轴）值。默认情况下，它将设备的 gamma 值（通常在 -90 到 90 度之间）缩放到 -0.5 到 0.5 之间。你可以提供自定义函数来调整灵敏度或反转方向。
    • deviceOrientationRollAdjust?: (i: number) => number：一个调整函数，用于处理来自 deviceOrientation 的滚动（beta 轴）值。默认情况下，它将设备的 beta 值（通常在 -180 到 180 度之间）缩放到 -0.5 到 0.5 之间。你可以提供自定义函数来调整灵敏度或反转方向。
    • mouseTiltAdjust?: (i: number) => number：一个调整函数，用于处理来自鼠标的倾斜（水平）值。默认情况下，它将鼠标在元素内的水平相对位置（-0.5到0.5）缩放。你可以自定义此函数。
    • mouseRollAdjust?: (i: number) => number：一个调整函数，用于处理来自鼠标的滚动（垂直）值。默认情况下，它将鼠标在元素内的垂直相对位置（-0.5到0.5）缩放。你可以自定义此函数。
    • window?: Window (来自 ConfigurableWindow)：可选。允许你指定一个自定义的 Window 对象，通常用于测试环境或 SSR。

返回值 ​

useParallax 函数返回一个包含以下计算属性的对象：

  • roll: ComputedRef<number>：一个计算属性，表示水平滚动或倾斜度。值被标准化到 -0.5 到 0.5 之间。当 source 为 'deviceOrientation' 时，它对应于设备的 beta 轴（上下倾斜）旋转，并根据屏幕方向进行调整。当 source 为 'mouse' 时，它对应于鼠标在目标元素内的垂直位置。
  • tilt: ComputedRef<number>：一个计算属性，表示垂直倾斜度。值被标准化到 -0.5 到 0.5 之间。当 source 为 'deviceOrientation' 时，它对应于设备的 gamma 轴（左右倾斜）旋转，并根据屏幕方向进行调整。当 source 为 'mouse' 时，它对应于鼠标在目标元素内的水平位置。
  • source: ComputedRef<'deviceOrientation' | 'mouse'>：一个计算属性，指示当前正在使用的传感器来源。如果设备支持且数据可用，则为 'deviceOrientation'；否则为 'mouse'。

工作原理 ​

useParallax 的核心在于它结合了 useDeviceOrientation 和 useMouseInElement，并根据设备的传感器支持情况进行智能切换：

  1. 设备方向检测：

    • 它首先使用 useDeviceOrientation 来监听设备的陀螺仪和加速计数据（alpha、beta、gamma）。
    • 同时，useScreenOrientation 用于获取屏幕的方向（portrait-primary, landscape-primary 等），以便正确地将设备倾斜数据映射到视差效果的 roll 和 tilt 值。
    • 只有当 useDeviceOrientation 支持且返回有效的 alpha 或 gamma 值时，source 才会被设置为 'deviceOrientation'。

  2. 鼠标回退机制：

    • 如果 DeviceOrientation API 不受支持或没有可用的设备方向数据，source 将回退到 'mouse'。
    • 在这种情况下，它会利用 useMouseInElement 来获取鼠标在指定 target 元素内部的相对位置 (elementX, elementY)。

  3. 标准化计算：

    • 无论是设备方向数据还是鼠标位置数据，都会被转换和标准化到 -0.5 到 0.5 的范围内，以便于在 CSS 变换中应用。
    • roll 值通常与设备的 beta 轴（或鼠标的 Y 轴）相关，tilt 值与设备的 gamma 轴（或鼠标的 X 轴）相关。
    • 提供了 deviceOrientationTiltAdjust、deviceOrientationRollAdjust、mouseTiltAdjust、mouseRollAdjust 等调整函数，允许开发者自定义这些值的转换逻辑，以实现不同的视差灵敏度或效果。

使用场景 ​

  • 网页背景视差：当用户滚动页面或移动鼠标时，背景图片或元素以不同速度移动，创造景深。
  • 交互式卡片/按钮：鼠标悬停在卡片上时，卡片内容根据鼠标位置轻微倾斜或移动，增加交互感。
  • 游戏元素：在简单的网页游戏中，模拟 3D 对象的轻微晃动效果。
  • 产品展示：在 3D 产品模型上应用视差效果，当用户移动鼠标时，产品视角略微变化。
  • 沉浸式体验：为网站或应用添加微妙的动态视觉效果，提升用户体验。

注意事项 ​

  • 浏览器兼容性：DeviceOrientation API 在桌面浏览器上支持不佳，主要用于移动设备。因此，useParallax 的回退机制非常重要，它确保在不支持设备方向的浏览器上也能提供视差效果。
  • HTTPS 环境：为了访问 DeviceOrientation API，您的应用程序必须在 HTTPS 环境下运行。在 http://localhost 上开发时通常不受此限制。
  • 用户权限：某些浏览器可能需要用户授权才能访问设备传感器数据。如果未获得权限，useParallax 将自动回退到使用鼠标。
  • 性能：持续监听设备传感器或鼠标事件可能会消耗一定的资源。确保在不需要时，例如组件卸载时，正确地清理资源。
  • 灵敏度调整：根据你的设计需求，可能需要通过 adjust 选项来调整 roll 和 tilt 的灵敏度，以达到最佳的视觉效果。

实际应用示例 ​

以下是一个简单的 Vue 组件，演示如何使用 useParallax 在一个 div 元素上创建视差效果。请在桌面浏览器上使用鼠标，在移动设备上使用倾斜设备来体验不同来源的视差效果。

      当前使用：<b>{{ source === 'mouse' ? '鼠标' : '设备方向' }}</b>
    </p>

<script setup>
import { ref, computed } from 'vue'
import { useParallax } from '@vueuse/core'

const targetRef = ref(null)
const {
  roll,
  tilt,
  source,
} = useParallax(targetRef)

// 定义不同层级的移动量，数值越大，移动越明显
const layer0 = 10 // 整个卡片的旋转
const layer1 = 20 // 最底层背景元素移动
const layer2 = 30 // 中间层元素移动
const layer3 = 40 // 最上层内容元素移动

// 计算卡片的整体旋转样式
const cardStyle = computed(() => ({
  transform: `perspective(1000px) rotateX(${-tilt.value * layer0}deg) rotateY(${roll.value * layer0}deg)`,
  transition: 'transform 0.1s ease-out',
}))

// 计算不同图层的位移样式
const layer1Style = computed(() => ({
  transform: `translateX(${roll.value * layer1}px) translateY(${tilt.value * layer1}px)`,
  transition: 'transform 0.1s ease-out',
}))

const layer2Style = computed(() => ({
  transform: `translateX(${roll.value * layer2}px) translateY(${tilt.value * layer2}px)`,
  transition: 'transform 0.1s ease-out',
}))

const layer3Style = computed(() => ({
  transform: `translateX(${roll.value * layer3}px) translateY(${tilt.value * layer3}px)`,
  transition: 'transform 0.1s ease-out',
}))
</script>
