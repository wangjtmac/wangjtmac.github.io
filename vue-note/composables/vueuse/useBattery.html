useBattery ​

响应式电池状态 API

useBattery 是 VueUse 库中的一个传感器类组合式函数，用于获取设备电池的状态信息，如充电状态、充电时间、放电时间和电量水平。这个函数封装了浏览器的 Battery Status API，提供了响应式的电池状态监控。

基本用法 ​

ts
import { useBattery } from '@vueuse/core'

const { charging, chargingTime, dischargingTime, level, isSupported } = useBattery()

// 检查是否支持 Battery API
console.log('是否支持电池 API:', isSupported.value)

// 获取电池状态
console.log('是否正在充电:', charging.value)
console.log('充电完成剩余时间(秒):', chargingTime.value)
console.log('电池耗尽剩余时间(秒):', dischargingTime.value)
console.log('电池电量水平(0-1):', level.value)

类型定义 ​

ts
export interface BatteryManager extends EventTarget {
  charging: boolean
  chargingTime: number
  dischargingTime: number
  level: number
}

/**
 * 响应式电池状态 API
 */
export function useBattery(options?: ConfigurableNavigator): UseBatteryReturn

export interface UseBatteryReturn {
  isSupported: Ref<boolean>
  charging: Ref<boolean>
  chargingTime: Ref<number>
  dischargingTime: Ref<number>
  level: Ref<number>
}

返回值 ​

useBattery 函数返回一个包含以下属性的对象：

属性类型描述

chargingRef<boolean>设备是否正在充电
chargingTimeRef<number>充电完成所需的剩余秒数
dischargingTimeRef<number>电池耗尽所需的剩余秒数
levelRef<number>电池电量水平，范围在 0 到 1 之间

工作原理 ​

useBattery 使用浏览器的 Battery Status API 来获取设备的电池信息。它通过 navigator.getBattery() 方法获取 BatteryManager 对象，然后监听相关事件来更新电池状态。

源码实现 ​

typescript
export function useBattery(options: ConfigurableNavigator = {}) {
  const { navigator = defaultNavigator } = options
  const events = ['chargingchange', 'chargingtimechange', 'dischargingtimechange', 'levelchange']

  const isSupported = useSupported(() => navigator && 'getBattery' in navigator && typeof navigator.getBattery === 'function')

  const charging = shallowRef(false)
  const chargingTime = shallowRef(0)
  const dischargingTime = shallowRef(0)
  const level = shallowRef(1)

  let battery: BatteryManager | null

  function updateBatteryInfo(this: BatteryManager) {
    charging.value = this.charging
    chargingTime.value = this.chargingTime || 0
    dischargingTime.value = this.dischargingTime || 0
    level.value = this.level
  }

  if (isSupported.value) {
    (navigator as NavigatorWithBattery)
      .getBattery()
      .then((_battery) => {
        battery = _battery
        updateBatteryInfo.call(battery)
        useEventListener(battery, events, updateBatteryInfo, { passive: true })
      })
  }

  return {
    isSupported,
    charging,
    chargingTime,
    dischargingTime,
    level,
  }
}

使用场景 ​

useBattery 主要用于以下场景：

  1. 电量感知型应用：根据电池状态调整应用行为，在低电量时节省资源
  2. 电池监控应用：显示设备电池状态和剩余使用时间
  3. 节能模式：在电池电量低时自动启用暗色主题或减少功能
  4. 性能优化：根据电池状态调整应用的性能配置，如减少网络请求或后台任务
  5. 用户体验增强：在电池即将耗尽时提醒用户保存工作或连接充电器

注意事项 ​

  1. 浏览器兼容性：Battery Status API 并非所有浏览器都支持，使用前应检查 isSupported 值
  2. 权限和隐私：某些浏览器可能需要用户授权才能访问电池信息
  3. 移动设备限制：在某些移动设备上，Battery API 可能受到限制或提供不准确的信息
  4. 节能模式影响：设备处于节能模式时，API 返回的值可能会受到影响
  5. SSR 兼容性：在服务器端渲染环境中，该函数将返回默认值，因为服务器没有电池

与其他函数的对比 ​

函数描述区别
useBattery监控设备电池状态提供完整的电池信息，包括充电状态和电量水平
useDevicesList获取连接的设备列表关注设备连接而非电池状态
useNetwork监控网络连接状态关注网络连接而非电池状态

示例1：电池状态显示 ​

vue
<template>
  <div class="battery-status">
    <h2>设备电池状态</h2>
    
    <div v-if="isSupported" class="battery-info">
      <div class="battery-level">
        <div class="battery-outer">
          <div class="battery-inner" :style="{ width: `${level * 100}%`, backgroundColor: batteryColor }"></div>
        </div>
        <div class="battery-percentage">{{ (level * 100).toFixed(0) }}%</div>
      </div>
      
      <div class="battery-details">
        <p>状态: {{ charging ? '充电中' : '放电中' }}</p>
        <p v-if="charging">充电完成剩余时间: {{ formatTime(chargingTime) }}</p>
        <p v-else>电池耗尽剩余时间: {{ formatTime(dischargingTime) }}</p>
      </div>
    </div>
    
    <div v-else class="not-supported">
      您的浏览器不支持电池状态 API
    </div>
  </div>
</template>

<script setup>
import { computed } from 'vue'
import { useBattery } from '@vueuse/core'

const { charging, chargingTime, dischargingTime, level, isSupported } = useBattery()

// 根据电量计算电池颜色
const batteryColor = computed(() => {
  if (level.value > 0.5) return '#4caf50' // 绿色
  if (level.value > 0.2) return '#ff9800' // 橙色
  return '#f44336' // 红色
})

// 格式化时间（秒转为小时和分钟）
function formatTime(seconds) {
  if (!seconds || seconds === Infinity) return '未知'
  
  const hours = Math.floor(seconds / 3600)
  const minutes = Math.floor((seconds % 3600) / 60)
  
  return hours > 0 
    ? `${hours}小时${minutes}分钟` 
    : `${minutes}分钟`
}
</script>

示例2：电量感知型应用 ​

vue
<template>
  <div :class="['app-container', { 'battery-saver': isBatterySaverMode }]">
    <div class="battery-info">
      <div v-if="isSupported">
        <p>电池电量: {{ (level * 100).toFixed(0) }}%</p>
        <p>状态: {{ charging ? '充电中' : '放电中' }}</p>
      </div>
      <div v-else>
        无法获取电池信息
      </div>
    </div>
    
    <div class="content">
      <h2>应用内容</h2>
      <div v-if="isBatterySaverMode" class="battery-saver-notice">
        电量低，已启用省电模式
      </div>
      
      <div class="video-section">
        <h3>视频内容</h3>
        <video 
          ref="videoPlayer" 
          src="https://example.com/video.mp4" 
          :autoplay="!isBatterySaverMode" 
          controls
        ></video>
      </div>
      
      <button @click="toggleBatterySaverMode">
        {{ isBatterySaverMode ? '关闭' : '开启' }}省电模式
      </button>
    </div>
  </div>
</template>

<script setup>
import { ref, computed, watch } from 'vue'
import { useBattery, usePreferredDark } from '@vueuse/core'

const { charging, level, isSupported } = useBattery()
const isDark = usePreferredDark()

// 是否启用省电模式（电量低于20%且未充电，或用户手动开启）
const manualBatterySaverMode = ref(false)
const isBatterySaverMode = computed(() => {
  // 如果手动设置了省电模式，则使用手动设置
  if (manualBatterySaverMode.value !== null)
    return manualBatterySaverMode.value
    
  // 否则，在电量低且未充电时自动启用
  return isSupported.value && level.value < 0.2 && !charging.value
})

// 视频播放器引用
const videoPlayer = ref(null)

// 监听省电模式变化
watch(isBatterySaverMode, (newValue) => {
  if (newValue) {
    // 启用省电模式时的操作
    if (videoPlayer.value) {
      videoPlayer.value.pause()
    }
    
    // 减少网络请求和后台任务
    pauseNonEssentialTasks()
  } else {
    // 关闭省电模式时的操作
    resumeNonEssentialTasks()
  }
})

// 切换省电模式
function toggleBatterySaverMode() {
  manualBatterySaverMode.value = !isBatterySaverMode.value
}

// 模拟暂停非必要任务
function pauseNonEssentialTasks() {
  console.log('已暂停非必要任务以节省电量')
  // 实际应用中，这里可以暂停定时器、减少网络请求频率等
}

// 模拟恢复任务
function resumeNonEssentialTasks() {
  console.log('已恢复所有任务')
  // 实际应用中，这里可以恢复之前暂停的任务
}
</script>
