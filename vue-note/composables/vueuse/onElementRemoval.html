onElementRemoval ​

监听元素从 DOM 中移除的事件

onElementRemoval 是 VueUse 库中的一个实用组合式函数，用于监听元素从 DOM 树中移除的事件。当目标元素或包含该元素的任何父元素被从 DOM 中移除时，它会触发回调函数。

基本用法 ​

ts
import { onElementRemoval } from '@vueuse/core'

// 监听元素移除
const el = ref<HTMLElement | null>(null)

onElementRemoval(el, () => {
  console.log('元素已被移除！')
})

类型定义 ​

ts
/**
 * Fires when the element or any element containing it is removed.
 */
export function onElementRemoval(
  target: MaybeElementRef,
  callback: (mutationRecords: MutationRecord[]) => void,
  options?: OnElementRemovalOptions
): Fn

参数 ​

参数类型描述
targetMaybeElementRef要监听的目标元素
callback(mutationRecords: MutationRecord[]) => void元素被移除时触发的回调函数
optionsOnElementRemovalOptions配置选项

配置选项 ​

OnElementRemovalOptions 接口继承了 ConfigurableWindow, ConfigurableDocumentOrShadowRoot 和 WatchOptionsBase，包含以下选项：

选项类型默认值描述
windowWindowdefaultWindow要使用的窗口对象
documentDocumentwindow?.document要使用的文档对象
flush'pre' | 'post' | 'sync''sync'控制回调的刷新时机

返回值 ​

onElementRemoval 函数返回一个停止监听的函数（类型为 Fn），调用该函数可以移除事件监听器。

ts
const stop = onElementRemoval(el, () => {
  console.log('元素已被移除！')
})

// 停止监听
stop()

工作原理 ​

onElementRemoval 使用 MutationObserver 来监视整个文档中的 DOM 变化，特别是节点移除操作。当观察到节点被移除时，它会检查这些节点是否包含目标元素，如果是，则触发回调函数。

源码实现 ​

typescript
export function onElementRemoval(
  target: MaybeElementRef,
  callback: (mutationRecords: MutationRecord[]) => void,
  options: OnElementRemovalOptions = {},
): Fn {
  const {
    window = defaultWindow,
    document = window?.document,
    flush = 'sync',
  } = options

  // SSR
  if (!window || !document)
    return noop

  let stopFn: Fn | undefined
  const cleanupAndUpdate = (fn?: Fn) => {
    stopFn?.()
    stopFn = fn
  }

  const stopWatch = watchEffect(() => {
    const el = unrefElement(target)
    if (el) {
      const { stop } = useMutationObserver(
        document as any,
        (mutationsList) => {
          const targetRemoved = mutationsList
            .map(mutation => [...mutation.removedNodes])
            .flat()
            .some(node => node === el || node.contains(el))
          
          if (targetRemoved) {
            callback(mutationsList)
          }
        },
        {
          window,
          childList: true,
          subtree: true,
        },
      )
      cleanupAndUpdate(stop)
    }
  }, { flush })

  const stopHandle = () => {
    stopWatch()
    cleanupAndUpdate()
  }

  tryOnScopeDispose(stopHandle)

  return stopHandle
}

使用场景 ​

onElementRemoval 主要用于以下场景：

  1. 资源清理：当元素被移除时释放相关资源或取消订阅
  2. 状态同步：在元素被移除时更新应用状态
  3. 动态组件管理：监控动态创建的组件何时被销毁
  4. 第三方库集成：在基于 DOM 的第三方库中，当相关元素被移除时执行清理操作
  5. 条件渲染监听：当条件渲染的组件被移除时执行逻辑

注意事项 ​

  1. 该函数使用 MutationObserver 实现，在非常老旧的浏览器中可能不受支持
  2. 由于监听整个文档的 DOM 变化，在大型应用中可能会有性能影响
  3. 当不再需要监听时，应调用返回的停止函数以避免内存泄漏
  4. 回调函数接收 MutationRecord[] 参数，包含了所有触发回调的 DOM 变化记录

与其他函数的对比 ​

函数描述区别
onElementRemoval监听元素从 DOM 中移除使用 MutationObserver 监听整个文档
tryOnUnmounted在组件卸载时执行回调基于 Vue 生命周期，只在组件卸载时触发
useMutationObserver监听 DOM 变化更通用的 API，可以监听各种 DOM 变化

实际应用示例 ​

示例1：资源清理 ​

vue
<template>
  <div>
    <div v-if="showVideo" ref="videoContainer">
      <video ref="videoEl" :src="videoSrc" controls></video>
    </div>
    <button @click="showVideo = !showVideo">
      {{ showVideo ? '隐藏视频' : '显示视频' }}
    </button>
  </div>
</template>

<script setup>
import { ref } from 'vue'
import { onElementRemoval } from '@vueuse/core'

const showVideo = ref(true)
const videoContainer = ref(null)
const videoEl = ref(null)
const videoSrc = 'https://example.com/video.mp4'

// 当视频元素被移除时停止视频并释放资源
onElementRemoval(videoEl, () => {
  console.log('视频元素已被移除，正在清理资源...')
  
  // 在实际应用中，可能需要取消网络请求、释放内存等
  if (videoEl.value) {
    videoEl.value.pause()
    videoEl.value.src = ''
    videoEl.value.load()
  }
})
</script>

示例2：动态组件状态同步 ​

vue
<template>
  <div>
    <div v-if="showComponent" ref="componentContainer">
      <dynamic-component />
    </div>
    <button @click="showComponent = !showComponent">
      {{ showComponent ? '移除组件' : '显示组件' }}
    </button>
    <div>
      组件状态: {{ componentStatus }}
    </div>
  </div>
</template>

<script setup>
import { ref } from 'vue'
import { onElementRemoval } from '@vueuse/core'
import DynamicComponent from './DynamicComponent.vue'

const showComponent = ref(true)
const componentContainer = ref(null)
const componentStatus = ref('活跃')

onElementRemoval(componentContainer, () => {
  componentStatus.value = '已移除'
  
  // 可以在这里执行其他状态同步操作
  console.log('组件已从DOM中移除')
})
</script>

示例3：第三方库集成 ​

vue
<template>
  <div>
    <div v-if="showChart" ref="chartContainer" class="chart-container"></div>
    <button @click="showChart = !showChart">
      {{ showChart ? '隐藏图表' : '显示图表' }}
    </button>
  </div>
</template>

<script setup>
import { ref, onMounted } from 'vue'
import { onElementRemoval } from '@vueuse/core'
import * as echarts from 'echarts'

const showChart = ref(true)
const chartContainer = ref(null)
let chartInstance = null

// 初始化图表
onMounted(() => {
  if (chartContainer.value) {
    chartInstance = echarts.init(chartContainer.value)
    chartInstance.setOption({
      title: { text: '示例图表' },
      xAxis: { type: 'category', data: ['Mon', 'Tue', 'Wed', 'Thu', 'Fri', 'Sat', 'Sun'] },
      yAxis: { type: 'value' },
      series: [{ data: [150, 230, 224, 218, 135, 147, 260], type: 'line' }]
    })
  }
})

// 当图表容器被移除时销毁图表实例
onElementRemoval(chartContainer, () => {
  console.log('图表容器已被移除，正在销毁图表实例...')
  
  if (chartInstance) {
    chartInstance.dispose()
    chartInstance = null
  }
})
</script>

<style scoped>
.chart-container {
  width: 600px;
  height: 400px;
}
</style>
