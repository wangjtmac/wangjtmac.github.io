UniApp 组件详解 ​

本文是对 组件系统 的补充和扩展，提供了更详细的组件使用指南、扩展组件(uni-ui)的介绍以及实战案例。如果您想了解组件系统的基础知识，请先阅读 组件系统。

一、内置组件 ​

UniApp 提供了丰富的内置组件，这些组件可以直接在项目中使用，无需额外安装。内置组件按功能可分为以下几类：

1. 视图容器组件 ​

1.1 view ​

最基础的视图容器，类似于 HTML 中的 div。

vue
<view class="container">
  <view class="item">项目1</view>
  <view class="item">项目2</view>
</view>

1.2 scroll-view ​

可滚动的视图区域，用于展示超出容器的内容。

vue
<scroll-view scroll-y="true" class="scroll-container" @scroll="onScroll">
  <view class="scroll-item" v-for="(item, index) in items" :key="index">
    {{ item.name }}
  </view>
</scroll-view>

常用属性：

  • scroll-x: 允许横向滚动
  • scroll-y: 允许纵向滚动
  • scroll-top: 设置竖向滚动条位置
  • scroll-left: 设置横向滚动条位置
  • scroll-into-view: 滚动到指定元素

1.3 swiper ​

滑块视图容器，用于实现轮播图等效果。

vue
<swiper class="banner" indicator-dots autoplay circular :interval="3000" :duration="500">
  <swiper-item v-for="(item, index) in banners" :key="index">
    <image class="banner-image" :src="item.image" mode="aspectFill"></image>
  </swiper-item>
</swiper>

常用属性：

  • indicator-dots: 是否显示指示点
  • autoplay: 是否自动切换
  • circular: 是否循环播放
  • interval: 自动切换时间间隔
  • duration: 滑动动画时长

2. 基础内容组件 ​

2.1 text ​

文本组件，用于显示文本内容。

vue
<text class="title" selectable>可选择的文本内容</text>

常用属性：

  • selectable: 文本是否可选
  • space: 显示连续空格
  • decode: 是否解码

2.2 rich-text ​

富文本组件，用于显示 HTML 内容。

vue
<rich-text :nodes="htmlContent"></rich-text>

2.3 progress ​

进度条组件。

vue
<progress :percent="80" show-info stroke-width="3" />

3. 表单组件 ​

3.1 button ​

按钮组件。

vue
<button type="primary" @tap="handleTap">主按钮</button>
<button type="default" size="mini">次按钮</button>

常用属性：

  • type: 按钮类型（primary, default, warn）
  • size: 按钮大小（default, mini）
  • plain: 是否镂空
  • loading: 是否显示加载图标

3.2 input ​

输入框组件。

vue
<input type="text" v-model="inputValue" placeholder="请输入内容" @input="onInput" />

常用属性：

  • type: 输入类型（text, number, idcard, digit）
  • password: 是否是密码类型
  • placeholder: 输入提示
  • focus: 是否自动聚焦

3.3 checkbox & radio ​

多选框和单选框。

vue
<checkbox-group @change="checkboxChange">
  <label v-for="(item, index) in items" :key="index">
    <checkbox :value="item.value" :checked="item.checked" />
    {{item.name}}
  </label>
</checkbox-group>

<radio-group @change="radioChange">
  <label v-for="(item, index) in items" :key="index">
    <radio :value="item.value" :checked="item.checked" />
    {{item.name}}
  </label>
</radio-group>

3.4 picker ​

选择器组件，支持普通选择器、多列选择器、时间选择器、日期选择器等。

vue
<picker @change="bindPickerChange" :value="index" :range="array">
  <view class="picker">
    当前选择: {{array[index]}}
  </view>
</picker>

4. 媒体组件 ​

4.1 image ​

图片组件。

vue
<image src="/static/logo.png" mode="aspectFit"></image>

常用属性：

  • src: 图片路径
  • mode: 图片裁剪、缩放模式
    • aspectFit: 保持纵横比缩放图片，使图片完整显示
    • aspectFill: 保持纵横比缩放图片，只保证图片的短边能完全显示出来
    • widthFix: 宽度不变，高度自动变化，保持原图宽高比不变

4.2 video ​

视频组件。

vue
<video src="https://example.com/video.mp4" controls></video>

5. 导航组件 ​

5.1 navigator ​

页面链接组件。

vue
<navigator url="/pages/detail/detail?id=1" hover-class="navigator-hover">
  跳转到详情页
</navigator>

常用属性：

  • url: 跳转页面路径
  • open-type: 跳转方式
    • navigate: 对应 uni.navigateTo
    • redirect: 对应 uni.redirectTo
    • switchTab: 对应 uni.switchTab
    • reLaunch: 对应 uni.reLaunch
    • navigateBack: 对应 uni.navigateBack

二、扩展组件（uni-ui） ​

uni-ui 是 DCloud 提供的一个专门为 uni-app 开发者设计的 UI 组件库，它具有以下特点：

  1. 高性能：组件经过性能优化，在多端运行更加流畅
  2. 全端：支持 H5/小程序/APP 等多个平台
  3. 风格统一：遵循统一的设计规范
  4. 扩展灵活：支持自定义主题
  5. 与 uniCloud 协作：搭配 uniCloud 使用更加高效
  6. 与 uni 统计自动集成：免打点即可实现统计功能

1. 安装 uni-ui ​

方式一：使用 npm 安装（推荐） ​

bash
# 安装
npm install @dcloudio/uni-ui

# 或使用 yarn
yarn add @dcloudio/uni-ui

# 或使用 pnpm
pnpm add @dcloudio/uni-ui

方式二：使用 HBuilderX 导入 ​

在 HBuilderX 中新建 uni-app 项目，在 uni-app 插件市场找到 uni-ui，点击导入到项目。

2. 配置 easycom ​

在 pages.json 中配置 easycom 规则，使组件可以免注册、免导入直接使用：

json
{
  "easycom": {
    "autoscan": true,
    "custom": {
      "^uni-(.*)": "@dcloudio/uni-ui/lib/uni-$1/uni-$1.vue"
    }
  }
}

3. 常用扩展组件 ​

3.1 uni-badge（数字角标） ​

在元素右上角显示数字或文字的小标记。

vue
<uni-badge text="1"></uni-badge>
<uni-badge text="2" type="primary"></uni-badge>
<uni-badge text="3" type="success"></uni-badge>
<uni-badge text="4" type="warning"></uni-badge>
<uni-badge text="5" type="error"></uni-badge>

3.2 uni-calendar（日历） ​

日历组件，可以查看日期，选择任意范围内的日期。

vue
<uni-calendar 
  :insert="false"
  :lunar="true" 
  :start-date="'2019-3-2'"
  :end-date="'2019-5-20'"
  @change="change" 
/>

3.3 uni-card（卡片） ​

卡片视图组件，可承载文字、列表、图片、段落等内容。

vue
<uni-card 
  title="标题文字" 
  sub-title="副标题文字" 
  extra="额外信息"
  thumbnail="https://example.com/image.jpg"
>
  <text>这是一个卡片组件</text>
</uni-card>

3.4 uni-collapse（折叠面板） ​

折叠面板，可以折叠/展开内容区域。

vue
<uni-collapse>
  <uni-collapse-item title="标题1">
    <text>内容1</text>
  </uni-collapse-item>
  <uni-collapse-item title="标题2">
    <text>内容2</text>
  </uni-collapse-item>
</uni-collapse>

3.5 uni-data-checkbox（数据选择器） ​

通过数据渲染的单选/多选组件。

vue
<uni-data-checkbox 
  v-model="value" 
  :localdata="range" 
  @change="change"
/>

3.6 uni-datetime-picker（日期时间选择器） ​

支持日期、时间或日期+时间的选择器。

vue
<uni-datetime-picker 
  v-model="dateTime" 
  type="datetime" 
  @change="changeDateTime"
/>

3.7 uni-easyinput（增强输入框） ​

对原生 input 组件的增强，提供了更多功能。

vue
<uni-easyinput 
  v-model="value" 
  placeholder="请输入内容" 
  @input="input"
/>

3.8 uni-forms（表单） ​

表单组件，提供表单验证等功能。

vue
<uni-forms :modelValue="formData" ref="form">
  <uni-forms-item label="姓名" name="name" required>
    <uni-easyinput v-model="formData.name" placeholder="请输入姓名" />
  </uni-forms-item>
  <uni-forms-item label="年龄" name="age" required>
    <uni-easyinput v-model="formData.age" placeholder="请输入年龄" />
  </uni-forms-item>
</uni-forms>

3.9 uni-popup（弹出层） ​

弹出层组件，支持从顶部、底部、中间弹出。

vue
<uni-popup ref="popup" type="center">
  <view class="popup-content">弹出层内容</view>
</uni-popup>

3.10 uni-table（表格） ​

表格组件，用于展示多列数据。

vue
<uni-table border stripe emptyText="暂无更多数据">
  <uni-tr>
    <uni-th align="center">日期</uni-th>
    <uni-th align="center">姓名</uni-th>
    <uni-th align="center">地址</uni-th>
  </uni-tr>
  <uni-tr v-for="(item, index) in tableData" :key="index">
    <uni-td>{{ item.date }}</uni-td>
    <uni-td>{{ item.name }}</uni-td>
    <uni-td>{{ item.address }}</uni-td>
  </uni-tr>
</uni-table>

三、组件使用最佳实践 ​

1. 组件选择原则 ​

  1. 优先使用内置组件：内置组件性能更好，兼容性更强
  2. 功能复杂时使用扩展组件：当需要更复杂的功能时，可以使用 uni-ui 提供的扩展组件
  3. 根据场景选择：根据具体业务场景选择最合适的组件

2. 性能优化 ​

  1. 避免组件过度嵌套：减少组件嵌套层级，提高渲染性能
  2. 合理使用条件渲染：使用 v-if 和 v-show 控制组件的显示和隐藏
  3. 使用虚拟列表：展示大量数据时，使用虚拟列表优化性能

3. 跨平台适配 ​

  1. 使用条件编译：针对不同平台进行适配
  2. 了解平台差异：了解各平台组件的差异和限制
  3. 统一样式处理：使用 rpx 等相对单位，适配不同屏幕尺寸

4. 自定义组件封装 ​

  1. 组件粒度适中：既不过大也不过小，便于复用和维护
  2. 使用插槽增强灵活性：通过插槽提供更灵活的定制能力
  3. 规范组件接口：定义清晰的 props 和事件接口

五、总结 ​

UniApp 提供了丰富的内置组件和扩展组件，可以满足大多数移动应用开发需求。在实际开发中，应当根据具体场景选择合适的组件，并注意性能优化和跨平台适配。通过合理使用组件，可以提高开发效率，提升用户体验。
