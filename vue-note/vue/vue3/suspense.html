Suspense 异步依赖处理 ​

<Suspense> 是 Vue 3 的内置组件，用于处理异步依赖的加载状态，提供优雅的加载和错误处理机制。

基础用法 ​

异步组件 ​

vue
<template>
  <div>
    <h2>异步组件示例</h2>
    <Suspense>
      <template #default>
        <AsyncComponent />
      </template>
      <template #fallback>
        <div class="loading">加载中...</div>
      </template>
    </Suspense>
  </div>
</template>

<script setup>
import { defineAsyncComponent } from 'vue'

const AsyncComponent = defineAsyncComponent(() => 
  import('./AsyncComponent.vue')
)
</script>

<style scoped>
.loading {
  display: flex;
  justify-content: center;
  align-items: center;
  height: 200px;
  color: #666;
}
</style>

错误处理 ​

错误边界 ​

vue
<template>
  <div>
    <Suspense @resolve="onResolve" @reject="onReject">
      <template #default>
        <AsyncComponent />
      </template>
      <template #fallback>
        <LoadingSpinner />
      </template>
    </Suspense>
    
    <!-- 错误显示 -->
    <div v-if="error" class="error-message">
      <h4>加载失败</h4>
      <p>{{ error.message }}</p>
      <button @click="retry">重试</button>
    </div>
  </div>
</template>

<script setup>
import { ref } from 'vue'

const error = ref(null)
const retryKey = ref(0)

function onResolve() {
  error.value = null
  console.log('异步组件加载成功')
}

function onReject(err) {
  error.value = err
  console.error('异步组件加载失败:', err)
}

function retry() {
  error.value = null
  retryKey.value++
}
</script>

javascript
// main.js

3. 组合式函数中的异步操作 ​

javascript
// composables/useAsyncData.js
import { ref } from 'vue'

export function useAsyncData(fetcher) {
  const data = ref(null)
  const error = ref(null)
  const loading = ref(true)
  
  const execute = async () => {
    try {
      loading.value = true
      error.value = null
      data.value = await fetcher()
    } catch (err) {
      error.value = err
    } finally {
      loading.value = false
    }
  }
  
  // 立即执行
  execute()
  
  return {
    data,
    error,
    loading,
    retry: execute
  }
}
vue
<template>
  <div>
    <div v-if="error" class="error">
      <p>{{ error.message }}</p>
      <button @click="retry">重试</button>
    </div>
    
    <div v-else-if="loading" class="loading">
      加载中...
    </div>
    
    <div v-else class="content">
      <h3>{{ data.title }}</h3>
      <p>{{ data.content }}</p>
    </div>
  </div>
</template>

<script setup>
import { useAsyncData } from '@/composables/useAsyncData'

const { data, error, loading, retry } = useAsyncData(async () => {

})
</script>

2. 缓存策略 ​

javascript
// utils/cache.js
const cache = new Map()

export async function cachedFetch(url, options = {}) {
  const cacheKey = `${url}:${JSON.stringify(options)}`
  
  if (cache.has(cacheKey)) {
    return cache.get(cacheKey)
  }
  
  const response = await fetch(url, options)
  const data = await response.json()
  
  cache.set(cacheKey, data)
  return data
}

3. 超时处理 ​

javascript
function fetchWithTimeout(url, timeout = 5000) {
  return Promise.race([
    fetch(url),
    new Promise((_, reject) =>
      setTimeout(() => reject(new Error('请求超时')), timeout)
    )
  ])
}

总结 ​

<Suspense> 是 Vue 3 中处理异步操作的强大工具：

  1. 异步处理：优雅处理异步组件和数据
  2. 加载状态：提供统一的加载状态管理
  3. 错误处理：支持错误边界和重试机制
  4. 嵌套支持：支持嵌套的异步依赖
  5. 性能优化：配合缓存和预加载提升性能

合理使用 <Suspense> 可以大大提升异步操作的用户体验。
