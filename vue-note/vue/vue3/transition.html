Transition 过渡 ​

Vue 3 的 <Transition> 组件提供了强大的过渡动画功能，可以为元素的进入和离开添加平滑的过渡效果。

基础用法 ​

简单过渡 ​

vue
<template>
  <button @click="show = !show">切换</button>
  <Transition name="fade">
    <p v-if="show">Hello Vue!</p>
  </Transition>
</template>

<script setup>
import { ref } from 'vue'

const show = ref(true)
</script>

<style>
.fade-enter-active,
.fade-leave-active {
  transition: opacity 0.5s ease;
}

.fade-enter-from,
.fade-leave-to {
  opacity: 0;
}
</style>

过渡类名 ​

Vue 会自动添加/移除 CSS 类名：

进入过渡 ​

  • v-enter-from：进入过渡的开始状态
  • v-enter-active：进入过渡的激活状态
  • v-enter-to：进入过渡的结束状态

离开过渡 ​

  • v-leave-from：离开过渡的开始状态
  • v-leave-active：离开过渡的激活状态
  • v-leave-to：离开过渡的结束状态

自定义类名前缀 ​

vue
<Transition name="slide">
  <div v-if="show">内容</div>
</Transition>
css
.slide-enter-active,
.slide-leave-active {
  transition: all 0.3s ease;
}

.slide-enter-from {
  transform: translateX(-100%);
}

.slide-leave-to {
  transform: translateX(100%);
}

<template>
  <Transition
    enter-active-class="animate__animated animate__fadeInUp"

JavaScript 钩子 ​

vue
<template>
  <Transition
    @before-enter="onBeforeEnter"
    @enter="onEnter"
    @after-enter="onAfterEnter"
    @enter-cancelled="onEnterCancelled"
    @before-leave="onBeforeLeave"
    @leave="onLeave"
    @after-leave="onAfterLeave"
    @leave-cancelled="onLeaveCancelled"
  >
    <div v-if="show">JavaScript 动画</div>
  </Transition>
</template>

<script setup>
function onBeforeEnter(el) {
  el.style.opacity = 0
  el.style.transform = 'scale(0)'
}

function onEnter(el, done) {
  // 强制重排
  el.offsetHeight
  
  el.style.transition = 'all 0.3s ease'
  el.style.opacity = 1
  el.style.transform = 'scale(1)'
  
  // 动画完成后调用 done
  setTimeout(done, 300)
}

function onAfterEnter(el) {
  console.log('进入动画完成')
}

function onBeforeLeave(el) {
  el.style.opacity = 1
  el.style.transform = 'scale(1)'
}

function onLeave(el, done) {
  el.style.transition = 'all 0.3s ease'
  el.style.opacity = 0
  el.style.transform = 'scale(0)'
  
  setTimeout(done, 300)
}

function onAfterLeave(el) {
  console.log('离开动画完成')
}
</script>

<template>

列表过渡 ​

基础列表过渡 ​

vue
<template>
  <TransitionGroup name="list" tag="ul">
    <li v-for="item in items" :key="item.id">
      {{ item.text }}
    </li>
  </TransitionGroup>
</template>

<style>
.list-enter-active,
.list-leave-active {
  transition: all 0.5s ease;
}

.list-enter-from,
.list-leave-to {
  opacity: 0;
  transform: translateX(30px);
}

.list-move {
  transition: transform 0.5s ease;
}
</style>

交错列表动画 ​

vue
<template>
  <TransitionGroup
    name="stagger"
    tag="ul"
    @before-enter="onBeforeEnter"
    @enter="onEnter"
    @leave="onLeave"
  >
    <li
      v-for="(item, index) in items"
      :key="item.id"
      :data-index="index"
    >
      {{ item.text }}
    </li>
  </TransitionGroup>
</template>

<script setup>
function onBeforeEnter(el) {
  el.style.opacity = 0
  el.style.transform = 'translateY(30px)'
}

function onEnter(el, done) {
  const delay = el.dataset.index * 150
  
  setTimeout(() => {
    el.style.transition = 'all 0.5s ease'
    el.style.opacity = 1
    el.style.transform = 'translateY(0)'
    
    setTimeout(done, 500)
  }, delay)
}

function onLeave(el, done) {
  const delay = el.dataset.index * 150
  
  setTimeout(() => {
    el.style.transition = 'all 0.5s ease'
    el.style.opacity = 0
    el.style.transform = 'translateY(-30px)'
    
    setTimeout(done, 500)
  }, delay)
}
</script>

高级技巧 ​

1. 条件过渡 ​

vue
<template>
  <Transition :name="shouldAnimate ? 'fade' : ''">
    <div v-if="show">条件动画</div>
  </Transition>
</template>

<script setup>
import { ref, computed } from 'vue'

const show = ref(true)
const enableAnimation = ref(true)

const shouldAnimate = computed(() => enableAnimation.value)
</script>

2. 过渡持续时间 ​

vue
<template>
  <Transition
    name="fade"
    :duration="{ enter: 500, leave: 800 }"
  >
    <div v-if="show">自定义持续时间</div>
  </Transition>
</template>

3. 过渡事件监听 ​

vue
<template>
  <Transition
    name="fade"
    @enter="onEnter"
    @leave="onLeave"
  >
    <div v-if="show">事件监听</div>
  </Transition>
</template>

<script setup>
function onEnter() {
  console.log('元素进入')
}

function onLeave() {
  console.log('元素离开')
}
</script>

性能优化 ​

1. 使用 CSS 动画 ​

css
/* 使用 CSS 动画而不是过渡 */
.bounce-enter-active {
  animation: bounce-in 0.5s;
}

.bounce-leave-active {
  animation: bounce-out 0.5s;
}

@keyframes bounce-in {
  0% {
    transform: scale(0);
  }
  50% {
    transform: scale(1.2);
  }
  100% {
    transform: scale(1);
  }
}

@keyframes bounce-out {
  0% {
    transform: scale(1);
  }
  50% {
    transform: scale(1.2);
  }
  100% {
    transform: scale(0);
  }
}

2. 硬件加速 ​

css
.gpu-accelerated {
  transform: translateZ(0); /* 触发硬件加速 */
  will-change: transform, opacity; /* 提示浏览器优化 */
}

最佳实践 ​

1. 语义化命名 ​

css
/* ✅ 好的命名 */
.modal-enter-active { /* ... */ }
.slide-up-enter-active { /* ... */ }
.fade-in-enter-active { /* ... */ }

/* ❌ 避免的命名 */
.v-enter-active { /* ... */ }
.transition1-enter-active { /* ... */ }

2. 合理的动画时长 ​

css
/* 快速交互 */
.quick-fade {
  transition: opacity 0.15s ease;
}

/* 普通过渡 */
.normal-slide {
  transition: transform 0.3s ease;
}

/* 复杂动画 */
.complex-animation {
  transition: all 0.5s cubic-bezier(0.4, 0, 0.2, 1);
}

3. 减少重排和重绘 ​

css
/* ✅ 只影响合成层 */
.optimized-transition {
  transition: transform 0.3s ease, opacity 0.3s ease;
}

/* ❌ 会触发重排 */
.expensive-transition {
  transition: width 0.3s ease, height 0.3s ease;
}

总结 ​

Vue 3 的 Transition 组件提供了丰富的过渡动画功能：

  1. CSS 过渡：基于 CSS 类名的自动过渡
  2. JavaScript 钩子：完全控制动画过程
  3. 过渡模式：控制元素切换的时机
  4. 列表过渡：为列表项添加动画效果
  5. 性能优化：合理使用硬件加速和动画属性

合理使用过渡动画可以提升用户体验和应用的视觉效果。
