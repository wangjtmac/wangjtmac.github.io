模板语法 ​

Vue3 使用基于 HTML 的模板语法，允许我们声明式地将渲染的 DOM 绑定到 Vue 实例的数据上。所有的 Vue 模板都是有效的 HTML，可以被符合规范的浏览器和 HTML 解析器解析。

文本插值 ​

双大括号语法 ​

最基本的数据绑定形式是使用"Mustache"语法（双大括号）的文本插值：

html
<span>消息: {{ message }}</span>

双大括号标签会被替换为相应组件实例中message属性的值。每当message属性变化时，插值处的内容也会更新。

v-text 指令 ​

也可以使用v-text指令实现相同的效果：

html
<span v-text="message"></span>

原始 HTML ​

双大括号会将数据解释为纯文本，而不是 HTML。如果要输出真正的 HTML，需要使用v-html指令：

html
<div v-html="rawHtml"></div>

安全警告

动态渲染 HTML 可能会导致 XSS 攻击，只对可信内容使用 HTML 插值，绝不要对用户提供的内容使用它。

属性绑定 ​

v-bind 指令 ​

不能在 HTML 属性中使用双大括号语法，应该使用v-bind指令：

html
<div v-bind:id="dynamicId"></div>

v-bind指令指示 Vue 将元素的id属性与组件的dynamicId属性保持一致。

简写语法 ​

v-bind可以简写为一个冒号：

html
<div :id="dynamicId"></div>

布尔属性 ​

对于布尔属性（存在即为 true 的属性），v-bind工作方式略有不同：

html
<button :disabled="isButtonDisabled">按钮</button>

如果isButtonDisabled的值是null、undefined或false，则disabled属性甚至不会被包含在渲染的<button>元素中。

动态绑定多个属性 ​

可以使用不带参数的v-bind，通过对象语法绑定多个属性：

html
<div v-bind="objectOfAttrs"></div>
js
data() {
  return {
    objectOfAttrs: {
      id: 'container',
      class: 'wrapper',
      style: 'color: red'
    }
  }
}

JavaScript 表达式 ​

Vue 支持在数据绑定中使用完整的 JavaScript 表达式：

html
{{ number + 1 }} {{ ok ? 'YES' : 'NO' }} {{ message.split('').reverse().join('')
}}

<div :id="`list-${id}`"></div>

这些表达式会在当前组件实例的数据作用域下作为 JavaScript 被解析。

限制

每个绑定只能包含单个表达式，下面的例子不会生效：

vue
<!-- 这是语句，不是表达式 -->
{{ var a = 1 }}

<!-- 流控制也不会生效，请使用三元表达式 -->
{{ if (ok) { return message } }}

指令 ​

指令是带有v-前缀的特殊属性。指令的值预期是单个 JavaScript 表达式（除了v-for、v-on和v-slot）。指令的职责是在表达式的值变化时响应式地更新 DOM。

参数 ​

某些指令可以接收一个"参数"，在指令名称之后以冒号表示：

html
<a v-bind:href="url"> ... </a>

<a v-on:click="doSomething"> ... </a>

动态参数 ​

也可以在指令参数中使用 JavaScript 表达式，方法是用方括号括起来：

html
<a v-bind:[attributeName]="url"> ... </a>

<a v-on:[eventName]="doSomething"> ... </a>

这里的attributeName和eventName会被作为 JavaScript 表达式进行求值，求值结果将会作为最终的参数来使用。

动态参数值的限制

动态参数预期会求出一个字符串，异常情况下值为null。null值可以被显性地用于移除绑定。任何其他非字符串类型的值都将会触发警告。

动态参数表达式的限制

动态参数表达式有一些语法限制，因为某些字符，如空格和引号，放在 HTML 属性名里是无效的。例如：

html
<!-- 这会触发一个编译警告 -->
<a v-bind:['foo' + bar]="value"> ... </a>

如果需要传入复杂的动态参数，推荐使用计算属性替代。

修饰符 ​

修饰符是以点开头的特殊后缀，用于指出一个指令应该以特殊方式绑定：

html
<form v-on:submit.prevent="onSubmit">...</form>

这里的.prevent修饰符告诉v-on指令对触发的事件调用event.preventDefault()。

常用指令详解 ​

v-bind ​

用于动态绑定 HTML 属性、组件 prop 或 HTML class 和 style：

html
<!-- 绑定属性 -->
<img v-bind:src="imageSrc" />
<img :src="imageSrc" />
<!-- 简写 -->

<!-- 动态属性名 -->
<button v-bind:[key]="value"></button>
<button :[key]="value"></button>
<!-- 简写 -->

<!-- 绑定对象 -->
<div v-bind="{ id: 'container', class: 'wrapper' }"></div>
<div v-bind="objectOfAttrs"></div>

绑定 Class ​

html
<!-- 对象语法 -->
<div :class="{ active: isActive, 'text-danger': hasError }"></div>

<!-- 数组语法 -->
<div :class="[activeClass, errorClass]"></div>

<!-- 数组中使用对象 -->
<div :class="[{ active: isActive }, errorClass]"></div>

绑定 Style ​

html
<!-- 对象语法 -->
<div :style="{ color: activeColor, fontSize: fontSize + 'px' }"></div>

<!-- 数组语法 -->
<div :style="[baseStyles, overridingStyles]"></div>

<!-- 自动添加前缀 -->
<div :style="{ display: ['-webkit-box', '-ms-flexbox', 'flex'] }"></div>

v-if, v-else-if, v-else ​

条件性地渲染元素：

html
<div v-if="type === 'A'">A</div>
<div v-else-if="type === 'B'">B</div>
<div v-else-if="type === 'C'">C</div>
<div v-else>Not A/B/C</div>

v-show ​

根据条件显示或隐藏元素（通过 CSS 的display属性）：

html
<h1 v-show="ok">Hello!</h1>

与v-if不同，v-show不支持在<template>元素上使用，也不支持v-else。

v-for ​

基于源数据多次渲染元素或模板块：

html
<!-- 遍历数组 -->
<div v-for="(item, index) in items" :key="item.id">
  {{ index }} - {{ item.name }}
</div>

<!-- 遍历对象 -->
<div v-for="(value, key, index) in object" :key="key">
  {{ index }}. {{ key }}: {{ value }}
</div>

<!-- 遍历范围 -->
<span v-for="n in 10" :key="n">{{ n }}</span>

v-on ​

绑定事件监听器：

html
<!-- 方法处理器 -->
<button v-on:click="doThis"></button>
<button @click="doThis"></button>
<!-- 简写 -->

<!-- 动态事件 -->
<button v-on:[event]="doThis"></button>
<button @[event]="doThis"></button>
<!-- 简写 -->

<!-- 内联语句 -->
<button @click="count++">Add 1</button>

<!-- 事件修饰符 -->
<form @submit.prevent="onSubmit">...</form>

<!-- 按键修饰符 -->
<input @keyup.enter="onEnter" />

<!-- 系统修饰符 -->
<button @click.ctrl="onClick">A</button>

事件修饰符 ​

  • .stop - 调用event.stopPropagation()
  • .prevent - 调用event.preventDefault()
  • .capture - 添加事件侦听器时使用 capture 模式
  • .self - 只当事件是从侦听器绑定的元素本身触发时才触发回调
  • .once - 只触发一次回调
  • .passive - 以{ passive: true }模式添加侦听器
html
<!-- 阻止单击事件继续传播 -->
<a @click.stop="doThis"></a>

<!-- 提交事件不再重载页面 -->
<form @submit.prevent="onSubmit">...</form>

<!-- 修饰符可以串联 -->
<a @click.stop.prevent="doThat"></a>

按键修饰符 ​

html
<!-- 只有在 `key` 是 `Enter` 时调用 `vm.submit()` -->
<input @keyup.enter="submit" />

<!-- 也可以使用按键码 -->
<input @keyup.13="submit" />

常用按键别名：

  • .enter
  • .tab
  • .delete (捕获"删除"和"退格"键)
  • .esc
  • .space
  • .up
  • .down
  • .left
  • .right

系统修饰键 ​

html
<!-- Alt + C -->
<input @keyup.alt.67="clear" />

<!-- Ctrl + Click -->
<div @click.ctrl="doSomething">Do something</div>

可用的系统修饰符：

  • .ctrl
  • .alt
  • .shift
  • .meta

v-model ​

在表单元素或组件上创建双向绑定：

html
<input v-model="message" placeholder="edit me" />
<p>Message is: {{ message }}</p>

修饰符 ​

  • .lazy - 在"change"时而非"input"时更新
  • .number - 输入字符串转为有效的数字
  • .trim - 输入首尾空格过滤
html
<!-- 在"change"时而非"input"时更新 -->
<input v-model.lazy="msg" />

<!-- 自动将用户的输入值转为数值类型 -->
<input v-model.number="age" type="number" />

<!-- 自动过滤用户输入的首尾空白字符 -->
<input v-model.trim="msg" />

v-slot ​

提供具名插槽或需要接收 prop 的插槽：

html
<!-- 具名插槽 -->
<base-layout>
  <template v-slot:header> Header content </template>

  <template v-slot:default> Default slot content </template>

  <template v-slot:footer> Footer content </template>
</base-layout>

<!-- 接收 prop 的插槽 -->
<infinite-scroll>
  <template v-slot:item="slotProps">
    <div class="item">{{ slotProps.item.text }}</div>
  </template>
</infinite-scroll>

<!-- v-slot 简写 -->
<base-layout>
  <template #header> Header content </template>
</base-layout>

v-pre ​

跳过这个元素和它的子元素的编译过程：

html
<span v-pre>{{ 这里的内容不会被编译 }}</span>

v-cloak ​

这个指令保持在元素上直到关联组件实例结束编译：

html
<div v-cloak>{{ message }}</div>

配合 CSS 规则使用：

css
[v-cloak] {
  display: none;
}

v-once ​

只渲染元素和组件一次，后续的重新渲染会被跳过：

html
<span v-once>This will never change: {{msg}}</span>

v-memo ​

记住一个模板的子树，在依赖的值没有变化时跳过更新：

html
<div v-memo="[valueA, valueB]">...</div>

特殊属性 ​

key ​

key特殊属性主要用在 Vue 的虚拟 DOM 算法，在新旧 nodes 对比时辨识 VNodes：

html
<ul>
  <li v-for="item in items" :key="item.id">...</li>
</ul>

ref ​

用来给元素或子组件注册引用信息：

html
<input ref="input" />
js
const input = ref(null);

onMounted(() => {
  input.value.focus();
});

is ​

用于动态组件：

html
<component :is="currentComponent"></component>

小结 ​

Vue3 的模板语法提供了丰富而灵活的方式来声明式地绑定 DOM 与 Vue 实例的数据。通过各种指令、特殊属性和表达式，我们可以轻松地创建动态的、响应式的用户界面。掌握这些模板语法是使用 Vue 进行高效开发的基础。
