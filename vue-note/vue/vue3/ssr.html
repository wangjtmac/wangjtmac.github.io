服务端渲染 (SSR) ​

Vue 3 提供了强大的服务端渲染支持，本文将介绍如何实现和优化 Vue 3 的 SSR。

SSR 基础概念 ​

什么是 SSR ​

服务端渲染 (Server-Side Rendering) 是在服务器上生成完整的 HTML 页面，然后发送给客户端的技术。

SSR vs CSR ​

CSR (客户端渲染):
浏览器 → 下载 JS → 执行 JS → 渲染页面

SSR (服务端渲染):
服务器 → 生成 HTML → 发送给浏览器 → 水合 (Hydration)

SSR 的优势 ​

  1. 更好的 SEO: 搜索引擎可以直接抓取内容
  2. 更快的首屏加载: 用户更快看到内容
  3. 更好的性能: 特别是在慢速网络下

通用应用入口 ​

javascript
// src/app.js
import { createSSRApp } from 'vue'
import { createRouter, createMemoryHistory, createWebHistory } from 'vue-router'
import App from './App.vue'
import { routes } from './router'

export function createApp() {
  const app = createSSRApp(App)
  
  const router = createRouter({
    history: import.meta.env.SSR 
      ? createMemoryHistory() 
      : createWebHistory(),
    routes
  })
  
  app.use(router)
  
  return { app, router }
}

import { createApp } from './app'
import { renderToString } from '@vue/server-renderer'

export async function render(url, manifest) {
  const { app, router } = createApp()
  
  // 设置服务器端路由位置
  await router.push(url)
  await router.isReady()
  
  // 渲染应用为字符串
  const html = await renderToString(app)
  
  return html
}

服务端数据预取 ​

javascript
// entry-server.js
export async function render(url, manifest) {
  const { app, router } = createApp()
  
  await router.push(url)
  await router.isReady()
  
  // 获取匹配的路由组件
  const matchedComponents = router.currentRoute.value.matched
  
  // 预取数据
  const asyncDataPromises = matchedComponents.map(route => {
    if (route.meta?.asyncData) {
      return route.meta.asyncData({
        params: router.currentRoute.value.params,
        query: router.currentRoute.value.query
      })
    }
  }).filter(Boolean)
  
  const asyncData = await Promise.all(asyncDataPromises)
  
  // 将数据注入到应用中
  const context = { asyncData }
  
  const html = await renderToString(app, context)
  
  return { html, context }
}

缓存策略 ​

页面级缓存 ​

javascript
// server.js
import LRU from 'lru-cache'

const cache = new LRU({
  max: 1000,
  ttl: 1000 * 60 * 15 // 15 分钟
})

app.get('*', async (req, res) => {
  const cacheKey = req.originalUrl
  
  // 检查缓存
  if (cache.has(cacheKey)) {
    return res.end(cache.get(cacheKey))
  }
  
  try {
    const { html, state } = await render(req.originalUrl)
    
    const fullHtml = `
      <!DOCTYPE html>
      <html>
        <head>
          <title>Vue SSR App</title>
        </head>
        <body>
          <div id="app">${html}</div>
          <script>window.__PINIA_STATE__ = ${state}</script>
          <script type="module" src="/entry-client.js"></script>
        </body>
      </html>
    `
    
    // 缓存结果
    cache.set(cacheKey, fullHtml)
    
    res.end(fullHtml)
  } catch (error) {
    console.error(error)
    res.status(500).end('Internal Server Error')
  }
})

流式渲染 ​

使用 renderToNodeStream ​

javascript
// entry-server.js
import { renderToNodeStream } from '@vue/server-renderer'

export function renderStream(url) {
  const { app, router } = createApp()
  
  return router.push(url)
    .then(() => router.isReady())
    .then(() => renderToNodeStream(app))
}
javascript
// server.js
app.get('*', async (req, res) => {
  try {
    const stream = await renderStream(req.originalUrl)
    
    res.write(`
      <!DOCTYPE html>
      <html>
        <head>
          <title>Vue SSR App</title>
        </head>
        <body>
          <div id="app">
    `)
    
    stream.pipe(res, { end: false })
    
    stream.on('end', () => {
      res.end(`
          </div>
          <script type="module" src="/entry-client.js"></script>
        </body>
      </html>
      `)
    })
    
    stream.on('error', (error) => {
      console.error(error)
      res.status(500).end('Internal Server Error')
    })
  } catch (error) {
    console.error(error)
    res.status(500).end('Internal Server Error')
  }
})

错误处理 ​

服务端错误处理 ​

javascript
// entry-server.js
export async function render(url) {
  try {
    const { app, router } = createApp()
    
    await router.push(url)
    await router.isReady()
    
    // 检查是否匹配到路由
    if (router.currentRoute.value.matched.length === 0) {
      throw new Error('404 - Page Not Found')
    }
    
    const html = await renderToString(app)
    return { html, statusCode: 200 }
  } catch (error) {
    console.error('SSR Error:', error)
    
    if (error.message.includes('404')) {
      return { 
        html: '<div>Page Not Found</div>', 
        statusCode: 404 
      }
    }
    
    return { 
      html: '<div>Internal Server Error</div>', 
      statusCode: 500 
    }
  }
}

import { ref, onErrorCaptured } from 'vue'

const error = ref(null)

  error.value = err
  return false // 阻止错误继续传播
})

function retry() {
  error.value = null
}
</script>

性能监控 ​

javascript
// server.js
import { performance } from 'perf_hooks'

app.get('*', async (req, res) => {
  const start = performance.now()
  
  try {
    const html = await render(req.originalUrl)
    const duration = performance.now() - start
    
    // 记录性能指标
    console.log(`SSR ${req.originalUrl}: ${duration.toFixed(2)}ms`)
    
    res.set('Server-Timing', `ssr;dur=${duration}`)
    res.end(html)

})
