KeepAlive 组件缓存 ​

<KeepAlive> 是 Vue 3 的内置组件，用于缓存组件实例，避免重复创建和销毁，提高应用性能。

基础用法 ​

缓存动态组件 ​

vue
<template>
  <div>
    <button 
      v-for="tab in tabs" 
      :key="tab"
      @click="currentTab = tab"
      :class="{ active: currentTab === tab }"
    >
      {{ tab }}
    </button>
    
    <KeepAlive>
      <component :is="currentTab" />
    </KeepAlive>
  </div>
</template>

<script setup>
import { ref } from 'vue'
import TabA from './TabA.vue'
import TabB from './TabB.vue'
import TabC from './TabC.vue'

const currentTab = ref('TabA')
const tabs = ['TabA', 'TabB', 'TabC']
</script>

缓存条件渲染 ​

vue
<template>
  <div>
    <button @click="show = !show">切换组件</button>
    
    <KeepAlive>
      <ExpensiveComponent v-if="show" />
    </KeepAlive>
  </div>
</template>

<script setup>
import { ref } from 'vue'
import ExpensiveComponent from './ExpensiveComponent.vue'

const show = ref(true)
</script>

包含/排除组件 ​

include 属性 ​

只缓存指定的组件：

vue
<template>
  <!-- 只缓存名为 TabA 和 TabB 的组件 -->
  <KeepAlive include="TabA,TabB">
    <component :is="currentTab" />
  </KeepAlive>
  
  <!-- 使用数组 -->
  <KeepAlive :include="['TabA', 'TabB']">
    <component :is="currentTab" />
  </KeepAlive>
  
  <!-- 使用正则表达式 -->
  <KeepAlive :include="/Tab[AB]/">
    <component :is="currentTab" />
  </KeepAlive>
</template>

exclude 属性 ​

排除指定的组件：

vue
<template>
  <!-- 不缓存名为 TabC 的组件 -->
  <KeepAlive exclude="TabC">
    <component :is="currentTab" />
  </KeepAlive>
  
  <!-- 使用数组 -->
  <KeepAlive :exclude="['TabC', 'TabD']">
    <component :is="currentTab" />
  </KeepAlive>
</template>

组件名称定义 ​

vue
<!-- TabA.vue -->
<script setup>
// 使用 defineOptions 定义组件名称
defineOptions({
  name: 'TabA'
})
</script>

<!-- 或者使用传统方式 -->
<script>
export default {
  name: 'TabA'
}
</script>

最大缓存数量 ​

使用 max 属性限制缓存的组件数量：

vue
<template>
  <!-- 最多缓存 3 个组件实例 -->
  <KeepAlive :max="3">
    <component :is="currentTab" />
  </KeepAlive>
</template>

当缓存数量超过限制时，最久未被访问的组件实例会被销毁。

生命周期钩子 ​

activated 和 deactivated ​

被 <KeepAlive> 缓存的组件有特殊的生命周期钩子：

vue
<template>
  <div>
    <h3>缓存组件</h3>
    <p>访问次数: {{ visitCount }}</p>
    <p>当前时间: {{ currentTime }}</p>
  </div>
</template>

<script setup>
import { ref, onMounted, onActivated, onDeactivated } from 'vue'

const visitCount = ref(0)
const currentTime = ref('')

onMounted(() => {
  console.log('组件挂载')
  updateTime()
})

onActivated(() => {
  console.log('组件激活')
  visitCount.value++
  updateTime()
})

onDeactivated(() => {
  console.log('组件失活')
})

function updateTime() {
  currentTime.value = new Date().toLocaleTimeString()
}
</script>

选项式 API ​

vue
<script>
export default {
  data() {
    return {
      visitCount: 0
    }
  },
  
  mounted() {
    console.log('组件挂载')
  },
  
  activated() {
    console.log('组件激活')
    this.visitCount++
  },
  
  deactivated() {
    console.log('组件失活')
  }
}
</script>

2. 路由缓存 ​

vue
<!-- App.vue -->
<template>
  <div id="app">
    <nav>
      <router-link to="/home">首页</router-link>
      <router-link to="/about">关于</router-link>
      <router-link to="/contact">联系</router-link>
    </nav>
    
    <router-view v-slot="{ Component, route }">
      <KeepAlive :include="cachedViews">
        <component :is="Component" :key="route.fullPath" />
      </KeepAlive>
    </router-view>
  </div>
</template>

<script setup>
import { computed } from 'vue'
import { useRoute } from 'vue-router'

const route = useRoute()

// 根据路由元信息决定是否缓存
const cachedViews = computed(() => {
  return route.meta.keepAlive ? [route.name] : []
})
</script>
javascript
// router/index.js
const routes = [
  {
    path: '/home',
    name: 'Home',
    component: () => import('../views/Home.vue'),
    meta: { keepAlive: true }
  },
  {
    path: '/about',
    name: 'About',
    component: () => import('../views/About.vue'),
    meta: { keepAlive: false }
  }
]

3. 表单缓存 ​

vue
<template>
  <div>
    <div class="form-tabs">
      <button
        v-for="step in steps"
        :key="step.id"
        @click="currentStep = step.id"
        :class="{ active: currentStep === step.id }"
      >
        {{ step.title }}
      </button>
    </div>
    
    <!-- 缓存表单组件，保持用户输入 -->
    <KeepAlive>
      <component :is="currentStepComponent" @next="nextStep" @prev="prevStep" />
    </KeepAlive>
  </div>
</template>

<script setup>
import { ref, computed } from 'vue'
import StepOne from './StepOne.vue'
import StepTwo from './StepTwo.vue'
import StepThree from './StepThree.vue'

const currentStep = ref(1)

const steps = [
  { id: 1, title: '基本信息', component: 'StepOne' },
  { id: 2, title: '详细信息', component: 'StepTwo' },
  { id: 3, title: '确认提交', component: 'StepThree' }
]

const currentStepComponent = computed(() => {
  const step = steps.find(s => s.id === currentStep.value)
  return step?.component
})

function nextStep() {
  if (currentStep.value < steps.length) {
    currentStep.value++
  }
}

function prevStep() {
  if (currentStep.value > 1) {
    currentStep.value--
  }
}
</script>

<template>

<script setup>
import { ref } from 'vue'

const enableCache = ref(true)
const currentComponent = ref('ComponentA')
</script>

性能优化 ​

1. 合理使用缓存 ​

vue
<template>
  <!-- ✅ 适合缓存：复杂的列表组件 -->
  <KeepAlive include="UserList,ProductList">
    <component :is="currentList" />
  </KeepAlive>
  
  <!-- ❌ 不适合缓存：简单的静态组件 -->
  <KeepAlive>
    <SimpleText />
  </KeepAlive>
</template>

2. 限制缓存数量 ​

vue
<template>
  <!-- 限制最多缓存 5 个组件 -->
  <KeepAlive :max="5">
    <component :is="currentComponent" />
  </KeepAlive>
</template>

3. 清理缓存数据 ​

vue
<script setup>
import { onDeactivated } from 'vue'

// 在组件失活时清理不必要的数据
onDeactivated(() => {
  // 清理定时器
  clearInterval(timer)
  
  // 清理事件监听器
  window.removeEventListener('resize', handleResize)
  
  // 清理大量数据
  largeDataArray.value = []
})
</script>

最佳实践 ​

  1. 明确缓存目标：只缓存真正需要的组件
  2. 设置合理的 max 值：避免内存泄漏
  3. 使用 include/exclude：精确控制缓存范围
  4. 处理生命周期：在 activated/deactivated 中处理状态
  5. 清理资源：在失活时清理不必要的资源

总结 ​

<KeepAlive> 是 Vue 3 中强大的组件缓存功能：

  1. 提高性能：避免重复创建和销毁组件
  2. 保持状态：保留组件的状态和用户输入
  3. 灵活控制：支持 include/exclude 和 max 属性
  4. 生命周期：提供 activated/deactivated 钩子
  5. 内存管理：合理使用避免内存泄漏

合理使用 <KeepAlive> 可以显著提升应用的用户体验和性能。
