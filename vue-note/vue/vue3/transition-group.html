TransitionGroup 列表过渡 ​

<TransitionGroup> 是 Vue 3 中专门用于处理列表过渡动画的组件，它可以为列表中的元素添加进入、离开和移动动画。

基础用法 ​

vue
<template>
  <div>
    <button @click="addItem">添加</button>
    <button @click="removeItem">删除</button>
    <button @click="shuffle">打乱</button>
  </div>
  
  <TransitionGroup name="list" tag="ul">
    <li v-for="item in items" :key="item.id" class="list-item">
      {{ item.text }}
      <button @click="remove(item.id)">×</button>
    </li>
  </TransitionGroup>
</template>

<script setup>
import { ref } from 'vue'

const items = ref([
  { id: 1, text: '项目 1' },
  { id: 2, text: '项目 2' },
  { id: 3, text: '项目 3' }
])

let nextId = 4

function addItem() {
  items.value.push({
    id: nextId++,
    text: `项目 ${nextId - 1}`
  })
}

function removeItem() {
  if (items.value.length > 0) {
    items.value.pop()
  }
}

function remove(id) {
  const index = items.value.findIndex(item => item.id === id)
  if (index > -1) {
    items.value.splice(index, 1)
  }
}

function shuffle() {
  items.value = items.value.sort(() => Math.random() - 0.5)
}
</script>

过渡类名 ​

进入/离开过渡 ​

与 <Transition> 相同的类名：

  • v-enter-from / v-enter-active / v-enter-to
  • v-leave-from / v-leave-active / v-leave-to

移动过渡 ​

  • v-move：元素位置改变时的过渡类名

JavaScript 钩子 ​

vue
<template>
  <TransitionGroup
    name="stagger"
    tag="div"
    @before-enter="onBeforeEnter"
    @enter="onEnter"
    @leave="onLeave"
  >
    <div
      v-for="(item, index) in items"
      :key="item.id"
      :data-index="index"
      class="stagger-item"
    >
      {{ item.text }}
    </div>
  </TransitionGroup>
</template>

<script setup>
function onBeforeEnter(el) {
  el.style.opacity = 0
  el.style.transform = 'translateY(30px)'
}

function onEnter(el, done) {
  const delay = el.dataset.index * 150
  
  setTimeout(() => {
    el.style.transition = 'all 0.5s ease'
    el.style.opacity = 1
    el.style.transform = 'translateY(0)'
    
    setTimeout(done, 500)
  }, delay)
}

function onLeave(el, done) {
  const delay = el.dataset.index * 150
  
  setTimeout(() => {
    el.style.transition = 'all 0.5s ease'
    el.style.opacity = 0
    el.style.transform = 'translateY(-30px)'
    
    setTimeout(done, 500)
  }, delay)
}
</script>

实际应用示例 ​

1. 待办事项列表 ​

vue
<template>
  <div class="todo-app">
    <div class="input-section">
      <input
        v-model="newTodo"
        @keyup.enter="addTodo"
        placeholder="添加新任务..."
      />
      <button @click="addTodo">添加</button>
    </div>
    
    <TransitionGroup name="todo" tag="ul" class="todo-list">
      <li
        v-for="todo in todos"
        :key="todo.id"
        class="todo-item"
        :class="{ completed: todo.completed }"
      >
        <input
          type="checkbox"
          v-model="todo.completed"
        />
        <span class="todo-text">{{ todo.text }}</span>
        <button @click="removeTodo(todo.id)" class="delete-btn">删除</button>
      </li>
    </TransitionGroup>
  </div>
</template>

<script setup>
import { ref } from 'vue'

const newTodo = ref('')
const todos = ref([])
let nextId = 1

function addTodo() {
  if (newTodo.value.trim()) {
    todos.value.push({
      id: nextId++,
      text: newTodo.value,
      completed: false
    })
    newTodo.value = ''
  }
}

function removeTodo(id) {
  const index = todos.value.findIndex(todo => todo.id === id)
  if (index > -1) {
    todos.value.splice(index, 1)
  }
}
</script>

<template>

性能优化 ​

1. 虚拟滚动 ​

对于大量列表项，考虑使用虚拟滚动：

vue
<template>
  <div class="virtual-list" ref="container">
    <TransitionGroup name="virtual" tag="div" :style="containerStyle">
      <div
        v-for="item in visibleItems"
        :key="item.id"
        class="virtual-item"
        :style="getItemStyle(item)"
      >
        {{ item.text }}
      </div>
    </TransitionGroup>
  </div>
</template>

<script setup>
import { ref, computed, onMounted } from 'vue'

const container = ref()
const scrollTop = ref(0)
const itemHeight = 50
const visibleCount = 10

const items = ref(Array.from({ length: 1000 }, (_, i) => ({
  id: i,
  text: `项目 ${i + 1}`
})))

const visibleItems = computed(() => {
  const start = Math.floor(scrollTop.value / itemHeight)
  const end = start + visibleCount
  return items.value.slice(start, end)
})

const containerStyle = computed(() => ({
  height: `${items.value.length * itemHeight}px`,
  position: 'relative'
}))

function getItemStyle(item) {
  const index = items.value.indexOf(item)
  return {
    position: 'absolute',
    top: `${index * itemHeight}px`,
    height: `${itemHeight}px`,
    width: '100%'
  }
}

onMounted(() => {
  container.value.addEventListener('scroll', (e) => {
    scrollTop.value = e.target.scrollTop
  })
})
</script>

2. 减少动画复杂度 ​

css
/* ✅ 高性能动画 */
.optimized-list-enter-active,
.optimized-list-leave-active {
  transition: transform 0.3s ease, opacity 0.3s ease;
}

.optimized-list-enter-from,
.optimized-list-leave-to {
  opacity: 0;
  transform: translateX(30px);
}

/* ❌ 低性能动画 */
.expensive-list-enter-active,
.expensive-list-leave-active {
  transition: all 0.3s ease; /* 避免使用 all */
}

最佳实践 ​

  1. 使用唯一 key：确保每个列表项都有唯一的 key
  2. 合理的动画时长：不要过长，通常 0.3s-0.5s 比较合适
  3. 避免复杂动画：在大列表中使用简单的动画效果
  4. 考虑性能：对于大量数据考虑虚拟滚动
  5. 测试用户体验：确保动画不会影响用户操作

总结 ​

<TransitionGroup> 为列表动画提供了强大的功能：

  1. 自动管理：自动处理列表项的进入、离开和移动
  2. 灵活配置：支持 CSS 和 JavaScript 钩子
  3. 性能优化：合理使用可以提供流畅的用户体验
  4. 实用场景：适用于各种列表交互场景

合理使用列表过渡可以大大提升应用的交互体验。
