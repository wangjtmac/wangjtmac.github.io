uView UI 主题定制 ​

uView UI 提供了灵活的主题定制功能，允许开发者根据项目需求自定义组件的样式和主题。通过主题定制，可以使应用具有统一的视觉风格，提升用户体验。本章将详细介绍 uView UI 的主题定制方法和最佳实践。

主题变量系统 ​

uView UI 的主题系统基于 SCSS 变量，通过修改这些变量，可以轻松定制组件的样式。

基本使用方法 ​

  1. 在项目根目录下创建 uni.scss 文件（如果已存在则直接编辑）
  2. 在 uni.scss 文件中引入 uView UI 的主题变量
  3. 覆盖需要自定义的变量
scss
/* uni.scss */

// 引入 uView UI 主题变量
@import 'uview-ui/theme.scss';

// 覆盖主题变量
// 主色
$u-primary: #3c9cff;
$u-warning: #ff9900;
$u-success: #19be6b;
$u-error: #fa3534;
$u-info: #909399;

// 文字颜色
$u-main-color: #303133;
$u-content-color: #606266;
$u-tips-color: #909399;
$u-light-color: #c0c4cc;

// 背景颜色
$u-bg-color: #f3f4f6;

// 边框颜色
$u-border-color: #e4e7ed;

组件样式定制 ​

除了通过主题变量定制全局样式外，uView UI 还支持通过 props 和 customStyle 属性定制单个组件的样式。

通过 Props 定制 ​

大多数 uView UI 组件都提供了样式相关的 props，可以通过这些 props 定制组件的样式：

vue
<template>
  <view>
    <!-- 定制按钮样式 -->
    <u-button 
      text="自定义按钮" 
      color="#8f2d56" 
      size="large" 
      shape="circle"
    ></u-button>
    
    <!-- 定制文本样式 -->
    <u-text 
      text="自定义文本" 
      color="#8f2d56" 
      size="32" 
      bold 
      decoration="underline"
    ></u-text>
    
    <!-- 定制徽标样式 -->
    <u-badge 
      value="99+" 
      type="error" 
      size="mini" 
      offset="[-10, -10]"
    ></u-badge>
  </view>
</template>

通过 customStyle 定制 ​

对于需要更加灵活的样式定制，可以使用 customStyle 属性：

vue
<template>
  <view>
    <!-- 使用 customStyle 定制按钮样式 -->
    <u-button 
      text="自定义按钮" 
      :customStyle="{
        marginTop: '20rpx',
        width: '400rpx',
        height: '100rpx',
        background: 'linear-gradient(to right, #3c9cff, #5ac8fa)',
        boxShadow: '0 10rpx 20rpx rgba(60, 156, 255, 0.3)'
      }"
    ></u-button>
    
    <!-- 使用 customStyle 定制文本样式 -->
    <u-text 
      text="自定义文本" 
      :customStyle="{
        textShadow: '2rpx 2rpx 5rpx rgba(0, 0, 0, 0.3)',
        letterSpacing: '5rpx',
        fontStyle: 'italic'
      }"
    ></u-text>
  </view>
</template>

全局样式定制 ​

如果需要对多个组件进行统一的样式定制，可以通过全局样式文件进行定制。

创建全局样式文件 ​

  1. 在项目中创建一个全局样式文件，例如 common/style/custom.scss
  2. 在 App.vue 中引入该样式文件
scss
/* common/style/custom.scss */

/* 定制按钮样式 */
.u-button {
  &--primary {
    background-color: #8f2d56 !important;
    
    &--plain {
      color: #8f2d56 !important;
      border-color: #8f2d56 !important;
    }
  }
  
  &--large {
    height: 100rpx !important;
    line-height: 100rpx !important;
  }
}

/* 定制文本样式 */
.u-text {
  &__value {
    font-family: 'Arial', sans-serif !important;
  }
}

/* 定制单元格样式 */
.u-cell {
  &__title {
    color: #8f2d56 !important;
  }
  
  &__value {
    color: #606266 !important;
  }
}
vue
<!-- App.vue -->
<style lang="scss">
@import "uview-ui/index.scss";
@import "common/style/custom.scss";
</style>

主题切换 ​

uView UI 支持动态切换主题，可以实现明暗主题切换等功能。

基于 CSS 变量的主题切换 ​

  1. 定义不同主题的 CSS 变量
scss
/* common/style/theme.scss */

/* 明亮主题 */
:root {
  --primary-color: #3c9cff;
  --bg-color: #ffffff;
  --text-color: #303133;
  --border-color: #e4e7ed;
}

/* 暗黑主题 */
[data-theme="dark"] {
  --primary-color: #3c9cff;
  --bg-color: #1b1b1b;
  --text-color: #f5f5f5;
  --border-color: #3a3a3a;
}
  1. 在组件中使用 CSS 变量
vue
<template>
  <view class="custom-component">
    <text class="custom-text">文本内容</text>
    <view class="custom-box">盒子内容</view>
  </view>
</template>

<style lang="scss">
.custom-component {
  background-color: var(--bg-color);
  
  .custom-text {
    color: var(--text-color);
  }
  
  .custom-box {
    border: 1px solid var(--border-color);
  }
}
</style>
  1. 切换主题
js
// 切换为暗黑主题
document.documentElement.setAttribute('data-theme', 'dark');

// 切换为明亮主题
document.documentElement.setAttribute('data-theme', 'light');

基于 Vuex 的主题管理 ​

  1. 在 Vuex 中管理主题状态
js
// store/index.js
import Vue from 'vue';
import Vuex from 'vuex';

Vue.use(Vuex);

export default new Vuex.Store({
  state: {
    theme: uni.getStorageSync('theme') || 'light'
  },
  mutations: {
    setTheme(state, theme) {
      state.theme = theme;
      uni.setStorageSync('theme', theme);
      
      // 设置 data-theme 属性
      document.documentElement.setAttribute('data-theme', theme);
    }
  },
  actions: {
    toggleTheme({ commit, state }) {
      const newTheme = state.theme === 'light' ? 'dark' : 'light';
      commit('setTheme', newTheme);
    }
  }
});
  1. 在组件中使用主题切换功能
vue
<template>
  <view>
    <u-button text="切换主题" @click="toggleTheme"></u-button>
  </view>
</template>

<script>
import { mapState, mapActions } from 'vuex';

export default {
  computed: {
    ...mapState(['theme'])
  },
  methods: {
    ...mapActions(['toggleTheme'])
  },
  onLoad() {
    // 初始化主题
    document.documentElement.setAttribute('data-theme', this.theme);
  }
}
</script>

自定义组件样式 ​

对于需要深度定制的组件，可以使用 ::v-deep 或 /deep/ 选择器进行样式覆盖。

vue
<template>
  <view class="custom-form">
    <u-form :model="form" ref="uForm">
      <u-form-item label="姓名" prop="name">
        <u-input v-model="form.name" placeholder="请输入姓名"></u-input>
      </u-form-item>
      
      <u-form-item label="手机号" prop="phone">
        <u-input v-model="form.phone" placeholder="请输入手机号"></u-input>
      </u-form-item>
    </u-form>
  </view>
</template>

<style lang="scss">
.custom-form {
  /* 使用 ::v-deep 选择器修改组件内部样式 */
  ::v-deep .u-form-item {
    margin-bottom: 30rpx;
    
    &__label {
      color: #8f2d56;
      font-weight: bold;
    }
    
    &__body {
      background-color: #f8f8f8;
      border-radius: 8rpx;
    }
  }
  
  /* 或者使用 /deep/ 选择器 */
  /deep/ .u-input {
    height: 80rpx;
    
    &__input {
      font-size: 28rpx;
      color: #606266;
    }
  }
}
</style>

主题定制最佳实践 ​

1. 建立设计规范 ​

在开始主题定制前，建立一套完整的设计规范，包括颜色、字体、圆角、间距等，确保应用的视觉风格统一。

2. 使用变量管理样式 ​

将所有样式值都使用变量管理，避免硬编码，便于后期维护和调整。

scss
/* common/style/variables.scss */

// 主题颜色
$primary-color: #3c9cff;
$success-color: #19be6b;
$warning-color: #ff9900;
$error-color: #fa3534;
$info-color: #909399;

// 文字颜色
$text-main-color: #303133;
$text-content-color: #606266;
$text-tips-color: #909399;
$text-light-color: #c0c4cc;

// 背景颜色
$bg-color: #f3f4f6;
$bg-color-light: #fafafa;

// 边框颜色
$border-color: #e4e7ed;

// 字体大小
$font-size-xs: 22rpx;
$font-size-sm: 24rpx;
$font-size-base: 28rpx;
$font-size-lg: 32rpx;
$font-size-xl: 36rpx;

// 圆角
$radius-sm: 4rpx;
$radius-base: 6rpx;
$radius-lg: 12rpx;
$radius-circle: 100rpx;

// 间距
$gap-xs: 10rpx;
$gap-sm: 20rpx;
$gap-base: 30rpx;
$gap-lg: 40rpx;
$gap-xl: 50rpx;

3. 分层管理样式 ​

将样式按照功能和作用范围分层管理，便于维护和扩展。

styles/
├── base/              # 基础样式
│   ├── reset.scss     # 重置样式
│   ├── variables.scss # 变量定义
│   └── mixins.scss    # 混合宏
├── themes/            # 主题样式
│   ├── light.scss     # 明亮主题
│   └── dark.scss      # 暗黑主题
├── components/        # 组件样式
│   ├── button.scss    # 按钮样式
│   ├── form.scss      # 表单样式
│   └── ...
└── index.scss         # 样式入口文件

4. 使用 Mixin 和函数 ​

使用 SCSS 的 Mixin 和函数，减少重复代码，提高样式的复用性。

scss
/* common/style/mixins.scss */

// 文本溢出省略号
@mixin text-ellipsis($lines: 1) {
  @if $lines == 1 {
    overflow: hidden;
    text-overflow: ellipsis;
    white-space: nowrap;
  } @else {
    display: -webkit-box;
    -webkit-box-orient: vertical;
    -webkit-line-clamp: $lines;
    overflow: hidden;
  }
}

// 弹性布局
@mixin flex($direction: row, $justify: flex-start, $align: center, $wrap: nowrap) {
  display: flex;
  flex-direction: $direction;
  justify-content: $justify;
  align-items: $align;
  flex-wrap: $wrap;
}

// 定位
@mixin position($position, $top: null, $right: null, $bottom: null, $left: null) {
  position: $position;
  top: $top;
  right: $right;
  bottom: $bottom;
  left: $left;
}

// 使用示例
.custom-text {
  @include text-ellipsis(2);
}

.custom-flex {
  @include flex(row, space-between, center);
}

.custom-position {
  @include position(absolute, 0, 0, 0, 0);
}

5. 组件样式隔离 ​

使用 scoped 或命名空间，避免样式冲突。

vue
<template>
  <view class="custom-component">
    <!-- 组件内容 -->
  </view>
</template>

<style lang="scss" scoped>
.custom-component {
  // 组件样式
}
</style>

6. 响应式设计 ​

使用 uView UI 的响应式工具，适配不同屏幕尺寸。

scss
/* 响应式设计 */
@media screen and (min-width: 768px) {
  .custom-component {
    padding: 40rpx;
  }
}

@media screen and (max-width: 767px) {
  .custom-component {
    padding: 20rpx;
  }
}
