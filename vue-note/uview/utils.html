uView UI 工具库 ​

uView UI 除了提供丰富的组件外，还提供了一系列实用的工具函数和工具类，帮助开发者更高效地开发应用。本章将详细介绍 uView UI 工具库的使用方法和最佳实践。

工具函数 ​

uView UI 提供了多种工具函数，涵盖了日期格式化、节流防抖、深拷贝等常用功能。

基本使用方法 ​

在使用 uView UI 的工具函数之前，需要先导入：

js
// 导入整个工具库
import { $u } from 'uview-ui';

// 使用工具函数
$u.toast('提示内容');

或者在组件中直接使用：

vue
<script>
export default {
  onLoad() {
    // 在组件中使用
    this.$u.toast('提示内容');
  }
}
</script>

提示类函数 ​

uView UI 提供了多种提示类函数，用于显示提示信息：

js
// 显示消息提示
this.$u.toast('提示内容');

// 显示成功提示
this.$u.toast('操作成功', 'success');

// 显示错误提示
this.$u.toast('操作失败', 'error');

// 显示加载提示
this.$u.toast('加载中', 'loading');

// 显示带图标的提示
this.$u.toast('提示内容', 'none', '图标名称');

// 显示模态框
this.$u.modal({
  title: '标题',
  content: '内容',
  showCancelButton: true,
  confirmText: '确定',
  cancelText: '取消',
  confirmColor: '#2979ff',
  cancelColor: '#606266',
  success: (res) => {
    if (res.confirm) {
      console.log('用户点击确定');
    } else if (res.cancel) {
      console.log('用户点击取消');
    }
  }
});

路由跳转函数 ​

uView UI 封装了路由跳转相关的函数，使页面跳转更加便捷：

js
// 普通页面跳转
this.$u.route('/pages/index/index');

// 带参数的页面跳转
this.$u.route('/pages/index/index', {
  id: 1,
  name: 'uView'
});

// 页面跳转方式
this.$u.route({
  url: '/pages/index/index',
  params: {
    id: 1,
    name: 'uView'
  },
  type: 'navigateTo', // 可选值：navigateTo、redirectTo、switchTab、reLaunch
  animationType: 'pop-in', // 动画类型
  animationDuration: 300 // 动画持续时间
});

// 返回上一页
this.$u.route({
  type: 'back',
  delta: 1 // 返回的页面数
});

时间格式化函数 ​

uView UI 提供了时间格式化函数，可以将时间戳转换为指定格式的时间字符串：

js
// 格式化时间戳
const timestamp = Date.now();
const dateStr = this.$u.timeFormat(timestamp, 'yyyy-mm-dd');
console.log(dateStr); // 例如：2023-07-21

// 自定义格式
const timeStr = this.$u.timeFormat(timestamp, 'yyyy年mm月dd日 hh:MM:ss');
console.log(timeStr); // 例如：2023年07月21日 15:30:45

// 时间格式说明：
// yyyy：年份，如 2023
// mm：月份，如 07
// dd：日期，如 21
// hh：小时，如 15
// MM：分钟，如 30
// ss：秒数，如 45
// q：季度，如 3
// S：毫秒，如 789

节流防抖函数 ​

uView UI 提供了节流和防抖函数，用于优化高频事件的处理：

js
// 防抖函数
// 在用户停止操作一段时间后才执行函数，适用于搜索框输入等场景
const debounceFunction = this.$u.debounce(() => {
  console.log('执行搜索');
}, 500);

// 节流函数
// 在一定时间内只执行一次函数，适用于滚动事件等场景
const throttleFunction = this.$u.throttle(() => {
  console.log('执行滚动处理');
}, 200);

// 使用方式
input.addEventListener('input', debounceFunction);
window.addEventListener('scroll', throttleFunction);

对象深度克隆 ​

uView UI 提供了对象深度克隆函数，用于复制复杂对象：

js
const obj = {
  name: 'uView',
  info: {
    version: '2.0.31',
    author: 'uView UI'
  },
  array: [1, 2, 3]
};

// 深度克隆对象
const newObj = this.$u.deepClone(obj);
newObj.info.version = '2.0.32';

console.log(obj.info.version); // 2.0.31
console.log(newObj.info.version); // 2.0.32

请求库 ​

uView UI 提供了基于 Promise 的网络请求库，封装了 uni.request，支持请求和响应拦截、全局配置等功能。

基本使用方法 ​

js
// 设置请求基地址
this.$u.http.setConfig({
  baseUrl: 'https://api.example.com',
  header: {
    'content-type': 'application/json'
  },
  dataType: 'json',
  showLoading: true,
  loadingText: '请求中...',
  loadingTime: 800,
  originalData: false,
  loadingMask: true
});

// 请求拦截器
this.$u.http.interceptor.request = (config) => {
  // 获取 token
  const token = uni.getStorageSync('token');
  // 添加 token 到请求头
  config.header.token = token;
  return config;
};

// 响应拦截器
this.$u.http.interceptor.response = (response) => {
  // 状态码判断
  if (response.statusCode === 200) {
    // 请求成功
    if (response.data.code === 0) {
      return response.data.data;
    } else {
      // 业务错误
      this.$u.toast(response.data.message);
      return Promise.reject(response.data);
    }
  } else if (response.statusCode === 401) {
    // 未授权，跳转登录
    uni.navigateTo({
      url: '/pages/login/login'
    });
    return Promise.reject(response);
  } else {
    // 其他错误
    this.$u.toast('网络异常，请稍后再试');
    return Promise.reject(response);
  }
};

// GET 请求
this.$u.http.get('/user/info', {
  id: 1
}).then(res => {
  console.log(res);
}).catch(err => {
  console.error(err);
});

// POST 请求
this.$u.http.post('/user/login', {
  username: 'admin',
  password: '123456'
}).then(res => {
  console.log(res);
}).catch(err => {
  console.error(err);
});

// PUT 请求
this.$u.http.put('/user/update', {
  id: 1,
  name: 'uView'
}).then(res => {
  console.log(res);
}).catch(err => {
  console.error(err);
});

// DELETE 请求
this.$u.http.delete('/user/delete', {
  id: 1
}).then(res => {
  console.log(res);
}).catch(err => {
  console.error(err);
});

// 自定义请求
this.$u.http.request({
  url: '/user/info',
  method: 'GET',
  data: {
    id: 1
  },
  header: {
    'custom-header': 'custom-value'
  },
  dataType: 'json',
  responseType: 'text',
  showLoading: true,
  loadingText: '加载中...',
  loadingTime: 800,
  loadingMask: true
}).then(res => {
  console.log(res);
}).catch(err => {
  console.error(err);
});

表单验证 ​

uView UI 提供了表单验证功能，可以对表单数据进行验证。

基本使用方法 ​

vue
<template>
  <view>
    <u-form :model="form" ref="uForm">
      <u-form-item label="姓名" prop="name">
        <u-input v-model="form.name" placeholder="请输入姓名"></u-input>
      </u-form-item>
      
      <u-form-item label="手机号" prop="phone">
        <u-input v-model="form.phone" placeholder="请输入手机号"></u-input>
      </u-form-item>
      
      <u-form-item label="邮箱" prop="email">
        <u-input v-model="form.email" placeholder="请输入邮箱"></u-input>
      </u-form-item>
    </u-form>
    
    <u-button text="提交" type="primary" @click="submit"></u-button>
  </view>
</template>

<script>
export default {
  data() {
    return {
      form: {
        name: '',
        phone: '',
        email: ''
      },
      rules: {
        name: [
          { required: true, message: '请输入姓名', trigger: 'blur' },
          { min: 2, max: 20, message: '姓名长度在2-20个字符之间', trigger: 'blur' }
        ],
        phone: [
          { required: true, message: '请输入手机号', trigger: 'blur' },
          { pattern: /^1[3-9]\d{9}$/, message: '请输入正确的手机号', trigger: 'blur' }
        ],
        email: [
          { required: true, message: '请输入邮箱', trigger: 'blur' },
          { type: 'email', message: '请输入正确的邮箱地址', trigger: 'blur' }
        ]
      }
    }
  },
  onReady() {
    this.$refs.uForm.setRules(this.rules);
  },
  methods: {
    submit() {
      this.$refs.uForm.validate(valid => {
        if (valid) {
          console.log('表单验证通过');
          console.log('表单数据：', this.form);
          this.$u.toast('表单验证通过');
        } else {
          console.log('表单验证失败');
        }
      });
    }
  }
}
</script>

自定义验证规则 ​

可以自定义表单验证规则：

js
// 在 data 中定义验证规则
data() {
  return {
    form: {
      password: '',
      confirmPassword: ''
    },
    rules: {
      password: [
        { required: true, message: '请输入密码', trigger: 'blur' },
        { min: 6, max: 20, message: '密码长度在6-20个字符之间', trigger: 'blur' }
      ],
      confirmPassword: [
        { required: true, message: '请确认密码', trigger: 'blur' },
        {
          validator: (rule, value, callback) => {
            if (value !== this.form.password) {
              callback(new Error('两次输入的密码不一致'));
            } else {
              callback();
            }
          },
          trigger: 'blur'
        }
      ]
    }
  }
}
