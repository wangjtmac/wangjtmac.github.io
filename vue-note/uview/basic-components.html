uView UI 基础组件 ​

基础组件是构建应用界面的基础元素，uView UI提供了丰富的基础组件，包括按钮、图标、文本、布局等。本章将详细介绍这些基础组件的使用方法和最佳实践。

按钮组件 (u-button) ​

按钮组件是最常用的交互元素之一，用于触发操作。

基本用法 ​

vue
<template>
  <view>
    <!-- 基础按钮 -->
    <u-button text="默认按钮"></u-button>
    
    <!-- 带图标的按钮 -->
    <u-button text="图标按钮" icon="map"></u-button>
    
    <!-- 加载中的按钮 -->
    <u-button text="加载中" loading loading-text="加载中"></u-button>
    
    <!-- 禁用状态 -->
    <u-button text="禁用按钮" disabled></u-button>
  </view>
</template>

按钮类型 ​

uView UI提供了多种预设的按钮类型：

vue
<template>
  <view>
    <u-button text="主要按钮" type="primary"></u-button>
    <u-button text="信息按钮" type="info"></u-button>
    <u-button text="警告按钮" type="warning"></u-button>
    <u-button text="错误按钮" type="error"></u-button>
    <u-button text="成功按钮" type="success"></u-button>
  </view>
</template>

按钮形状 ​

按钮支持多种形状：

vue
<template>
  <view>
    <!-- 默认按钮 -->
    <u-button text="默认按钮" type="primary"></u-button>
    
    <!-- 圆形按钮 -->
    <u-button text="圆形按钮" type="primary" shape="circle"></u-button>
    
    <!-- 方形按钮 -->
    <u-button text="方形按钮" type="primary" shape="square"></u-button>
  </view>
</template>

按钮尺寸 ​

按钮支持多种尺寸：

vue
<template>
  <view>
    <u-button text="大号按钮" type="primary" size="large"></u-button>
    <u-button text="默认按钮" type="primary" size="normal"></u-button>
    <u-button text="小型按钮" type="primary" size="mini"></u-button>
  </view>
</template>

自定义样式 ​

可以通过props自定义按钮样式：

vue
<template>
  <view>
    <u-button 
      text="自定义按钮" 
      color="#8f2d56" 
      :custom-style="{
        marginTop: '20px',
        width: '200px',
        height: '80px'
      }"
    ></u-button>
  </view>
</template>

按钮事件 ​

按钮支持点击事件：

vue
<template>
  <view>
    <u-button text="点击按钮" @click="handleClick"></u-button>
  </view>
</template>

<script>
export default {
  methods: {
    handleClick() {
      console.log('按钮被点击');
      uni.showToast({
        title: '按钮被点击',
        icon: 'none'
      });
    }
  }
}
</script>

按钮组 ​

可以使用按钮组实现多个按钮的排列：

vue
<template>
  <view class="button-group">
    <u-button text="按钮1" type="primary" :custom-style="{ marginRight: '10px' }"></u-button>
    <u-button text="按钮2" type="info"></u-button>
  </view>
</template>

<style>
.button-group {
  display: flex;
  flex-direction: row;
}
</style>

图标组件 (u-icon) ​

图标是界面设计中不可或缺的元素，uView UI提供了丰富的图标集合。

基本用法 ​

vue
<template>
  <view>
    <u-icon name="photo"></u-icon>
    <u-icon name="level"></u-icon>
    <u-icon name="star"></u-icon>
  </view>
</template>

图标颜色 ​

可以自定义图标颜色：

vue
<template>
  <view>
    <u-icon name="photo" color="#3c9cff"></u-icon>
    <u-icon name="level" color="#f56c6c"></u-icon>
    <u-icon name="star" color="#f9ae3d"></u-icon>
  </view>
</template>

图标大小 ​

可以自定义图标大小：

vue
<template>
  <view>
    <u-icon name="photo" size="20"></u-icon>
    <u-icon name="level" size="30"></u-icon>
    <u-icon name="star" size="40"></u-icon>
  </view>
</template>

自定义图标 ​

可以使用自定义图标：

vue
<template>
  <view>
    <u-icon name="/static/icons/custom-icon.png" size="40"></u-icon>
  </view>
</template>

图标点击事件 ​

图标支持点击事件：

vue
<template>
  <view>
    <u-icon name="map" @click="handleIconClick"></u-icon>
  </view>
</template>

<script>
export default {
  methods: {
    handleIconClick() {
      console.log('图标被点击');
      uni.showToast({
        title: '图标被点击',
        icon: 'none'
      });
    }
  }
}
</script>

文本组件 (u-text) ​

文本组件用于展示各种文本内容，支持多种样式和效果。

基本用法 ​

vue
<template>
  <view>
    <u-text text="这是一段普通文本"></u-text>
    <u-text text="这是一段主要文本" type="primary"></u-text>
    <u-text text="这是一段警告文本" type="warning"></u-text>
    <u-text text="这是一段错误文本" type="error"></u-text>
    <u-text text="这是一段成功文本" type="success"></u-text>
    <u-text text="这是一段信息文本" type="info"></u-text>
  </view>
</template>

文本修饰 ​

文本支持多种修饰效果：

vue
<template>
  <view>
    <u-text text="下划线文本" decoration="underline"></u-text>
    <u-text text="删除线文本" decoration="line-through"></u-text>
    <u-text text="过长省略文本过长省略文本过长省略文本" mode="ellipsis"></u-text>
    <u-text text="价格文本" mode="price"></u-text>
  </view>
</template>

文本大小和粗细 ​

可以自定义文本大小和粗细：

vue
<template>
  <view>
    <u-text text="小号文本" size="12"></u-text>

    <u-text text="大号文本" size="18"></u-text>
    <u-text text="粗体文本" bold></u-text>
  </view>
</template>

文本图标 ​

文本可以配合图标使用：

vue
<template>

布局组件 (u-row, u-col) ​

uView UI提供了栅格布局系统，可以轻松实现复杂的页面布局。

基本用法 ​

vue
<template>
  <view>
    <u-row>
      <u-col span="6">
        <view class="demo-layout bg-purple">span: 6</view>
      </u-col>
      <u-col span="6">
        <view class="demo-layout bg-purple-light">span: 6</view>
      </u-col>
    </u-row>
    
    <u-row>
      <u-col span="4">
        <view class="demo-layout bg-purple">span: 4</view>
      </u-col>
      <u-col span="8">
        <view class="demo-layout bg-purple-light">span: 8</view>
      </u-col>
    </u-row>
  </view>
</template>

<style>
.demo-layout {
  height: 40px;
  border-radius: 4px;
  margin: 5px 0;
  text-align: center;
  line-height: 40px;
}
.bg-purple {
  background: #d3dce6;
}
.bg-purple-light {
  background: #e5e9f2;
}
</style>

栅格间隔 ​

可以设置栅格之间的间隔：

vue
<template>
  <view>
    <u-row gutter="10">
      <u-col span="6">
        <view class="demo-layout bg-purple">span: 6</view>
      </u-col>
      <u-col span="6">
        <view class="demo-layout bg-purple-light">span: 6</view>
      </u-col>
    </u-row>
  </view>
</template>

栅格偏移 ​

可以设置栅格的偏移量：

vue
<template>
  <view>
    <u-row>
      <u-col span="6" offset="3">
        <view class="demo-layout bg-purple">span: 6, offset: 3</view>
      </u-col>
    </u-row>
  </view>
</template>

对齐方式 ​

可以设置栅格的对齐方式：

vue
<template>
  <view>
    <u-row justify="start">
      <u-col span="4">
        <view class="demo-layout bg-purple">左对齐</view>
      </u-col>
    </u-row>
    
    <u-row justify="center">
      <u-col span="4">
        <view class="demo-layout bg-purple">居中对齐</view>
      </u-col>
    </u-row>
    
    <u-row justify="end">
      <u-col span="4">
        <view class="demo-layout bg-purple">右对齐</view>
      </u-col>
    </u-row>
  </view>
</template>

分割线组件 (u-divider) ​

分割线用于分隔内容区域，增强视觉效果和可读性。

基本用法 ​

vue
<template>
  <view>
    <u-divider></u-divider>
    <u-divider text="文本分割线"></u-divider>
    <u-divider text="虚线" dashed></u-divider>
    <u-divider text="自定义颜色" color="#2979ff"></u-divider>
    <u-divider text="自定义样式" textStyle="color: #909399; font-size: 14px"></u-divider>
  </view>
</template>

分割线方向 ​

分割线支持水平和垂直两种方向：

vue
<template>
  <view>
    <u-divider text="水平分割线" direction="row"></u-divider>
    
    <view style="height: 100px; display: flex;">
      <u-divider text="垂直分割线" direction="col"></u-divider>
    </view>
  </view>
</template>

徽标组件 (u-badge) ​

徽标组件用于展示新消息提示、数量统计等信息。

基本用法 ​

vue
<template>
  <view>
    <u-badge value="5"></u-badge>
    <u-badge value="10" type="primary"></u-badge>
    <u-badge value="Hot" type="error"></u-badge>
    <u-badge value="New" type="success"></u-badge>
    <u-badge dot type="warning"></u-badge>
  </view>
</template>

徽标位置 ​

徽标可以作为子组件使用，放置在其他组件上：

vue
<template>
  <view>
    <u-badge value="5" :offset="[20, 20]">
      <u-button text="按钮" type="primary"></u-button>
    </u-badge>
    
    <u-badge dot :offset="[10, 10]">
      <u-icon name="map" size="40"></u-icon>
    </u-badge>
  </view>
</template>

徽标最大值 ​

可以设置徽标的最大值，超过最大值会显示为"最大值+"：

vue
<template>
  <view>
    <u-badge value="99" max="99"></u-badge>
    <u-badge value="100" max="99"></u-badge>
    <u-badge value="1000" max="999"></u-badge>
  </view>
</template>

总结 ​

本章介绍了uView UI提供的基础组件，包括按钮、图标、文本、布局和分割线等。这些组件是构建应用界面的基础元素，掌握它们的使用方法对于快速开发高质量的UniApp应用至关重要。
