uView UI 数据展示组件 ​

数据展示组件用于向用户展示各种类型的信息，包括列表、卡片、轮播图、徽标等。这些组件帮助开发者以清晰、直观的方式呈现数据，提升用户体验。本章将详细介绍uView UI提供的数据展示组件的使用方法和最佳实践。

列表组件 (u-list, u-list-item) ​

列表组件用于展示一组相关的内容，支持下拉刷新和上拉加载更多功能。

基本用法 ​

vue
<template>
  <view>
    <u-list
      @scrolltolower="scrolltolower"
      @refresh="refresh"
      :enablePullDown="true"
      :enablePullUp="true"
    >
      <u-list-item v-for="(item, index) in list" :key="index">
        <view class="list-item">
          <image :src="item.avatar" class="avatar"></image>
          <view class="content">
            <view class="title">{{ item.title }}</view>
            <view class="desc">{{ item.desc }}</view>
          </view>
        </view>
      </u-list-item>
    </u-list>
  </view>
</template>

<script>
export default {
  data() {
    return {
      list: [],
      page: 1,
      limit: 10,
      loading: false,
      finished: false
    }
  },
  created() {
    this.loadData();
  },
  methods: {
    async loadData() {
      if (this.loading || this.finished) return;
      
      this.loading = true;
      
      // 模拟接口请求
      await new Promise(resolve => setTimeout(resolve, 1000));
      
      // 模拟数据
      const newData = Array.from({ length: this.limit }, (_, i) => {
        const index = (this.page - 1) * this.limit + i + 1;
        return {
          id: index,
          title: `标题 ${index}`,
          desc: `这是第 ${index} 条数据的描述信息`,
          avatar: 'https://via.placeholder.com/50'
        };
      });
      
      // 更新数据
      this.list = [...this.list, ...newData];
      
      // 判断是否还有更多数据
      if (this.page >= 3) {
        this.finished = true;
      } else {
        this.page++;
      }
      
      this.loading = false;
    },
    scrolltolower() {
      console.log('滚动到底部');
      this.loadData();
    },
    refresh() {
      console.log('下拉刷新');
      this.list = [];
      this.page = 1;
      this.finished = false;
      this.loadData();
    }
  }
}
</script>

轮播图组件 (u-swiper) ​

轮播图组件用于展示一组图片或内容，支持自动轮播和手动切换。

基本用法 ​

vue
<template>
  <view>
    <u-swiper
      :list="list"
      :height="300"
      :autoplay="true"
      :interval="3000"
      :duration="500"
      :circular="true"
      :indicator="true"
      :indicatorMode="indicatorMode"
      @change="change"
      @click="click"
    ></u-swiper>
  </view>
</template>

<script>
export default {
  data() {
    return {
      list: [
        { image: 'https://via.placeholder.com/750x300/3c9cff/ffffff?text=Slide1', title: '标题1' },
        { image: 'https://via.placeholder.com/750x300/f56c6c/ffffff?text=Slide2', title: '标题2' },
        { image: 'https://via.placeholder.com/750x300/5ac725/ffffff?text=Slide3', title: '标题3' }
      ],
      indicatorMode: 'dot'
    }
  },
  methods: {
    change(index) {
      console.log('当前轮播图索引：', index);
    },
    click(index) {
      console.log('点击了轮播图：', index);
      uni.showToast({
        title: `点击了第${index + 1}张轮播图`,
        icon: 'none'
      });
    }
  }
}
</script>

徽标组件 (u-badge) ​

徽标组件用于展示数量、状态等信息，通常用于消息提醒、数量统计等场景。

基本用法 ​

vue
<template>
  <view>
    <u-badge :value="value" :max="max" :isDot="isDot" :type="type"></u-badge>
    
    <u-badge :value="8" :offset="[20, 20]">
      <u-button text="按钮" type="primary"></u-button>
    </u-badge>
    
    <u-badge value="新消息" type="error" :offset="[10, 10]">
      <u-icon name="chat" size="40"></u-icon>
    </u-badge>
  </view>
</template>

<script>
export default {
  data() {
    return {
      value: 5,
      max: 99,
      isDot: false,
      type: 'error'
    }
  }
}
</script>

徽标类型 ​

徽标支持多种类型：

vue
<template>
  <view>
    <u-badge value="5" type="primary"></u-badge>
    <u-badge value="5" type="success"></u-badge>
    <u-badge value="5" type="warning"></u-badge>
    <u-badge value="5" type="error"></u-badge>
    <u-badge value="5" type="info"></u-badge>
  </view>
</template>

徽标形状 ​

徽标支持数字和圆点两种形状：

vue
<template>
  <view>
    <u-badge value="5"></u-badge>
    <u-badge isDot></u-badge>
  </view>
</template>

徽标最大值 ​

可以设置徽标的最大值，超过最大值会显示为"最大值+"：

vue
<template>
  <view>
    <u-badge value="99" max="99"></u-badge>
    <u-badge value="100" max="99"></u-badge>
    <u-badge value="1000" max="999"></u-badge>
  </view>
</template>

可关闭标签 ​

标签支持关闭功能：

vue
<template>
  <view>
    <u-tag 
      v-if="show" 
      text="可关闭标签" 
      type="primary" 
      closable 
      @close="close"
    ></u-tag>
    
    <u-button 
      v-if="!show" 
      text="显示标签" 
      type="primary" 
      @click="show = true"
    ></u-button>
  </view>
</template>

<script>
export default {
  data() {
    return {
      show: true
    }
  },
  methods: {
    close() {
      this.show = false;
    }
  }
}
</script>

折叠面板组件 (u-collapse, u-collapse-item) ​

折叠面板组件用于将内容区域折叠/展开，节省页面空间。

基本用法 ​

vue
<template>
  <view>
    <u-collapse :value="value">
      <u-collapse-item title="标题1" name="1">
        <view class="collapse-content">
          这是折叠面板的内容区域1
        </view>
      </u-collapse-item>
      
      <u-collapse-item title="标题2" name="2">
        <view class="collapse-content">
          这是折叠面板的内容区域2
        </view>
      </u-collapse-item>
      
      <u-collapse-item title="标题3" name="3">
        <view class="collapse-content">
          这是折叠面板的内容区域3
        </view>
      </u-collapse-item>
    </u-collapse>
  </view>
</template>

<script>
export default {
  data() {
    return {
      value: ['1']
    }
  }
}
</script>

<style>
.collapse-content {
  padding: 20rpx;
  font-size: 28rpx;
  color: #666;
}

手风琴模式 ​

折叠面板支持手风琴模式，即一次只能展开一个面板：

vue
<template>
  <view>
    <u-collapse :value="value" accordion @change="change">
      <u-collapse-item title="标题1" name="1">
        <view class="collapse-content">
          这是折叠面板的内容区域1
        </view>
      </u-collapse-item>
      
      <u-collapse-item title="标题2" name="2">
        <view class="collapse-content">
          这是折叠面板的内容区域2
        </view>
      </u-collapse-item>
      
      <u-collapse-item title="标题3" name="3">
        <view class="collapse-content">
          这是折叠面板的内容区域3
        </view>
      </u-collapse-item>
    </u-collapse>
  </view>
</template>

<script>
export default {
  data() {
    return {
      value: '1'
    }
  },
  methods: {
    change(value) {
      console.log('当前展开的面板：', value);
