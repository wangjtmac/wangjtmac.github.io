Vue 表单处理相关面试题。

v-model 基础 ​

1. 什么是 v-model？ v-model 是 Vue 提供的双向数据绑定指令，用于在表单控件元素上创建双向数据绑定。

2. v-model 的原理是什么？ v-model 是语法糖，本质上是 v-bind 绑定 value 属性和 v-on 绑定 input 事件的结合。

3. v-model 可以用在哪些元素上？

  • 文本输入框（<input type="text">）
  • 文本域（<textarea>）
  • 复选框（<input type="checkbox">）
  • 单选按钮（<input type="radio">）
  • 下拉选择框（<select>）

4. v-model 的基本用法？

vue
<template>
  <div>
    <input v-model="message" placeholder="输入消息">
    <p>消息：{{ message }}</p>
  </div>
</template>

<script>
export default {
  data() {
    return {
      message: ''
    }
  }
}
</script>

v-model 修饰符 ​

1. v-model 的修饰符有哪些？

  • .lazy：在 change 事件而非 input 事件中更新
  • .number：自动将输入值转为数字类型
  • .trim：自动过滤首尾空格

2. 如何使用 .lazy 修饰符？

vue
<template>
  <input v-model.lazy="message">
  <p>消息：{{ message }}</p>
</template>

3. 如何使用 .number 修饰符？

vue
<template>
  <input v-model.number="age" type="number">
  <p>年龄：{{ age }}（类型：{{ typeof age }}）</p>
</template>

4. 如何使用 .trim 修饰符？

vue
<template>
  <input v-model.trim="username">
  <p>用户名：{{ username }}</p>
</template>

5. 修饰符的使用场景？

  • .lazy：适用于需要用户完成输入后再处理的场景
  • .number：适用于需要数字类型的场景
  • .trim：适用于需要去除空格的场景

复选框的 v-model ​

1. 单个复选框的 v-model？

vue
<template>
  <div>
    <input type="checkbox" id="checkbox" v-model="checked">
    <label for="checkbox">{{ checked }}</label>
  </div>
</template>

<script>
export default {
  data() {
    return {
      checked: false
    }
  }
}
</script>

2. 多个复选框绑定到数组？

vue
<template>
  <div>
    <input type="checkbox" id="jack" value="Jack" v-model="checkedNames">
    <label for="jack">Jack</label>

    <input type="checkbox" id="john" value="John" v-model="checkedNames">
    <label for="john">John</label>

    <input type="checkbox" id="mike" value="Mike" v-model="checkedNames">
    <label for="mike">Mike</label>

    <p>选中的名字：{{ checkedNames }}</p>
  </div>
</template>

<script>
export default {
  data() {
    return {
      checkedNames: []
    }
  }
}
</script>

3. 复选框的注意事项？

  • 多个复选框绑定到数组时，value 属性是必须的
  • 复选框的值会作为数组元素
  • 复选框的选中状态与数组中是否包含该值相关

3. 使用 v-for 渲染选项？

vue
<template>
  <div>
    <select v-model="selected">
      <option v-for="option in options" :value="option.value" :key="option.value">
        {{ option.text }}
      </option>
    </select>
    <p>选中的值：{{ selected }}</p>
  </div>
</template>

<script>
export default {
  data() {
    return {
      selected: 'A',
      options: [
        { text: 'One', value: 'A' },
        { text: 'Two', value: 'B' },
        { text: 'Three', value: 'C' }
      ]
    }
  }
}
</script>

4. 下拉选择框的注意事项？

  • 多选下拉选择框需要添加 multiple 属性
  • 使用 v-for 渲染选项时，需要设置 value 属性
  • 选项的 value 可以是任意类型

自定义组件的 v-model ​

1. 如何在自定义组件中使用 v-model？

vue
<!-- 子组件 -->
<template>
  <input
    :value="value"
    @input="$emit('input', $event.target.value)"
  />
</template>

<script>
export default {
  props: {
    value: {
      type: String,
      default: ''
    }
  }
}
</script>

<!-- 父组件 -->
<template>
  <div>
    <custom-input v-model="message"></custom-input>
    <p>消息：{{ message }}</p>
  </div>
</template>

<script>
import CustomInput from './CustomInput.vue'

export default {
  components: {
    CustomInput
  },
  data() {
    return {
      message: ''
    }
  }
}
</script>

2. Vue 2 中自定义 v-model 的 prop 和 event？

javascript
export default {
  model: {
    prop: 'value',
    event: 'input'
  },
  props: {
    value: String
  }
}

3. Vue 3 中自定义 v-model 的 prop 和 event？

javascript
export default {
  props: {
    modelValue: String
  },
  emits: ['update:modelValue'],
  setup(props, { emit }) {
    const updateValue = (event) => {
      emit('update:modelValue', event.target.value)
    }
    return { updateValue }
  }
}

4. 自定义 v-model 的注意事项？

  • Vue 2 中默认使用 value prop 和 input 事件
  • Vue 3 中默认使用 modelValue prop 和 update:modelValue 事件
  • 可以通过 model 选项自定义 prop 和 event

表单验证 ​

1. 如何实现表单验证？

vue
<template>
  <div>
    <form @submit.prevent="handleSubmit">
      <div>
        <label for="username">用户名：</label>
        <input
          id="username"
          v-model="form.username"
          type="text"
        >
        <span v-if="errors.username" class="error">{{ errors.username }}</span>
      </div>

      <div>
        <label for="email">邮箱：</label>
        <input
          id="email"
          v-model="form.email"
          type="email"
        >
        <span v-if="errors.email" class="error">{{ errors.email }}</span>
      </div>

      <button type="submit">提交</button>
    </form>
  </div>
</template>

<script>
export default {
  data() {
    return {
      form: {
        username: '',
        email: ''
      },
      errors: {}
    }
  },
  methods: {
    validate() {
      this.errors = {}

      if (!this.form.username) {
        this.errors.username = '用户名不能为空'
      } else if (this.form.username.length < 3) {
        this.errors.username = '用户名至少 3 个字符'
      }

      if (!this.form.email) {
        this.errors.email = '邮箱不能为空'
      } else if (!this.isValidEmail(this.form.email)) {
        this.errors.email = '邮箱格式不正确'
      }

      return Object.keys(this.errors).length === 0
    },
    isValidEmail(email) {
      const re = /^[^\s@]+@[^\s@]+\.[^\s@]+$/
      return re.test(email)
    },
    handleSubmit() {
      if (this.validate()) {
        console.log('表单提交成功', this.form)
      }
    }
  }
}
</script>

<style>
.error {
  color: red;
}
</style>

2. 如何使用计算属性进行表单验证？

vue
<template>
  <div>
    <form @submit.prevent="handleSubmit">
      <div>
        <label for="username">用户名：</label>
        <input
          id="username"
          v-model="form.username"
          type="text"
        >
        <span v-if="errors.username" class="error">{{ errors.username }}</span>
      </div>

      <button type="submit" :disabled="!isValid">提交</button>
    </form>
  </div>
</template>

<script>
export default {
  data() {
    return {
      form: {
        username: ''
      }
    }
  },
  computed: {
    errors() {
      const errors = {}

      if (!this.form.username) {
        errors.username = '用户名不能为空'
      } else if (this.form.username.length < 3) {
        errors.username = '用户名至少 3 个字符'
      }

      return errors
    },
    isValid() {
      return Object.keys(this.errors).length === 0
    }
  },
  methods: {
    handleSubmit() {
      if (this.isValid) {
        console.log('表单提交成功', this.form)
      }
    }
  }
}
</script>

3. 如何使用第三方验证库？

vue
<template>
  <div>
    <form @submit.prevent="handleSubmit">
      <div>
        <label for="username">用户名：</label>
        <input
          id="username"
          v-model="form.username"
          type="text"
          v-validate="'required|min:3'"
          data-vv-name="username"
        >
        <span v-if="errors.has('username')" class="error">{{ errors.first('username') }}</span>
      </div>

      <button type="submit">提交</button>
    </form>
  </div>
</template>

<script>
import { Validator } from 'vee-validate'

export default {
  data() {
    return {
      form: {
        username: ''
      }
    }
  },
  methods: {
    handleSubmit() {
      this.$validator.validateAll().then((result) => {
        if (result) {
          console.log('表单提交成功', this.form)
        }
      })
    }
  }
}
</script>

4. 表单验证的最佳实践？

  • 实时验证：在用户输入时进行验证
  • 提交验证：在表单提交时进行验证
  • 错误提示：提供清晰的错误提示
  • 用户体验：提供良好的用户体验
  • 性能优化：避免频繁的验证操作

表单的高级用法 ​

1. 如何实现表单重置？

vue
<template>
  <div>
    <form @submit.prevent="handleSubmit">
      <input v-model="form.username" type="text">
      <input v-model="form.email" type="email">
      <button type="submit">提交</button>
      <button type="button" @click="handleReset">重置</button>
    </form>
  </div>
</template>

<script>
export default {
  data() {
    return {
      form: {
        username: '',
        email: ''
      }
    }
  },
  methods: {
    handleSubmit() {
      console.log('表单提交成功', this.form)
    },
    handleReset() {
      this.form = {
        username: '',
        email: ''
      }
    }
  }
}
</script>

2. 如何实现表单的动态字段？

vue
<template>
  <div>
    <form @submit.prevent="handleSubmit">
      <div v-for="(field, index) in fields" :key="index">
        <input v-model="field.value" type="text">
        <button type="button" @click="removeField(index)">删除</button>
      </div>
      <button type="button" @click="addField">添加字段</button>
      <button type="submit">提交</button>
    </form>
  </div>
</template>

<script>
export default {
  data() {
    return {
      fields: [
        { value: '' }
      ]
    }
  },
  methods: {
    addField() {
      this.fields.push({ value: '' })
    },
    removeField(index) {
      this.fields.splice(index, 1)
    },
    handleSubmit() {
      console.log('表单提交成功', this.fields)
    }
  }
}
</script>

3. 如何实现表单的文件上传？

vue
<template>
  <div>
    <form @submit.prevent="handleSubmit">
      <input type="file" @change="handleFileChange">
      <button type="submit">上传</button>
    </form>
  </div>
</template>

<script>
export default {
  data() {
    return {
      file: null
    }
  },
  methods: {
    handleFileChange(event) {
      this.file = event.target.files[0]
    },
    handleSubmit() {
      if (this.file) {
        const formData = new FormData()
        formData.append('file', this.file)

        fetch('/api/upload', {
          method: 'POST',
          body: formData
        })
          .then(response => response.json())
          .then(data => {
            console.log('上传成功', data)
          })
      }
    }
  }
}
</script>

4. 如何实现表单的防抖？

vue
<template>

    }
  },

    handleSearch: debounce(function() {
      console.log('搜索', this.search)
    }, 300)
  }
}
</script>

表单的常见问题 ​

1. v-model 不生效的原因？

  • 没有正确绑定 v-model
  • 组件没有正确触发事件
  • 组件的 prop 和 event 不匹配

2. 如何解决 v-model 不生效的问题？

  • 检查 v-model 的绑定是否正确
  • 检查组件是否正确触发事件
  • 检查组件的 prop 和 event 是否匹配

3. 表单验证不生效的原因？

  • 验证规则不正确
  • 验证时机不正确
  • 验证逻辑有误

4. 如何解决表单验证不生效的问题？

  • 检查验证规则是否正确
  • 检查验证时机是否正确
  • 检查验证逻辑是否有误
