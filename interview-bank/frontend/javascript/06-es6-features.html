JavaScript ES6+ 新特性 ​

JavaScript ES6+ 新特性相关面试题，包括 let/const、解构赋值、模板字符串、Symbol、Map/Set、Proxy 等。

let 和 const ​

基本用法 ​

javascript
let a = 10;
const b = 20;

a = 20;  // 可以重新赋值
b = 30;  // TypeError: Assignment to constant variable

块级作用域 ​

javascript
{
  let a = 10;
  const b = 20;
}

console.log(a);  // ReferenceError
console.log(b);  // ReferenceError

暂时性死区（TDZ） ​

javascript
console.log(a);  // ReferenceError
let a = 10;

常见面试问题 ​

Q: const 声明的对象可以修改吗？

javascript
const obj = { a: 1 };
obj.a = 2;  // 可以
obj = { b: 2 };  // TypeError

const arr = [1, 2, 3];
arr.push(4);  // 可以
arr = [4, 5];  // TypeError

解构赋值 ​

数组解构 ​

javascript
const [a, b, c] = [1, 2, 3];
console.log(a, b, c);  // 1 2 3

const [first, ...rest] = [1, 2, 3, 4];
console.log(first, rest);  // 1 [2, 3, 4]

对象解构 ​

javascript
const { name, age } = { name: 'Alice', age: 20 };
console.log(name, age);  // Alice 20

const { name: userName, age: userAge } = { name: 'Alice', age: 20 };
console.log(userName, userAge);  // Alice 20

函数参数解构 ​

javascript
function fn({ name, age }) {
  console.log(name, age);
}

fn({ name: 'Alice', age: 20 });  // Alice 20

默认值 ​

javascript
const [a = 1, b = 2] = [];
console.log(a, b);  // 1 2

const { name = 'Guest' } = {};
console.log(name);  // Guest

常见面试问题 ​

Q: 解构赋值的应用场景？

  • 交换变量
  • 函数参数
  • 提取对象属性
  • 数组操作

标签模板 ​

javascript
function tag(strings, ...values) {
  console.log(strings);  // ['Hello, ', '!']
  console.log(values);   // ['Alice']
  return strings[0] + values[0] + strings[1];
}

const name = 'Alice';
const result = tag`Hello, ${name}!`;
console.log(result);  // Hello, Alice!

Symbol ​

基本用法 ​

javascript
const sym = Symbol('description');
const obj = {
  [sym]: 'value'
};

console.log(obj[sym]);  // value
console.log(sym === Symbol('description'));  // false

Symbol 作为对象属性键 ​

javascript
const sym = Symbol('id');
const obj = {
  name: 'Alice',
  [sym]: '123'
};

console.log(Object.keys(obj));  // ['name']
console.log(Object.getOwnPropertySymbols(obj));  // [Symbol(id)]

Map ​

基本用法 ​

javascript
const map = new Map();

map.set('key1', 'value1');
map.set('key2', 'value2');

console.log(map.get('key1'));  // value1
console.log(map.has('key2'));  // true
console.log(map.size);        // 2

map.delete('key1');
map.clear();

Map vs Object ​

对比项MapObject
键类型任意值字符串或 Symbol
键顺序插入顺序无序
大小size 属性手动计算
迭代可迭代不可迭代

WeakMap ​

javascript
const weakMap = new WeakMap();
const obj = { name: 'Alice' };

weakMap.set(obj, 'value');
console.log(weakMap.get(obj));  // value

// obj 被垃圾回收后，weakMap 中的条目自动删除

Proxy ​

基本用法 ​

javascript
const target = { name: 'Alice' };
const handler = {
  get: function(target, prop) {
    console.log(`Getting ${prop}`);
    return target[prop];
  },
  set: function(target, prop, value) {
    console.log(`Setting ${prop} to ${value}`);
    target[prop] = value;
    return true;
  }
};

const proxy = new Proxy(target, handler);

proxy.name;  // Getting name
proxy.name = 'Bob';  // Setting name to Bob

Optional Chaining ​

基本用法 ​

javascript
const user = {
  name: 'Alice',
  age: 20,
  address: {
    city: 'Beijing'
  }
};

console.log(user?.address?.city);  // Beijing
console.log(user?.phone?.number);  // undefined

空值合并 ​

javascript
const user = null;
const city = user?.address?.city ?? 'Unknown';
console.log(city);  // Unknown

其他新特性 ​

展开运算符 ​

javascript
const arr = [1, 2, 3];
const newArr = [...arr, 4, 5];
console.log(newArr);  // [1, 2, 3, 4, 5]

const obj = { a: 1, b: 2 };
const newObj = { ...obj, c: 3 };
console.log(newObj);  // { a: 1, b: 2, c: 3 }

剩余参数 ​

javascript
function fn(...args) {
  console.log(args);  // 数组
}

fn(1, 2, 3);  // [1, 2, 3]

默认参数 ​

javascript
function fn(a = 1, b = 2) {
  console.log(a, b);
}

fn();      // 1 2
fn(10);    // 10 2
fn(10, 20);  // 10 20

类的私有属性 ​

javascript
class Person {
  #name = 'Alice';  // 私有属性
  
  getName() {
    return this.#name;
  }
}

const person = new Person();
console.log(person.getName());  // Alice
console.log(person.#name);  // SyntaxError
