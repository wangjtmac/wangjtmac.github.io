常见面试问题 ​

Q: innerHTML 和 textContent 的区别？

  • innerHTML：解析 HTML，会执行脚本
  • textContent：纯文本，不会执行脚本

Q: offsetWidth 和 clientWidth 的区别？

  • offsetWidth：包括边框、内边距
  • clientWidth：不包括边框、内边距

事件 ​

事件流 ​

捕获阶段 ─→ 目标阶段 ←─ 冒泡阶段

事件监听 ​

javascript
// 添加事件监听
element.addEventListener('click', handler, false);

// 移除事件监听
element.removeEventListener('click', handler);

// 事件对象
function handler(event) {
  event.preventDefault();  // 阻止默认行为
  event.stopPropagation(); // 阻止事件冒泡
}

事件冒泡与捕获 ​

javascript
// 捕获阶段（true）
element.addEventListener('click', handler, true);

// 冒泡阶段（false）
element.addEventListener('click', handler, false);

常见面试问题 ​

Q: 事件冒泡和捕获的区别？

  • 捕获：从 document 到目标元素
  • 冒泡：从目标元素到 document

Q: preventDefault 和 stopPropagation 的区别？

  • preventDefault()：阻止默认行为（如表单提交）
  • stopPropagation()：阻止事件传播（冒泡或捕获）

事件委托 ​

基本原理 ​

利用事件冒泡，在父元素上监听事件，通过 event.target 判断触发元素。

javascript
<ul id="list">
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>

<script>
document.getElementById('list').addEventListener('click', function(event) {
  if (event.target.tagName === 'LI') {
    console.log(event.target.textContent);
  }
});
</script>

优势 ​

  • 减少事件监听器数量
  • 动态添加元素无需重新绑定
  • 提高性能

常见面试问题 ​

Q: 事件委托的应用场景？

  • 列表项点击
  • 表格行操作
  • 动态内容交互

防抖与节流 ​

防抖（Debounce） ​

javascript
function debounce(fn, delay) {
  let timer = null;
  
  return function(...args) {
    clearTimeout(timer);
    timer = setTimeout(() => {
      fn.apply(this, args);
    }, delay);
  };
}

// 使用
const handleInput = debounce(e => {
  console.log(e.target.value);
}, 300);

input.addEventListener('input', handleInput);

节流（Throttle） ​

javascript
function throttle(fn, delay) {
  let lastTime = 0;
  
  return function(...args) {
    const now = Date.now();
    if (now - lastTime >= delay) {
      fn.apply(this, args);
      lastTime = now;
    }
  };
}

// 使用
const handleScroll = throttle(e => {
  console.log('scroll');
}, 300);

window.addEventListener('scroll', handleScroll);

常见面试问题 ​

Q: 防抖和节流的区别？

  • 防抖：延迟执行，多次触发只执行最后一次
  • 节流：固定时间执行，多次触发按固定频率执行

Q: 防抖和节流的应用场景？

  • 防抖：输入框搜索、窗口 resize
  • 节流：滚动事件、鼠标移动

DOM 性能优化 ​

减少 DOM 操作 ​

javascript
// 不好的做法
for (let i = 0; i < 100; i++) {
  document.body.appendChild(createElement());
}

// 好的做法
const fragment = document.createDocumentFragment();
for (let i = 0; i < 100; i++) {
  fragment.appendChild(createElement());
}
document.body.appendChild(fragment);

避免重排重绘 ​

javascript
// 不好的做法
element.style.width = '100px';
element.style.height = '100px';
element.style.background = 'red';

// 好的做法
element.style.cssText = 'width: 100px; height: 100px; background: red;';

使用文档片段 ​

javascript
const fragment = document.createDocumentFragment();
fragment.appendChild(element1);
fragment.appendChild(element2);
parent.appendChild(fragment);

常见面试问题 ​

Q: 什么是重排和重绘？

  • 重排（Reflow）：元素尺寸、位置改变，需要重新计算布局
  • 重绘（Repaint）：元素外观改变，不需要重新计算布局

Q: 如何减少重排重绘？

  • 批量修改 DOM
  • 使用文档片段
  • 避免逐条修改样式
  • 使用 transform 和 opacity
