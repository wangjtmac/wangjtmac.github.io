JavaScript 函数与 this ​

JavaScript 函数与 this 相关面试题，包括函数定义、this 指向、call/apply/bind、箭头函数等。

函数定义 ​

函数声明 ​

javascript
function fn(a, b) {
  return a + b;
}

fn(1, 2);  // 3

函数表达式 ​

javascript
var fn = function(a, b) {
  return a + b;
};

fn(1, 2);  // 3

箭头函数 ​

javascript
const fn = (a, b) => a + b;
fn(1, 2);  // 3

// 简写
const square = x => x * x;
square(5);  // 25

立即执行函数（IIFE） ​

javascript
(function() {
  console.log('立即执行');
})();

(() => {
  console.log('箭头函数立即执行');
})();

常见面试问题 ​

Q: 函数声明和函数表达式的区别？

  • 函数声明：会提升，可以在声明前调用
  • 函数表达式：不会提升，必须在声明后调用

this 指向 ​

this 的指向规则 ​

调用方式this 指向
直接调用全局对象（严格模式下 undefined）
对象方法调用调用对象
构造函数调用新创建的实例
call/apply/bind指定的对象
箭头函数定义时的外层 this

示例 ​

javascript
// 1. 直接调用
function fn() {
  console.log(this);
}
fn();  // window（非严格模式）/ undefined（严格模式）

// 2. 对象方法调用
var obj = {
  name: 'Alice',
  sayName: function() {
    console.log(this.name);
  }
};
obj.sayName();  // Alice

// 3. 构造函数调用
function Person(name) {
  this.name = name;
}
var person = new Person('Bob');
console.log(person.name);  // Bob

// 4. call/apply/bind
function greet() {
  console.log('Hello, ' + this.name);
}
var obj = { name: 'Charlie' };
greet.call(obj);  // Hello, Charlie
greet.apply(obj);  // Hello, Charlie
var boundGreet = greet.bind(obj);
boundGreet();  // Hello, Charlie

// 5. 箭头函数
var obj = {
  name: 'David',
  sayName: () => {
    console.log(this.name);  // undefined（this 指向全局）
  }
};
obj.sayName();

常见面试问题 ​

Q: 箭头函数的 this 指向？

箭头函数没有自己的 this，它会捕获定义时外层的 this。

call、apply、bind ​

call ​

javascript
function greet(greeting, punctuation) {
  console.log(greeting + ', ' + this.name + punctuation);
}

var person = { name: 'Alice' };
greet.call(person, 'Hello', '!');  // Hello, Alice!

apply ​

javascript
function greet(greeting, punctuation) {
  console.log(greeting + ', ' + this.name + punctuation);
}

var person = { name: 'Bob' };
greet.apply(person, ['Hi', '.']);  // Hi, Bob.

bind ​

javascript
function greet() {
  console.log('Hello, ' + this.name);
}

var person = { name: 'Charlie' };
var boundGreet = greet.bind(person);
boundGreet();  // Hello, Charlie

手动实现 call ​

javascript
Function.prototype.myCall = function(context, ...args) {
  context = context || window;
  context.fn = this;
  const result = context.fn(...args);
  delete context.fn;
  return result;
};

手动实现 apply ​

javascript
Function.prototype.myApply = function(context, args) {
  context = context || window;
  context.fn = this;
  const result = context.fn(...args);
  delete context.fn;
  return result;
};

手动实现 bind ​

javascript
Function.prototype.myBind = function(context, ...bindArgs) {
  const self = this;
  return function(...callArgs) {
    return self.apply(context, [...bindArgs, ...callArgs]);
  };
};

常见面试问题 ​

Q: call、apply、bind 的区别？

方法参数立即执行返回值
call参数列表是函数返回值
apply参数数组是函数返回值
bind参数列表否新函数

箭头函数 ​

箭头函数特性 ​

  • 没有 arguments 对象
  • 没有 this（继承外层）
  • 不能用作构造函数
  • 没有 prototype 属性

箭头函数 vs 普通函数 ​

javascript
// 普通函数
function fn() {
  console.log(arguments);  // Arguments 对象
  console.log(this);      // 调用时的 this
}

// 箭头函数
const arrowFn = () => {
  console.log(arguments);  // ReferenceError
  console.log(this);      // 定义时的 this
};

常见面试问题 ​

Q: 箭头函数不能使用场景？

  • 不能用作构造函数
  • 不能使用 arguments
  • 不能使用 yield（不能作为 Generator 函数）

函数柯里化 ​

基本概念 ​

将多参数函数转换为单参数函数的序列。

javascript
function add(a, b, c) {
  return a + b + c;
}

function curry(fn) {
  return function curried(...args) {
    if (args.length >= fn.length) {
      return fn.apply(this, args);
    } else {
      return function(...moreArgs) {
        return curried.apply(this, args.concat(moreArgs));
      };
    }
  };
}

var curriedAdd = curry(add);
console.log(curriedAdd(1)(2)(3));  // 6

简单实现 ​

javascript
function add(a) {
  return function(b) {
    return a + b;
  };
}

var add5 = add(5);
console.log(add5(3));  // 8

常见面试问题 ​

Q: 函数柯里化的应用场景？

  • 参数复用
  • 延迟计算
  • 动态创建函数

高阶函数 ​

基本概念 ​

接收函数作为参数或返回函数的函数。

javascript
// map
[1, 2, 3].map(x => x * 2);  // [2, 4, 6]

// filter
[1, 2, 3, 4].filter(x => x % 2 === 0);  // [2, 4]

// reduce
[1, 2, 3].reduce((sum, x) => sum + x, 0);  // 6

手动实现 map ​

javascript
Array.prototype.myMap = function(fn) {
  const result = [];
  for (let i = 0; i < this.length; i++) {
    result.push(fn(this[i], i, this));
  }
  return result;
};

手动实现 filter ​

javascript
Array.prototype.myFilter = function(fn) {
  const result = [];
  for (let i = 0; i < this.length; i++) {
    if (fn(this[i], i, this)) {
      result.push(this[i]);
    }
  }
  return result;
};

手动实现 reduce ​

javascript
Array.prototype.myReduce = function(fn, initialValue) {
  let accumulator = initialValue;
  for (let i = 0; i < this.length; i++) {
    accumulator = fn(accumulator, this[i], i, this);
  }
  return accumulator;
};

常见面试问题 ​

Q: 什么是高阶函数？

接收函数作为参数或返回函数的函数，如 map、filter、reduce 等。

arguments 对象 ​

基本用法 ​

javascript
function fn() {
  console.log(arguments);  // Arguments 对象
  console.log(arguments.length);  // 参数个数
  console.log(arguments[0]);  // 第一个参数
}

fn(1, 2, 3);

转为数组 ​

javascript
function fn() {
  const args = Array.from(arguments);
  // 或
  const args = [...arguments];
  // 或
  const args = Array.prototype.slice.call(arguments);
}

箭头函数没有 arguments ​

javascript
const arrowFn = () => {
  console.log(arguments);  // ReferenceError
};

常见面试问题 ​

Q: 如何在箭头函数中使用 arguments？

使用剩余参数：

javascript
const arrowFn = (...args) => {
  console.log(args);  // 数组
};
