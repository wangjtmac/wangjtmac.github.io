HTML 列表 ​

HTML 列表相关面试题，包括无序列表、有序列表、定义列表等。

无序列表 ​

基本用法 ​

html
<ul>
  <li>苹果</li>
  <li>香蕉</li>
  <li>橙子</li>
</ul>

嵌套列表 ​

html
<ul>
  <li>水果
    <ul>
      <li>苹果</li>
      <li>香蕉</li>
    </ul>
  </li>
  <li>蔬菜
    <ul>
      <li>白菜</li>
      <li>萝卜</li>
    </ul>
  </li>
</ul>

type 属性 ​

html
<ul type="disc">    <!-- 实心圆点（默认） -->
<ul type="circle">  <!-- 空心圆点 -->
<ul type="square">  <!-- 方块 -->

CSS 自定义样式 ​

css
ul {
  list-style-type: disc;    /* 实心圆点 */
  list-style-type: circle;  /* 空心圆点 */
  list-style-type: square;  /* 方块 */
  list-style-type: none;    /* 无标记 */
  list-style-image: url('bullet.png');  /* 自定义图片 */
}

常见面试问题 ​

Q: 如何移除列表的默认样式？

css
ul {
  list-style: none;
  padding: 0;
  margin: 0;
}

有序列表 ​

基本用法 ​

html
<ol>
  <li>第一步</li>
  <li>第二步</li>
  <li>第三步</li>
</ol>

嵌套列表 ​

html
<ol>
  <li>准备工作
    <ol>
      <li>准备材料</li>
      <li>准备工具</li>
    </ol>
  </li>
  <li>开始制作
    <ol>
      <li>混合材料</li>
      <li>加热</li>
    </ol>
  </li>
</ol>

type 属性 ​

html
<ol type="1">      <!-- 数字（默认） -->
<ol type="A">      <!-- 大写字母 -->
<ol type="a">      <!-- 小写字母 -->
<ol type="I">      <!-- 大写罗马数字 -->
<ol type="i">      <!-- 小写罗马数字 -->

start 属性 ​

html
<ol start="5">
  <li>第五步</li>
  <li>第六步</li>
  <li>第七步</li>
</ol>

reversed 属性 ​

html
<ol reversed>
  <li>第三步</li>
  <li>第二步</li>
  <li>第一步</li>
</ol>

CSS 自定义样式 ​

css
ol {
  list-style-type: decimal;       /* 数字 */
  list-style-type: upper-alpha;   /* 大写字母 */
  list-style-type: lower-alpha;   /* 小写字母 */
  list-style-type: upper-roman;   /* 大写罗马数字 */
  list-style-type: lower-roman;   /* 小写罗马数字 */
}

</ol>

html

</ol>

定义列表 ​

基本用法 ​

html
<dl>
  <dt>HTML</dt>
  <dd>超文本标记语言</dd>
  
  <dt>CSS</dt>
  <dd>层叠样式表</dd>
  
  <dt>JavaScript</dt>
  <dd>一种脚本语言</dd>
</dl>

多个定义 ​

html
<dl>
  <dt>苹果</dt>
  <dd>一种水果</dd>
  <dd>一家科技公司</dd>
  
  <dt>香蕉</dt>
  <dd>一种水果</dd>
</dl>

嵌套列表 ​

html
<dl>
  <dt>水果
    <dl>
      <dt>苹果</dt>
      <dd>红色水果</dd>
      <dt>香蕉</dt>
      <dd>黄色水果</dd>
    </dl>
  </dt>
</dl>

常见面试问题 ​

Q: dt 和 dd 的关系？

  • dt：定义术语（Definition Term）
  • dd：定义描述（Definition Description）
  • 一个 dt 可以有多个 dd

列表样式 ​

list-style-type ​

css
ul {
  list-style-type: disc;      /* 实心圆点 */
  list-style-type: circle;    /* 空心圆点 */
  list-style-type: square;    /* 方块 */
  list-style-type: decimal;   /* 数字 */
  list-style-type: none;      /* 无标记 */
}

list-style-position ​

css
ul {
  list-style-position: outside;  /* 外部（默认） */
  list-style-position: inside;   /* 内部 */
}

list-style-image ​

css
ul {
  list-style-image: url('bullet.png');
  list-style-image: none;  /* 无图片 */
}

简写属性 ​

css
ul {
  list-style: disc outside url('bullet.png');
}

常见面试问题 ​

Q: list-style-position 的作用？

  • outside：列表标记在内容外部（默认）
  • inside：列表标记在内容内部

常见面试问题 ​

Q: ul 和 ol 的区别？

对比项ulol
类型无序列表有序列表
标记圆点、方块等数字、字母等
顺序无序有序
用途不需要顺序的项目需要顺序的步骤

Q: 什么时候使用定义列表？

  • 术语和解释
  • 问题和答案
  • 名称和描述

Q: 如何实现自定义列表标记？

css
ul {
  list-style: none;
  padding-left: 20px;
}

ul li::before {
  content: "→ ";
  color: red;
}

Q: 如何实现横向列表？

css
ul {
  list-style: none;
  padding: 0;
  margin: 0;
  display: flex;
}

ul li {
  margin-right: 10px;
}

Q: 如何实现多列列表？

css
ul {
  column-count: 3;
  column-gap: 20px;
}

Q: 如何实现响应式列表？

css
ul {
  display: flex;
  flex-wrap: wrap;
  gap: 10px;
}

ul li {
  flex: 1 1 200px;
}
