Canvas ​

基本使用 ​

html
<canvas id="myCanvas" width="400" height="300"></canvas>

<script>
const canvas = document.getElementById('myCanvas');
const ctx = canvas.getContext('2d');

// 绘制矩形
ctx.fillStyle = 'red';
ctx.fillRect(10, 10, 100, 50);

// 绘制圆形
ctx.beginPath();
ctx.arc(200, 150, 50, 0, Math.PI * 2);
ctx.fillStyle = 'blue';
ctx.fill();

// 绘制线条
ctx.beginPath();
ctx.moveTo(10, 200);
ctx.lineTo(100, 250);
ctx.strokeStyle = 'green';
ctx.stroke();
</script>

Canvas 方法 ​

方法说明
getContext('2d')获取 2D 绘图上下文
fillRect(x, y, width, height)填充矩形
strokeRect(x, y, width, height)绘制矩形边框
clearRect(x, y, width, height)清除矩形区域
beginPath()开始路径
moveTo(x, y)移动到指定点
lineTo(x, y)绘制直线到指定点
arc(x, y, radius, startAngle, endAngle)绘制圆弧
fill()填充路径
stroke()绘制路径边框
fillText(text, x, y)填充文本
strokeText(text, x, y)绘制文本边框

Canvas 动画 ​

javascript
function animate() {
  ctx.clearRect(0, 0, canvas.width, canvas.height);
  
  // 绘制内容
  ctx.fillStyle = 'red';
  ctx.fillRect(x, y, 50, 50);
  
  // 更新位置
  x += 2;
  y += 2;
  
  requestAnimationFrame(animate);
}

Canvas 图片处理 ​

javascript
const img = new Image();
img.onload = function() {
  ctx.drawImage(img, 0, 0);
};
img.src = 'image.jpg';

// 获取像素数据
const imageData = ctx.getImageData(0, 0, canvas.width, canvas.height);
const data = imageData.data;

// 修改像素
for (let i = 0; i < data.length; i += 4) {
  data[i] = 255 - data[i];     // 反转红色
  data[i + 1] = 255 - data[i + 1]; // 反转绿色
  data[i + 2] = 255 - data[i + 2]; // 反转蓝色
}

// 放回画布
ctx.putImageData(imageData, 0, 0);

常见面试问题 ​

Q: Canvas 的优势？

  • 适合大量图形和动画
  • 性能好，像素级操作
  • 支持复杂的图像处理
  • 文件体积小

Q: Canvas 的劣势？

  • 不支持事件监听（需要手动计算坐标）
  • 缩放会失真
  • 不利于 SEO
  • 不利于可访问性

Service Worker ​

Service Worker 是现代的离线存储方案：

javascript
if ('serviceWorker' in navigator) {
  navigator.serviceWorker.register('/sw.js')
    .then(registration => {
      console.log('Service Worker 注册成功');
    });
}

优势：

  • 支持离线访问
  • 可以拦截网络请求
  • 支持推送通知
  • 支持后台同步

常见面试问题 ​

Q: 如何实现网页离线访问？

使用 Service Worker + Cache API：

  1. 注册 Service Worker
  2. 在 install 事件中缓存静态资源
  3. 在 fetch 事件中拦截请求，优先返回缓存

Q: Service Worker 和 Web Worker 的区别？

对比项Service WorkerWeb Worker
生命周期独立生命周期，可长期存在随页面关闭而销毁
作用域可控制多个页面仅当前页面
功能拦截请求、离线缓存、推送后台计算
DOM 访问不能访问不能访问

HTML5 拖拽 API ​

基本使用 ​

html
<div id="draggable" draggable="true">可拖拽元素</div>
<div id="droppable">放置区域</div>

<script>
const draggable = document.getElementById('draggable');
const droppable = document.getElementById('droppable');

draggable.addEventListener('dragstart', (e) => {
  e.dataTransfer.setData('text/plain', e.target.id);
});

droppable.addEventListener('dragover', (e) => {
  e.preventDefault();
});

droppable.addEventListener('drop', (e) => {
  e.preventDefault();
  const id = e.dataTransfer.getData('text/plain');
  droppable.appendChild(document.getElementById(id));
});
</script>

拖拽事件 ​

事件触发时机
dragstart开始拖拽
drag拖拽过程中
dragend拖拽结束
dragenter进入放置区域
dragover在放置区域内移动
dragleave离开放置区域
drop放置

常见面试问题 ​

Q: 为什么需要 preventDefault？

浏览器默认禁止拖放操作，需要在 dragover 和 drop 事件中调用 e.preventDefault() 才能允许放置。

常见面试问题 ​

Q: 为什么要用语义化标签？

  • 提升 SEO：搜索引擎更容易理解页面结构
  • 可访问性：屏幕阅读器等辅助设备能更好地解析
  • 代码可读性：代码结构更清晰，便于维护
  • 减少 CSS 依赖：标签本身具有语义
