HTML 高级特性 ​

HTML 高级特性相关面试题，包括 iframe、WebSocket、响应式图片等。

iframe 的优缺点 ​

优点 ​

  • 可以嵌入第三方内容
  • 实现异步文件上传
  • 解决跨域问题（配合 postMessage）
  • 隔离样式和脚本

缺点 ​

  • 阻塞父页面 onload 事件
  • 不利于 SEO
  • 增加页面复杂度
  • 移动端兼容性差
  • 安全问题（点击劫持）

安全属性 ​

html
<iframe 
  src="https://example.com" 
  sandbox="allow-scripts allow-same-origin"
  referrerpolicy="no-referrer">
</iframe>

sandbox 属性值 ​

值说明
allow-scripts允许执行脚本
allow-same-origin允许同源访问
allow-forms允许表单提交
allow-popups允许弹出窗口
allow-top-navigation允许导航到顶层窗口

常见面试问题 ​

Q: 如何防止 iframe 点击劫持？

使用 X-Frame-Options 或 CSP 头：

http
X-Frame-Options: DENY
X-Frame-Options: SAMEORIGIN

Content-Security-Policy: frame-ancestors 'self'

Q: iframe 和主页面如何通信？

使用 postMessage：

javascript
parent.postMessage({ type: 'message' }, '*');

window.addEventListener('message', (e) => {
  if (e.origin !== 'https://trusted-origin.com') return;
  console.log(e.data);
});

WebSocket ​

WebSocket 简介 ​

WebSocket 是 HTML5 提供的在单个 TCP 连接上进行全双工通信的协议。

WebSocket 特点 ​

  • 全双工通信：客户端和服务端可以同时发送和接收消息
  • 实时性：建立连接后可以随时发送消息，无需轮询
  • 低开销：相比 HTTP，WebSocket 的头部更小
  • 跨域支持：支持跨域通信

WebSocket 与 HTTP 的区别 ​

对比项HTTPWebSocket
通信方式请求-响应模式全双工通信
连接短连接长连接
实时性需要轮询实时推送
协议HTTP/HTTPSws:// 或 wss://
状态无状态有状态

WebSocket 使用示例 ​

javascript
const socket = new WebSocket('ws://localhost:8080');

socket.onopen = function() {
  console.log('连接已建立');
};

socket.onmessage = function(event) {
  console.log('收到消息:', event.data);
};

socket.onclose = function() {
  console.log('连接已关闭');
};

socket.onerror = function(error) {
  console.log('错误:', error);
};

socket.send('Hello Server');

常见面试问题 ​

Q: WebSocket 的应用场景？

  • 实时聊天应用
  • 在线协作编辑
  • 实时数据大屏
  • 在线游戏
  • 股票/金融实时行情

Q: WebSocket 和 Socket.io 的区别？

  • WebSocket：浏览器原生支持的协议，底层 API
  • Socket.io：基于 WebSocket 的封装库，提供更多功能（如自动重连、房间管理、心跳检测等）

响应式图片 ​

picture 标签 ​

<picture> 标签允许根据不同设备条件加载不同的图片：

html
<picture>
  <source media="(min-width: 800px)" srcset="large.jpg">
  <source media="(min-width: 480px)" srcset="medium.jpg">
  <img src="small.jpg" alt="响应式图片">
</picture>

srcset 属性 ​

srcset 允许指定多个图片源，浏览器根据设备像素比选择：

html
<img 
  src="image-320w.jpg" 
  srcset="image-320w.jpg 320w,
          image-640w.jpg 640w,
          image-1280w.jpg 1280w"
  sizes="(max-width: 600px) 100vw, 50vw"
  alt="响应式图片">

常见面试问题 ​

Q: srcset 和 sizes 的作用？

  • srcset：提供多个不同分辨率的图片，浏览器根据设备选择
  • sizes：定义图片在不同屏幕宽度下的显示尺寸，帮助浏览器选择合适的图片

Q: 响应式图片的优化策略？

  • 使用 WebP 等现代格式
  • 提供多个分辨率版本
  • 使用 CDN 加速
  • 延迟加载（lazy loading）
  • 压缩图片大小
