HTML 表格 ​

HTML 表格相关面试题，包括 table、thead、tbody、tfoot、tr、td、th 等元素的使用。

基本结构 ​

简单表格 ​

html
<table>
  <tr>
    <th>姓名</th>
    <th>年龄</th>
    <th>城市</th>
  </tr>
  <tr>
    <td>Alice</td>
    <td>20</td>
    <td>北京</td>
  </tr>
  <tr>
    <td>Bob</td>
    <td>25</td>
    <td>上海</td>
  </tr>
</table>

完整结构 ​

html
<table>
  <caption>员工信息表</caption>
  <thead>
    <tr>
      <th>姓名</th>
      <th>年龄</th>
      <th>城市</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Alice</td>
      <td>20</td>
      <td>北京</td>
    </tr>
    <tr>
      <td>Bob</td>
      <td>25</td>
      <td>上海</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <td colspan="3">共 2 人</td>
    </tr>
  </tfoot>
</table>

表格元素 ​

table ​

表格容器元素。

html
<table border="1" cellpadding="10" cellspacing="0">
  <!-- 表格内容 -->
</table>

常用属性 ​

属性说明示例
border边框宽度border="1"
cellpadding单元格内边距cellpadding="10"
cellspacing单元格间距cellspacing="0"
width表格宽度width="100%"
align对齐方式align="center"

caption ​

表格标题。

html
<table>
  <caption>员工信息表</caption>
  <!-- 表格内容 -->
</table>

thead ​

表头部分。

html
<thead>
  <tr>
    <th>姓名</th>
    <th>年龄</th>
  </tr>
</thead>

tbody ​

表体部分。

html
<tbody>
  <tr>
    <td>Alice</td>

tfoot ​

<tfoot>

tr ​

表格行。

html

  <td>Alice</td>

td ​

表格单元格。

html
<td>Alice</td>

表头单元格。

html
<th>姓名</th>

单元格合并 ​

colspan（跨列） ​

html
<table border="1">
  <tr>
    <td colspan="2">跨两列</td>
  </tr>
  <tr>
    <td>单元格1</td>
    <td>单元格2</td>
  </tr>
</table>

rowspan（跨行） ​

html
<table border="1">
  <tr>
    <td rowspan="2">跨两行</td>
    <td>单元格1</td>
  </tr>
  <tr>
    <td>单元格2</td>
  </tr>
</table>

复杂合并 ​

html
<table border="1">
  <tr>
    <th colspan="3">标题</th>
  </tr>
  <tr>
    <td rowspan="2">A</td>
    <td>B1</td>
    <td>C1</td>
  </tr>
  <tr>
    <td>B2</td>
    <td>C2</td>
  </tr>
</table>

常见面试问题 ​

Q: colspan 和 rowspan 的区别？

  • colspan：横向合并单元格，跨列
  • rowspan：纵向合并单元格，跨行

Q: 合并单元格时需要注意什么？

  • 合并后要删除被合并的单元格
  • colspan 和 rowspan 不能同时使用

表格样式 ​

基本样式 ​

css
table {
  width: 100%;
  border-collapse: collapse;
  border-spacing: 0;
}

th, td {
  border: 1px solid #ddd;
  padding: 8px;
  text-align: left;
}

th {
  background-color: #f5f5f5;
  font-weight: bold;
}

tr:nth-child(even) {
  background-color: #f9f9f9;
}

tr:hover {
  background-color: #f5f5f5;
}

边框合并 ​

css
table {
  border-collapse: collapse;  /* 合并边框 */
}

table {
  border-collapse: separate;  /* 分离边框 */
  border-spacing: 10px;       /* 边框间距 */
}

常见面试问题 ​

Q: border-collapse 的作用？

  • collapse：合并相邻边框
  • separate：分离相邻边框

响应式表格 ​

横向滚动 ​

html
<div style="overflow-x: auto;">
  <table>
    <!-- 表格内容 -->
  </table>
</div>

卡片式布局（移动端） ​

html
<style>
  @media (max-width: 600px) {
    table, thead, tbody, th, td, tr {
      display: block;
    }
    thead tr {
      position: absolute;
      top: -9999px;
      left: -9999px;
    }
    tr {
      border: 1px solid #ccc;
      margin-bottom: 10px;
    }
    td {
      border: none;
      border-bottom: 1px solid #eee;
      position: relative;
      padding-left: 50%;
    }
    td:before {
      position: absolute;
      top: 6px;
      left: 6px;
      width: 45%;
      padding-right: 10px;
      white-space: nowrap;
      content: attr(data-label);
      font-weight: bold;
    }
  }
</style>

<table>
  <thead>
    <tr>
      <th>姓名</th>
      <th>年龄</th>
      <th>城市</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td data-label="姓名">Alice</td>
      <td data-label="年龄">20</td>
      <td data-label="城市">北京</td>
    </tr>
  </tbody>
</table>

常见面试问题 ​

Q: 如何实现响应式表格？

  • 横向滚动：在小屏幕上允许横向滚动
  • 卡片式布局：将表格转为卡片式显示
  • 隐藏列：在小屏幕上隐藏不重要的列

表格可访问性 ​

scope 属性 ​

html
<table>
  <thead>
    <tr>
      <th scope="col">姓名</th>
      <th scope="col">年龄</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <th scope="row">Alice</th>
      <td>20</td>
    </tr>
  </tbody>
</table>

caption ​

html
<table>
  <caption>员工信息表</caption>
  <!-- 表格内容 -->
</table>

常见面试问题 ​

Q: scope 属性的作用？

  • scope="col"：列标题
  • scope="row"：行标题
  • 帮助屏幕阅读器理解表格结构

常见面试问题 ​

Q: 表格的语义化结构？

html
<table>
  <caption>表格标题</caption>
  <thead>
    <tr>
      <th>表头1</th>
      <th>表头2</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>数据1</td>
      <td>数据2</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <td colspan="2">表尾</td>
    </tr>
  </tfoot>
</table>

Q: 什么时候使用表格？

  • 展示二维数据
  • 数据对比
  • 不应用于页面布局

Q: 表格和 div 布局的区别？

对比项表格div
用途展示数据页面布局
语义语义化无语义
性能较慢较快
响应式较难容易
