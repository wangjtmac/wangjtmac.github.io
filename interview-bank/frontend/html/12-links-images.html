HTML 链接和图片 ​

HTML 链接和图片相关面试题，包括 a 标签、img 标签、picture 标签等。

a 标签（链接） ​

基本用法 ​

html
<a href="https://example.com">访问网站</a>

常用属性 ​

属性说明示例
href链接地址href="https://example.com"
target打开方式target="_blank"
title提示文本title="提示信息"
download下载文件download="filename"
rel关系属性rel="noopener"

target 属性 ​

值说明
_self当前窗口（默认）
_blank新窗口
_parent父窗口
_top顶层窗口

常见面试问题 ​

Q: target="_blank" 的安全问题？

html
<!-- 不安全 -->
<a href="https://example.com" target="_blank">链接</a>

<!-- 安全 -->
<a href="https://example.com" target="_blank" rel="noopener noreferrer">链接</a>

Q: rel="noopener" 的作用？

防止新窗口访问原窗口的 window.opener，避免安全漏洞。

锚点链接 ​

页面内锚点 ​

html
<!-- 定义锚点 -->
<h2 id="section1">第一节</h2>

<!-- 跳转到锚点 -->
<a href="#section1">跳转到第一节</a>

跨页面锚点 ​

html
<!-- 跳转到其他页面的锚点 -->
<a href="page.html#section1">跳转到第一节</a>

平滑滚动 ​

html
<!-- HTML5 方式 -->
<a href="#section1" style="scroll-behavior: smooth;">跳转</a>

<!-- CSS 方式 -->
<style>
  html {
    scroll-behavior: smooth;
  }
</style>

常见面试问题 ​

Q: 如何实现平滑滚动？

css
html {
  scroll-behavior: smooth;
}

Q: 锚点链接的格式？

html
<!-- 页面内 -->
<a href="#id">跳转</a>

<!-- 跨页面 -->
<a href="page.html#id">跳转</a>

下载链接 ​

基本用法 ​

html
<a href="file.pdf" download>下载 PDF</a>
<a href="file.pdf" download="filename.pdf">下载 PDF</a>

常见面试问题 ​

Q: download 属性的作用？

  • 指示浏览器下载文件而非打开
  • 可以指定下载文件名

邮件链接 ​

基本用法 ​

html
<a href="mailto:example@example.com">发送邮件</a>

带主题和内容 ​

html
<a href="mailto:example@example.com?subject=主题&body=内容">发送邮件</a>

多个收件人 ​

html
<a href="mailto:example1@example.com,example2@example.com">发送邮件</a>

常见面试问题 ​

Q: mailto 链接的格式？

html
<a href="mailto:email?subject=主题&body=内容">发送邮件</a>

电话链接 ​

基本用法 ​

html
<a href="tel:1234567890">拨打电话</a>

常见面试问题 ​

Q: tel 链接的作用？

在移动设备上点击可以直接拨打电话。

img 标签（图片） ​

基本用法 ​

html
<img src="image.jpg" alt="图片描述">

常用属性 ​

属性说明示例
src图片路径src="image.jpg"
alt替代文本alt="图片描述"
title提示文本title="提示信息"
width宽度width="200"
height高度height="150"
loading加载方式loading="lazy"
decoding解码方式decoding="async"

loading 属性 ​

html
<!-- 懒加载 -->
<img src="image.jpg" alt="图片" loading="lazy">

<!-- 立即加载 -->
<img src="image.jpg" alt="图片" loading="eager">

decoding 属性 ​

html
<!-- 异步解码 -->
<img src="image.jpg" alt="图片" decoding="async">

<!-- 同步解码 -->
<img src="image.jpg" alt="图片" decoding="sync">

响应式图片 ​

html
<img src="image-800.jpg"
     srcset="image-400.jpg 400w,
             image-800.jpg 800w,
             image-1200.jpg 1200w"
     sizes="(max-width: 600px) 400px,
            (max-width: 1200px) 800px,
            1200px"
     alt="响应式图片">

常见面试问题 ​

Q: alt 属性的作用？

  • 图片无法显示时显示替代文本
  • 提升可访问性（屏幕阅读器）
  • 提升 SEO（搜索引擎理解图片）

Q: loading="lazy" 的作用？

图片懒加载，进入视口才加载，提升页面性能。

Q: srcset 和 sizes 的作用？

  • srcset：提供不同分辨率的图片
  • sizes：根据视口宽度选择合适的图片

picture 标签 ​

基本用法 ​

html
<picture>
  <source media="(min-width: 800px)" srcset="large.jpg">
  <source media="(min-width: 400px)" srcset="medium.jpg">
  <img src="small.jpg" alt="响应式图片">
</picture>

根据屏幕尺寸选择 ​

html
<picture>
  <source media="(min-width: 1200px)" srcset="desktop.jpg">
  <source media="(min-width: 768px)" srcset="tablet.jpg">
  <img src="mobile.jpg" alt="响应式图片">
</picture>

根据图片格式选择 ​

html
<picture>
  <source srcset="image.webp" type="image/webp">
  <source srcset="image.jpg" type="image/jpeg">
  <img src="image.jpg" alt="图片">
</picture>

常见面试问题 ​

Q: picture 标签的作用？

根据不同条件（媒体查询、图片格式）选择最合适的图片。

Q: picture 和 img srcset 的区别？

  • picture：更灵活，支持媒体查询和格式选择
  • img srcset：更简单，只支持分辨率选择

响应式图片 ​

基本原则 ​

  1. 使用合适的图片格式（WebP、AVIF）
  2. 提供不同分辨率的图片
  3. 根据设备选择合适的图片
  4. 使用懒加载

实现方式 ​

html
<!-- 方式 1：srcset -->
<img src="image-800.jpg"
     srcset="image-400.jpg 400w,
             image-800.jpg 800w,
             image-1200.jpg 1200w"
     sizes="(max-width: 600px) 400px,
            (max-width: 1200px) 800px,
            1200px"
     alt="响应式图片">

<!-- 方式 2：picture -->
<picture>
  <source media="(min-width: 800px)" srcset="large.jpg">
  <source media="(min-width: 400px)" srcset="medium.jpg">
  <img src="small.jpg" alt="响应式图片">
</picture>

<!-- 方式 3：CSS -->
<style>
  @media (max-width: 600px) {
    .responsive-img {
      background-image: url(small.jpg);
    }
  }
  @media (min-width: 601px) {
    .responsive-img {
      background-image: url(large.jpg);
    }
  }
</style>

常见面试问题 ​

Q: 如何优化图片加载？

  • 使用合适的格式（WebP、AVIF）
  • 压缩图片
  • 使用懒加载
  • 使用 CDN
  • 提供不同分辨率的图片

图片格式 ​

常见格式 ​

格式特点适用场景
JPEG有损压缩，体积小照片、复杂图像
PNG无损压缩，支持透明图标、简单图形
GIF支持动画，256 色简单动画
WebP高压缩率，支持透明现代浏览器
AVIF更高压缩率最新浏览器
SVG矢量图，可缩放图标、Logo

常见面试问题 ​

Q: 什么时候使用 SVG？

  • 图标
  • Logo
  • 需要缩放的图形
  • 简单的矢量图形

Q: 什么时候使用 WebP？

  • 现代浏览器
  • 需要高压缩率
  • 需要透明背景

Q: 如何实现图片懒加载？

html
<!-- HTML5 原生 -->
<img src="image.jpg" loading="lazy" alt="图片">

<!-- JavaScript -->
<img data-src="image.jpg" alt="图片">

<script>
  const img = document.querySelector('img[data-src]');
  img.src = img.dataset.src;
</script>

Q: 如何实现图片占位符？

html
<!-- 使用 src -->
<img src="placeholder.jpg" data-src="real.jpg" alt="图片">

<!-- 使用 CSS -->
<style>
  .lazy-img {
    background-color: #f0f0f0;
    min-height: 200px;
  }
</style>
<img class="lazy-img" data-src="real.jpg" alt="图片">
