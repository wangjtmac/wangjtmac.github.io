HTML 表单 ​

HTML 表单相关面试题，包括 form 元素、input 类型、表单验证、表单提交等。

form 表单元素 ​

基本结构 ​

html
<form action="/submit" method="POST" enctype="multipart/form-data">
  <label for="username">用户名：</label>
  <input type="text" id="username" name="username" required>

  <label for="password">密码：</label>
  <input type="password" id="password" name="password" required>

  <button type="submit">提交</button>
</form>

form 常用属性 ​

属性说明示例
action表单提交的 URLaction="/submit"
method请求方法（GET/POST）method="POST"
enctype编码类型enctype="multipart/form-data"
target打开方式target="_blank"
autocomplete自动完成autocomplete="on"
novalidate禁用验证novalidate

enctype 属性 ​

值说明适用场景
application/x-www-form-urlencoded默认编码普通表单
multipart/form-data文件上传包含文件上传
text/plain纯文本不推荐使用

Q: GET 和 POST 的区别？

对比项GETPOST
数据位置URL 参数请求体
数据大小有限制无限制
安全性较低较高

label 标签 ​

基本用法 ​

html
<!-- 隐式关联 -->
<label>
  用户名：<input type="text" name="username">
</label>

<!-- 显式关联（推荐） -->
<label for="username">用户名：</label>
<input type="text" id="username" name="username">

作用 ​

  • 提升可访问性（屏幕阅读器）
  • 扩大点击区域（点击 label 聚焦 input）

常见面试问题 ​

Q: label 的 for 属性和 input 的 id 属性的关系？

for 属性的值应该与对应 input 的 id 属性值相同。

表单提交 ​

提交方式 ​

html
<!-- 表单提交 -->
<form action="/submit" method="POST">
  <button type="submit">提交</button>
</form>

<!-- JavaScript 提交 -->
<form id="myForm">
  <button type="button" onclick="submitForm()">提交</button>
</form>

<script>
function submitForm() {
  const form = document.getElementById('myForm');
  form.submit();
}
</script>

<!-- AJAX 提交 -->
<form id="myForm">
  <button type="button" onclick="ajaxSubmit()">提交</button>
</form>

<script>
function ajaxSubmit() {
  const form = document.getElementById('myForm');
  const formData = new FormData(form);

  fetch('/submit', {
    method: 'POST',
    body: formData
  });
}
</script>

表单数据获取 ​

FormData ​

javascript
const form = document.getElementById('myForm');
const formData = new FormData(form);

// 获取单个值
const username = formData.get('username');

// 获取所有值
const values = formData.getAll('hobbies');

// 遍历
for (const [key, value] of formData) {
  console.log(key, value);
}

// 转为对象
const obj = Object.fromEntries(formData);

传统方式 ​

javascript
const form = document.getElementById('myForm');

// 获取所有 input
const inputs = form.querySelectorAll('input, select, textarea');

// 手动构建数据
const data = {};
inputs.forEach(input => {
  data[input.name] = input.value;
});
