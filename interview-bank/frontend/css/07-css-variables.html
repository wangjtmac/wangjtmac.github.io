CSS 变量 ​

CSS 变量（自定义属性）相关面试题，包括变量定义、变量使用、变量作用域、变量继承等。

基本概念 ​

什么是 CSS 变量 ​

CSS 变量（Custom Properties）是 CSS 中定义的自定义属性，可以在整个文档中重复使用。

变量定义 ​

css
/* 定义变量 */
:root {
  --primary-color: #333;
  --secondary-color: #666;
  --font-size: 16px;
  --spacing: 10px;
}

/* 局部变量 */
.container {
  --container-width: 1200px;
}

变量使用 ​

css
/* 使用变量 */
.button {
  color: var(--primary-color);
  font-size: var(--font-size);
  padding: var(--spacing);
}

/* 变量默认值 */
.button {
  color: var(--primary-color, #333);
  font-size: var(--font-size, 16px);
}

变量作用域 ​

全局变量 ​

css
/* :root 伪类定义全局变量 */
:root {
  --primary-color: #333;
  --secondary-color: #666;
}

/* 所有元素都可以使用 */
body {
  color: var(--primary-color);
}

局部变量 ​

css
.container {
  --container-width: 1200px;
  --container-padding: 20px;
  width: var(--container-width);
  padding: var(--container-padding);
}

/* 子元素可以继承 */
.container .content {
  padding: var(--container-padding);
}

/* 外部元素无法访问 */
.header {
  width: var(--container-width); /* 无效 */
}

变量优先级 ​

css
/* 全局变量 */
:root {
  --color: red;
}

/* 局部变量覆盖全局变量 */
.container {
  --color: blue;
  color: var(--color); /* blue */
}

/* 优先级：内联样式 > ID > 类 > 标签 > 全局 */

变量类型 ​

颜色变量 ​

css
:root {
  --primary-color: #333;
  --secondary-color: rgb(51, 51, 51);
  --accent-color: rgba(51, 51, 51, 0.5);
  --text-color: hsl(0, 0%, 20%);
}

.button {
  color: var(--primary-color);
  background: var(--secondary-color);
}

数值变量 ​

css
:root {
  --font-size: 16px;
  --spacing: 10px;
  --border-radius: 4px;
  --opacity: 0.8;
}

.box {
  font-size: var(--font-size);
  padding: var(--spacing);
  border-radius: var(--border-radius);
  opacity: var(--opacity);
}

字符串变量 ​

css
:root {
  --font-family: 'Arial', sans-serif;
  --content: 'Hello World';
}

body {
  font-family: var(--font-family);
}

.element::before {
  content: var(--content);
}

URL 变量 ​

css
:root {
  --background-image: url('background.jpg');
  --icon-image: url('icon.png');
}

.box {
  background: var(--background-image);
}

变量计算 ​

calc() 函数 ​

css
:root {
  --base-size: 16px;
  --spacing: 10px;
  --border-width: 2px;
}

.box {
  /* 变量计算 */
  font-size: calc(var(--base-size) * 1.5);
  padding: calc(var(--spacing) * 2);
  border: var(--border-width) solid #333;
  
  /* 组合计算 */
  width: calc(100% - var(--spacing) * 2);
}

变量拼接 ​

css
:root {
  --prefix: 'prefix-';
  --suffix: '-suffix';
}

.element {
  /* 字符串拼接 */
  content: var(--prefix) 'value' var(--suffix);
}

变量继承 ​

变量继承规则 ​

css
:root {
  --color: red;
}

.parent {
  --color: blue;
}

/* 子元素继承父元素的变量 */
.child {
  color: var(--color); /* blue */
}

/* 重置变量 */
.child {
  --color: green;
  color: var(--color); /* green */
}

变量重置 ​

css
:root {
  --color: red;
}

/* 重置变量为初始值 */
.container {
  --color: initial;
  color: var(--color); /* 继承默认值 */
}

/* 重置变量为继承值 */
.container {
  --color: inherit;
  color: var(--color); /* 继承父元素的值 */
}

变量与 JavaScript ​

获取变量值 ​

javascript
// 获取元素的计算样式
const element = document.querySelector('.element');
const styles = getComputedStyle(element);

// 获取变量值
const color = styles.getPropertyValue('--color');
console.log(color); // '#333'

设置变量值 ​

javascript
// 设置元素变量
const element = document.querySelector('.element');
element.style.setProperty('--color', '#333');

// 设置根元素变量
document.documentElement.style.setProperty('--color', '#333');

监听变量变化 ​

javascript
// 监听变量变化（需要 MutationObserver）
const observer = new MutationObserver((mutations) => {
  mutations.forEach((mutation) => {
    if (mutation.type === 'attributes' && mutation.attributeName === 'style') {
      console.log('变量已更新');
    }
  });
});

observer.observe(document.documentElement, {
  attributes: true,
  attributeFilter: ['style']
});

变量应用场景 ​

主题切换 ​

css
:root {
  --primary-color: #333;
  --background-color: #fff;
  --text-color: #333;
}

[data-theme="dark"] {
  --primary-color: #fff;
  --background-color: #333;
  --text-color: #fff;
}

body {
  background: var(--background-color);
  color: var(--text-color);
}
javascript
// 切换主题
function toggleTheme() {
  const currentTheme = document.documentElement.getAttribute('data-theme');
  const newTheme = currentTheme === 'dark' ? 'light' : 'dark';
  document.documentElement.setAttribute('data-theme', newTheme);
}

响应式设计 ​

css
:root {
  --font-size: 16px;
  --spacing: 10px;
}

@media (min-width: 768px) {
  :root {
    --font-size: 18px;
    --spacing: 15px;
  }
}

@media (min-width: 1200px) {
  :root {
    --font-size: 20px;
    --spacing: 20px;
  }
}

.container {
  font-size: var(--font-size);
  padding: var(--spacing);
}

组件样式 ​

css
:root {
  --button-primary-color: #007bff;
  --button-primary-hover: #0056b3;
  --button-padding: 10px 20px;
  --button-border-radius: 4px;
}

.button-primary {
  background: var(--button-primary-color);
  color: white;
  padding: var(--button-padding);
  border-radius: var(--button-border-radius);
}

.button-primary:hover {
  background: var(--button-primary-hover);
}

常见面试问题 ​

Q: CSS 变量和 Sass 变量的区别？

对比项CSS 变量Sass 变量
运行时运行时编译时
作用域CSS 作用域Sass 作用域
动态性可动态修改不可动态修改
JavaScript 交互可交互不可交互
浏览器支持现代浏览器需要编译

Q: CSS 变量的命名规范？

  • 必须以 -- 开头
  • 区分大小写
  • 不能包含 $、{、}、;、(、) 等特殊字符
css
/* 正确 */
--primary-color: #333;
--fontSize: 16px;

/* 错误 */
$primary-color: #333;
--primary-color: #333;
--primary color: #333;

Q: 如何实现 CSS 变量的默认值？

使用 var() 函数的第二个参数：

css
.button {
  color: var(--primary-color, #333);
  font-size: var(--font-size, 16px);
}

Q: CSS 变量的浏览器兼容性？

  • 现代浏览器都支持
  • IE 不支持
  • 可以使用 @supports 检测支持
css
@supports (--css: variables) {
  /* 支持 CSS 变量的样式 */
  .box {
    color: var(--color);
  }
}

@supports not (--css: variables) {
  /* 不支持 CSS 变量的回退样式 */
  .box {
    color: #333;
  }
}

Q: 如何在 JavaScript 中操作 CSS 变量？

javascript
// 获取变量值
const styles = getComputedStyle(element);
const color = styles.getPropertyValue('--color');

// 设置变量值
element.style.setProperty('--color', '#333');

// 删除变量值
element.style.removeProperty('--color');
