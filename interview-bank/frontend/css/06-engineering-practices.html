常用特性 ​

变量

scss

.button {
  color: $primary-color;
}

嵌套

scss
.nav {
  ul {
    list-style: none;
  }
  li {
    display: inline-block;
  }
}

混入（Mixin）

scss
@mixin flex-center {
  display: flex;
  justify-content: center;
  align-items: center;
}

.container {
  @include flex-center;
}

带参数的混入

scss
@mixin border-radius($radius) {
  -webkit-border-radius: $radius;
  -moz-border-radius: $radius;
  border-radius: $radius;
}

.box {
  @include border-radius(10px);
}

继承

scss
.button {
  padding: 10px 20px;
}

.primary-button {
  @extend .button;
  background: blue;
}

运算

scss
.container {
  width: 100% / 3;
  margin: 10px + 5px;
}

条件判断

scss
@mixin responsive($breakpoint) {
  @if $breakpoint == mobile {
    @media (max-width: 767px) {
      @content;
    }
  } @else if $breakpoint == tablet {
    @media (min-width: 768px) and (max-width: 991px) {
      @content;
    }
  }
}

.container {
  @include responsive(mobile) {
    width: 100%;
  }
}

循环

scss
@for $i from 1 through 3 {
  .col-#{$i} {
    width: 100% / 3 * $i;
  }
}

CSS 性能优化 ​

优化策略 ​

策略说明
压缩 CSS减小文件体积
合并文件减少 HTTP 请求
使用 CDN加速资源加载
异步加载非关键 CSS 异步加载
避免深层嵌套减少选择器匹配时间
使用简写属性margin: 10px 代替四个方向
避免使用 @import阻塞渲染
使用 will-change提前告知浏览器优化
减少重排重绘使用 transform 和 opacity

关键渲染路径优化 ​

html
<!-- 内联关键 CSS -->
<style>
  .header { ... }
  .hero { ... }
</style>

<!-- 异步加载非关键 CSS -->
<link rel="preload" href="style.css" as="style" onload="this.rel='stylesheet'">

减少重排重绘 ​

css
/* 触发重排的属性 */
width, height, padding, margin, border, display, position, top, left, right, bottom

/* 触发重绘的属性 */
color, background, background-image, box-shadow, text-decoration

/* 只触发合成的属性 */
transform, opacity, filter

/* 优化示例 */
.bad {
  left: 100px; /* 触发重排 */
}

.good {
  transform: translateX(100px); /* 只触发合成 */
}

GPU 加速 ​

css
.gpu-accelerated {
  transform: translateZ(0);
  /* 或 */
  will-change: transform;
}

常见面试问题 ​

Q: 为什么不建议使用 @import？

  • @import 会阻塞渲染
  • 无法并行下载
  • 建议使用 <link> 标签

Q: 如何优化动画性能？

  • 使用 transform 和 opacity（触发合成层）
  • 避免使用 left/top（触发重排）
  • 使用 will-change 提示浏览器
css
.animated {
  will-change: transform;
  transform: translateX(0);
}

Q: 什么是重排和重绘？

  • 重排（Reflow）：元素尺寸、位置改变，需要重新计算布局
  • 重绘（Repaint）：元素外观改变，不需要重新计算布局

Q: 如何减少重排重绘？

  1. 批量修改 DOM
  2. 使用文档片段（DocumentFragment）
  3. 使用 transform 和 opacity
  4. 避免逐条修改样式
javascript
// 不好的做法
element.style.width = '100px';
element.style.height = '100px';
element.style.background = 'red';

// 好的做法
element.style.cssText = 'width: 100px; height: 100px; background: red;';

CSS 单位区别 ​

长度单位分类 ​

类型单位说明
绝对单位px像素，最常用
相对字体em相对于父元素字体大小
相对字体rem相对于根元素字体大小
相对视口vw视口宽度的 1%
相对视口vh视口高度的 1%
相对视口vminvw 和 vh 中较小值
相对视口vmaxvw 和 vh 中较大值
百分比%相对于父元素

em 与 rem 区别 ​

css
html {
  font-size: 16px;
}
.parent {
  font-size: 20px;
}
.child {
  /* em 相对于父元素 */
  padding: 2em;  /* 20px * 2 = 40px */
  
  /* rem 相对于根元素 */
  margin: 2rem;  /* 16px * 2 = 32px */
}

使用场景 ​

单位场景
px边框、固定尺寸
em按钮内边距、图标大小
rem移动端适配、整体布局
vw/vh全屏布局、响应式设计

常见面试问题 ​

Q: rem 和 em 的区别？

  • rem：相对于根元素（<html>）的字体大小
  • em：相对于父元素的字体大小

Q: vw 和 % 的区别？

  • vw：相对于视口宽度
  • %：相对于父元素宽度

</div>

  padding: 20px;
`;

const Title = styled.h1`
  font-size: 24px;
`;

<Container>
  <Title>Title</Title>
</Container>

  • 实现样式局部化
  • 支持模块化开发
