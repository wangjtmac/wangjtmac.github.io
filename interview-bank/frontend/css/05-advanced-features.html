CSS 高级特性 ​

CSS 高级特性相关面试题，包括层叠上下文、隐藏元素、CSS 绘制图形、多行文本省略等。

层叠上下文 ​

什么是层叠上下文 ​

层叠上下文（Stacking Context）是 HTML 元素的三维概念，决定元素的层叠顺序。

触发条件 ​

条件示例
根元素<html>
position + z-indexposition: relative; z-index: 1;
position: fixed/sticky自动创建
opacity < 1opacity: 0.9;
transform 非 nonetransform: scale(1);
filter 非 nonefilter: blur(0);
isolation: isolate创建新的层叠上下文
will-changewill-change: transform;

层叠顺序 ​

从下到上：

  1. 层叠上下文的 background/border
  2. 负 z-index
  3. block 块级元素
  4. float 浮动元素
  5. inline/inline-block 元素
  6. z-index: 0 / auto
  7. 正 z-index

常见面试问题 ​

Q: 为什么 z-index 不生效？

z-index 只在定位元素（非 static）或创建了层叠上下文的元素上生效。

css
.box {
  position: relative;
  z-index: 1;
}

Q: 如何解决 z-index 层叠问题？

  1. 确保元素已定位（非 static）
  2. 检查是否创建了新的层叠上下文
  3. 调整父元素的 z-index

隐藏元素的方式 ​

方式是否占位是否可交互是否触发重排
display: none否否是
visibility: hidden是否否
opacity: 0是是否
position: absolute + 移出屏幕否是是
clip-path: circle(0)是否否
height: 0 + overflow: hidden否否是

常见面试问题 ​

Q: display: none 和 visibility: hidden 的区别？

  • display: none：不占位，触发重排，绑定事件不触发
  • visibility: hidden：占位，只触发重绘，绑定事件不触发

Q: opacity: 0 和 visibility: hidden 的区别？

  • opacity: 0：占位，可交互（可点击）
  • visibility: hidden：占位，不可交互

常见面试问题 ​

Q: 多行省略的兼容性问题？

-webkit-line-clamp 是 WebKit 私有属性，非标准。兼容方案：

  • 使用 JavaScript 截断文本
  • 使用 CSS 渐变遮罩
css
.fallback {
  display: -webkit-box;
  -webkit-box-orient: vertical;
  -webkit-line-clamp: 3;
  overflow: hidden;
  max-height: 4.5em; /* 回退方案 */
}

CSS 遮罩 ​

基本遮罩 ​

css
.box {
  /* 图片遮罩 */
  mask-image: url('mask.png');
  mask-mode: alpha;
  mask-repeat: no-repeat;
  mask-position: center;
  mask-size: cover;
}

/* 简写 */
.box {
  mask: url('mask.png') center / cover no-repeat alpha;
}

渐变遮罩 ​

css
.box {
  /* 线性渐变遮罩 */
  mask-image: linear-gradient(to bottom, black, transparent);
  
  /* 径向渐变遮罩 */
  mask-image: radial-gradient(circle, black 0%, transparent 70%);
}

SVG 遮罩 ​

html
<svg>
  <defs>
    <mask id="mask1">
      <rect x="0" y="0" width="100" height="100" fill="white" />
      <circle cx="50" cy="50" r="30" fill="black" />
    </mask>
  </defs>
</svg>

<style>
.box {
  mask: url(#mask1);
}
</style>

遮罩属性 ​

属性说明值
mask-image遮罩图像url()、linear-gradient()、radial-gradient()
mask-mode遮罩模式alpha、luminance
mask-repeat重复方式repeat、no-repeat、space、round
mask-position位置center、top、left、right、bottom
mask-size尺寸cover、contain、100px、50%
mask-clip裁剪区域border-box、padding-box、content-box
mask-origin定位区域border-box、padding-box、content-box
mask-composite组合方式add、subtract、intersect、exclude

遮罩示例 ​

css
/* 淡出效果 */
.fade-out {
  mask-image: linear-gradient(to bottom, black, transparent);
}

/* 圆形遮罩 */
.circle-mask {
  mask-image: radial-gradient(circle, black 0%, transparent 70%);
}

/* 图片遮罩 */
.image-mask {
  mask-image: url('mask.png');
  mask-size: cover;
  mask-position: center;
}

CSS 滚动捕捉 ​

基本用法 ​

css
.container {
  scroll-snap-type: x mandatory;
  overflow-x: scroll;
  display: flex;
}

.item {
  scroll-snap-align: start;
  flex-shrink: 0;
  width: 100%;
}

滚动捕捉类型 ​

css
.container {
  /* X 轴捕捉 */
  scroll-snap-type: x mandatory;
  
  /* Y 轴捕捉 */
  scroll-snap-type: y mandatory;
  
  /* 两个轴捕捉 */
  scroll-snap-type: both mandatory;
  
  /* 非强制捕捉 */
  scroll-snap-type: x proximity;
}

滚动捕捉对齐 ​

css
.item {
  /* 起始对齐 */
  scroll-snap-align: start;
  
  /* 结束对齐 */
  scroll-snap-align: end;
  
  /* 中心对齐 */
  scroll-snap-align: center;
  
  /* 无对齐 */
  scroll-snap-align: none;
}

滚动捕捉停止 ​

css
.container {
  /* 总是停止 */
  scroll-snap-stop: always;
  
  /* 正常停止 */
  scroll-snap-stop: normal;
}

滚动捕捉属性 ​

属性说明值
scroll-snap-type捕捉类型none、x mandatory、y mandatory、both mandatory
scroll-snap-align对齐方式none、start、end、center
scroll-snap-stop停止方式normal、always

滚动捕捉示例 ​

css
/* 横向轮播 */

  scroll-behavior: smooth;
}

/* 纵向滚动 */

CSS 打印样式 ​

基本打印样式 ​

css

  /* 隐藏不需要打印的元素 */

  /* 设置打印背景 */

  }

  }

分页控制 ​

css
@media print {
  /* 分页前 */
  .page-break-before {
    page-break-before: always;
  }
  
  /* 分页后 */
  .page-break-after {
    page-break-after: always;
  }
  
  /* 避免分页 */
  .no-break {
    page-break-inside: avoid;
  }
  
  /* 新属性 */
  .break-before {
    break-before: page;
  }
  
  .break-after {
    break-after: page;
  }

    break-inside: avoid;
  }
}
