CSS 响应式设计 ​

CSS 响应式设计相关面试题，包括媒体查询、移动端适配方案、常见断点等。

媒体查询 ​

基本语法 ​

css
/* 移动端优先 */
.container {
  width: 100%;
}

@media (min-width: 768px) {
  .container {
    width: 750px;
  }
}

@media (min-width: 992px) {
  .container {
    width: 970px;
  }
}

@media (min-width: 1200px) {
  .container {
    width: 1170px;
  }
}

常见断点 ​

设备断点
手机< 768px
平板768px - 991px
桌面992px - 1199px
大桌面≥ 1200px

媒体查询类型 ​

css
/* 屏幕设备 */
@media screen and (min-width: 768px) { }

/* 打印设备 */
@media print { }

/* 多个条件 */
@media screen and (min-width: 768px) and (max-width: 991px) { }

/* 或条件 */
@media screen and (min-width: 768px), (orientation: landscape) { }

/* 否定条件 */
@media not screen { }

移动端适配方案 ​

方案对比 ​

方案说明优点缺点
rem基于根元素字体大小，需配合 JS 或媒体查询灵活、可缩放需要额外计算
vw/vh基于视口宽高，1vw = 视口宽度的 1%简单直接兼容性问题
%基于父元素，适合流式布局简单不适合精确控制

rem 适配方案 ​

css
/* 方案 1：媒体查询 */
html {
  font-size: 16px;
}

@media (min-width: 375px) {
  html {
    font-size: 16px;
  }
}

@media (min-width: 414px) {
  html {
    font-size: 17.6px;
  }
}

/* 方案 2：JS 动态设置 */
html {
  font-size: calc(100vw / 3.75); /* 设计稿 375px */
}

.box {
  width: 1rem; /* 设计稿 100px */
}

vw/vh 适配方案 ​

css
.container {
  width: 100vw;
  height: 100vh;
}

.box {
  width: 50vw;  /* 视口宽度的 50% */
  height: 50vh; /* 视口高度的 50% */
}

常见面试问题 ​

Q: rem 和 em 的区别？

  • rem：相对于根元素（<html>）的字体大小
  • em：相对于父元素的字体大小

Q: 如何实现移动端 1px 边框？

css
/* 方案 1：使用 transform */
.border-1px {
  position: relative;
}
.border-1px::after {
  content: "";
  position: absolute;
  top: 0;
  left: 0;
  width: 200%;
  height: 200%;
  border: 1px solid #000;
  transform: scale(0.5);
  transform-origin: 0 0;
}

/* 方案 2：使用伪元素 + 缩放 */
.border-1px {
  position: relative;
}
.border-1px::before {
  content: "";
  position: absolute;
  top: 0;
  left: 0;
  width: 100%;
  height: 1px;
  background: #000;
  transform: scaleY(0.5);
}

响应式图片 ​

srcset 属性 ​

html
<img 
  src="image-320w.jpg" 
  srcset="image-320w.jpg 320w,
          image-640w.jpg 640w,
          image-1280w.jpg 1280w"
  sizes="(max-width: 600px) 100vw, 50vw"
  alt="响应式图片">

picture 标签 ​

html
<picture>
  <source media="(min-width: 800px)" srcset="large.jpg">
  <source media="(min-width: 480px)" srcset="medium.jpg">
  <img src="small.jpg" alt="响应式图片">
</picture>

常见面试问题 ​

Q: srcset 和 sizes 的作用？

  • srcset：提供多个不同分辨率的图片，浏览器根据设备选择
  • sizes：定义图片在不同屏幕宽度下的显示尺寸，帮助浏览器选择合适的图片
