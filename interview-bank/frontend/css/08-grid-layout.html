Grid 布局 ​

Grid 布局相关面试题，包括 Grid 容器属性、Grid 项目属性、Grid 模板、Grid 对齐等。

基本概念 ​

什么是 Grid 布局 ​

Grid 布局（CSS Grid Layout）是 CSS 中最强大的布局系统，是一个二维布局系统，可以同时处理行和列。

基本语法 ​

css
.container {
  display: grid;
}

.inline-grid {
  display: inline-grid;
}

Grid 容器属性 ​

grid-template-columns 和 grid-template-rows ​

css
.container {
  /* 固定值 */
  grid-template-columns: 100px 100px 100px;
  grid-template-rows: 100px 100px;
  
  /* 百分比 */
  grid-template-columns: 33.33% 33.33% 33.33%;
  
  /* fr 单位 */
  grid-template-columns: 1fr 1fr 1fr;
  
  /* 混合使用 */
  grid-template-columns: 200px 1fr 100px;
  
  /* repeat 函数 */
  grid-template-columns: repeat(3, 1fr);
  grid-template-columns: repeat(3, 100px);
  
  /* auto-fill 和 auto-fit */
  grid-template-columns: repeat(auto-fill, minmax(100px, 1fr));
  grid-template-columns: repeat(auto-fit, minmax(100px, 1fr));
}

grid-template-areas ​

css
.container {
  grid-template-areas:
    "header header header"
    "sidebar content content"
    "footer footer footer";
}

.header {
  grid-area: header;
}

.sidebar {
  grid-area: sidebar;
}

.content {
  grid-area: content;
}

.footer {
  grid-area: footer;
}

grid-template ​

css
.container {
  /* 简写形式 */
  grid-template:
    "header header header" 100px
    "sidebar content content" 1fr
    "footer footer footer" 50px
    / 200px 1fr 1fr;
}

grid-column-gap 和 grid-row-gap ​

css
.container {
  /* 列间距 */
  grid-column-gap: 20px;
  
  /* 行间距 */
  grid-row-gap: 20px;
  
  /* 简写形式 */
  gap: 20px;
  gap: 20px 30px; /* 行 列 */
}

grid-auto-rows 和 grid-auto-columns ​

css
.container {
  /* 自动行高 */
  grid-auto-rows: 100px;
  grid-auto-rows: minmax(100px, auto);
  
  /* 自动列宽 */
  grid-auto-columns: 100px;
  grid-auto-columns: minmax(100px, auto);
}

grid-auto-flow ​

css
.container {
  /* 默认：行优先 */
  grid-auto-flow: row;
  
  /* 列优先 */
  grid-auto-flow: column;
  
  /* 稠密排列 */
  grid-auto-flow: row dense;
  grid-auto-flow: column dense;
}

Grid 项目属性 ​

grid-column 和 grid-row ​

css
.item {
  /* 跨列 */
  grid-column: 1 / 3;
  grid-column: 1 / span 2;
  grid-column: span 2;
  
  /* 跨行 */
  grid-row: 1 / 3;
  grid-row: 1 / span 2;
  grid-row: span 2;
  
  /* 简写形式 */
  grid-column: 1 / span 2;
  grid-row: 1 / span 2;
}

grid-area ​

css
.item {
  /* 使用命名区域 */
  grid-area: header;
  
  /* 使用行/列 */
  grid-area: 1 / 1 / 2 / 3;
  grid-area: 1 / 1 / span 1 / span 2;
}

justify-self 和 align-self ​

css
.item {
  /* 水平对齐 */
  justify-self: start;
  justify-self: end;
  justify-self: center;
  justify-self: stretch;
  
  /* 垂直对齐 */
  align-self: start;
  align-self: end;
  align-self: center;
  align-self: stretch;
}

place-self ​

css
.item {
  /* 简写形式 */
  place-self: center center;
  place-self: stretch center;
}

Grid 和 Flex 对比 ​

对比项GridFlex
维度二维（行和列）一维（行或列）
适用场景复杂布局、二维布局简单布局、一维布局
对齐方式justify/align-items/contentjustify/align-items
重叠支持不支持
浏览器支持现代浏览器现代浏览器

何时使用 Grid ​

  • 需要二维布局（行和列）
  • 需要精确控制元素位置
  • 需要重叠元素
  • 需要复杂的网格布局

何时使用 Flex ​

  • 需要一维布局（行或列）
  • 需要简单的对齐
  • 需要动态调整元素大小
  • 需要响应式布局

常见面试问题 ​

Q: Grid 和 Flex 的区别？

Grid 是二维布局系统，可以同时处理行和列；Flex 是一维布局系统，只能处理行或列。Grid 适合复杂布局，Flex 适合简单布局。

Q: fr 单位是什么？

fr 是 Grid 布局中的分数单位，表示可用空间的份数。例如 1fr 1fr 1fr 表示三等分。

css
.container {
  grid-template-columns: 1fr 2fr 1fr;
  /* 第一列和第三列各占 1/4，第二列占 1/2 */
}

Q: minmax() 函数的作用？

minmax() 函数定义一个长度范围，最小值和最大值之间。

css
.container {
  grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));
  /* 最小 200px，最大 1fr */
}

Q: auto-fill 和 auto-fit 的区别？

  • auto-fill：尽可能多地填充列，即使有空列
  • auto-fit：尽可能多地填充列，但会收缩空列
css
.container {
  grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));
  /* 可能会有空列 */
}

.container {
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
  /* 不会有空列 */
}

Q: 如何实现响应式 Grid 布局？

使用 auto-fit 和 minmax() 函数：

css
.container {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
  gap: 20px;
}

或者使用媒体查询：

css
.container {
  display: grid;
  grid-template-columns: 1fr;
}

@media (min-width: 768px) {
  .container {
    grid-template-columns: repeat(2, 1fr);
  }
}

@media (min-width: 1200px) {
  .container {
    grid-template-columns: repeat(3, 1fr);
  }
}

Q: Grid 布局的浏览器兼容性？

  • 现代浏览器都支持
  • IE 部分支持（需要 -ms- 前缀）
  • 可以使用 @supports 检测支持
css
@supports (display: grid) {
  /* 支持 Grid 的样式 */
  .container {
    display: grid;
  }
}

@supports not (display: grid) {
  /* 不支持 Grid 的回退样式 */
  .container {
    display: flex;
    flex-wrap: wrap;
  }
}
