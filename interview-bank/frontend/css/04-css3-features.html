新增选择器 ​

属性选择器 ​

css
E[attr^="val"]  /* 属性值以 val 开头 */
E[attr$="val"]  /* 属性值以 val 结尾 */
E[attr*="val"]  /* 属性值包含 val */

结构伪类 ​

css
E:first-of-type  /* 当前类型第一个子元素 */
E:last-of-type   /* 当前类型最后一个子元素 */
E:only-of-type    /* 唯一的当前类型子元素 */
E:nth-child(n)   /* 第 n 个子元素 */
E:nth-of-type(n) /* 第 n 个指定类型的子元素 */
E:not(selector)  /* 不匹配选择器的元素 */
E:empty          /* 空元素（无子元素或文本） */

css

变换 ​

基本变换 ​

css
.box {
  /* 平移 */
  transform: translate(100px, 50px);  /* X, Y */
  transform: translateX(100px);
  transform: translateY(50px);
  transform: translateZ(50px);       /* 3D */
  
  /* 旋转 */
  transform: rotate(45deg);          /* 2D 旋转 */
  transform: rotateX(45deg);        /* 3D X 轴旋转 */
  transform: rotateY(45deg);        /* 3D Y 轴旋转 */
  transform: rotateZ(45deg);        /* 3D Z 轴旋转 */
  
  /* 缩放 */
  transform: scale(1.5);            /* X, Y 同时缩放 */
  transform: scaleX(1.5);
  transform: scaleY(1.5);
  transform: scaleZ(1.5);          /* 3D */
  
  /* 倾斜 */
  transform: skew(30deg, 20deg);    /* X, Y */
  transform: skewX(30deg);
  transform: skewY(20deg);
  
  /* 组合变换 */
  transform: translate(100px) rotate(45deg) scale(1.5);
}

变换原点 ​

css
.box {
  transform-origin: center center;  /* 默认 */
  transform-origin: left top;
  transform-origin: 50% 50%;
  transform-origin: 10px 10px;
}

3D 变换 ​

css
.container {
  perspective: 1000px;  /* 透视距离 */
}

.box {
  transform-style: preserve-3d;  /* 保留 3D 空间 */
  transform: rotateX(45deg) rotateY(45deg);
}

过渡 ​

基本语法 ​

css
.box {
  transition: all 0.3s ease;
  /* 属性 时间 缓动函数 延迟 */
}

过渡属性 ​

属性说明示例
transition-property过渡的属性all、width、transform
transition-duration过渡时间0.3s、500ms
transition-timing-function缓动函数ease、linear、ease-in
transition-delay延迟时间0.1s、100ms

缓动函数 ​

css
.box {
  /* 预定义函数 */
  transition-timing-function: ease;        /* 默认 */
  transition-timing-function: linear;      /* 线性 */
  transition-timing-function: ease-in;     /* 慢速开始 */
  transition-timing-function: ease-out;    /* 慢速结束 */
  transition-timing-function: ease-in-out; /* 慢速开始和结束 */
  
  /* 自定义贝塞尔曲线 */
  transition-timing-function: cubic-bezier(0.25, 0.1, 0.25, 1.0);
  transition-timing-function: cubic-bezier(0.68, -0.55, 0.265, 1.55); /* 弹跳效果 */
}

过渡示例 ​

css
.button {
  background: blue;
  color: white;
  transition: background 0.3s ease, transform 0.3s ease;
}

.button:hover {
  background: darkblue;
  transform: scale(1.1);
}

动画 ​

基本语法 ​

css
.box {
  animation: fadeIn 1s ease forwards;
  /* 动画名 时间 缓动函数 延迟 次数 方向 填充模式 */
}

动画属性 ​

属性说明示例
animation-name动画名称fadeIn
animation-duration动画时间1s
animation-timing-function缓动函数ease、linear
animation-delay延迟时间0.5s
animation-iteration-count循环次数1、infinite
animation-direction动画方向normal、alternate
animation-fill-mode填充模式forwards、backwards
animation-play-state播放状态running、paused

关键帧 ​

css
/* 简单动画 */
@keyframes fadeIn {
  from { opacity: 0; }
  to { opacity: 1; }
}

/* 百分比动画 */
@keyframes slideIn {
  0% { transform: translateX(-100%); }
  50% { transform: translateX(50%); }
  100% { transform: translateX(0); }
}

/* 多属性动画 */
@keyframes bounce {
  0%, 100% {
    transform: translateY(0);
  }
  50% {
    transform: translateY(-20px);
  }
}

/* 复杂动画 */
@keyframes complex {
  0% {
    transform: scale(1) rotate(0deg);
    opacity: 0;
  }
  50% {
    transform: scale(1.2) rotate(180deg);
    opacity: 1;
  }
  100% {
    transform: scale(1) rotate(360deg);
    opacity: 0;
  }
}

动画示例 ​

css
/* 淡入动画 */
.fade-in {

/* 无限旋转 */

@keyframes pulse {
  0%, 100% { transform: scale(1); }

滤镜 ​

基本滤镜 ​

css
.box {
  /* 模糊 */
  filter: blur(5px);
  
  /* 亮度 */
  filter: brightness(1.5);  /* 0-1 暗, >1 亮 */
  
  /* 对比度 */
  filter: contrast(200%);   /* 100% 原始 */
  
  /* 灰度 */
  filter: grayscale(100%);  /* 0-100% */
  
  /* 色相旋转 */
  filter: hue-rotate(90deg); /* 0-360deg */
  
  /* 反转 */
  filter: invert(100%);      /* 0-100% */
  
  /* 透明度 */
  filter: opacity(0.5);     /* 0-1 */
  
  /* 饱和度 */
  filter: saturate(200%);   /* 0-100%+, 100% 原始 */
  
  /* 褐色 */
  filter: sepia(100%);     /* 0-100% */
}

高级滤镜 ​

css
.box {
  /* 投影 */
  filter: drop-shadow(10px 10px 5px rgba(0, 0, 0, 0.5));
  
  /* 组合滤镜 */
  filter: blur(5px) grayscale(100%) brightness(1.2);
  
  /* 滤镜函数 */
  filter: url(#filter-id);  /* 引用 SVG 滤镜 */
}

SVG 滤镜 ​

html

常见面试问题 ​

Q: transition 和 animation 的区别？

对比项transitionanimation
触发方式需要触发（如 hover）自动执行
循环不能循环可以循环
复杂度简单动画复杂动画
性能较好较好

Q: 如何优化动画性能？

  • 使用 transform 和 opacity（触发合成层）
  • 避免使用 left/top（触发重排）
  • 使用 will-change 提示浏览器
css
.animated {
  will-change: transform;
  transform: translateX(0);
}

Q: 什么是 GPU 加速？

通过将元素提升到独立的合成层，利用 GPU 进行渲染，提升性能。

css
.gpu-accelerated {
  transform: translateZ(0);
  /* 或 */
  will-change: transform;
}
