模块列表 ​

01. 基础概念 ​

CSS 的基础概念，包括盒模型、选择器、优先级与继承。

主要内容：

  • 标准盒模型与 IE 盒模型
  • 选择器分类与优先级
  • 伪类和伪元素的区别
  • CSS 优先级与继承

02. 布局原理 ​

CSS 布局原理，包括 BFC、Flex 布局、position 定位等。

主要内容：

  • BFC（块级格式化上下文）
  • Flex 布局
  • 水平垂直居中
  • 清除浮动
  • position 定位
  • margin 重叠

03. 响应式设计 ​

CSS 响应式设计，包括媒体查询、移动端适配方案等。

主要内容：

  • 媒体查询
  • 常见断点
  • 移动端适配方案（rem、vw/vh、%）
  • 响应式图片

04. CSS3 新特性 ​

CSS3 的新特性，包括新增选择器、新增属性、动画等。

主要内容：

  • 新增选择器（属性选择器、结构伪类、UI 伪类）
  • 新增属性（圆角、阴影、渐变、变换）
  • 过渡与动画
  • 弹性盒子与网格布局
  • 滤镜

05. 高级特性 ​

CSS 高级特性，包括层叠上下文、隐藏元素、CSS 绘制图形等。

主要内容：

  • 层叠上下文
  • 隐藏元素的方式
  • CSS 绘制图形（三角形、圆形、椭圆等）
  • 多行文本省略
  • 其他高级特性

06. 工程化实践 ​

CSS 工程化实践，包括 CSS 预处理器、CSS 性能优化等。

主要内容：

  • CSS 预处理器（Sass、Less、Stylus）
  • CSS 性能优化
  • 单位区别（px、em、rem、vw、vh）
  • CSS 模块化（BEM、CSS Modules、CSS-in-JS）

07. CSS 变量 ​

CSS 变量（自定义属性）相关内容。

主要内容：

  • 变量定义与使用
  • 变量作用域与继承
  • 变量类型与计算
  • 变量与 JavaScript 交互
  • 变量应用场景（主题切换、响应式设计）

08. Grid 布局 ​

Grid 布局相关内容。

主要内容：

  • Grid 容器属性
  • Grid 项目属性
  • Grid 模板与对齐
  • Grid 布局示例
  • Grid 与 Flex 对比

学习建议 ​

  1. 循序渐进：从基础概念开始，逐步深入到高级特性
  2. 实践结合：在理论学习的同时，多动手实践，加深理解
  3. 关注标准：关注 CSS 标准的最新发展，特别是 CSS3 的新特性
  4. 项目实战：通过实际项目应用所学知识，提升实战能力
  5. 性能优化：重视 CSS 性能优化，提升用户体验
