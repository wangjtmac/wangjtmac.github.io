动态轨迹动画 ​

本示例展示如何在 Cesium 中创建动态轨迹动画，包括移动实体、轨迹路径、速度变化等功能。

核心功能 ​

1. 轨迹动画系统 ​

  • SampledPositionProperty - 采样位置属性，用于定义实体的运动轨迹
  • VelocityOrientationProperty - 速度方向属性，使实体朝向运动方向
  • PathGraphics - 路径图形，显示实体的运动轨迹

2. 时间控制 ​

  • Clock - 时间控制器，控制动画的播放
  • JulianDate - 儒略日期，Cesium 的时间表示方式
  • TimeInterval - 时间区间，定义动画的时间范围

3. 动画类型 ​

  • 飞机航线 - 模拟飞机沿航线飞行
  • 车辆路径 - 模拟车辆在道路上行驶
  • 卫星轨道 - 模拟卫星绕地球运行

技术要点 ​

SampledPositionProperty 使用 ​

javascript
// 创建采样位置属性
const positionProperty = new Cesium.SampledPositionProperty();

// 添加采样点（时间、位置）
positionProperty.addSample(
  Cesium.JulianDate.fromIso8601('2024-01-01T00:00:00Z'),
  Cesium.Cartesian3.fromDegrees(116.4, 39.9, 1000)
);

positionProperty.addSample(
  Cesium.JulianDate.fromIso8601('2024-01-01T00:01:00Z'),
  Cesium.Cartesian3.fromDegrees(116.5, 40.0, 1500)
);

// 设置插值算法
positionProperty.setInterpolationOptions({
  interpolationDegree: 5,
  interpolationAlgorithm: Cesium.LagrangePolynomialApproximation
});

时间控制器配置 ​

javascript
// 配置时间控制器
viewer.clock.startTime = startTime.clone();
viewer.clock.stopTime = stopTime.clone();
viewer.clock.currentTime = startTime.clone();
viewer.clock.clockRange = Cesium.ClockRange.LOOP_STOP; // 循环播放
viewer.clock.multiplier = 1; // 播放速度倍数

// 设置时间轴
viewer.timeline.zoomTo(startTime, stopTime);

实体朝向控制 ​

javascript
// 使实体朝向运动方向
entity.orientation = new Cesium.VelocityOrientationProperty(position);

// 或手动计算朝向
entity.orientation = new Cesium.CallbackProperty((time) => {
  const position = entity.position.getValue(time);
  const velocity = Cesium.Cartesian3.subtract(
    entity.position.getValue(Cesium.JulianDate.addSeconds(time, 1, new Cesium.JulianDate())),
    position,
    new Cesium.Cartesian3()
  );
  return Cesium.Transforms.headingPitchRollQuaternion(
    position,
    new Cesium.HeadingPitchRoll(
      Math.atan2(velocity.y, velocity.x),
      Math.atan2(velocity.z, Math.sqrt(velocity.x * velocity.x + velocity.y * velocity.y)),
      0
    )
  );
}, false);

应用场景 ​

  1. 交通监控 - 实时显示车辆、飞机的运动轨迹
  2. 物流追踪 - 追踪货物运输路径
  3. 运动分析 - 分析运动员、动物的移动模式
  4. 历史回放 - 回放历史轨迹数据
  5. 路径规划 - 展示规划的路径和预计到达时间
