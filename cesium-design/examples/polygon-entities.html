面实体示例 ​

这是一个演示如何使用 Cesium Entity API 创建和自定义面实体的示例。

功能说明 ​

  • 实现不同样式的面实体创建
  • 演示面实体的各种属性设置
  • 实现面实体的动态更新
  • 提供交互式控制面实体样式

核心 API ​

  • viewer.entities.add() - 添加实体到场景
  • Cesium.PolygonGraphics - 面图形类
  • Cesium.Color - 颜色类
  • Cesium.Cartesian3 - 笛卡尔坐标类

<template>
  <div class="cesium-container-wrapper">

      <!-- 面实体类型选择 -->
      <div class="control-section">

          <button @click="createHollowPolygon" :class="{ active: currentType === 'hollow' }">带孔多边形</button>
          <button @click="createExtrudedPolygon" :class="{ active: currentType === 'extruded' }">拉伸多边形</button>
        </div>
      </div>
      
      <!-- 面实体样式设置 -->
      <div class="control-section">
        <h4>面实体样式</h4>
        <div class="input-group">
          <label>填充透明度：</label>
          <input type="range" v-model.number="fillAlpha" min="0" max="1" step="0.1">
          <span>{{ fillAlpha }}</span>
        </div>
        <div class="color-group">
          <label>填充颜色：</label>
          <input type="color" v-model="fillColor" @change="updatePolygonStyle">
          <span>{{ fillColor }}</span>
        </div>
        <div class="color-group">
          <label>边框颜色：</label>
          <input type="color" v-model="outlineColor" @change="updatePolygonStyle">
          <span>{{ outlineColor }}</span>
        </div>
        <div class="input-group">
          <label>边框宽度：</label>
          <input type="range" v-model.number="outlineWidth" min="1" max="5" step="1">
          <span>{{ outlineWidth }}px</span>
        </div>
        <div v-if="currentType === 'extruded'" class="input-group">
          <label>拉伸高度：</label>
          <input type="range" v-model.number="extrudedHeight" min="100" max="10000" step="100">
          <span>{{ extrudedHeight }}米</span>
        </div>
        <div class="button-group">
          <button @click="clearAllPolygons">清除所有面</button>
        </div>
      </div>
      
      <!-- 面实体信息 -->
      <div class="info-section">
        <h4>面实体信息</h4>
        <div class="info-group">
          <p>当前面实体数量：{{ polygonCount }}</p>
          <p>当前类型：{{ currentTypeName }}</p>
        </div>
      </div>
    </div>
    
    <div ref="cesiumContainer" class="cesium-container">
      <div v-if="loading" class="loading-overlay">
        <div class="loading-spinner">加载 Cesium 中...</div>
      </div>
    </div>
  </div>
</template>

<script setup>
import { ref, onMounted, onUnmounted, computed, getCurrentInstance } from 'vue'

const { proxy } = getCurrentInstance()

const cesiumContainer = ref(null)
const loading = ref(true)
const currentType = ref('simple')
const fillColor = ref('#ff0000')
const fillAlpha = ref(0.5)
const outlineColor = ref('#ffffff')
const outlineWidth = ref(2)
const extrudedHeight = ref(1000)

// 面实体列表
let viewer = null
let polygons = []

/**
 * 计算属性：获取当前面实体数量
 */
const polygonCount = computed(() => polygons.length)

/**
 * 计算属性：获取当前类型名称
 */
const currentTypeName = computed(() => {
  const names = {
    'simple': '简单多边形',
    'hollow': '带孔多边形', 
    'extruded': '拉伸多边形'
  }
  return names[currentType.value] || '未知类型'
})

/**
 * 组件挂载时初始化 Cesium 查看器
 */
onMounted(async () => {
  if (!cesiumContainer.value) return
  
  try {
    // 动态加载 Cesium
    await loadCesium()
    
    Cesium.Ion.defaultAccessToken = 'eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJqdGkiOiI5N2UyMjcwOS00MDY1LTQxYjEtYjZjMy00YTU0ZTg1YmJjMGEiLCJpZCI6MjU5MCwic2NvcGVzIjpbImFzciIsImdjIl0sImlhdCI6MTU0Mjg2MTUzMX0.Hvl3oBH8t1pJOOkOXJJ0M0pHcJJ8Gx8Hb3Ks3k7Ks3k'
    
    // 初始化 Cesium 查看器
    viewer = new Cesium.Viewer(cesiumContainer.value, {
      animation: false,
      baseLayerPicker: false,
      fullscreenButton: true,
      geocoder: false,
      homeButton: true,
      infoBox: false,
      sceneModePicker: true,
      selectionIndicator: false,
      timeline: false,
      navigationHelpButton: false,
      scene3DOnly: true,
      requestRenderMode: true,
      fullscreenElement: cesiumContainer.value
    })
    
    // 设置相机位置到中国区域
    viewer.camera.setView({
      destination: Cesium.Cartesian3.fromDegrees(104.0, 35.0, 5000000.0)
    })
    
    // 创建示例面实体
    createExamplePolygons()
    
    loading.value = false
  } catch (error) {
    console.error('Cesium 初始化失败:', error)
    loading.value = false
  }
})

onUnmounted(() => {
  if (viewer) {
    viewer.destroy()
  }
})

/**
 * 动态加载 Cesium 库
 */
async function loadCesium() {
  return new Promise((resolve, reject) => {
    if (window.Cesium) {
      resolve()
      return
    }
    
    // 加载 Cesium CSS
    const link = document.createElement('link')
    link.rel = 'stylesheet'
    link.href = 'https://cesium.com/downloads/cesiumjs/releases/1.109/Build/Cesium/Widgets/widgets.css'
    document.head.appendChild(link)
    
    // 加载 Cesium JavaScript
    const script = document.createElement('script')
    script.src = 'https://cesium.com/downloads/cesiumjs/releases/1.109/Build/Cesium/Cesium.js'
    script.onload = () => {
      // 设置 Cesium Ion 访问令牌
      window.Cesium.Ion.defaultAccessToken = 'eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJqdGkiOiI5N2UyMjcwOS00MDY1LTQxYjEtYjZjMy00YTU0ZTg1YmJjMGEiLCJpZCI6MjU5MCwic2NvcGVzIjpbImFzciIsImdjIl0sImlhdCI6MTU0Mjg2MTUzMX0.Hvl3oBH8t1pJOOkOXJJ0M0pHcJJ8Gx8Hb3Ks3k7Ks3k'
      resolve()
    }
    script.onerror = () => {
      reject(new Error('Cesium 加载失败'))
    }
    document.body.appendChild(script)
  })
}

/**
 * 创建示例面实体
 */
function createExamplePolygons() {
  createSimplePolygon()
}

/**
 * 创建简单多边形
 */
function createSimplePolygon() {
  currentType.value = 'simple'
  clearAllPolygons()
  
  // 生成随机多边形坐标
  const positions = generateRandomPolygonPositions()
  
  // 创建面实体
  const polygonEntity = viewer.entities.add({
    polygon: {
      hierarchy: positions,
      material: Cesium.Color.fromCssColorString(fillColor.value).withAlpha(fillAlpha.value),
      outline: true,
      outlineColor: Cesium.Color.fromCssColorString(outlineColor.value),
      outlineWidth: outlineWidth.value
    }
  })
  
  polygons.push(polygonEntity)
  
  // 聚焦到新创建的面实体
  focusOnEntity(polygonEntity)
}

/**
 * 创建带孔多边形
 */
function createHollowPolygon() {
  currentType.value = 'hollow'
  clearAllPolygons()
  
  // 生成外部多边形坐标
  const outerPositions = generateRandomPolygonPositions()
  
  // 生成内部多边形坐标（孔）
  const center = Cesium.Cartesian3.midpoint(outerPositions[0], outerPositions[2], new Cesium.Cartesian3())
  const innerPositions = generateRandomPolygonPositions(center, 50000)
  
  // 创建带孔的多边形
  const polygonEntity = viewer.entities.add({
    polygon: {
      hierarchy: {
        positions: outerPositions,
        holes: [innerPositions]
      },
      material: Cesium.Color.fromCssColorString(fillColor.value).withAlpha(fillAlpha.value),
      outline: true,
      outlineColor: Cesium.Color.fromCssColorString(outlineColor.value),
      outlineWidth: outlineWidth.value
    }
  })
  
  polygons.push(polygonEntity)
  
  // 聚焦到新创建的面实体
  focusOnEntity(polygonEntity)
}

/**
 * 创建拉伸多边形
 */
function createExtrudedPolygon() {
  currentType.value = 'extruded'
  clearAllPolygons()
  
  // 生成随机多边形坐标
  const positions = generateRandomPolygonPositions()
  
  // 创建拉伸多边形
  const polygonEntity = viewer.entities.add({
    polygon: {
      hierarchy: positions,
      material: Cesium.Color.fromCssColorString(fillColor.value).withAlpha(fillAlpha.value),
      outline: true,
      outlineColor: Cesium.Color.fromCssColorString(outlineColor.value),
      outlineWidth: outlineWidth.value,
      extrudedHeight: extrudedHeight.value
    }
  })
  
  polygons.push(polygonEntity)
  
  // 聚焦到新创建的面实体
  focusOnEntity(polygonEntity)
}

/**
 * 更新面实体样式
 */
function updatePolygonStyle() {
  polygons.forEach(polygon => {
    if (polygon.polygon) {
      polygon.polygon.material = Cesium.Color.fromCssColorString(fillColor.value).withAlpha(fillAlpha.value)
      polygon.polygon.outlineColor = Cesium.Color.fromCssColorString(outlineColor.value)
      polygon.polygon.outlineWidth = outlineWidth.value
      
      // 如果是拉伸多边形，更新拉伸高度
      if (polygon.polygon.extrudedHeight) {
        polygon.polygon.extrudedHeight = extrudedHeight.value
      }
    }
  })
}

/**
 * 清除所有面实体
 */
function clearAllPolygons() {
  polygons.forEach(polygon => {
    viewer.entities.remove(polygon)
  })
  polygons = []
}

/**
 * 生成随机多边形坐标
 * @param {Cesium.Cartesian3} center 中心点坐标（可选）
 * @param {number} radius 半径（米，可选）
 * @returns {Cesium.Cartesian3[]} 多边形坐标数组
 */
function generateRandomPolygonPositions(center = null, radius = 100000) {
  const positions = []
  const sides = 6 // 六边形
  
  // 如果没有提供中心点，随机生成一个
  if (!center) {
    const lon = 100 + Math.random() * 10 // 100-110°E
    const lat = 30 + Math.random() * 10 // 30-40°N
    center = Cesium.Cartesian3.fromDegrees(lon, lat)
  }
  
  // 生成多边形顶点
  for (let i = 0; i < sides; i++) {
    const angle = (i / sides) * Math.PI * 2
    const offsetLon = (Math.cos(angle) * radius) / (111319.9 * Math.cos(Cesium.Math.toRadians(35)))
    const offsetLat = (Math.sin(angle) * radius) / 111319.9
    
    // 获取中心点的经纬度
    const centerCartographic = Cesium.Cartographic.fromCartesian(center)
    const centerLon = Cesium.Math.toDegrees(centerCartographic.longitude)
    const centerLat = Cesium.Math.toDegrees(centerCartographic.latitude)
    
    positions.push(Cesium.Cartesian3.fromDegrees(
      centerLon + offsetLon,
      centerLat + offsetLat
    ))
  }
  
  return positions
}

/**
 * 聚焦到指定实体
 * @param {Cesium.Entity} entity 要聚焦的实体
 */
function focusOnEntity(entity) {
  viewer.flyTo(entity, {
    duration: 1.5
  })
}

</script>

说明 ​

这个示例展示了如何在 Vue 组件中实现 Cesium 面实体功能。主要功能包括：

  • 创建不同样式的面实体
  • 演示面实体的各种属性设置
  • 实现面实体的动态更新
  • 提供交互式控制界面

技术要点 ​

  1. 面样式配置：使用 reactive 创建响应式面样式对象，包括线宽、颜色、透明度等
  2. 面实体创建：实现了多种面类型（矩形、三角形、自定义多边形）
  3. 交互功能：支持创建、更新、删除和清除面实体
  4. 相机控制：实现相机聚焦到面实体的功能
  5. 动态加载：使用 Promise 异步加载 Cesium JS 和 CSS 文件

注意事项 ​

  • 示例需要网络连接才能加载 Cesium CDN 资源
  • 面实体的性能受顶点数量影响，大量面或复杂面可能影响性能
  • 组件会自动清理资源，避免内存泄漏

代码解释 ​

  1. 填充样式配置：

    • 使用 reactive 创建响应式填充样式对象，包括填充颜色和填充透明度
    • 通过颜色选择器和滑块实现交互式样式调整

  2. 轮廓样式配置：

    • 使用 reactive 创建响应式轮廓样式对象，包括轮廓颜色和轮廓宽度
    • 通过颜色选择器和滑块实现交互式样式调整

  3. 面实体创建：

    • 实现了三种面类型：矩形、三角形、自定义多边形
    • 每种面类型都有不同的创建方法和参数
    • 为每个面实体设置唯一 ID 和名称

  4. 初始数据：

    • 默认创建了北京区域的矩形面
    • 默认创建了长三角地区的三角面
    • 新创建的面使用随机位置

  5. 交互功能：

    • 支持创建新面（矩形、三角形、自定义多边形）
    • 支持更新选中面的样式
    • 支持删除选中面和清除所有面
    • 实现了相机聚焦到面的功能
