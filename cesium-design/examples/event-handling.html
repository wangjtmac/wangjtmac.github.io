事件处理示例 ​

这是一个演示如何使用 Cesium 事件处理系统的综合示例。

功能说明 ​

  • 演示鼠标事件处理（点击、移动、右键、滚轮）
  • 演示相机事件处理（移动、缩放）
  • 演示实体交互事件（点击、悬停）
  • 实时事件日志显示
  • 可配置的事件监听开关

核心 API ​

  • Cesium.ScreenSpaceEventHandler - 屏幕空间事件处理器
  • viewer.scene.pick() - 拾取场景对象
  • viewer.camera.pickEllipsoid() - 拾取地球表面位置
  • viewer.camera.moveEnd - 相机移动结束事件

组件代码 ​

vue
<template>
  <div class="cesium-event-demo">
    <div v-if="loading" class="loading">加载 Cesium 中...</div>
    <div v-if="error" class="error">{{ error }}</div>

    <div v-show="!loading && !error" class="w-full h-full">
      <!-- Cesium 容器 -->
      <div ref="cesiumContainer" class="cesium-container"></div>
      <!-- 控制面板 -->
      <div class="control-panel">
        <div class="panel-header">
          <div>🎯 事件处理示例</div>
          <button @click="toggleFullscreen" class="btn-icon" title="全屏">⛶</button>
        </div>


        <!-- 事件日志 -->
        <div class="event-log-section">
          <div class="section-header">
            <span>📋 事件日志</span>
            <button @click="clearLog" class="btn-clear">清空</button>
          </div>
          <div class="event-log" ref="eventLogRef">
            <div v-for="(log, index) in eventLogs" :key="index" :class="['log-item', `log-${log.type}`]">
              <span class="log-time">{{ log.time }}</span>
              <span class="log-type">{{ log.eventType }}</span>
              <span class="log-message">{{ log.message }}</span>
            </div>
            <div v-if="eventLogs.length === 0" class="log-empty">
              暂无事件，请在地图上进行操作
            </div>
          </div>
        </div>

        <!-- 事件开关 -->
        <div class="event-switches">
          <h4>事件监听开关</h4>
          <label class="switch-item">
            <input type="checkbox" v-model="eventEnabled.click" />
            <span>鼠标点击</span>
          </label>
          <label class="switch-item">
            <input type="checkbox" v-model="eventEnabled.move" />
            <span>鼠标移动</span>
          </label>
          <label class="switch-item">
            <input type="checkbox" v-model="eventEnabled.camera" />
            <span>相机移动</span>
          </label>
          <label class="switch-item">
            <input type="checkbox" v-model="eventEnabled.entity" />
            <span>实体交互</span>
          </label>
        </div>

        <!-- 测试实体 -->
        <div class="test-entities">
          <h4>测试实体</h4>
          <button @click="addTestEntity" class="btn-action">添加测试点</button>
          <button @click="clearEntities" class="btn-action">清除实体</button>
        </div>
      </div>
    </div>
  </div>
</template>

<script setup>
import { ref, onMounted, onUnmounted, reactive, watch, nextTick, getCurrentInstance } from 'vue'

const cesiumContainer = ref(null)
const eventLogRef = ref(null)
const loading = ref(true)
const error = ref('')
let viewer = null
let eventHandlers = []
let entityCounter = 0
const { proxy } = getCurrentInstance()
// 事件日志
const eventLogs = ref([])
const maxLogs = 50

// 事件开关
const eventEnabled = reactive({
  click: true,
  move: false,
  camera: true,
  entity: true
})
const toggleFullscreen = () => {
  const element = proxy.$el;
  if (!document.fullscreenElement) {
    element.requestFullscreen()
  } else {
    document.exitFullscreen()
  }
}

// 添加日志
const addLog = (eventType, message, type = 'info') => {
  const now = new Date()
  const time = `${now.getHours().toString().padStart(2, '0')}:${now.getMinutes().toString().padStart(2, '0')}:${now.getSeconds().toString().padStart(2, '0')}`

  eventLogs.value.unshift({
    time,
    eventType,
    message,
    type
  })

  // 限制日志数量
  if (eventLogs.value.length > maxLogs) {
    eventLogs.value.pop()
  }

  // 滚动到顶部
  nextTick(() => {
    if (eventLogRef.value) {
      eventLogRef.value.scrollTop = 0
    }
  })
}

// 清空日志
const clearLog = () => {
  eventLogs.value = []
}

// 格式化坐标
const formatPosition = (cartesian) => {
  if (!cartesian) return '未知位置'

  const cartographic = window.Cesium.Cartographic.fromCartesian(cartesian)
  const lon = window.Cesium.Math.toDegrees(cartographic.longitude).toFixed(4)
  const lat = window.Cesium.Math.toDegrees(cartographic.latitude).toFixed(4)
  const height = cartographic.height.toFixed(2)

  return `经度: ${lon}°, 纬度: ${lat}°, 高度: ${height}m`
}

// 添加测试实体
const addTestEntity = () => {
  if (!viewer) return

  entityCounter++
  const lon = 116.4 + (Math.random() - 0.5) * 2
  const lat = 39.9 + (Math.random() - 0.5) * 2

  viewer.entities.add({
    id: `test-entity-${entityCounter}`,
    name: `测试点 ${entityCounter}`,
    position: window.Cesium.Cartesian3.fromDegrees(lon, lat, 0),
    point: {
      pixelSize: 15,
      color: window.Cesium.Color.fromRandom({ alpha: 0.8 }),
      outlineColor: window.Cesium.Color.WHITE,
      outlineWidth: 2
    },
    label: {
      text: `点 ${entityCounter}`,
      font: '14px sans-serif',
      fillColor: window.Cesium.Color.WHITE,
      outlineColor: window.Cesium.Color.BLACK,
      outlineWidth: 2,
      style: window.Cesium.LabelStyle.FILL_AND_OUTLINE,
      verticalOrigin: window.Cesium.VerticalOrigin.BOTTOM,
      pixelOffset: new window.Cesium.Cartesian2(0, -20)
    }
  })

  addLog('实体', `添加了测试点 ${entityCounter}`, 'success')
}

// 清除实体
const clearEntities = () => {
  if (!viewer) return

  const count = viewer.entities.values.length
  viewer.entities.removeAll()
  entityCounter = 0

  addLog('实体', `清除了 ${count} 个实体`, 'warning')
}

// 设置事件监听
const setupEventHandlers = () => {
  if (!viewer) return

  const handler = new window.Cesium.ScreenSpaceEventHandler(viewer.scene.canvas)

  // 鼠标左键点击
  handler.setInputAction((click) => {
    if (!eventEnabled.click) return

    const pickedObject = viewer.scene.pick(click.position)
    const cartesian = viewer.camera.pickEllipsoid(click.position, viewer.scene.globe.ellipsoid)

    if (pickedObject && pickedObject.id) {
      const entity = pickedObject.id
      addLog('点击', `点击了实体: ${entity.name || entity.id}`, 'entity')
    } else if (cartesian) {
      addLog('点击', formatPosition(cartesian), 'click')
    }
  }, window.Cesium.ScreenSpaceEventType.LEFT_CLICK)

  // 鼠标移动
  let lastMoveTime = 0
  handler.setInputAction((movement) => {
    if (!eventEnabled.move) return

    // 限制移动事件频率
    const now = Date.now()
    if (now - lastMoveTime < 500) return
    lastMoveTime = now

    const pickedObject = viewer.scene.pick(movement.endPosition)
    const cartesian = viewer.camera.pickEllipsoid(movement.endPosition, viewer.scene.globe.ellipsoid)

    if (pickedObject && pickedObject.id) {
      const entity = pickedObject.id
      addLog('悬停', `悬停在实体: ${entity.name || entity.id}`, 'entity')
    } else if (cartesian) {
      addLog('移动', formatPosition(cartesian), 'move')
    }
  }, window.Cesium.ScreenSpaceEventType.MOUSE_MOVE)

  // 鼠标右键点击
  handler.setInputAction((click) => {
    if (!eventEnabled.click) return

    const cartesian = viewer.camera.pickEllipsoid(click.position, viewer.scene.globe.ellipsoid)
    if (cartesian) {
      addLog('右键', formatPosition(cartesian), 'click')
    }
  }, window.Cesium.ScreenSpaceEventType.RIGHT_CLICK)

  // 鼠标滚轮
  handler.setInputAction((wheelIncrement) => {
    if (!eventEnabled.camera) return

    const direction = wheelIncrement > 0 ? '放大' : '缩小'
    addLog('滚轮', `${direction} (${Math.abs(wheelIncrement)})`, 'camera')
  }, window.Cesium.ScreenSpaceEventType.WHEEL)

  eventHandlers.push(handler)

  // 相机移动事件
  let lastCameraMoveTime = 0
  const cameraMoveListener = viewer.camera.moveEnd.addEventListener(() => {
    if (!eventEnabled.camera) return

    const now = Date.now()
    if (now - lastCameraMoveTime < 1000) return
    lastCameraMoveTime = now

    const position = viewer.camera.positionCartographic
    const height = position.height.toFixed(2)
    addLog('相机', `相机移动结束，高度: ${height}m`, 'camera')
  })

  eventHandlers.push(cameraMoveListener)
}

// 加载 Cesium
const loadCesium = () => {
  return new Promise((resolve, reject) => {
    if (window.Cesium) {
      resolve(window.Cesium)
      return
    }

    const script = document.createElement('script')
    script.src = 'https://cesium.com/downloads/cesiumjs/releases/1.136/Build/Cesium/Cesium.js'
    script.onload = () => {
      const link = document.createElement('link')
      link.rel = 'stylesheet'
      link.href = 'https://cesium.com/downloads/cesiumjs/releases/1.136/Build/Cesium/Widgets/widgets.css'
      link.onload = () => resolve(window.Cesium)
      link.onerror = () => reject(new Error('加载 Cesium CSS 失败'))
      document.head.appendChild(link)
    }
    script.onerror = () => reject(new Error('加载 Cesium JS 失败'))
    document.head.appendChild(script)
  })
}

// 初始化 Cesium
const initCesium = async () => {
  try {
    loading.value = true
    const Cesium = await loadCesium()

    viewer = new Cesium.Viewer(cesiumContainer.value, {
      animation: false,
      timeline: false,
      baseLayerPicker: false,
      geocoder: false,
      homeButton: false,
      sceneModePicker: false,
      navigationHelpButton: false,
      fullscreenButton: false,
      imageryProvider: false // 禁用默认影像层
    })

    // 手动添加高德地图影像图层
    viewer.imageryLayers.addImageryProvider(
      new Cesium.UrlTemplateImageryProvider({
        url: "https://webst01.is.autonavi.com/appmaptile?style=6&x={x}&y={y}&z={z}",
        minimumLevel: 0,
        maximumLevel: 18
      })
    )

    // 设置初始视角（北京）
    viewer.camera.setView({
      destination: Cesium.Cartesian3.fromDegrees(116.4, 39.9, 1000000)
    })

    // 设置事件监听
    setupEventHandlers()
    // 添加初始测试实体
    addTestEntity()
    addTestEntity()

    addLog('系统', 'Cesium 初始化成功', 'success')
    loading.value = false
  } catch (err) {
    error.value = err.message
    loading.value = false
  }
}

// 清理资源
const cleanup = () => {
  if (eventHandlers.length > 0) {
    eventHandlers.forEach(handler => {
      if (handler && handler.destroy) {
        handler.destroy()
      } else if (handler && typeof handler === 'function') {
        handler()
      }
    })
    eventHandlers = []
  }

  if (viewer) {
    viewer.destroy()
    viewer = null
  }
}

onMounted(() => {
  initCesium()
})

onUnmounted(() => {
  cleanup()
})
</script>

代码解释 ​

1. 事件处理器初始化 ​

使用 ScreenSpaceEventHandler 创建事件处理器，它可以监听各种鼠标和触摸事件：

javascript
const handler = new Cesium.ScreenSpaceEventHandler(viewer.scene.canvas)

2. 鼠标点击事件 ​

监听鼠标左键点击，可以拾取实体或获取地球表面坐标：

javascript
handler.setInputAction((click) => {
  const pickedObject = viewer.scene.pick(click.position)
  const cartesian = viewer.camera.pickEllipsoid(click.position)
  
  if (pickedObject && pickedObject.id) {
    // 点击了实体
    console.log('点击实体:', pickedObject.id)
  } else if (cartesian) {
    // 点击了地球表面
    console.log('点击位置:', cartesian)
  }
}, Cesium.ScreenSpaceEventType.LEFT_CLICK)

3. 鼠标移动事件 ​

监听鼠标移动，实现悬停效果：

javascript
handler.setInputAction((movement) => {
  const pickedObject = viewer.scene.pick(movement.endPosition)
  // 处理悬停逻辑
}, Cesium.ScreenSpaceEventType.MOUSE_MOVE)

4. 相机事件 ​

监听相机移动结束事件：

javascript
viewer.camera.moveEnd.addEventListener(() => {
  const position = viewer.camera.positionCartographic
  console.log('相机高度:', position.height)
})

5. 坐标转换 ​

将笛卡尔坐标转换为地理坐标：

javascript
const cartographic = Cesium.Cartographic.fromCartesian(cartesian)
const lon = Cesium.Math.toDegrees(cartographic.longitude)
const lat = Cesium.Math.toDegrees(cartographic.latitude)
const height = cartographic.height

事件类型说明 ​

鼠标事件 ​

  • LEFT_CLICK - 鼠标左键点击
  • LEFT_DOUBLE_CLICK - 鼠标左键双击
  • RIGHT_CLICK - 鼠标右键点击
  • MIDDLE_CLICK - 鼠标中键点击
  • MOUSE_MOVE - 鼠标移动
  • WHEEL - 鼠标滚轮

触摸事件 ​

  • PINCH_START - 捏合开始
  • PINCH_MOVE - 捏合移动
  • PINCH_END - 捏合结束

拖拽事件 ​

  • LEFT_DOWN - 鼠标左键按下
  • LEFT_UP - 鼠标左键抬起

实用技巧 ​

1. 性能优化 ​

对于高频事件（如鼠标移动），应该限制处理频率：

javascript
let lastMoveTime = 0
handler.setInputAction((movement) => {
  const now = Date.now()
  if (now - lastMoveTime < 100) return // 限制为每100ms处理一次
  lastMoveTime = now
  
  // 处理移动事件
}, Cesium.ScreenSpaceEventType.MOUSE_MOVE)

2. 实体拾取 ​

使用 scene.pick() 拾取实体时，可以获取更多信息：

javascript
const pickedObject = viewer.scene.pick(position)
if (pickedObject && pickedObject.id) {
  const entity = pickedObject.id
  console.log('实体名称:', entity.name)
  console.log('实体ID:', entity.id)
  console.log('实体属性:', entity.properties)
}

3. 禁用默认行为 ​

有时需要禁用 Cesium 的默认交互行为：

javascript
// 禁用双击飞行
viewer.cesiumWidget.screenSpaceEventHandler.removeInputAction(
  Cesium.ScreenSpaceEventType.LEFT_DOUBLE_CLICK
)

// 禁用右键菜单
viewer.cesiumWidget.screenSpaceEventHandler.setInputAction(
  () => {},
  Cesium.ScreenSpaceEventType.RIGHT_CLICK
)

4. 资源清理 ​

使用完事件处理器后，记得销毁以避免内存泄漏：

javascript
handler.destroy()

应用场景 ​

  1. 地图标注：点击地图添加标记点
  2. 测量工具：点击多个点进行距离或面积测量
  3. 信息查询：点击实体显示详细信息
  4. 交互式编辑：拖拽实体进行位置调整
  5. 悬停提示：鼠标悬停显示实体信息
