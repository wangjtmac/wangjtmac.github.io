相机控制示例 ​

这是一个演示如何使用 Cesium 相机控制方法的综合示例。

功能说明 ​

  • 快速定位到预设城市（北京、上海、福建）
  • 切换不同的视角模式（3D、2D、哥伦布模式）
  • 基本的相机操作（放大、缩小、左转、右转）
  • 实时相机高度调节（滑块控制）
  • 实时显示相机位置和角度信息

核心 API ​

  • viewer.camera - 相机对象
  • viewer.camera.flyTo() - 飞行到指定位置
  • viewer.camera.setView() - 设置相机视图
  • viewer.scene.mode - 场景模式设置
  • Cesium.SceneMode - 场景模式枚举

const { proxy } = getCurrentInstance()

    document.exitFullscreen()

/**
 * 组件挂载时初始化 Cesium 查看器
 * 功能：动态加载 Cesium 资源并初始化地球视图
 * 参数：无
 * 返回值：无
 */
onMounted(async () => {
  if (!cesiumContainer.value) return

  try {
    // 动态加载 Cesium
    await loadCesium()

    // 初始化 Cesium 查看器
    viewer = new Cesium.Viewer(cesiumContainer.value, {
      animation: false,
      baseLayerPicker: false,
      fullscreenButton: false,
      geocoder: false,
      homeButton: true,
      infoBox: false,
      sceneModePicker: true,
      selectionIndicator: false,
      timeline: false,
      navigationHelpButton: false,
      scene3DOnly: true,
      requestRenderMode: true,
      imageryProvider: false // 禁用默认影像层
    })

    // 手动添加高德地图影像图层
    viewer.imageryLayers.addImageryProvider(
      new Cesium.UrlTemplateImageryProvider({
        url: "https://webst01.is.autonavi.com/appmaptile?style=6&x={x}&y={y}&z={z}",
        minimumLevel: 0,
        maximumLevel: 18
      })
    )

    // 设置相机位置到中国区域
    viewer.camera.setView({
      destination: Cesium.Cartesian3.fromDegrees(104.0, 35.0, 3000000.0)
    })



    // 开始更新相机信息
    startCameraInfoUpdate()

    loading.value = false
  } catch (error) {
    console.error('Cesium 初始化失败:', error)
    loading.value = false
  }
})

onUnmounted(() => {
  if (viewer) {
    viewer.destroy()
  }

  if (cameraUpdateInterval) {
    clearInterval(cameraUpdateInterval)
  }
})

/**
 * 开始更新相机信息
 * 功能：定期更新相机位置和角度信息
 * 参数：无
 * 返回值：无
 */
function startCameraInfoUpdate() {
  cameraUpdateInterval = setInterval(() => {
    if (!viewer) return

    const camera = viewer.camera
    const position = camera.position

    // 将笛卡尔坐标转换为地理坐标
    const cartographic = Cesium.Cartographic.fromCartesian(position)

    cameraInfo.value = {
      longitude: Cesium.Math.toDegrees(cartographic.longitude),
      latitude: Cesium.Math.toDegrees(cartographic.latitude),
      height: cartographic.height,
      pitch: Cesium.Math.toDegrees(camera.pitch),
      heading: Cesium.Math.toDegrees(camera.heading)
    }

    // 更新滑块值
    cameraHeight.value = cartographic.height
  }, 100)
}

 * 更新相机高度
 * 功能：根据滑块值更新相机高度
 * 参数：无
 * 返回值：无
 */
function updateCameraHeight() {
  if (!viewer) return

  const camera = viewer.camera
  const position = camera.position
  const cartographic = Cesium.Cartographic.fromCartesian(position)

  // 创建新的位置，只改变高度
  const newPosition = Cesium.Cartesian3.fromRadians(
    cartographic.longitude,
    cartographic.latitude,
    cameraHeight.value
  )

  camera.setView({
    destination: newPosition,
    orientation: {
      heading: camera.heading,
      pitch: camera.pitch,
      roll: camera.roll
    }
  })
}

/**
 * 动态加载 Cesium 资源
 * 功能：异步加载 Cesium JS 和 CSS 文件
 * 参数：无
 * 返回值：Promise 对象，加载成功时 resolve
 */
async function loadCesium() {
  return new Promise((resolve, reject) => {
    if (window.Cesium) {
      resolve()
      return
    }

    const link = document.createElement('link')
    link.rel = 'stylesheet'
    link.href = 'https://cdn.staticfile.net/cesium/1.113.0/Widgets/widgets.css'
    document.head.appendChild(link)

    const script = document.createElement('script')
    script.src = 'https://cdn.staticfile.net/cesium/1.113.0/Cesium.js'
    script.onload = () => resolve()
    script.onerror = () => reject(new Error('Cesium 加载失败'))
    document.head.appendChild(script)

  })
}

代码解释 ​

1. 相机飞行动画 ​

使用 flyTo() 方法实现平滑的相机飞行：

javascript
viewer.camera.flyTo({
  destination: Cesium.Cartesian3.fromDegrees(116.4074, 39.9042, 2000),
  duration: 2.0,  // 飞行持续时间（秒）
  orientation: {
    heading: 0.0,   // 航向角（弧度）
    pitch: -0.5,    // 俯仰角（弧度）
    roll: 0.0       // 翻滚角（弧度）
  }
})

2. 场景模式切换 ​

Cesium 支持三种场景模式：

javascript
// 3D 模式（默认）
viewer.scene.mode = Cesium.SceneMode.SCENE3D

// 2D 模式（平面地图）
viewer.scene.mode = Cesium.SceneMode.SCENE2D

// 哥伦布模式（2.5D，平面但保留高度）
viewer.scene.mode = Cesium.SceneMode.COLUMBUS_VIEW

3. 相机基本操作 ​

javascript
// 放大/缩小
viewer.camera.zoomIn(10000)   // 参数为移动距离（米）
viewer.camera.zoomOut(10000)

// 旋转
viewer.camera.rotateLeft(Cesium.Math.toRadians(10))   // 参数为角度（弧度）
viewer.camera.rotateRight(Cesium.Math.toRadians(10))

// 上下倾斜
viewer.camera.rotateUp(Cesium.Math.toRadians(10))
viewer.camera.rotateDown(Cesium.Math.toRadians(10))

4. 相机位置信息获取 ​

javascript
const camera = viewer.camera
const position = camera.position

// 将笛卡尔坐标转换为地理坐标
const cartographic = Cesium.Cartographic.fromCartesian(position)

// 获取经纬度和高度
const longitude = Cesium.Math.toDegrees(cartographic.longitude)
const latitude = Cesium.Math.toDegrees(cartographic.latitude)
const height = cartographic.height

// 获取相机姿态
const heading = Cesium.Math.toDegrees(camera.heading)  // 航向角
const pitch = Cesium.Math.toDegrees(camera.pitch)      // 俯仰角
const roll = Cesium.Math.toDegrees(camera.roll)        // 翻滚角

5. 动态调整相机高度 ​

javascript
function updateCameraHeight(newHeight) {
  const camera = viewer.camera
  const position = camera.position
  const cartographic = Cesium.Cartographic.fromCartesian(position)
  
  // 创建新位置，只改变高度
  const newPosition = Cesium.Cartesian3.fromRadians(
    cartographic.longitude,
    cartographic.latitude,
    newHeight
  )
  
  // 设置新位置，保持原有姿态
  camera.setView({
    destination: newPosition,
    orientation: {
      heading: camera.heading,
      pitch: camera.pitch,
      roll: camera.roll
    }
  })
}

相机控制方法详解 ​

flyTo() - 飞行动画 ​

最常用的相机移动方法，提供平滑的飞行动画：

javascript
viewer.camera.flyTo({
  destination: Cesium.Cartesian3.fromDegrees(lon, lat, height),
  orientation: {
    heading: Cesium.Math.toRadians(0),
    pitch: Cesium.Math.toRadians(-90),
    roll: 0.0
  },
  duration: 3.0,
  complete: () => {
    console.log('飞行完成')
  },
  cancel: () => {
    console.log('飞行取消')
  }
})

setView() - 瞬间切换 ​

不带动画的相机位置设置：

javascript
viewer.camera.setView({
  destination: Cesium.Cartesian3.fromDegrees(lon, lat, height),
  orientation: {
    heading: 0.0,
    pitch: -0.5,
    roll: 0.0
  }
})

lookAt() - 观察目标 ​

让相机观察某个目标点：

javascript
const target = Cesium.Cartesian3.fromDegrees(116.4, 39.9, 0)
const offset = new Cesium.HeadingPitchRange(
  0,                              // 航向角
  Cesium.Math.toRadians(-45),     // 俯仰角
  5000                            // 距离
)

viewer.camera.lookAt(target, offset)

场景模式说明 ​

SCENE3D - 3D 模式 ​

  • 完整的三维地球视图
  • 支持所有相机操作
  • 最常用的模式

SCENE2D - 2D 模式 ​

  • 平面地图视图
  • 类似传统 Web 地图
  • 不支持俯仰角调整

COLUMBUS_VIEW - 哥伦布模式 ​

  • 2.5D 视图
  • 平面投影但保留高度信息
  • 适合展示地形和建筑

2. 禁用相机控制 ​

// 禁用所有相机控制
viewer.scene.screenSpaceCameraController.enableRotate = false
viewer.scene.screenSpaceCameraController.enableZoom = false
viewer.scene.screenSpaceCameraController.enableTilt = false
viewer.scene.screenSpaceCameraController.enableLook = false

3. 保存和恢复相机位置 ​

javascript
// 保存相机位置
const cameraState = {
  position: viewer.camera.position.clone(),
  heading: viewer.camera.heading,
  pitch: viewer.camera.pitch,
  roll: viewer.camera.roll
}

// 恢复相机位置
viewer.camera.setView({
  destination: cameraState.position,
  orientation: {
    heading: cameraState.heading,
    pitch: cameraState.pitch,
    roll: cameraState.roll
  }
})

4. 相机跟随实体 ​

javascript
// 跟随实体
viewer.trackedEntity = entity

// 取消跟随
viewer.trackedEntity = undefined

说明 ​

这个示例展示了如何在 Vue 组件中实现 Cesium 相机控制功能。主要功能包括：

  • 快速定位到预设城市（北京、上海、福建）
  • 切换不同的视角模式（3D、2D、哥伦布模式）
  • 基本的相机操作（放大、缩小、左转、右转）
  • 实时相机高度调节（通过滑块）
  • 实时显示相机位置和角度信息
  • 动态加载 Cesium CDN 资源
  • 完整的错误处理和加载状态

技术要点 ​

  1. 相机控制：使用 viewer.camera 对象进行各种相机操作
  2. 飞行动画：使用 flyTo() 方法实现平滑的相机移动效果
  3. 场景模式：切换 SceneMode 实现不同的视角模式
  4. 实时更新：使用定时器（100ms 间隔）实时更新相机位置和角度信息
  5. 动态加载：使用 Promise 异步加载 Cesium JS 和 CSS 文件
  6. 组件生命周期：在 onMounted 中初始化，onUnmounted 中清理资源

应用场景 ​

  1. 城市导览：快速切换到不同城市进行展示
  2. 视角切换：根据需求切换 2D/3D 视图
  3. 交互式导航：提供用户友好的相机控制界面
  4. 位置监控：实时显示当前相机位置信息
  5. 教学演示：展示 Cesium 相机控制的各种功能
