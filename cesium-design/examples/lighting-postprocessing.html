光照与后期处理 ​

本示例展示如何在 Cesium 中使用光照系统和后期处理效果，增强场景的视觉表现力。

核心功能 ​

1. 光照系统 ​

  • 太阳光照 - 模拟太阳光的方向和强度
  • 环境光 - 全局环境光照
  • 阴影 - 实时阴影渲染
  • 光照强度 - 调节光照亮度

2. 后期处理效果 ​

  • 辉光效果 - Bloom 效果
  • 环境光遮蔽 - SSAO (Screen Space Ambient Occlusion)
  • 景深效果 - Depth of Field
  • 色调映射 - HDR Tone Mapping
  • 颜色调整 - 亮度、对比度、饱和度

3. 大气效果 ​

  • 大气散射 - 天空颜色和雾效
  • 地平线效果 - 地平线模糊
  • 天空盒 - 自定义天空背景

技术要点 ​

光照配置 ​

javascript
// 启用光照
viewer.scene.globe.enableLighting = true;

// 配置太阳光照
viewer.scene.light = new Cesium.DirectionalLight({
  direction: new Cesium.Cartesian3(1, 0, 0), // 光照方向
  intensity: 2.0 // 光照强度
});

// 或使用太阳作为光源
viewer.scene.light = new Cesium.SunLight();

// 配置环境光
viewer.scene.light.intensity = 1.5;

阴影配置 ​

javascript
// 启用阴影
viewer.shadows = true;

// 配置阴影质量
viewer.shadowMap.size = 2048; // 阴影贴图大小
viewer.shadowMap.softShadows = true; // 软阴影
viewer.shadowMap.darkness = 0.6; // 阴影暗度

// 设置阴影范围
viewer.shadowMap.maximumDistance = 5000.0;

// 为实体启用阴影
entity.model.shadows = Cesium.ShadowMode.ENABLED;
// 或
entity.model.shadows = Cesium.ShadowMode.CAST_ONLY; // 只投射阴影
entity.model.shadows = Cesium.ShadowMode.RECEIVE_ONLY; // 只接收阴影

后期处理 - 辉光效果 ​

javascript
// 创建后期处理阶段
const stages = viewer.scene.postProcessStages;

// 添加辉光效果
const bloom = stages.add(Cesium.PostProcessStageLibrary.createBloomStage());

// 配置辉光参数
bloom.uniforms.contrast = 128; // 对比度
bloom.uniforms.brightness = -0.3; // 亮度
bloom.uniforms.glowOnly = false; // 是否只显示辉光
bloom.uniforms.delta = 1.0; // 模糊半径
bloom.uniforms.sigma = 2.0; // 模糊强度
bloom.uniforms.stepSize = 1.0; // 采样步长

// 启用/禁用效果
bloom.enabled = true;

后期处理 - 环境光遮蔽 (SSAO) ​

javascript
// 添加 SSAO 效果
const ssao = stages.add(Cesium.PostProcessStageLibrary.createAmbientOcclusionStage());

// 配置 SSAO 参数
ssao.uniforms.intensity = 3.0; // 强度
ssao.uniforms.bias = 0.1; // 偏移
ssao.uniforms.lengthCap = 0.03; // 长度上限
ssao.uniforms.stepSize = 1.0; // 步长
ssao.uniforms.frustumLength = 1000.0; // 视锥体长度
ssao.uniforms.randomTexture = undefined; // 随机纹理

// 启用/禁用效果
ssao.enabled = true;

后期处理 - 景深效果 ​

javascript
// 添加景深效果
const depthOfField = stages.add(
  Cesium.PostProcessStageLibrary.createDepthOfFieldStage()
);

// 配置景深参数
depthOfField.uniforms.focalDistance = 50.0; // 焦距
depthOfField.uniforms.delta = 1.0; // 模糊半径
depthOfField.uniforms.sigma = 2.0; // 模糊强度
depthOfField.uniforms.stepSize = 1.0; // 采样步长

// 启用/禁用效果
depthOfField.enabled = true;

后期处理 - 自定义着色器 ​

javascript
// 创建自定义后期处理效果
const customStage = new Cesium.PostProcessStage({
  fragmentShader: `
    uniform sampler2D colorTexture;
    varying vec2 v_textureCoordinates;
    
    void main() {
      vec4 color = texture2D(colorTexture, v_textureCoordinates);
      
      // 反色效果
      color.rgb = vec3(1.0) - color.rgb;
      
      gl_FragColor = color;
    }
  `,
  uniforms: {}
});

// 添加到场景
stages.add(customStage);

后期处理 - 颜色调整 ​

javascript
// 亮度对比度饱和度调整
const brightnessContrast = stages.add(
  Cesium.PostProcessStageLibrary.createBrightnessStage()
);
brightnessContrast.uniforms.brightness = 0.5; // 亮度 (-1 到 1)

const contrast = stages.add(
  Cesium.PostProcessStageLibrary.createContrastStage()
);
contrast.uniforms.contrast = 1.2; // 对比度 (0 到 无穷)

// 色调调整
const hue = stages.add(
  Cesium.PostProcessStageLibrary.createHueStage()
);
hue.uniforms.hue = 0.5; // 色调偏移 (0 到 1)

// 饱和度调整
const saturation = stages.add(
  Cesium.PostProcessStageLibrary.createSaturationStage()
);
saturation.uniforms.saturation = 1.5; // 饱和度 (0 到 无穷)

大气效果配置 ​

javascript
// 配置大气效果
viewer.scene.skyAtmosphere.show = true;

// 大气散射参数
viewer.scene.skyAtmosphere.hueShift = 0.0; // 色调偏移
viewer.scene.skyAtmosphere.saturationShift = 0.0; // 饱和度偏移
viewer.scene.skyAtmosphere.brightnessShift = 0.0; // 亮度偏移

// 地平线效果
viewer.scene.globe.showGroundAtmosphere = true;

// 雾效
viewer.scene.fog.enabled = true;
viewer.scene.fog.density = 0.0002; // 雾密度
viewer.scene.fog.minimumBrightness = 0.03; // 最小亮度

天空盒配置 ​

javascript
// 使用自定义天空盒
viewer.scene.skyBox = new Cesium.SkyBox({
  sources: {
    positiveX: 'path/to/right.jpg',
    negativeX: 'path/to/left.jpg',
    positiveY: 'path/to/front.jpg',
    negativeY: 'path/to/back.jpg',
    positiveZ: 'path/to/top.jpg',
    negativeZ: 'path/to/bottom.jpg'

// 或使用单张全景图
viewer.scene.skyBox = new Cesium.SkyBox({
  sources: {
    positiveX: 'path/to/panorama.jpg',
    negativeX: 'path/to/panorama.jpg',
    positiveY: 'path/to/panorama.jpg',

    negativeZ: 'path/to/panorama.jpg'
  }
});

// 禁用天空盒
viewer.scene.skyBox.show = false;

时间控制与光照 ​

javascript

const currentTime = Cesium.JulianDate.fromDate(new Date(2024, 5, 21, 12, 0, 0));
viewer.clock.currentTime = currentTime;

// 动画时间以观察光照变化
viewer.clock.shouldAnimate = true;
viewer.clock.multiplier = 3600; // 1小时/秒

// 监听时间变化
viewer.clock.onTick.addEventListener((clock) => {
  // 根据时间调整光照
  const time = clock.currentTime;
  // ...
});

性能优化 ​

javascript
// 降低后期处理质量以提高性能
viewer.scene.postProcessStages.fxaa.enabled = false; // 禁用抗锯齿

// 降低阴影质量
viewer.shadowMap.size = 1024; // 使用较小的阴影贴图

// 限制阴影距离
viewer.shadowMap.maximumDistance = 1000.0;

// 禁用不必要的效果
viewer.scene.globe.showGroundAtmosphere = false;
viewer.scene.skyAtmosphere.show = false;
