图层切换示例 ​

这是一个演示如何在 Cesium 中切换不同影像图层和地形图层的示例。

功能说明 ​

  • 实现影像图层的切换（默认图层、天地图、高德地图等）
  • 实现地形图层的切换（默认地形、Cesium World Terrain 等）
  • 支持图层的叠加显示
  • 提供直观的图层控制界面

核心 API ​

  • viewer.imageryLayers - 管理影像图层的集合
  • viewer.terrainProvider - 设置地形数据源
  • Cesium.UrlTemplateImageryProvider - 自定义影像图层
  • Cesium.createWorldTerrain - 创建世界地形

<script setup>
import { ref, onMounted, onUnmounted, getCurrentInstance } from "vue";

const currentTerrain = ref("none");

/**
 * 切换影像图层
 * 功能：切换不同的影像图层（默认、天地图、高德地图）
 * 参数：layerType - 图层类型（'default' | 'tianditu' | 'gaode'）
 * 返回值：无
 */
function switchImageryLayer(layerType) {
  if (!viewer) return;

  currentImagery.value = layerType;

  // 移除所有影像图层
  viewer.imageryLayers.removeAll();

  switch (layerType) {
    case "default":
      // 使用默认影像图层
      viewer.imageryLayers.addImageryProvider(
        new Cesium.UrlTemplateImageryProvider({
          url: "https://webst04.is.autonavi.com/appmaptile?style=6&x={x}&y={y}&z={z}",
        })
      );
      viewer;
      break;
    case "tianditu":
      // 天地图影像图层
      viewer.imageryLayers.addImageryProvider(
        new Cesium.UrlTemplateImageryProvider({
          url: "https://t{s}.tianditu.gov.cn/DataServer?T=vec_w&x={x}&y={y}&l={z}&tk=305787e7e50d55fa42275025f382d1e5",
          subdomains: ["0", "1", "2", "3", "4", "5", "6", "7"],
          tilingScheme: new Cesium.WebMercatorTilingScheme(),
          maximumLevel: 18,
        })
      );
      let imgProviderTemp = new Cesium.UrlTemplateImageryProvider({
        url: "https://t{s}.tianditu.gov.cn/DataServer?T=cva_w&x={x}&y={y}&l={z}&tk=305787e7e50d55fa42275025f382d1e5",
        subdomains: ["0", "1", "2", "3", "4", "5", "6", "7"],
        tilingScheme: new Cesium.WebMercatorTilingScheme(),
        maximumLevel: 18,
      });
      viewer.imageryLayers.addImageryProvider(imgProviderTemp);
      break;
    case "gaode":
      // 高德地图影像图层
      viewer.imageryLayers.addImageryProvider(
        new Cesium.UrlTemplateImageryProvider({
          url: "https://webst01.is.autonavi.com/appmaptile?style=6&x={x}&y={y}&z={z}",
          credit: "高德地图",
        })
      );
      break;
  }
}

/**
 * 切换地形图层
 * 功能：切换不同的地形图层（无地形、世界地形）
 * 参数：layerType - 地形类型（'none' | 'world'）
 * 返回值：无
 */
function switchTerrainLayer(layerType) {
  if (!viewer) return;

  currentTerrain.value = layerType;

  switch (layerType) {
    case "none":
      viewer.terrainProvider = new Cesium.EllipsoidTerrainProvider();
      break;
    case "world":
      // 由于世界地形需要Cesium Ion访问令牌，这里使用简单的椭球体地形代替
      // 如果需要真实地形，可以申请自己的Cesium Ion访问令牌并设置
      viewer.terrainProvider = new Cesium.EllipsoidTerrainProvider();
      // 提示用户需要访问令牌
      console.log("世界地形需要Cesium Ion访问令牌，当前使用简单椭球体地形代替");
      break;
  }
}

/**
 * 动态加载 Cesium 资源
 * 功能：异步加载 Cesium JS 和 CSS 文件
 * 参数：无
 * 返回值：Promise 对象，加载成功时返回 Cesium 对象
 */
async function loadCesium() {
  return new Promise((resolve, reject) => {
    if (window.Cesium) {
      resolve(window.Cesium);
      return;
    }

    const link = document.createElement("link");
    link.rel = "stylesheet";
    link.href = "https://cdn.staticfile.net/cesium/1.113.0/Widgets/widgets.css";
    document.head.appendChild(link);

    const script = document.createElement("script");
    script.src = "https://cdn.staticfile.net/cesium/1.113.0/Cesium.js";
    script.onload = () => resolve(window.Cesium);
    script.onerror = () => reject(new Error("Cesium 加载失败"));
    document.head.appendChild(script);
  });
}


</script>

说明 ​

这个示例展示了如何在 Vue 组件中实现 Cesium 图层切换功能。主要功能包括：

  • 动态加载 Cesium CDN 资源
  • 创建 Cesium.Viewer 实例
  • 实现影像图层切换（默认、天地图、高德地图）
  • 实现地形图层切换（无地形、世界地形）
  • 完整的错误处理和加载状态

技术要点 ​

  1. 图层管理：使用 viewer.imageryLayers 和 viewer.terrainProvider 管理图层
  2. 动态加载：使用 Promise 异步加载 Cesium JS 和 CSS 文件
  3. 组件生命周期：在 onMounted 中初始化，onUnmounted 中清理资源
  4. 全屏功能：手动修复 Cesium 全屏按钮，支持跨浏览器全屏 API

注意事项 ​

  • 示例需要网络连接才能加载 Cesium CDN 资源
  • 天地图需要申请密钥才能正常使用
  • 全屏功能需要浏览器支持 Fullscreen API
  • 组件会自动清理资源，避免内存泄漏
