视图动画示例 ​

这是一个演示如何使用 Cesium 实现各种视图动画效果的综合示例。

功能说明 ​

  • 实现平滑的相机飞行动画
  • 演示环绕动画效果
  • 演示路径动画效果
  • 演示缩放动画效果
  • 可配置的动画参数（持续时间、高度、角度）

核心 API ​

  • viewer.camera.flyTo() - 相机飞行动画
  • viewer.camera.setView() - 设置相机视图
  • Cesium.Cartesian3.fromDegrees() - 创建笛卡尔坐标
  • Cesium.Math.toRadians() - 角度转弧度

vue
<template>
  <div class="cesium-view-animation-demo">
    <div v-if="loading" class="loading">加载 Cesium 中...</div>
    <div v-if="error" class="error">{{ error }}</div>

    <div v-show="!loading && !error">
      <!-- Cesium 容器 -->
      <div ref="cesiumContainer" class="cesium-container"></div>

      <!-- 控制面板 -->
      <div class="control-panel">
        <div class="panel-header">
          <h3>🎬 视图动画示例</h3>
          <button @click="toggleFullscreen" class="btn-icon" title="全屏">⛶</button>
        </div>

        <!-- 预设动画 -->
        <h4>预设动画</h4>
        <div class="animation-section">

          <button @click="flyToCity('beijing')" class="btn-animation">飞往北京</button>
          <button @click="flyToCity('shanghai')" class="btn-animation">飞往上海</button>
          <button @click="flyToCity('guangzhou')" class="btn-animation">飞往广州</button>
          <button @click="flyToCity('chengdu')" class="btn-animation">飞往成都</button>
        </div>

        <!-- 动画参数 -->
        <div class="animation-params">
          <h4>动画参数</h4>

          <div class="param-item">
            <label>持续时间 (秒)</label>
            <input type="range" v-model.number="animationParams.duration" min="1" max="10" step="0.5" />
            <span class="param-value">{{ animationParams.duration }}s</span>
          </div>

          <div class="param-item">
            <label>目标高度 (米)</label>
            <input type="range" v-model.number="animationParams.height" min="1000" max="5000000" step="1000" />
            <span class="param-value">{{ formatNumber(animationParams.height) }}m</span>
          </div>

          <div class="param-item">
            <label>俯仰角 (度)</label>
            <input type="range" v-model.number="animationParams.pitch" min="-90" max="0" step="5" />
            <span class="param-value">{{ animationParams.pitch }}°</span>
          </div>

          <div class="param-item">
            <label>航向角 (度)</label>
            <input type="range" v-model.number="animationParams.heading" min="0" max="360" step="15" />
            <span class="param-value">{{ animationParams.heading }}°</span>
          </div>
        </div>

        <!-- 动画类型 -->
        <div class="animation-types">
          <h4>动画类型</h4>
          <button @click="circleAnimation" class="btn-action">环绕动画</button>
          <button @click="pathAnimation" class="btn-action">路径动画</button>
          <button @click="zoomAnimation" class="btn-action">缩放动画</button>
          <button @click="stopAnimation" class="btn-stop">停止动画</button>
        </div>

        <!-- 动画状态 -->
        <div class="animation-status">
          <div class="status-item">
            <span class="status-label">动画状态:</span>
            <span :class="['status-value', isAnimating ? 'status-active' : '']">
              {{ isAnimating ? '运行中' : '已停止' }}
            </span>
          </div>
        </div>
      </div>
    </div>
  </div>
</template>

<script setup>
import { ref, onMounted, onUnmounted, reactive, getCurrentInstance } from 'vue'

const { proxy } = getCurrentInstance()

const cesiumContainer = ref(null)
const loading = ref(true)
const error = ref('')
const isAnimating = ref(false)
let viewer = null
let animationHandle = null

// 动画参数
const animationParams = reactive({
  duration: 3,
  height: 1000000,
  pitch: -45,
  heading: 0
})

// 城市坐标
const cities = {
  beijing: { lon: 116.4074, lat: 39.9042, name: '北京' },
  shanghai: { lon: 121.4737, lat: 31.2304, name: '上海' },
  guangzhou: { lon: 113.2644, lat: 23.1291, name: '广州' },
  chengdu: { lon: 104.0668, lat: 30.5728, name: '成都' }
}

// 格式化数字
const formatNumber = (num) => {
  if (num >= 1000000) {
    return (num / 1000000).toFixed(1) + 'M'
  } else if (num >= 1000) {
    return (num / 1000).toFixed(1) + 'K'
  }
  return num.toString()
}

// 飞往城市
const flyToCity = (cityKey) => {
  if (!viewer) return

  const city = cities[cityKey]
  if (!city) return

  stopAnimation()

  viewer.camera.flyTo({
    destination: window.Cesium.Cartesian3.fromDegrees(
      city.lon,
      city.lat,
      animationParams.height
    ),
    orientation: {
      heading: window.Cesium.Math.toRadians(animationParams.heading),
      pitch: window.Cesium.Math.toRadians(animationParams.pitch),
      roll: 0.0
    },
    duration: animationParams.duration,
    complete: () => {
      console.log(`已到达${city.name}`)
    }
  })
}

// 环绕动画
const circleAnimation = () => {
  if (!viewer) return

  stopAnimation()
  isAnimating.value = true

  const center = viewer.camera.positionCartographic
  const centerLon = window.Cesium.Math.toDegrees(center.longitude)
  const centerLat = window.Cesium.Math.toDegrees(center.latitude)
  const radius = 0.5 // 环绕半径（度）

  let angle = 0
  const speed = 0.5 // 旋转速度

  animationHandle = setInterval(() => {
    if (!viewer) {
      stopAnimation()
      return
    }

    angle += speed
    const radians = window.Cesium.Math.toRadians(angle)

    const lon = centerLon + radius * Math.cos(radians)
    const lat = centerLat + radius * Math.sin(radians)

    viewer.camera.setView({
      destination: window.Cesium.Cartesian3.fromDegrees(lon, lat, animationParams.height),
      orientation: {
        heading: radians + Math.PI / 2,
        pitch: window.Cesium.Math.toRadians(animationParams.pitch),
        roll: 0.0
      }
    })
  }, 50)
}

// 路径动画
const pathAnimation = () => {
  if (!viewer) return

  stopAnimation()
  isAnimating.value = true

  // 定义路径点（中国主要城市）
  const path = [
    cities.beijing,
    cities.shanghai,
    cities.guangzhou,
    cities.chengdu,
    cities.beijing
  ]

  let currentIndex = 0

  const flyToNext = () => {
    if (currentIndex >= path.length) {
      stopAnimation()
      return
    }

    const city = path[currentIndex]

    viewer.camera.flyTo({
      destination: window.Cesium.Cartesian3.fromDegrees(
        city.lon,
        city.lat,
        animationParams.height
      ),
      orientation: {
        heading: window.Cesium.Math.toRadians(animationParams.heading),
        pitch: window.Cesium.Math.toRadians(animationParams.pitch),
        roll: 0.0
      },
      duration: animationParams.duration,
      complete: () => {
        currentIndex++
        if (currentIndex < path.length) {
          setTimeout(flyToNext, 1000) // 停留1秒后飞往下一个点
        } else {
          stopAnimation()
        }
      }
    })
  }

  flyToNext()
}

// 缩放动画
const zoomAnimation = () => {
  if (!viewer) return

  stopAnimation()
  isAnimating.value = true

  const startHeight = viewer.camera.positionCartographic.height
  const minHeight = 10000
  const maxHeight = 5000000

  let zoomingIn = startHeight > (minHeight + maxHeight) / 2
  let step = 0
  const totalSteps = 100

  animationHandle = setInterval(() => {
    if (!viewer) {
      stopAnimation()
      return
    }

    step++
    if (step >= totalSteps) {
      zoomingIn = !zoomingIn
      step = 0
    }

    const progress = step / totalSteps
    const currentHeight = zoomingIn
      ? startHeight - (startHeight - minHeight) * progress
      : minHeight + (maxHeight - minHeight) * progress

    const position = viewer.camera.positionCartographic
    viewer.camera.setView({
      destination: window.Cesium.Cartesian3.fromRadians(
        position.longitude,
        position.latitude,
        currentHeight
      ),
      orientation: {
        heading: viewer.camera.heading,
        pitch: viewer.camera.pitch,
        roll: viewer.camera.roll
      }
    })
  }, 50)
}

// 停止动画
const stopAnimation = () => {
  if (animationHandle) {
    clearInterval(animationHandle)
    animationHandle = null
  }
  isAnimating.value = false
}

// 全屏切换
const toggleFullscreen = () => {
  const element = proxy.$el
  if (!document.fullscreenElement) {
    element.requestFullscreen()
  } else {
    document.exitFullscreen()
  }
}


// 加载 Cesium
const loadCesium = () => {
  return new Promise((resolve, reject) => {
    // 检查是否已加载
    if (window.Cesium) {
      resolve(window.Cesium)
      return
    }

    // 动态加载 Cesium CSS
    const link = document.createElement('link')
    link.rel = 'stylesheet'
    link.href = 'https://cdn.staticfile.net/cesium/1.113.0/Widgets/widgets.css'
    document.head.appendChild(link)

    // 动态加载 Cesium JS
    const script = document.createElement('script')
    script.src = 'https://cdn.staticfile.net/cesium/1.113.0/Cesium.js'
    script.onload = () => resolve(window.Cesium)
    script.onerror = () => reject(new Error('Cesium 加载失败'))
    document.head.appendChild(script)
  })
}

// 初始化 Cesium
const initCesium = async () => {
  try {
    loading.value = true
    await loadCesium()

    // 不需要设置 Ion token，我们使用第三方影像服务
    viewer = new window.Cesium.Viewer(cesiumContainer.value, {
      animation: false,
      timeline: false,
      baseLayerPicker: false,
      geocoder: false,
      homeButton: true,
      sceneModePicker: false,
      navigationHelpButton: false,
      fullscreenButton: true,
      imageryProvider: false // 禁用默认影像层
    })

    // 手动添加高德地图影像图层
    viewer.imageryLayers.addImageryProvider(
      new window.Cesium.UrlTemplateImageryProvider({
        url: "https://webst01.is.autonavi.com/appmaptile?style=6&x={x}&y={y}&z={z}",
        minimumLevel: 0,
        maximumLevel: 18
      })
    )

    // 设置初始视角（中国全景）
    viewer.camera.setView({
      destination: window.Cesium.Cartesian3.fromDegrees(105.0, 35.0, 5000000),
      orientation: {
        heading: 0.0,
        pitch: window.Cesium.Math.toRadians(-45),
        roll: 0.0
      }
    })

    loading.value = false
  } catch (err) {
    error.value = err.message
    loading.value = false
  }
}

// 清理资源
const cleanup = () => {
  stopAnimation()

  if (viewer) {
    viewer.destroy()
    viewer = null
  }
}

onMounted(() => {
  initCesium()
})

onUnmounted(() => {
  cleanup()
})
</script>

代码解释 ​

1. 相机飞行动画 ​

使用 flyTo() 方法实现平滑的相机飞行：

javascript
viewer.camera.flyTo({
  destination: Cesium.Cartesian3.fromDegrees(lon, lat, height),
  orientation: {
    heading: Cesium.Math.toRadians(0),    // 航向角
    pitch: Cesium.Math.toRadians(-45),    // 俯仰角
    roll: 0.0                              // 翻滚角
  },
  duration: 3,                             // 持续时间（秒）
  complete: () => {
    console.log('飞行完成')
  }
})

2. 环绕动画 ​

通过定时器不断更新相机位置，实现环绕效果：

javascript
const circleAnimation = () => {
  const center = viewer.camera.positionCartographic
  const centerLon = Cesium.Math.toDegrees(center.longitude)
  const centerLat = Cesium.Math.toDegrees(center.latitude)
  const radius = 0.5 // 环绕半径
  
  let angle = 0
  const speed = 0.5 // 旋转速度
  
  setInterval(() => {
    angle += speed
    const radians = Cesium.Math.toRadians(angle)
    
    const lon = centerLon + radius * Math.cos(radians)
    const lat = centerLat + radius * Math.sin(radians)
    
    viewer.camera.setView({
      destination: Cesium.Cartesian3.fromDegrees(lon, lat, height),
      orientation: {
        heading: radians + Math.PI / 2,
        pitch: Cesium.Math.toRadians(-45),
        roll: 0.0
      }
    })
  }, 50)
}

3. 路径动画 ​

通过连续的 flyTo() 调用实现路径动画：

javascript
const pathAnimation = () => {
  const path = [
    { lon: 116.4, lat: 39.9, name: '北京' },
    { lon: 121.5, lat: 31.2, name: '上海' },
    { lon: 113.3, lat: 23.1, name: '广州' }
  ]
  
  let currentIndex = 0
  
  const flyToNext = () => {
    if (currentIndex >= path.length) return
    
    const point = path[currentIndex]
    
    viewer.camera.flyTo({
      destination: Cesium.Cartesian3.fromDegrees(
        point.lon, point.lat, height
      ),
      duration: 3,
      complete: () => {
        currentIndex++
        if (currentIndex < path.length) {
          setTimeout(flyToNext, 1000) // 停留1秒后继续
        }
      }
    })
  }
  
  flyToNext()
}

4. 缩放动画 ​

通过逐步改变相机高度实现缩放效果：

javascript
const zoomAnimation = () => {
  const startHeight = viewer.camera.positionCartographic.height
  const minHeight = 10000
  const maxHeight = 5000000
  
  let step = 0
  const totalSteps = 100
  
  setInterval(() => {
    step++
    const progress = step / totalSteps
    const currentHeight = minHeight + (maxHeight - minHeight) * progress
    
    const position = viewer.camera.positionCartographic
    viewer.camera.setView({
      destination: Cesium.Cartesian3.fromRadians(
        position.longitude,
        position.latitude,
        currentHeight
      )
    })
  }, 50)
}

动画参数说明 ​

持续时间 (duration) ​

  • 控制动画的播放速度
  • 单位：秒
  • 建议范围：1-10秒

目标高度 (height) ​

  • 控制相机的观察高度
  • 单位：米
  • 建议范围：1000-5000000米

俯仰角 (pitch) ​

  • 控制相机的俯仰角度
  • 单位：度
  • 范围：-90°（垂直向下）到 0°（水平）

航向角 (heading) ​

  • 控制相机的朝向
  • 单位：度
  • 范围：0°-360°（0°为正北）

实用技巧 ​

1. 平滑过渡 ​

使用 flyTo() 而不是 setView() 可以实现平滑过渡：

javascript
// 平滑过渡
viewer.camera.flyTo({ destination, duration: 3 })

// 瞬间切换
viewer.camera.setView({ destination })

2. 动画回调 ​

利用 complete 和 cancel 回调处理动画状态：

javascript
viewer.camera.flyTo({
  destination,
  complete: () => {
    console.log('动画完成')
  },
  cancel: () => {
    console.log('动画被取消')
  }
})

3. 取消动画 ​

使用 cancelFlight() 取消正在进行的飞行动画：

javascript
viewer.camera.cancelFlight()

4. 动画缓动 ​

Cesium 支持自定义缓动函数：

javascript
viewer.camera.flyTo({
  destination,
  easingFunction: Cesium.EasingFunction.CUBIC_IN_OUT
})

应用场景 ​

  1. 城市导览：自动飞行浏览多个城市
  2. 建筑展示：环绕建筑物展示不同角度
  3. 路线演示：沿着路线飞行展示
  4. 缩放演示：从全球视角缩放到具体位置
  5. 交互式导航：响应用户操作的动画效果
