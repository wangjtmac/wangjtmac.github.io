点实体示例 ​

这是一个演示如何使用 Cesium Entity API 创建和自定义点实体的示例。

功能说明 ​

  • 实现不同样式的点实体创建
  • 演示点实体的各种属性设置
  • 实现点实体的动态更新
  • 提供交互式控制点实体样式

核心 API ​

  • viewer.entities.add() - 添加实体到场景
  • Cesium.PointGraphics - 点图形类
  • Cesium.Color - 颜色类
  • Cesium.Cartesian3 - 笛卡尔坐标类

组件代码 ​

vue
<template>
  <div class="cesium-point-demo">
    <div v-if="loading" class="loading">加载 Cesium 中...</div>
    <div v-if="error" class="error">{{ error }}</div>

    <div v-show="!loading && !error" class="w-full h-full">
      <!-- Cesium 容器 -->
      <div ref="cesiumContainer" class="cesium-container"></div>

      <!-- 控制面板 -->
      <div class="control-panel">
        <div class="panel-header">
          <h3>🎯 点实体示例</h3>
          <button @click="toggleFullscreen" class="btn-icon" title="全屏">⛶</button>
        </div>

        <!-- 点样式控制 -->
        <div class="style-controls">
          <h4>点样式设置</h4>

          <div class="control-group">
            <label>点大小: {{ pointStyle.pixelSize }}</label>
            <input type="range" v-model.number="pointStyle.pixelSize" min="5" max="50" step="1" />
          </div>

          <div class="control-group">
            <label>点颜色:</label>
            <input type="color" v-model="pointStyle.color" />
          </div>

          <div class="control-group">
            <label>透明度: {{ pointStyle.alpha }}</label>
            <input type="range" v-model.number="pointStyle.alpha" min="0" max="1" step="0.1" />
          </div>

          <div class="control-group">
            <label>轮廓宽度: {{ pointStyle.outlineWidth }}</label>
            <input type="range" v-model.number="pointStyle.outlineWidth" min="0" max="10" step="1" />
          </div>

          <div class="control-group">
            <label>轮廓颜色:</label>
            <input type="color" v-model="pointStyle.outlineColor" />
          </div>
        </div>

        <!-- 操作按钮 -->
        <div class="actions">
          <h4>操作</h4>
          <button @click="createRandomPoint" class="btn-action">创建随机点</button>
          <button @click="createLabeledPoint" class="btn-action">创建带标签点</button>
          <button @click="updateSelectedPoint" class="btn-action">更新选中点</button>
          <button @click="clearAllPoints" class="btn-action btn-danger">清除所有点</button>
        </div>

        <!-- 点列表 -->
        <div class="point-list">
          <h4>点列表 ({{ pointEntities.length }})</h4>
          <div class="point-items">
            <div v-for="point in pointEntities" :key="point.id"
              :class="['point-item', { active: selectedPointId === point.id }]" @click="selectPoint(point.id)">
              <span class="point-name">{{ point.name }}</span>
              <button @click.stop="deletePoint(point.id)" class="btn-delete">删除</button>
            </div>
            <div v-if="pointEntities.length === 0" class="empty-message">
              暂无点实体，点击上方按钮创建
            </div>
          </div>
        </div>
      </div>
    </div>
  </div>
</template>

<script setup>
import { ref, onMounted, onUnmounted, reactive, watch, getCurrentInstance } from 'vue'

const { proxy } = getCurrentInstance()

const cesiumContainer = ref(null)
const loading = ref(true)
const error = ref('')
let viewer = null
let pointCounter = 0

// 点样式配置
const pointStyle = reactive({
  pixelSize: 15,
  color: '#ff0000',
  alpha: 0.8,
  outlineWidth: 2,
  outlineColor: '#ffffff'
})

// 点实体列表
const pointEntities = ref([])
const selectedPointId = ref('')

// 全屏切换
const toggleFullscreen = () => {
  const element = proxy.$el
  if (!document.fullscreenElement) {
    element.requestFullscreen()
  } else {
    document.exitFullscreen()
  }
}

// 监听样式变化，更新选中的点
watch(pointStyle, () => {
  updateSelectedPoint()
}, { deep: true })

// 创建随机点
const createRandomPoint = () => {
  if (!viewer) return

  pointCounter++
  const lon = 116.4 + (Math.random() - 0.5) * 2
  const lat = 39.9 + (Math.random() - 0.5) * 2

  const entity = viewer.entities.add({
    id: `point-${pointCounter}`,
    name: `随机点 ${pointCounter}`,
    position: window.Cesium.Cartesian3.fromDegrees(lon, lat, 0),
    point: {
      pixelSize: pointStyle.pixelSize,
      color: window.Cesium.Color.fromCssColorString(pointStyle.color).withAlpha(pointStyle.alpha),
      outlineColor: window.Cesium.Color.fromCssColorString(pointStyle.outlineColor),
      outlineWidth: pointStyle.outlineWidth,
      heightReference: window.Cesium.HeightReference.CLAMP_TO_GROUND
    }
  })

  pointEntities.value.push({
    id: entity.id,
    name: entity.name,
    entity: entity
  })

  // 自动选中新创建的点
  selectedPointId.value = entity.id
}

// 创建带标签的点
const createLabeledPoint = () => {
  if (!viewer) return

  pointCounter++
  const lon = 116.4 + (Math.random() - 0.5) * 2
  const lat = 39.9 + (Math.random() - 0.5) * 2

  const entity = viewer.entities.add({
    id: `labeled-point-${pointCounter}`,
    name: `标签点 ${pointCounter}`,
    position: window.Cesium.Cartesian3.fromDegrees(lon, lat, 0),
    point: {
      pixelSize: pointStyle.pixelSize,
      color: window.Cesium.Color.fromCssColorString(pointStyle.color).withAlpha(pointStyle.alpha),
      outlineColor: window.Cesium.Color.fromCssColorString(pointStyle.outlineColor),
      outlineWidth: pointStyle.outlineWidth,
      heightReference: window.Cesium.HeightReference.CLAMP_TO_GROUND
    },
    label: {
      text: `标签点 ${pointCounter}`,
      font: '14px sans-serif',
      fillColor: window.Cesium.Color.WHITE,
      outlineColor: window.Cesium.Color.BLACK,
      outlineWidth: 2,
      style: window.Cesium.LabelStyle.FILL_AND_OUTLINE,
      verticalOrigin: window.Cesium.VerticalOrigin.BOTTOM,
      pixelOffset: new window.Cesium.Cartesian2(0, -20)
    }
  })

  pointEntities.value.push({
    id: entity.id,
    name: entity.name,
    entity: entity
  })

  selectedPointId.value = entity.id
}

// 更新选中的点
const updateSelectedPoint = () => {
  if (!viewer || !selectedPointId.value) return

  const entity = viewer.entities.getById(selectedPointId.value)
  if (entity && entity.point) {
    entity.point.pixelSize = pointStyle.pixelSize
    entity.point.color = window.Cesium.Color.fromCssColorString(pointStyle.color).withAlpha(pointStyle.alpha)
    entity.point.outlineColor = window.Cesium.Color.fromCssColorString(pointStyle.outlineColor)
    entity.point.outlineWidth = pointStyle.outlineWidth
  }
}

// 选择点
const selectPoint = (pointId) => {
  selectedPointId.value = pointId

  // 飞行到选中的点
  const entity = viewer.entities.getById(pointId)
  if (entity) {
    viewer.camera.flyTo({
      destination: window.Cesium.Cartesian3.fromDegrees(
        window.Cesium.Math.toDegrees(window.Cesium.Cartographic.fromCartesian(entity.position.getValue()).longitude),
        window.Cesium.Math.toDegrees(window.Cesium.Cartographic.fromCartesian(entity.position.getValue()).latitude),
        50000
      ),
      duration: 1.5
    })
  }
}

// 删除点
const deletePoint = (pointId) => {
  if (!viewer) return

  viewer.entities.removeById(pointId)
  pointEntities.value = pointEntities.value.filter(p => p.id !== pointId)

  if (selectedPointId.value === pointId) {
    selectedPointId.value = ''
  }
}

// 清除所有点
const clearAllPoints = () => {
  if (!viewer) return

  pointEntities.value.forEach(point => {
    viewer.entities.removeById(point.id)
  })

  pointEntities.value = []
  selectedPointId.value = ''
  pointCounter = 0
}

// 加载 Cesium
const loadCesium = () => {
  return new Promise((resolve, reject) => {
    // 检查是否已加载
    if (window.Cesium) {
      resolve(window.Cesium)
      return
    }

    // 动态加载 Cesium CSS
    const link = document.createElement('link')
    link.rel = 'stylesheet'
    link.href = 'https://cdn.staticfile.net/cesium/1.113.0/Widgets/widgets.css'
    document.head.appendChild(link)

    // 动态加载 Cesium JS
    const script = document.createElement('script')
    script.src = 'https://cdn.staticfile.net/cesium/1.113.0/Cesium.js'
    script.onload = () => resolve(window.Cesium)
    script.onerror = () => reject(new Error('Cesium 加载失败'))
    document.head.appendChild(script)
  })
}

// 初始化 Cesium
const initCesium = async () => {
  try {
    loading.value = true
    const Cesium = await loadCesium()

    viewer = new Cesium.Viewer(cesiumContainer.value, {
      animation: false,
      timeline: false,
      baseLayerPicker: false,
      geocoder: false,
      homeButton: false,
      sceneModePicker: false,
      navigationHelpButton: false,
      fullscreenButton: false,
      imageryProvider: false // 禁用默认影像层
    })

    // 手动添加高德地图影像图层
    viewer.imageryLayers.addImageryProvider(
      new Cesium.UrlTemplateImageryProvider({
        url: "https://webst01.is.autonavi.com/appmaptile?style=6&x={x}&y={y}&z={z}",
        minimumLevel: 0,
        maximumLevel: 18
      })
    )

    // 设置初始视角（北京）
    viewer.camera.setView({
      destination: Cesium.Cartesian3.fromDegrees(116.4, 39.9, 1000000)
    })

    // 创建初始点
    createInitialPoints()

    loading.value = false
  } catch (err) {
    error.value = err.message
    loading.value = false
  }
}

// 创建初始点
const createInitialPoints = () => {
  // 北京
  const beijing = viewer.entities.add({
    id: 'beijing',
    name: '北京',
    position: window.Cesium.Cartesian3.fromDegrees(116.4, 39.9, 0),
    point: {
      pixelSize: 20,
      color: window.Cesium.Color.RED.withAlpha(0.8),
      outlineColor: window.Cesium.Color.WHITE,
      outlineWidth: 3,
      heightReference: window.Cesium.HeightReference.CLAMP_TO_GROUND
    },
    label: {
      text: '北京',
      font: '16px sans-serif',
      fillColor: window.Cesium.Color.WHITE,
      outlineColor: window.Cesium.Color.BLACK,
      outlineWidth: 2,
      style: window.Cesium.LabelStyle.FILL_AND_OUTLINE,
      verticalOrigin: window.Cesium.VerticalOrigin.BOTTOM,
      pixelOffset: new window.Cesium.Cartesian2(0, -25)
    }
  })

  // 上海
  const shanghai = viewer.entities.add({
    id: 'shanghai',
    name: '上海',
    position: window.Cesium.Cartesian3.fromDegrees(121.5, 31.2, 0),
    point: {
      pixelSize: 18,
      color: window.Cesium.Color.BLUE.withAlpha(0.8),
      outlineColor: window.Cesium.Color.WHITE,
      outlineWidth: 2,
      heightReference: window.Cesium.HeightReference.CLAMP_TO_GROUND
    },
    label: {
      text: '上海',
      font: '14px sans-serif',
      fillColor: window.Cesium.Color.WHITE,
      outlineColor: window.Cesium.Color.BLACK,
      outlineWidth: 2,
      style: window.Cesium.LabelStyle.FILL_AND_OUTLINE,
      verticalOrigin: window.Cesium.VerticalOrigin.BOTTOM,
      pixelOffset: new window.Cesium.Cartesian2(0, -22)
    }
  })

  pointEntities.value = [
    { id: 'beijing', name: '北京', entity: beijing },
    { id: 'shanghai', name: '上海', entity: shanghai }
  ]

  pointCounter = 2
}

// 清理资源
const cleanup = () => {
  if (viewer) {
    viewer.destroy()
    viewer = null
  }
}

onMounted(() => {
  initCesium()
})

onUnmounted(() => {
  cleanup()
})
</script>

代码解释 ​

1. 点实体基础创建 ​

javascript
viewer.entities.add({
  id: 'point-1',
  name: '基础点',
  position: Cesium.Cartesian3.fromDegrees(116.4, 39.9, 0),
  point: {
    pixelSize: 10,                          // 点的大小（像素）
    color: Cesium.Color.RED,                // 点的颜色
    outlineColor: Cesium.Color.WHITE,       // 轮廓颜色
    outlineWidth: 2,                        // 轮廓宽度
    heightReference: Cesium.HeightReference.CLAMP_TO_GROUND // 高度参考
  }
})

2. 点实体样式属性 ​

基础样式 ​

  • pixelSize - 点的大小（像素）
  • color - 点的颜色
  • show - 是否显示

轮廓样式 ​

  • outlineColor - 轮廓颜色
  • outlineWidth - 轮廓宽度

高度参考 ​

  • NONE - 绝对高度
  • CLAMP_TO_GROUND - 贴地
  • RELATIVE_TO_GROUND - 相对地面高度

3. 动态点实体 ​

javascript
// 创建动态颜色的点
viewer.entities.add({
  position: Cesium.Cartesian3.fromDegrees(116.4, 39.9, 0),
  point: {
    pixelSize: 20,
    color: new Cesium.CallbackProperty(() => {
      return Cesium.Color.fromRandom({ alpha: 0.8 })
    }, false)
  }
})

实用技巧 ​

1. 点实体聚合 ​

对于大量点实体，使用聚合功能提升性能：

javascript
// 启用聚合
viewer.entities.clustering.enabled = true
viewer.entities.clustering.pixelRange = 50
viewer.entities.clustering.minimumClusterSize = 3

// 自定义聚合样式
viewer.entities.clustering.clusterBillboards = false
viewer.entities.clustering.clusterLabels = false
viewer.entities.clustering.clusterPoints = true

2. 点实体拾取 ​

javascript
// 点击事件处理
const handler = new Cesium.ScreenSpaceEventHandler(viewer.scene.canvas)
handler.setInputAction((click) => {
  const pickedObject = viewer.scene.pick(click.position)
  if (pickedObject && pickedObject.id && pickedObject.id.point) {
    console.log('点击了点实体:', pickedObject.id.name)
  }
}, Cesium.ScreenSpaceEventType.LEFT_CLICK)

3. 批量创建点 ​

javascript
const points = [
  { lon: 116.4, lat: 39.9, name: '北京' },
  { lon: 121.5, lat: 31.2, name: '上海' },
  { lon: 113.3, lat: 23.1, name: '广州' }
]

points.forEach((point, index) => {
  viewer.entities.add({
    id: `city-${index}`,
    name: point.name,
    position: Cesium.Cartesian3.fromDegrees(point.lon, point.lat, 0),
    point: {
      pixelSize: 12,
      color: Cesium.Color.fromRandom(),
      outlineColor: Cesium.Color.WHITE,
      outlineWidth: 2
    }
  })
})

  4. 路径节点：标记路径上的关键节点
  5. 事件标记：标记历史事件发生地点
