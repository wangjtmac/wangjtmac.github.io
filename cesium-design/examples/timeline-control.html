时间轴控制与实时数据 ​

本示例展示如何在 Cesium 中使用时间轴控制器，实现时间控制、实时数据更新和历史数据回放功能。

核心功能 ​

1. 时间轴控制 ​

  • 播放/暂停 - 控制动画的播放状态
  • 速度调节 - 调整时间流逝速度（0.1x - 10x）
  • 时间跳转 - 快速跳转到指定时间点
  • 循环模式 - 设置时间轴的循环行为

2. 实时数据更新 ​

  • WebSocket 模拟 - 模拟实时数据流
  • 动态实体更新 - 实时更新实体位置和属性
  • 数据缓冲 - 管理实时数据的缓冲和显示

3. 历史数据回放 ​

  • 时间范围查询 - 查询指定时间范围的历史数据
  • 数据回放 - 回放历史轨迹数据
  • 时间标记 - 在时间轴上标记重要事件

技术要点 ​

Clock 时间控制器 ​

javascript
// 配置时间控制器
viewer.clock.startTime = startTime.clone();
viewer.clock.stopTime = stopTime.clone();
viewer.clock.currentTime = startTime.clone();

// 设置时钟范围模式
viewer.clock.clockRange = Cesium.ClockRange.LOOP_STOP; // 循环播放
// 其他选项:
// - UNBOUNDED: 不限制
// - CLAMPED: 到达终点后停止
// - LOOP_STOP: 循环播放

// 设置播放速度
viewer.clock.multiplier = 1; // 1x 实时速度

// 控制播放状态
viewer.clock.shouldAnimate = true; // 开始播放

时间轴事件监听 ​

javascript
// 监听时钟变化
viewer.clock.onTick.addEventListener((clock) => {
  const currentTime = clock.currentTime;
  // 处理时间变化
});

// 监听时钟停止
viewer.clock.onStop.addEventListener(() => {
  console.log('动画已停止');
});

实时数据更新 ​

javascript
// 创建动态位置属性
const position = new Cesium.CallbackProperty((time) => {
  // 从数据源获取当前时间的位置
  const data = getRealtimeData(time);
  return Cesium.Cartesian3.fromDegrees(data.lon, data.lat, data.alt);
}, false);

// 应用到实体
entity.position = position;

// 或使用 SampledPositionProperty 添加新采样点
const sampledPosition = new Cesium.SampledPositionProperty();
// 实时添加新数据点
function addNewDataPoint(time, lon, lat, alt) {
  const position = Cesium.Cartesian3.fromDegrees(lon, lat, alt);
  sampledPosition.addSample(time, position);
}

时间区间管理 ​

javascript
// 创建时间区间集合
const intervals = new Cesium.TimeIntervalCollection();

// 添加时间区间
intervals.addInterval(new Cesium.TimeInterval({
  start: startTime,
  stop: stopTime,
  isStartIncluded: true,
  isStopIncluded: false,
  data: { status: 'active' }
}));

// 查询指定时间的区间
const interval = intervals.findIntervalContainingDate(currentTime);
if (interval) {
  console.log('当前状态:', interval.data.status);
}

应用场景 ​

  1. 实时监控 - GPS 追踪、传感器数据监控
  2. 历史回放 - 事件回放、轨迹分析
  3. 时间分析 - 时空数据分析、模式识别
  4. 动画制作 - 时间序列动画、演示视频
  5. 数据可视化 - 时间序列数据的可视化展示
