Cesium 基础概念学习笔记 ​

1. Cesium 简介 ​

Cesium 是一个开源的 3D 地球和地图可视化库，基于 WebGL 技术，能够在浏览器中渲染高质量的 3D 地理空间数据。它支持多种地理数据格式，包括地形、影像、矢量数据等，广泛应用于 GIS、虚拟地球、航空航天等领域。

1.1 Cesium 的核心特点 ​

  • 高性能渲染：基于 WebGL，能够高效渲染大规模地理数据
  • 跨平台兼容：支持所有现代浏览器，无需插件
  • 开源免费：基于 Apache 2.0 许可证，社区活跃
  • 丰富的功能：支持地形、影像、矢量数据、3D 模型、粒子系统等
  • 强大的 API：提供丰富的 JavaScript API，方便扩展和定制
  • 时间动态：内置时间轴系统，支持时间序列数据可视化

1.2 Cesium 的应用场景 ​

  • 数字地球：构建全球性的三维地理信息系统
  • 智慧城市：城市规划、建筑可视化、交通监控
  • 航空航天：卫星轨道可视化、飞行路径规划
  • 军事国防：态势感知、作战指挥、装备展示
  • 科学研究：地质勘探、气象分析、海洋研究
  • 教育培训：地理教学、历史演示、科普展示

2. Cesium 架构体系 ​

2.1 整体架构 ​

Cesium 采用分层架构设计：

┌─────────────────────────────────────┐
│           应用层 (Application)        │  ← 用户应用代码
├─────────────────────────────────────┤
│           高级 API (Entity API)      │  ← 实体系统、数据源
├─────────────────────────────────────┤
│           中级 API (Primitive API)   │  ← 图元、材质、几何体
├─────────────────────────────────────┤
│           底层 API (Renderer)        │  ← WebGL 渲染器
├─────────────────────────────────────┤
│           核心层 (Core)              │  ← 数学库、工具类
└─────────────────────────────────────┘

2.2 核心模块 ​

  • Core：数学库、坐标系统、时间系统
  • Scene：场景管理、相机控制、渲染管线
  • DataSources：数据源系统、实体管理
  • Widgets：UI 组件、控制器
  • Workers：Web Workers 支持，提升性能

3.2 VitePress 集成配置 ​

在 VitePress 项目中集成 Cesium：

javascript
// docs/.vitepress/config.js
import { defineConfig } from 'vitepress'

export default defineConfig({
  vite: {
    // 优化 Cesium 加载
    optimizeDeps: {
      exclude: ['cesium']
    },
    define: {
      // 定义 Cesium 基础路径
      CESIUM_BASE_URL: JSON.stringify('/cesium/')
    }
  }
})

3.3 动态加载方案 ​

为了避免打包体积过大，推荐使用动态加载：

javascript
// 动态加载 Cesium
const loadCesium = () => {
  return new Promise((resolve, reject) => {
    if (window.Cesium) {
      resolve(window.Cesium)
      return
    }

    // 加载 CSS
    const link = document.createElement('link')
    link.rel = 'stylesheet'
    link.href = 'https://cesium.com/downloads/cesiumjs/releases/1.136/Build/Cesium/Widgets/widgets.css'
    document.head.appendChild(link)

    // 加载 JS
    const script = document.createElement('script')
    script.src = 'https://cesium.com/downloads/cesiumjs/releases/1.136/Build/Cesium/Cesium.js'
    script.onload = () => resolve(window.Cesium)
    script.onerror = () => reject(new Error('Cesium 加载失败'))
    document.head.appendChild(script)
  })
}

4. 核心概念详解 ​

4.1 Cesium.Viewer - 视图管理器 ​

Cesium.Viewer 是 Cesium 的核心类，负责创建和管理整个 3D 场景：

javascript
const viewer = new Cesium.Viewer('cesiumContainer', {
  // 基础配置
  animation: false,           // 动画控件
  timeline: false,           // 时间轴控件
  baseLayerPicker: false,    // 图层选择器
  geocoder: false,           // 地理编码搜索
  homeButton: true,          // 主页按钮
  sceneModePicker: true,     // 场景模式选择器
  navigationHelpButton: false, // 导航帮助按钮
  fullscreenButton: true,    // 全屏按钮
  
  // 高级配置
  terrainProvider: Cesium.createWorldTerrain(), // 地形提供者
  imageryProvider: new Cesium.IonImageryProvider({ assetId: 2 }), // 影像提供者
  skyBox: new Cesium.SkyBox({
    sources: {
      positiveX: 'path/to/px.jpg',
      negativeX: 'path/to/nx.jpg',
      // ... 其他方向
    }
  }),
  
  // 性能优化
  requestRenderMode: true,   // 按需渲染
  maximumRenderTimeChange: Infinity // 渲染时间限制
})

4.2 Scene - 场景系统 ​

Scene 管理整个 3D 场景的渲染和交互：

javascript
const scene = viewer.scene

// 场景模式
scene.mode = Cesium.SceneMode.SCENE3D      // 3D 模式
scene.mode = Cesium.SceneMode.SCENE2D      // 2D 模式
scene.mode = Cesium.SceneMode.COLUMBUS_VIEW // 哥伦布模式

// 渲染设置
scene.fog.enabled = true                   // 启用雾效
scene.globe.enableLighting = true         // 启用光照
scene.globe.depthTestAgainstTerrain = true // 地形深度测试

// 后期处理
scene.postProcessStages.fxaa.enabled = true // 抗锯齿

4.3 Camera - 相机系统 ​

相机控制观察者的视角和位置：

javascript
const camera = viewer.camera

// 位置设置
camera.position = Cesium.Cartesian3.fromDegrees(lon, lat, height)
camera.direction = Cesium.Cartesian3.UNIT_Z.negate()  // 观察方向
camera.up = Cesium.Cartesian3.UNIT_Y                  // 上方向
camera.right = Cesium.Cartesian3.UNIT_X               // 右方向

// 姿态角度
camera.heading = Cesium.Math.toRadians(0)    // 航向角（偏航）
camera.pitch = Cesium.Math.toRadians(-90)    // 俯仰角
camera.roll = Cesium.Math.toRadians(0)       // 翻滚角

// 视锥体参数
camera.frustum.fov = Cesium.Math.toRadians(60)  // 视场角
camera.frustum.near = 1.0                       // 近裁剪面
camera.frustum.far = 5906376272000.0             // 远裁剪面

4.4 坐标系统 ​

Cesium 使用多种坐标系统来表示位置：

地理坐标系（Geographic） ​

  • 经度（Longitude）：东西方向，范围 -180° 到 180°
  • 纬度（Latitude）：南北方向，范围 -90° 到 90°
  • 高度（Height）：相对于椭球面的高度，单位米
javascript
// 创建地理坐标
const position = Cesium.Cartesian3.fromDegrees(116.4, 39.9, 1000)

笛卡尔坐标系（Cartesian） ​

  • X 轴：指向经度 0°，纬度 0°
  • Y 轴：指向经度 90°，纬度 0°
  • Z 轴：指向北极
javascript
// 创建笛卡尔坐标
const cartesian = new Cesium.Cartesian3(x, y, z)

// 坐标转换
const geographic = Cesium.Cartographic.fromCartesian(cartesian)
const longitude = Cesium.Math.toDegrees(geographic.longitude)
const latitude = Cesium.Math.toDegrees(geographic.latitude)

屏幕坐标系（Screen） ​

  • 像素坐标：屏幕上的二维坐标，原点在左上角
javascript
// 世界坐标转屏幕坐标
const screenPosition = Cesium.SceneTransforms.wgs84ToWindowCoordinates(
  scene, cartesian
)

4.5 时间系统 ​

Cesium 内置强大的时间系统，支持时间序列数据：

javascript
// 创建时间
const start = Cesium.JulianDate.fromDate(new Date('2024-01-01T00:00:00Z'))
const stop = Cesium.JulianDate.fromDate(new Date('2024-12-31T23:59:59Z'))

// 时间区间
const timeInterval = new Cesium.TimeInterval({
  start: start,
  stop: stop
})

// 时钟控制
viewer.clock.startTime = start
viewer.clock.stopTime = stop
viewer.clock.currentTime = start
viewer.clock.clockRange = Cesium.ClockRange.LOOP_STOP
viewer.clock.multiplier = 1 // 时间倍速

5. Hello World 示例详解 ​

5.1 创建基础容器 ​

html
<div id="cesiumContainer" style="width: 100%; height: 600px;"></div>

5.2 初始化 Cesium 视图 ​

javascript
// 基础初始化
const viewer = new Cesium.Viewer('cesiumContainer')

// 设置初始视角
viewer.camera.setView({
  destination: Cesium.Cartesian3.fromDegrees(116.4074, 39.9042, 10000)
})

5.3 Vue 组件封装 ​

vue
<template>
  <div ref="cesiumContainer" class="cesium-container">
    <div v-if="loading" class="loading">加载中...</div>
  </div>
</template>

<script setup>
import { ref, onMounted, onUnmounted } from 'vue'

const cesiumContainer = ref(null)
const loading = ref(true)
let viewer = null

onMounted(async () => {
  try {
    const Cesium = await loadCesium()
    viewer = new Cesium.Viewer(cesiumContainer.value)
    viewer.camera.setView({
      destination: Cesium.Cartesian3.fromDegrees(116.4, 39.9, 10000)
    })
    loading.value = false
  } catch (error) {
    console.error('初始化失败:', error)
  }
})

onUnmounted(() => {
  if (viewer) {
    viewer.destroy()
    viewer = null
  }
})
</script>

6. 常见问题与解决方案 ​

6.1 Cesium Ion 访问令牌 ​

javascript
// 设置访问令牌（可选）
Cesium.Ion.defaultAccessToken = 'your_access_token_here'

// 或者使用离线数据
const viewer = new Cesium.Viewer('cesiumContainer', {
  imageryProvider: new Cesium.UrlTemplateImageryProvider({
    url: 'https://webst04.is.autonavi.com/appmaptile?style=6&x={x}&y={y}&z={z}'
  }),
  terrainProvider: new Cesium.EllipsoidTerrainProvider()
})

6.2 性能优化策略 ​

javascript
// 启用按需渲染
viewer.scene.requestRenderMode = true
viewer.scene.maximumRenderTimeChange = Infinity

// 禁用不必要的功能
const viewer = new Cesium.Viewer('cesiumContainer', {
  animation: false,
  timeline: false,
  baseLayerPicker: false,
  geocoder: false,
  navigationHelpButton: false
})

// 设置渲染距离
viewer.scene.screenSpaceCameraController.minimumZoomDistance = 1000
viewer.scene.screenSpaceCameraController.maximumZoomDistance = 20000000

6.3 跨域问题解决 ​

javascript
// 方案一：使用代理
const imageryProvider = new Cesium.UrlTemplateImageryProvider({
  url: '/api/proxy?url=https://example.com/tiles/{z}/{x}/{y}.png'
})

// 方案二：使用 CORS 友好的服务
const imageryProvider = new Cesium.UrlTemplateImageryProvider({
  url: 'https://webst04.is.autonavi.com/appmaptile?style=6&x={x}&y={y}&z={z}'

6.4 内存管理 ​

javascript
// 正确的资源清理
onUnmounted(() => {
  // 清理实体
  viewer.entities.removeAll()
  
  // 清理数据源
  viewer.dataSources.removeAll()
  
  // 销毁视图
  if (viewer && !viewer.isDestroyed()) {
    viewer.destroy()
    viewer = null
  }
})
