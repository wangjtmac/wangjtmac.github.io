Cesium 实体系统学习笔记 ​

概述 ​

Cesium 的实体系统（Entity API）是一个高级的数据可视化 API，它提供了一种简单直观的方式来创建和管理地理空间数据的可视化。相比于底层的 Primitive API，Entity API 更加易用，适合大多数应用场景。

核心概念 ​

1. Entity（实体） ​

Entity 是 Cesium 中表示地理对象的基本单位，每个实体可以包含多种图形属性：

  • 位置属性：position - 实体在三维空间中的位置
  • 图形属性：point、polyline、polygon、billboard、label 等
  • 描述属性：name、description、id 等
  • 可见性：show - 控制实体是否显示

2. EntityCollection（实体集合） ​

viewer.entities 是一个 EntityCollection 对象，用于管理场景中的所有实体：

javascript
// 添加实体
const entity = viewer.entities.add({
  id: 'my-entity',
  name: '我的实体',
  position: Cesium.Cartesian3.fromDegrees(116.4, 39.9, 0)
})

// 获取实体
const entity = viewer.entities.getById('my-entity')

// 删除实体
viewer.entities.remove(entity)

// 清空所有实体
viewer.entities.removeAll()

3. Graphics（图形对象） ​

每种图形类型都有对应的 Graphics 类，用于定义图形的样式和属性：

  • PointGraphics - 点图形
  • PolylineGraphics - 线图形
  • PolygonGraphics - 面图形
  • BillboardGraphics - 广告牌图形
  • LabelGraphics - 标签图形
  • ModelGraphics - 3D 模型图形

点实体（Point Entity） ​

点实体是 Cesium 中最基础的地理对象，用于在地图上标记位置。

基本用法 ​

javascript
viewer.entities.add({
  id: 'my-point',
  name: '我的点',
  position: Cesium.Cartesian3.fromDegrees(116.4, 39.9, 0),
  point: {
    pixelSize: 10,                          // 点的大小（像素）
    color: Cesium.Color.RED,                // 点的颜色
    outlineColor: Cesium.Color.WHITE,       // 轮廓颜色
    outlineWidth: 2,                        // 轮廓宽度
    heightReference: Cesium.HeightReference.CLAMP_TO_GROUND // 高度参考
  }
})

常用属性 ​

  • pixelSize - 点的大小（像素）
  • color - 点的颜色
  • outlineColor - 轮廓颜色
  • outlineWidth - 轮廓宽度
  • heightReference - 高度参考（贴地、相对地面、绝对高度）
  • disableDepthTestDistance - 禁用深度测试的距离
  • scaleByDistance - 根据距离缩放
  • translucencyByDistance - 根据距离调整透明度

高度参考类型 ​

javascript
// 绝对高度（默认）
heightReference: Cesium.HeightReference.NONE

// 贴地
heightReference: Cesium.HeightReference.CLAMP_TO_GROUND

// 相对地面高度
heightReference: Cesium.HeightReference.RELATIVE_TO_GROUND

距离相关属性 ​

javascript
point: {
  pixelSize: 20,
  color: Cesium.Color.YELLOW,
  
  // 根据距离缩放
  scaleByDistance: new Cesium.NearFarScalar(1000, 2.0, 10000, 0.5),
  
  // 根据距离调整透明度
  translucencyByDistance: new Cesium.NearFarScalar(1000, 1.0, 10000, 0.1),
  
  // 禁用深度测试距离
  disableDepthTestDistance: Number.POSITIVE_INFINITY
}

应用场景 ​

  • 标记兴趣点（POI）
  • 显示设备位置
  • 标注地理坐标
  • 数据点可视化
  • 传感器位置标记

线实体（Polyline Entity） ​

线实体用于绘制线条，可以是直线、折线或曲线。

基本用法 ​

javascript
viewer.entities.add({
  polyline: {
    positions: Cesium.Cartesian3.fromDegreesArray([
      116.4, 39.9,
      121.5, 31.2
    ]),
    width: 5,                               // 线宽
    material: Cesium.Color.BLUE,            // 线的材质/颜色
    clampToGround: true                     // 贴地
  }
})

常用属性 ​

  • positions - 线的顶点坐标数组
  • width - 线宽（像素）
  • material - 线的材质（颜色、图案等）
  • clampToGround - 是否贴地
  • arcType - 弧线类型（大地线、测地线、直线）
  • granularity - 弧线的细分粒度

材质类型 ​

javascript
// 实线
material: Cesium.Color.BLUE

// 发光线
material: new Cesium.PolylineGlowMaterialProperty({
  glowPower: 0.2,
  color: Cesium.Color.BLUE
})

// 虚线
material: new Cesium.PolylineDashMaterialProperty({
  color: Cesium.Color.BLUE,
  dashLength: 16
})

// 箭头线
material: new Cesium.PolylineArrowMaterialProperty(
  Cesium.Color.BLUE
)

应用场景 ​

  • 绘制路线
  • 显示管线
  • 标注边界
  • 轨迹可视化

面实体（Polygon Entity） ​

面实体用于绘制多边形区域。

基本用法 ​

javascript
viewer.entities.add({
  polygon: {
    hierarchy: Cesium.Cartesian3.fromDegreesArray([
      116.3, 39.8,
      116.5, 39.8,
      116.5, 40.0,
      116.3, 40.0
    ]),
    material: Cesium.Color.RED.withAlpha(0.5),  // 填充材质
    outline: true,                               // 显示轮廓
    outlineColor: Cesium.Color.BLACK,            // 轮廓颜色
    outlineWidth: 2                              // 轮廓宽度
  }
})

常用属性 ​

  • hierarchy - 多边形的顶点坐标
  • material - 填充材质
  • outline - 是否显示轮廓
  • outlineColor - 轮廓颜色
  • outlineWidth - 轮廓宽度
  • height - 多边形的高度
  • extrudedHeight - 挤出高度（创建立体效果）
  • perPositionHeight - 每个顶点使用独立高度

带洞的多边形 ​

javascript
polygon: {
  hierarchy: {
    positions: Cesium.Cartesian3.fromDegreesArray([
      // 外环
      116.3, 39.8,
      116.5, 39.8,
      116.5, 40.0,
      116.3, 40.0
    ]),
    holes: [
      {
        positions: Cesium.Cartesian3.fromDegreesArray([
          // 内环（洞）
          116.35, 39.85,
          116.45, 39.85,
          116.45, 39.95,
          116.35, 39.95
        ])
      }
    ]
  }
}

应用场景 ​

  • 绘制行政区划
  • 标注建筑轮廓
  • 显示影响范围
  • 区域分析可视化

坐标系统 ​

1. 经纬度坐标（Degrees） ​

最常用的坐标表示方式：

javascript
// 单个坐标
Cesium.Cartesian3.fromDegrees(lon, lat, height)

// 坐标数组
Cesium.Cartesian3.fromDegreesArray([
  lon1, lat1,
  lon2, lat2,
  lon3, lat3
])

// 带高度的坐标数组
Cesium.Cartesian3.fromDegreesArrayHeights([
  lon1, lat1, height1,
  lon2, lat2, height2,
  lon3, lat3, height3
])

2. 笛卡尔坐标（Cartesian3） ​

Cesium 内部使用的三维坐标系统：

javascript
const cartesian = new Cesium.Cartesian3(x, y, z)

3. 坐标转换 ​

javascript
// 笛卡尔坐标转地理坐标
const cartographic = Cesium.Cartographic.fromCartesian(cartesian)
const lon = Cesium.Math.toDegrees(cartographic.longitude)
const lat = Cesium.Math.toDegrees(cartographic.latitude)
const height = cartographic.height

// 地理坐标转笛卡尔坐标
const cartesian = Cesium.Cartesian3.fromDegrees(lon, lat, height)

颜色系统 ​

1. 预定义颜色 ​

javascript
Cesium.Color.RED
Cesium.Color.GREEN
Cesium.Color.BLUE
Cesium.Color.WHITE
Cesium.Color.BLACK

2. 自定义颜色 ​

javascript
// RGB（0-1范围）
Cesium.Color.fromCssColorString('#ff0000')
Cesium.Color.fromBytes(255, 0, 0, 255)

// 带透明度
Cesium.Color.RED.withAlpha(0.5)

// 随机颜色
Cesium.Color.fromRandom()
Cesium.Color.fromRandom({ alpha: 0.8 })

实体样式管理 ​

1. 动态更新样式 ​

javascript
const entity = viewer.entities.add({
  point: {
    pixelSize: 10,
    color: Cesium.Color.RED
  }
})

// 更新样式
entity.point.pixelSize = 20
entity.point.color = Cesium.Color.BLUE

2. 批量样式设置 ​

javascript
viewer.entities.values.forEach(entity => {
  if (entity.point) {
    entity.point.color = Cesium.Color.RED
  }
})

3. 条件样式 ​

javascript
viewer.entities.add({
  point: {
    pixelSize: new Cesium.CallbackProperty(() => {
      // 根据条件返回不同的值
      return someCondition ? 10 : 20
    }, false),
    color: new Cesium.CallbackProperty(() => {
      return someCondition ? Cesium.Color.RED : Cesium.Color.BLUE
    }, false)
  }
})

性能优化 ​

1. 使用实体集合 ​

对于大量实体，考虑使用 DataSource：

javascript
const dataSource = new Cesium.CustomDataSource('myData')
viewer.dataSources.add(dataSource)

// 添加实体到 DataSource
dataSource.entities.add({
  // 实体配置
})

2. 控制可见性 ​

javascript
// 根据距离控制可见性
entity.show = new Cesium.CallbackProperty(() => {
  const distance = Cesium.Cartesian3.distance(
    viewer.camera.position,
    entity.position.getValue()
  )
  return distance < 1000000
}, false)

3. 使用聚合 ​

对于大量点实体，使用 EntityCluster：

javascript
viewer.entities.clustering.enabled = true
viewer.entities.clustering.pixelRange = 50
viewer.entities.clustering.minimumClusterSize = 3

最佳实践 ​

  1. 合理使用 ID：为重要实体设置有意义的 ID，便于后续查找和管理
  2. 及时清理：不需要的实体要及时删除，避免内存泄漏
  3. 批量操作：使用 suspendEvents() 和 resumeEvents() 优化批量操作
  4. 材质选择：简单场景使用 Color，复杂效果使用 Material
  5. 坐标精度：注意浮点数精度问题，避免坐标偏移
  6. 高度参考：根据需求选择合适的 heightReference
  7. 性能监控：使用 Cesium Inspector 监控性能

常见问题 ​

1. 实体不显示 ​

  • 检查坐标是否正确
  • 检查 show 属性是否为 true
  • 检查相机位置是否合适
  • 检查材质颜色的透明度

2. 线条不贴地 ​

  • 设置 clampToGround: true
  • 确保地形数据已加载
  • 注意 clampToGround 对性能的影响

3. 多边形显示异常 ​

  • 检查顶点顺序（逆时针）
  • 检查坐标是否闭合
  • 检查是否有自相交

4. 颜色显示不正确 ​

  • 检查颜色值范围（0-1 或 0-255）
  • 检查透明度设置
  • 检查材质类型是否正确

总结 ​

Cesium 的实体系统提供了一套完整的地理数据可视化解决方案：

  • 点实体：适合标记位置和数据点可视化
  • 线实体：适合路线、管线和轨迹展示
  • 面实体：适合区域、建筑和范围标注

掌握实体系统的核心概念和使用方法，是开发 Cesium 应用的基础。在实际应用中，要根据具体需求选择合适的实体类型和样式配置，并注意性能优化。
